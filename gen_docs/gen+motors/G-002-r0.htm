Introduction

This document lists the equipment inspection I1 for GE frame 5, Air Cooled Generator, which must be performed on an annually bases as indicated by Y1 in the title block. The first draft initiated by eng Saleh and tested at 13 June 02 on generator unit 6 at Talkha GT PS, EDEPC and approved by Sector Manager of Talkha PS eng. Mostafa Abd El Hameed Aalam.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category,

  • Maintenance staff should be well trained, qualified or directly supervised by some one who is

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used (safety shoes, Helmet, breathing mask, safety glasses).

Tools and Equipment

  • Personal protective equipment as safety shoes, Helmet, breathing mask, and safety glasses

  • Brushes, Blower, Megger.

  • Cleaning fluids such as freon - TF, and chlorothene.

Work to be Carried Out

 

    Reference Test
Stator Inspection value results
1. Primary visual inspections. 

 Ok - Not Ok

  
2. All surfaces should be cleaned by using brushes and air dry compressor Ok - Not Ok  
3. Clean the all surfaces by using cleaning fluids such as freon - TF, chlorothene Ok - Not Ok  
4. Paint the all surfaces of the core if it needed. Ok - Not Ok  
5. Inspect the ends of the armature winding for deposits of oil and evidence of corona discharge Ok - Not Ok  
6. Inspect the winding and insulation care fully. Ok - Not Ok  
7. Check the insulation winding resistance. up to 80MW  
       
Rotor Inspection      
1. Check the insulation winding resistance. up to 20MW  
       
Exciter Inspection         
1. Clean the out surface of all component by using clothes, brushes Ok - Not Ok  
2. Clean all parts by using air blower Ok - Not Ok  
3. All contactors and relays should be inspected Ok - Not Ok  
4. The automatic and manual  regulator voltage adjuster contact brushes should be inspected Ok - Not Ok  
5. Inspected bridge rectifier ( diode - SCR )   Ok - Not Ok  
6. Check all screw type connections regularly Ok - Not Ok  
       
Brush Rigging and Collector Rings Inspection    
1. The collector brush rigging should be cleaned by using brushes air blower Ok - Not Ok  
2. Check the side clearance between brush and brushes holder  Ok - Not Ok  
3. Inspect all nuts bolts and capscrews for tightness Ok - Not Ok  
4. Inspect the collector ring wear   Ok - Not Ok  
5. Inspect sparking brushes  Ok - Not Ok  
6. Check the clearance between brush holder and collector ring Ok - Not Ok  
7. Check the brush wear brush tap at least 1/8" above top of brush box Ok - Not Ok  
       
Shaft- Grounding Brushes Inspection    
1. Remove the brushes from their holders      Ok - Not Ok  
2. Clean brush and holder so brush may move freely Ok - Not Ok  
3. Clean brush clip contacts Ok - Not Ok  
4. Use a soft brush seating stone or a canvas - padded bort to remove excess film from shaft Ok - Not Ok  
5. Replace brushes before the pigtail is in contact with top of brush box Ok - Not Ok  

 

      
Resistance Temperature Detectors ( R.T.D ) Inspection    
1. Clean terminal banal board Ok - Not Ok  
2. Check the connection  of wiring of board Ok - Not Ok  
3. Check the Calibration of the temperature meter Ok - Not Ok  
4. Test resistance of all resistance temperature detectors ( R.T.D ) it should be 11.73  W Ok - Not Ok
