Introduction

This document outlines the testing procedure P1, which must be performed on an annually bases as indicated by Y1 in the title block.

The GCAG14 relay may be used for any type of high impedance circulating current protection. It is used for protecting generator windings and transformer earthed star side.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category.

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • Safety and Healthy equipment must be used (safety Helmet, safety shoes, ...etc).

  • The transformer must be isolated.

  • Make sure that the CT's secondary circuits are shorted at the incoming blocks before isolating the relay current circuit for test.

Tools and Equipment

  • Secondary injection current  write here the test set type

  • Multifiger test plug type MMLB01 for use with test block type MMLG if fitted.

  • Test leads, and Multi-meter.

Work to be Carried Out

  1. Check the operating current and voltage.

  2. Check the tripping of all breakers associated with the relay.

Preparation

  1. The test set should be capable of providing voltage up to 120% of the relay setting.

  2. Isolate all the trip signals.

  3. If the relay is wired through an MMLG test block it is recommended that all secondary injection tests should be carried out using this test block.

  4. Ensure that the main system current transformers are shorted before isolating the relay from the current transformers.

  5. When type MMLG test block facilities are installed, it is important that the sockets in the type MMLB01test plug, which correspond the current transformer secondary windings are shorted together.

  6. Connect the circuits as the following circuit:

Checking the Operating Current and Voltage

  1. Slowly increase the voltage and notice the drawn current until the relay pick up.

  2. Record the current value and compare it with the setting.

  3. Reset the relay.

Checking Trip all Breakers Associated with the Relay

  1. Close all breakers associated with this relay.

  2. Reconnect the trip signals again.

  3. Apply voltage equal to the value which operates the relay.

  4. Check the breakers are tripped.
