Introduction

This document lists the testing procedures P1, to be performed every three months as indicated by M3 in the title block.

The purpose of the relay is basically to give an indication that over load the transformer or motor, this means the current flow is greater than the rated value cusses the load is greater than the rated load. This device operate when the thermal contact operate after defiant time according the vale of current this relay give trip signal.

Tools and Equipment

  • Clip-on ammeter type KEW SNAP 2002

  • Current test plug

Work to be Carried Out

Prepare for Relay Testing

  1. Review technical reference material and Manufacture’s manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Look at the MRB panel nameplate and check this MRB system designation nameplate agrees with the line that has been taken out of service for maintenance

  4. Perform one of the following measuring checks to verify that there is not current following through this relay and the testing will occur on the correct relay.

  5. Using clip-on ammeter checks that there is not current following in the STA relay circuit

  6. Opens the STA relay over load.

  7. Check all connections leads and wiring to the relay are in a good condition

  8. Attach all connections between the relay and the test set.

  9. Supply AC power to the SVERKER 650 test set.

Testing the Function Operation of the Over Load Relay type (STA)

  1. Connect the connection leads between the test set and the over load relay.

  2. Connect the DC (b10) of the line under test.

  3. Measure the AC injection current using clip-on ammeter type KEW. SNAP 2002.

  4. Increase the current until the relay operates after delay time     I (60). 

  5. Make reset to the relay on phase. 

  6. Check the time element of the relay.

  7. Tripping test of the line circuit breaker performed by

  8. Connect the CB of the line under test Adjust the current tripping value on the test set.

  9. Turn on the test set switch until the relay operates and the C.B disconnected

  10. Make reset to the three phases of the relay.

Return the Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal and the test set.

  2. Make sure that the relay terminals are in normal condition.

  3. Make sure that any equipment associated with the relay that isolated during tests restored to service.

  4. Make sure that there is no signals appeared on the signals panel

Power Line to be returned in Service

  1. Take crossed reading of all AC currents and voltages and compare them to those for normal load condition.

  2. Place a copy of the test results in the field maintenance file.
