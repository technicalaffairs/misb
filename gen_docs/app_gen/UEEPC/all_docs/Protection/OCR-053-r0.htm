Introduction

MPS-C-3G (Long time, shot time, and ground over current relay) for 380 v power center. The instantaneous and time delay over current relays of the series MPS ...  have been designed for the phase and fault protection of electrical plant and networks, there is three -phase unit.

Its fields of application include        Short- time setting.             Long- time setting.

                                              Instantaneous setting.         Ground current setting.

Safety Precautions

  • A work permit must be issued, the maintenance staff should carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required.

  • Wear your Safety and healthy equipment as safety helmet, shoes, safety belt.... etc

  • Make sure that CT secondary circuits are shorted before isolating the current circuit for test current injection.

  • Ensure that the Line isolation is done.

     

Tools and Equipment

  •  BTSB  Test set

 

 

 

 

Work to be Carried Out    

Preliminary Steps to Prepare the Relay Testing

  1. Review technical reference material, manufacturers manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are opened and tagged

Testing the Relay and Protection Outputs

  1. Verify the test results match the relay setting, if not consult technical services.

  2. Test trip the breaker associated with the relay

Preparation

  1. Connect the BTSB Test set to the relay as shown in figure

  2. From main menu select setting menu

  • Select 1 to adjust the test type (test to All, mag to on).

  • Select 2 to adjust date and time.

  1. From main menu select test menu.

  • Select enter to begin.

  • Follow the various status screens, depending on the tests selected.

  1. From main menu select print menu.

  • Select 1 (use a Ref. No.)

  • Enter Ref. No. 1 (which was selected).

Return Relay and Associated Equipment to Service

  1. Verify the test results match the relay setting, if not consult technical services.

  2. Trip the circuit breaker associated with the relay.

  3. Return relay to service by removing the test leads from the relay terminals block and the test set. Make sure CT secondary circuit is in normal condition.

  4. Any equipment associated with the relay which has been isolated
