Introduction

The type SFF relays are digital frequency relays that measure the system frequency The under frequency trip features of the SFF relays used in load conservation schemes.

Safety Precautions

  • A work order must be issued.

  • The maintenance staff should be qualified and carry the suitable safety ID category.

  • Wear safety and healthy equipment as safety helmet, safety shoes, anti static hand probe to avoid electrical shock …etc

  • Safety fence with caution marks surrounding the work area is required       

  • The generator must be isolated and work permit required.

  • Switch off DC power supply before inserting or withdrawing any card from the relay module.

Tools and Equipment

  • Test set MANTA

  • Clip on ammeter

  • Voltmeter

  • Test leads

Work to be Carried Out

  • Pick up frequency

  • Time test

  • Cut off voltage

  • Function test

To Get Time

  1. Use the same connection

  2. Adjust the frequency at 45 HZ and voltage to 100V

  3. Start inject the time

To Get Cut Off Voltage

  1. Set voltage under cut off voltage nearly 20 volt for example

  2. Set F under pick up frequency then increase V until trip occurs then record V
