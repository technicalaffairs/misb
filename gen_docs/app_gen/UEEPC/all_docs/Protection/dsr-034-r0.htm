Introduction

These procedure details the steps required to completely check the proper operation and calibration of the LH1W relay

The first preliminary steps to prepare for relay testing describes the activities required prior to testing.

Some of these steps could be performed a day before the actual test,

The second section is the preparation for testing describes. The steps to set up the equipment to perform the actual tests, the third sections is the actual test.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Line isolation must be done.

  • Review the relay test sheet to know the required settings

    •

  • Make sure that CT secondary circuits are shorted before isolating the current circuit for test current injection

Tools and Equipment

  • Safety and healthy equipment as safety helmet, safety shoes ... etc.

  • ABB test set type AK 103966

  • Phase rotation meter for the first time only. 

  • Small slot screwdriver, test leads, Timer

  • 5Amp Clip-on meter

 

Work to be Carried Out

 

Preparation      

This procedure should be used for performing off load line

  1. Look at the MRB panel nameplate and check this MRB system designation nameplate agrees with the line that Has been taken out of service for maintenance.

  2. Perform one of the following measuring checks to verify that there is not current following through this relay and that the testing will occur on the correct relay.

  3. Using clip on ammeter checks there is no current following in the LH1W relay circuits. wires going to the rear terminals 1,2,3 and 4 should have no current follows in them.

  4. Carefully opens the LH1W relay covers and connects ammeter across one of the following test block terminals (across terminal 2 or across terminal 3, or across terminal4.using a small slot screwdriver, turn the same test block pin until it is in the ammeter, return the pin to the normal vertical  (I position) and check the other terminals   

  5. Connect the CT shorting link in the test block terminals.

  6. Use a small slot screwdriver to turn all test terminals to horizontal  (-) position

  7. Check all connections and wiring to the relay are in a good condition

  8. Attach all connections between the relay and the test set  

  9. Supply A.C power the AK103966 test set.

  10. Observe the correct polarity and connect the other end of these leads to a three phases 110 v ac supply

Testing the Function Operation of the Distance Relay type LH1W

  1. Place the %Z switch (h) in the zero “o” position

  2. Place the multipliers of the % Z switches ”I” in the “x1”position and switch (k) in the zero “o” position.

  3. Operate the LH1W relay by pressing push button ”n”

Checking the pick up value for measuring element for Phase to Ground Fault

  1. Using the curves given and the calculated values on the test sheet to find the corresponding value of z% of the first time step.

  2. Turn the z % switches to agree with the calculated value

  3. Select the single phase to ground faults by turning "g" to position 2

  4. Select the fault by turning switch "f" to position R, S, T respectively.

  5. Presses push button (n) and observe when PA/R, PE and PD pick up.

  6. Change z% step by step until PSII pick up with PA/R, PE and PD.

  7. Repeat the above test for the other phases S, T.

  8. Repeat the above test but increase z% setting until the relay operates on the second time step. Change the z% until PSIII pick up with PA/R, PE and PD

  9. Repeat the above test but increase z% until the relay operate on the third time step .the contactor PSIII WILL pick up with the above flag. change z% switches step by step until the relay operate in the last step, PD will pick up.

Checking the Pick Up Value for Measuring Element for Phase to Phase Fault 

  1. From the curve given and the values calculated determines the value of z% of the first step.

  2. Turn the z% switches on the test set until agree with calculated zone 1setting.

  3. Select phase to phase faults by turning switch “g” to position “I”.

  4. Turn switch “I” to R, S, T respectively.

  5. Presses push button (n1) and observe when PA/R, PA/T, and PD pick up.

  6. Change the z% step by step on the test set until Psii picks up with last flag.  

  7. Repeats the above test but change switch F for the other phases (S, T). 

  8. Repeat the above test but increase Z% until the relay operates in the second

  9. time step Psii with the above flag. Change the Z% step by step until Psiii pick up with PA/R, and PD.

  10. Repeat the above test but increase the Z% on the tester until

  11. the relay operates in the third time step the contactor Psiii will pick up.

  12. Change the Z% step by step until the relay operate in the last time step, the contractor PD will pick up by the fourth time contact of timer.                            

The Steps for Checking the Tripping Times

  1. Use two test leads to connect the digital timer inputs to the terminals 12, 13 in the top right of the tester                          

  2. Connect an A.C supply to the digital timer display.                           

  3. Press the reset button for the digital timer display.

Time Test for Phase to Ground Fault            

  1. Repeat the test of checking the pick up value for measuring element for phase to ground fault as the above steps to measure the Zone I operation time for each phase to ground unit.

  2. to perform the time test press switch in the closed position

  3. Press down voltage control switch nI to the on position the time will start and then stop once the Pd trip relay has picked up.

  4. Record the time, which appear on the timer display on the test sheet.

  5. Repeats the above test for the other two phases

  6. Repeat the above test but change Z% setting on the AK103966 tester until the relay operates on the 2nd zone time.

  7. Repeat the above test but change the Z% setting on the AK103966 tester until the relay operates on the third zone time.

Time Tests for Phase to Phase Unit

  1. Select phase to phase faults by turning switch “g” to position “I” and perform time tests for the phase to phase unit by repeating the above steps.

  2. Switch off the tester.      

  3. Remove all the test lead between the relay and the tester.

 

 

Return Relay and Associated Equipment to Service 

  1. Return relay to service by removing the test leads from the relay terminal block and the test set.

  2. Make sure that CT and VT secondary circuit are in normal condition.

  3. Make sure that any equipment associated with the relay which isolated during tests be returned to service.

 Power Line to be Returned in Service   

  1. Take crossed reading of all AC currents and voltages and compare them to those for normal load condition.                                                 

  2. Place a copy of the test results in the field maintenance file.
