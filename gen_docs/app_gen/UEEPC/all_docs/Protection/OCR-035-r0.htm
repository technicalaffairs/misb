Introduction

This document outlines the testing procedure the steps required to completely checking the proper operation and calibration of the over current relay type ST , which must be performed on an annually based as indicated by Y1 in the above title block .

The first draft initiated and tested by eng. Badry Atea, eng. Kamel Tawfeek, eng. Safwat Mahrous, accepted by general director eng. Mostafa Abdel Nazeer Qotb, approved by eng. Hessain Ahmed Mahdy Head Sector of Assiut steam power station, and certified by eng. Gamal Abdel Wahab: UEEPC inspection and quality control general director

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Line isolation must be done.

  • Review the relay test sheet to know the required settings

  • Make sure that CT secondary circuits are shorted before isolating the current circuit for test current injection

Tools and Equipment

  • ABB test set type SVERKER 650

  • Clip-on ammeter type KEW SNAP 2002

  • Test leads.

  • Current test plug

  • Clamp ampere

Work to be Carried Out

 Prepare for Relay Testing

  1. Review technical reference material and Manufacture`s manual .

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay .

  3. Look at the MRB panel nameplate and check this MRB system designation nameplate agrees with the line that has been taken out of service for maintenance

  4. Perform one of the following measuring checks to verify that there is not current following through this relay and the testing will occur on the correct relay.

  5. Using clip-on ammeter checks that there is not current following in the ST relay circuit

  6. Opens the ST relay cover.

  7. Check all connections leads and wiring to the relay are in a good condition

  8. Attach all connections between the relay and the test set.

  9. Supply A.C power to the SVERKER 650 test set.

 Testing The Function Operation Of The Over Current Relay Type (ST)

  1. Connect the connection leads between the test set and the over current relay on phase (R).

  2. Connect the D.C (b10) of the line under test.

  3. Measure the A.C injection current using clip-on ammeter type KEW. SNAP 2002.

  4. Increase the current until the relay operates.

  5. Decreases the current until the relay reset.

  6. Make reset to the relay on phase R.

  7. Repeat the above steps on phases S, T.

  8. Check the time element of the relay .

  9. Tripping test of the line circuit breaker performed by

  • Connect the C.B of the line under test

  • Adjust the current tripping value on the test set.

  • Turn on the test set switch until the relay operates and the C.B disconnected

  1. Make reset to the three phases of the relay.

 Return the Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal and the test set.

  2. Make sure that the relay terminals are in normal condition.

  3. Make sure that any equipment associated with the relay that isolated during tests restored to service.

  4. Make sure that there is no signals appeared on the signals panel

Power Line to be Returned in Service 

  1. Take crossed reading of all AC currents and voltages and compare them to those for normal load condition.

  2. Place a copy of the test results in the field maintenance file.
