Introduction

This document outlines testing procedures P1, to be preformed every six months as indicated by M6 in the title block.

The purpose of the relay is basically to give an indication that an unbalancing (Asymmetrical load) occur between two phases (R&T) in the generator winding which give:

  1. Alarm signal for light asymmetrical load (10% for 8 sec).
    The relay contains over current relay set at 10% amp., Timer operates at eigh-second-delay time.

  2. Trip signal for heavy asymmetrical load (30% for 3 sec).
    The relay contains reverse current relay (ZH)
    12v from the testing transformer when the unit off, there is a resistance which convert the voltage to current. When the fault occur the over current will give alarm or trip signal after eight or three second delay

Safety Precautions

  • A work order must be issued.

  • The maintenance staff should be qualified and carry the suitable safety ID category.

  • Wear safety and healthy equipment as safety helmet, safety shoes, anti static hand probe to avoid electrical shock …etc

  • Safety fence with caution marks surrounding the work area is required.

  • The unit should be out of service

Tools and Equipment

  • Special test key

  • A vometer and connecting leads

  • Test transformer to give the test voltage (12v)

  • Selector switch to get the test voltage from the test transformer to the relay and isolate the voltage coming from the main unit transformer

 

Work to be Carried Out

The unit is out of service

  1. Turn the supply voltage switch from the off position to the auxiliary voltage position.

  2.  Insert the special test key into the test selector switch (b10).

  3. Change the position of the test selector switch by the test key on the test panel to:

  • Test light position for alarm signal

  • Test heavy position for trip signal

  1. Press on the test key button until signal appears.

  2. Release the button of the test key and return the selector back to its first position (action)

  3. Return the supply voltage selector switch back to its off position

  4. Reset the unbalancing relay to its normal position

  5. Record the values (results) on the test sheet
