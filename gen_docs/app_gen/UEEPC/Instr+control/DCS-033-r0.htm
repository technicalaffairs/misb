Introduction

DSAI 130 is an analog input card (4 to 20 mA) used to connect current and voltage signals

Analog input boards function in general as follows, the boards are provided with a no. of channels, interference suppression, filtration and adaptation of The signal with respect to zero point and gain are performed individually in a buffer amplifier. The A/D converts is multiplexed between the I/P channels. An I/P channel is selected by the controller software by writing to a parameter register, which also starts the A/d conversion the value converted to digital form is then transferred via the parallel data bus to the controller software for further signal processing. Zero point and gain must be adjusted manually or by the controller software

Safety Precautions

  • A work permit must be issued and the qualified maintenance stuff should be carrying the suitable safety category.

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • Wear the safety helmet and shoes.

  • Unit must be shutdown.

Tools and Equipment

  • Safety helmet, shoes

  • Hand held radio wireless communications means

  • mA source to inject mA

  • Screw driver, nut driver

  • Isolating tape and test leads

Work to be carried out 

Steps to prepare for test

  1. Review technical reference material and manufacture's manual.

  2. Open panel no. 4

  3. Using panel terminations identify the location for the desired card.

Start test

  1. Using panel termination list for each card identify the two input wires for the first channel

  2. Using the screw driver, remove the two wires and isolate with tape

  3. Connect the mA source leads to the first channel

  4. Inject 4 mA, 8 mA, 12 mA, 16 mA and 20 mA according to 0%, 25%,75% and 100%

  5. Compare the mV injected using radio with the mV reading in the DCS room (eng. station)

  6. If the reading is ok, or within the range (-+ 0.01 mV), repeat steps for another channel

  7. If any reading for any channel in DCS not correct replace the card

Return card to service

  1. Remove the mA source

  2. Reconnect the input wires to terminal strip

  3. Close the cabinet

  4. Place a copy of test result in the field maintenance file
