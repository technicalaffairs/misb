  • The maintenance staff should be qualified and carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear safety helmet, shoes, breathing mask, safety glass ... etc

Tools and Equipment

  • Safety helmet, shoes, breathing mask, safety glass ... etc

  • Safety fence, caution marks.

  • Digital Ammeter.

  • Screw Driver.

  • Computer.

Work to be Carried Out

Preparation

  1. Review technical reference material and manufactures manual.

  2. Obtain copy of LCB field maintenance file.

  3. first make dry clean for all cabinets ( CBB01:CBB06 ) using vacuum cleaner.

  4. Retighten the terminals in ( CBB01:CBB06 )

Power Supply

  1. Check and measure input and output voltage for DC/DC converter, regulator and battery

  2. DC/DC converter ( 230/48/24 Vdc )  (    )

  3. Voltage regulator ( 24/5 Vdc )            (    )

  4. Backup battery ( 230Vac / 24Vdc )   (    )

MP200/1

Cards Front Panels

CPU

  1. Check LED's on front panel of CPU as:

  2. Run ( green led ) ON             (    )

  3. PBC ( yellow led ) ON          (    )

  4. BATT ( green led ) ON         (    )

  5. HLT ( yellow led ) OFF         (    )

  6. TO ( yellow led )  OFF          (    )

  7. P1 ( operation mode )         P (    )

Communication Board

  • HLDA ( yellow led ) flickering               (    )

  • A1, B1  A2, B2 ( yellow led ) flickering (    )

Analog input Boards

  • C ( yellow led ) flickering  (    )

  • F  ( red led ) OFF            (    )

Digital Board

  • F ( red led ) OFF                      (    )

  • Active signal ( yellow led ) ON  (    )

Bus Repeater Board

  • F ( red led ) OFF            (    )

  • DC ok ( green led ) ON  (    )

Digital input Signals

  1. Check the signals between control devices for generator and digital input boards.

  2. Check the signals between control devices for turbine and digital input boards.

  3. Check the signals between control devices for transformer and digital input boards.

  4. Check the signals between control devices for hydraulic governor and digital input boards.

Contactors

  1. Clean the contacts for contactors using dry spray.

  2. Retighten the terminals.

Alarm System

  1. Switch off power supply.

  2. Dismantle all electronic boards and clean using vacuum cleaner.

  3. Check the jumpers.

  4. Install the boards.

  5. Switch on power supply.

  6. Carryout the function test for alarm system.

Ventilation Fans

  1. Switch off power supply.

  2. Dismantle the fans.

  3. Clean and lubricate the fan using spray with oil.

  4. Install the fan.

  5. Connect the power supply.
