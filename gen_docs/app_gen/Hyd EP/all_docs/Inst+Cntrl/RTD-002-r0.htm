Introduction

This document outlines the testing procedures P1, to be carried out on an annually bases as indicated by Y1 in the above title block

This first draft initiated & tested by eng. Ahmed Mohamed Hamada, accepted by general director eng. Mohamed Diaa El-Gabalawe, approved by eng. Abdel Karem Abu Zaid: High Dam Hydraulic PS Head Sector, and certified by eng. Kamel Ragheb: HPGC quality control general director & MPIS Coordinator

Safety Precautions

  • A work permit must be issued, and the maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipment as safety helmet, shoes ... etc

Tools and Equipment

  • Safety and healthy equipment

  • Oven temperature tester

  • Ohmmeter

Work to be Carried Out

Preparation

  1. Review technical reference material and manufacture's manual

  2. Obtain copy of the setting value of RTD and record at MRDS

  3. Disconnect RTD wires and isolate the wires using tape

  4. Remove RTD from thermowell and take it to lab

Testing

  1. Put the RTD into the oven

  2. Connect RTD terminals to the ohmmeter

  3. Adjust the oven temperature as stated at the MRDS (from 20°C to 100°C)

  4. Record the readings of the oven temperature and ohmmeter at the MRDS

Return RTD to Service

  1. Return RTD to service by removing the ohmmeter leads from the RTD terminal

  2. Remove TD from oven tester

  3. Return the RTD back to its location

  4. Connect the RTD wires

  5. Place a copy of test result in the field maintenance file

Automatic Calibration

An RS 232 computer interface supplied as standard allows a calibration management software such as INTECAL-W. to provide automatic calibration procedures, typically with the following capabilities

  • Creation of instrument details

  • Creation of test procedures and work orders

  • Analysis of devices due for calibration

  • Printing of calibration reports

  • Export functions

Please refer to the SERIAL PROTOCOL section for all the information related to the computer management of this instrument

Parts Identification

  1. Equalizer block, calibration block with insert

  2. Protection grid

  3. Switch test connection, suitable to connect test leads supplied as standard

  4. Lemo connection for external reference probe: suitable for connecting PT 100 probes with 3 or 4 wires: supplied with standard mating connector (B1, B2 and B3 options)

  5. Digital display with multi-language user interface

  6. Tactile keypad for direct selection of all the operating function (ramps, steps, preset values, switch test, help, setup) and to set temperature values

  7. Cursor (arrow) keys to select operating functions from the main menu and to move the selection cursor through the menus

  8. RS 232 serial communications interface

  9. 4 mm socket for voltage and current input with 24 v dc auxiliary excitation

  10. 4 mm socket for T/C`s, TRD`s and ohm connection, with CJ compensation

 

Calibration

 

Using a certified and high accuracy ohms simulator, apply the required value to the related electrical connection on the front panel of the instrument (the nominal value is the recommended value for the test)

Pressing the ent key will cause the microprocessor to combine the resistance measurement on the display (True value and effective reading)
