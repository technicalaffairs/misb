Introduction

This document outlines the inspection checklist I1 for auxiliary board, common part turbine house which must be carried out every six months as indicated by M6 in the title block.

The first draft initiated & tested by eng. Zakaria Mohamed Abd El-Reheem, accepted by eng. Osama Ahmad Hassanein, approved by eng. Khalaf Allah Mahmoud Abd El-Gaber head sector of Aswan Dam 1,2 and certified by eng. Mohamed Osama Soliman HPGC MPIS coordinator and inspection & quality control's general director.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category.

  • Maintenance staff should be will trained and qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear your safety and healthy equipment as safety helmet, safety shoes, goggles, and rubber glove. ...etc

Tools and Equipment

  • Digital ammeter

  • Screw driver

  • Computer

  • Vaccum cleaner

Work to be carried out 

Preparation

  1. Review technical reference material and manufacture's manual

  2. First make dry clean (0CBD 01: 0CBD06) by vacuum cleaner

  3. Retighten the terminals in (0CBD01 : 0CBD 06)

Steps of maintenance work
Power supply

  1. Check and measure input and and output voltage for DC/DC converter regulator and battery

    • DC/DC converter (230/48/24 Vdc)            (       )

    • Voltage regulator (24/5 Vdc)                       (      )

    • Backup battery (230 Vac / 24 Vdc)           (      )

MP 200/1 cards front panels

CPU

  1. Check led"s on front panel of CPU as

  2. Run (green led) on                                        (      )

  3. PBC (yellow led) on                                     (      )

  4. BATT (green led) on                                    (      )

  5. HLT (yellow led) off                                     (     )

  6. To (yellow led) off                                        (     )

  7. (P1 operation mode)                                     (     )

 

Communication Board

  1. HLDA (yellow led) flickering

  2. A1,B1 A2,B2 (yellow led) flickering             (       )

Analog Boards

  1. C (yellow led) flickering                              (        )

  2. F (red led) off                                             (       )

Digital Boards

  1. F (red led) off                                            (       )

  2. Active signal (yellow led) on                       (      )

BUS Repeater Board

  1. DC ok (green led) on                                (       )

  2. F (red led) off                                            (       )

Alarm System

  1. Switch off power supply

  2. Dismantle all electronic boards and clean it by using vacuum cleaner

  3. Check the jumpers

  4. Install the boards

  5. Switch on power supply

  6. Make a function test for alarm system

Ventilation Fans

  1. Switch off power supply

  2. Dismantle the fans

  3. Make a clean and used spray with oil to lubricate the fan

  4. Install the fan

  5. Connect the power supply

Function Test

  1. Use a computer to check the change over time (11 second)

  2. Check and record the actual value for change over time between the signal of first stage under voltage for 0.4 kv bus bar appear and stand by circuit breaker on during the maintenance of 0.4 kv CB and auxiliary transformers (11/0.4 kv)
