Introduction

This document outlines the inspection checklist   for Automatic Telephone System type BOSCH INTEGRAL33XE  which must be carried out on an annually

The first draft is initiated by eng. Elsayed Sobhy Saadand and tested by eng. Usama Ahmed, accepted by eng. Haseeb Heseen general director, approved by eng. Mohamed AbdEl Paset

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  •  A work order must be issued, the maintenance staff should be qualified and carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required.

  •  Wear safety helmet, shoes.

Tools and Equipment

  •  Safety helmet, shoes, and rags

  • Digital Ammeter.

  • Screw Driver.

  • Open end Spanner set.

Work to be Carried Out

Steps to Prepare for test

  1. Review technical reference material and manufacture’s manual.

  2.  Obtain copy of Automatic Telephone System field maintenance file.

Testing Procedure

  1. Disconnect power supply

  2. Check and clean the electronic card as follow

  • Analog subscriber circuit ( ASCEU )

  • Analog trunk circuit   ( ATA )

  • Digital line card ( DUPO )

  • Computer board  ( CB )

  • Control module ( CFD )

  1. Connect power supply and check the function

  2. Make a function test of the system   

  3. Place a copy of test result in the field maintenance
