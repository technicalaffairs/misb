Introduction

This document outlines the testing procedure using secondary injection test set (Sverker 650)

The purpose of the relay is basically to provide a trip and alarm settings capability corresponding to the generator capability sense the salient pole gen with damper winding ca be damaged, if the negative phase sequence currents exceed 5% of the generator rated current for long periods of time.

Safety Precautions

  • A work order must be issued and the qualified maintenance staff should carry the suitable safety ID category.

  • The generator is out of service

  • Safety fence with caution marks surrounding the work area is required.

  • Wear safety and healthy equipment as safety helmet, safety shoes

Tools and Equipment

  • Secondary 1-ph injection test set (sverker 650 - sweden)

  • Multimeter

  • Test - plug handle (RTXH 18)

  • Test wires

  • Variable resistance (0-450)ohm

 

 

 

 

Work to be Carried Out

The Generator is out Service

  1. Connect the circuit as shown in fig (1).

  2. Connect the test plug handle (RTXH 18).

  3. Increase the current gradualy until reach the operating value.

  4. Ensure that the relay operates at the setting alarm current value (0.27) A.

  5. Ensure that the relay operates at the setting alarm time value 6sec.

  6. Repeat for trip setting values ( I = 2.7 A,  t = 20 sec.)

  7. Repeat for phases S & T

  8. Record this values on the test sheet.

  9. Take off the equipments and the wires, then return the relay to its normal position.
