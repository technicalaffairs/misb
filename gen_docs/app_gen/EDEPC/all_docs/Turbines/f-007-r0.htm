Safety Precautions

  • Issue work order including the teamwork names

  • Use Safety fences with caution marks to surround the work area.

  • Wear safety helmets, shoes, earplug and gloves pairs.

  • Stop the fanes.

  • Be sure that the dust evacuation system is out of service

 Tools and Equipment

  • Compassed air hose.

  • Articles.

  • Special cleaning system.

  • Maintenance tool set.

Work to be Carried Out 

  1. Check the differential pressure. The reading must be between [ 50 mm -100 mm ] WG.

  2. Check the compressed air [6 – 7 bar].

  3. Check the function of the diaphragm valves.

  4. Check the dust supply line and dust discharge system.

  5. Clean the filter elements.

  6. Check the filter element gasket and replace it if it is required.

  7. Check the filter element sealing washers at the bottom of the filter lament.

  8. Check and repair the filter element leaks.

  9. Check the fan balancing.

  10. Remove the dust deposition on the fan blades.

  11. Replace fan worn blades.

  12. Replace the worn bearing.
