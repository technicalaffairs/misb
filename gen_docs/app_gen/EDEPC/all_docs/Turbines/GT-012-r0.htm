Safety Precautions

  • A work permit must be issued, the maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipment as safety helmets, shoes, gloves ..  etc

  • Turbine must be shut down

Tools and Equipment

  • Complete maintenance tool set

  • Cleaning fluid ( WD 40 )

  • Rags

Work to be Carried Out 

  1. Dismantle the flame monitor ( loosing 4 bolts 19 )  

  2. Inspect the HP sight glass  

  3. Clean the HP sight glass if it is dirty  

  4. Change the HP sight glass if it is broken  

  5. Change all gasket up and down the HP sight glass

  6. Install the flame monitor  ( tightening  4 bolts 19 )
