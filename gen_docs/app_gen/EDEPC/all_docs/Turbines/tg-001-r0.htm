Introduction

This document outlines the equipment inspection list I1, which must be performed  every six months as indicated by M6 or after 1000 operating hours in the above titled block.  This device used for slowly motion.

Safety Precautions

  • Issue work order including the teamwork names.

  • Use Safety fences (ropes) with caution marks to surround the work area.

  • Wear safety helmets, shoes, earplug and gloves pairs.

Tools and Equipment

  • Complete set of tools.

  • Grease gun

  • Soft brush, Oil can.

  • Cleaning articles.

Work to be Carried Out 

  1. Lubricate movable parts. (The type of oil is VG46).

  2. Check the oil level; add if it is required.
