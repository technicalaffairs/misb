Introduction

These document outlines the testing procedure P1, to be performed on an annually bases as indicated by Y1 in the title block.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment and is required to completely check the proper operation and calibration of the relay.

The first draft initiated by eng. Mohamed Osman, tested by eng  Mahmoud Youssef, reversed by EDEPC quality control eng. Adel Abd-Elqader Elnaggar, and approved by eng. Sawsan El_Lethy: general director of Port Said Gas Turbine.

This document accepted by eng. Samir Abdel Gilel: EDEPC inspection and quality control general director & MPIS coordinator, and certified by accountant: Mahmoud Mohamed Soliman EDEPC planning & financial predicting head sector and supervisor of inspection & quality control's head sector.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

Tools and Equipment

  • Secondary injection test set.

  • Voltmeter, clamp ammeter.

  • Test leads, complete tool set.

    •

Work to be Carried Out

  1. Disconnect the circuit breaker for generator.

  2. Removed the plug of connection for Time-Over current relay.

  3. Put the test plug of  connection in Time-Over current relay

  4. Connect test leads between the test set and the current terminal (5&6) for phase ( R ) in the test plug  for the Time-Over current relay.

  5. Connect test leads between the test set and the time terminal (1&2) for phase (R) in the test plug for the Time-Over current relay.

  6. Connect test leads between the test set and the voltage terminal (7&8) for phase (R) in the test   plug for the Time-Over current relay.

  7. At zero restraint volts Increase the injected current slowly until the red indication on the Time-Over current element phase ( R ) on the relay appears.  

  8. Record this value (of the current and the time) on the test sheet
    ( adjust it if needed ).

  9. Switch off the test set.

  10. Repeat the above test from step ( 4 ) to ( 8 ) but inject current between terminal ( 5 &6) for phase( S ) and terminal ( 5&6 )for phase ( T ) and time terminal (1&2) for phase (S) and terminal (1&2) for phase (T) and voltage terminal (7&8) for phase (S) and terminal (7&8) for phase (T).

  11. At { 48% of rated restraint voltage(60volt a.c)} Increase the injected current slowly until the red indication on the Time-Over current element phase ( R ) on the relay appears.

  12. Record this value (of the Current and the time) on the test sheet ( adjust it if needed ).

  13. Switch off the test set.

  14. Repeat the above test from step ( 11 ) to (12 ) but inject current between terminal ( 5 &6) for phase ( S ) and terminal ( 5&6 )for phase ( T ) and time terminal (1&2) for phase (S) and  terminal (1&2) for phase (T) and voltage terminal (7&8) for phase (S) and terminal (7&8) for phase (T).

  15. 15.At{100% of rated restraint voltage(120volta.c)}Increase the injected current slowly until the red indication on the Time-Over Current element phase (R) on the relay appears.

  16. Record this value (of the Current and the time) on the test sheet ( adjust it if needed ).

  17. Switch off the test set.

  18. Repeat the above test from step (15) to (16) but inject current between terminal ( 5 &6) for phase ( S ) and terminal ( 5&6 )for phase ( T ) and time terminal (1&2) for phase (S) and terminal (1&2) for phase (T) and voltage terminal (7&8) for phase (S) and terminal (7&8) for phase (T).

Return Relay to Service

  1. Disconnect the test leads from terminal on the relay.

  2. Remove the test plug from the relay.

  3. Put the plug of connection for Time-Over current relay.

If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  • Breaker trip coil

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.

  1. Repeat the circuit breaker of the generator to the normal position.
