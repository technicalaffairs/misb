Introduction

This document outlines the testing procedure P1, to be performed on an annually bases as indicated by Y1 in the title block.

The relay 51 GN with type IAC produce generator earth fault protection, and this achieved by mounted it on the neutral phase. This procedure details the steps required to completely check the proper operation and calibration of the relay using secondary injection test set.

The document is formatted  the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initiated by eng. Eslam Mahmoud Abas, tested by eng. Tamer Mohamed Mokhtar, accepted by eng. Sawsan Mohamed El-Lethy: Port Said general director, approved by eng Mohamed Hesain Ahmed Gomaa: head sector, and certified by eng. Zakaria El-Shahat: EDEPC MPIS Coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Equipment isolation (disconnect switches are opened and tagged).

  • Measure the current circuit of the relay by clip ammeter to be sure that is no current.

  • Make sure CT secondary are shorted before isolating  relay current circuits for test current injection.

Tools and Equipment

  • Secondary injection test set with timer.

  • Test leads, clamp ammeter, voltmeter, Test plug.

  • Complete tool set.

    •

Work to be Carried Out

Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand

  4. Check the AC source is available to the tester

  5. Check the existence of the DC sources in panels

  6. Obtain the input terminals of the relay (current terminals) and also the output terminals (trip terminals) from the station from the station drawing

  7. Remove the relay plug and insert the test plug

  8. Disconnect the trip leads of the relay and connect one of them to the timer of the tester

  9. Prepare the test sheet for the tested relay

Testing the over current element

Current Setting Test

  1. Connect test between the current terminals on the relay (5&6) and the current terminals on the test set

  2. Slowly increase the current until the relay pick up and record it as a pick up value

  3. Record this value on the test sheet (adjust it if needed)

  4. Increase the injected current to a value 25% greater than the operate value

  5. Record this value on the test sheet (adjust it if needed)

Time setting test

  1. Inject current into relay until the relay picks up. Keep this value and switch off the test set.

  2. Connect the relay time delayed output contacts (1.2) to the timer contacts of the test set.

  3. Switch on the test set and inject current. The timer will stop once the relay has picked up.

  4. Record this value on the test sheet (adjust it if needed).

  5. Repeat steps (1.2.3.4) on a various pick up values and record these values on the test sheet.

  6. Remove the leads from the time delayed contact and the test set.

Circuit breaker trip test

  1. Set up the circuit breaker for a trip test.

  2. Inject current into the relay until the circuit breaker trips at the time delayed setting.

  3. Close the circuit breaker again.

  4. Remove the wire from the relay time delayed contact terminals No. 2 or 1.

  5. Inject a value of current greater than the instantaneous setting suddenly into relay until the circuit breaker trips at the instantaneous setting.

Return relay to service

  1. Disconnect the test leads from terminal on the relay.

  2. Remove the test plug from the relay.

  3. Put the plug of connection for current relay.

  4. Re-install the wire which was removed.

  5. Remove the shorts from the CT secondary circuits.

  6. Return the relay and power equipment to service.

  7. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  • Breaker trip coil.

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.
