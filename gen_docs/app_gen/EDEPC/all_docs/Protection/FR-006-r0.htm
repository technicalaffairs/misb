Introduction

    This document lists the equipment inspection I1, which must be performed on an annually bases as indicated by Y1 in the above title block.

    The EMS 1000 equipment provides chronological recording of data presented to it as logic states. These data are identified and dated on each change of state. An alphanumeric printer prints them out, together with the label identifying the unit concerned in plain language.

    EMS 1000 comprises the following cards:

  • Control processing unit cards.

  • Printer control card.

  • Memory extension card.

  • 16 TS card.

  • Master slave cards.

  • Printer power card

  • Front panel card.

  • Converters.

  • Built in printer.

Safety Precautions

  • A work permit must be issued

  • The qualified maintenance staff should carry the suitable ID safety category,

  • Safety fence with caution marks surrounding the work area is required,

Tools and Equipment

  • Complete tool kit of screw drivers and common keys.

  • Digital multi-meter

  • Fine clothes for cleaning purpose

  • Spray, contact cleaner.

Work to be Carried Out

  1. Check that the internal cables are not disconnected.

  2. Check that the connection cables are serviceable.

  3. Check the fuses, if fault are found.

  4. Check that all cards required to be present are actually present.

  5. Check that there is sufficient paper for the printer (from time to time )

  6. Clean the mechanical parts of the printer.

  7. Check the state of the self-test: The equipment failure contact informs the operator in the event of test results being incorrect. The key CTRL can be used to identify the type of failure.
