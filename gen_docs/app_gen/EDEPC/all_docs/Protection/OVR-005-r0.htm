Introduction

This document outlines the testing procedure P1, to be carried out on an annually bases as indicated by Y1 in the title block.

This procedure details the steps required to completely check the proper operation and calibration of the relay.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment

The first draft initiated by eng. Mohamed Mahmoud Osman, tested by eng. Eslam Mphamed Abas, accepted by eng. Sawsan Mohamed El-Lethy: Port Said general director, approved by eng Hamdy Abd El-Moly Melegy: Head sector, and certified by eng. Zakaria El-Shahat: EDEPC MPIS Coordinator.

Safety Precautions

  • A work permit must be issued

  • The qualified maintenance staff should carry the suitable ID safety category

  • Safety fence with caution marks surrounding the work area is required

  • Equipment isolation (disconnect switches are opened and tagged)

  • Measure the current circuit of the relay by clip ammeter to be sure that is no current

  • Make sure ct secondary are shorted before isolating  relay current circuits for test current injection

Tools and Equipment

  • Secondary injection test set

  • Test leads, clamp ammeter, voltmeter, Test plug

  • Complete tool set

Work to be Carried Out

  1. Disconnect the circuit breaker for generator

  2. Removed the plug of connection for over voltage relay

  3. Put the test plug of connection in over voltage relay

  4. Connect test leads between the test set and the voltage terminal (5&6) for the over voltage relay as shown in fig 1

  5. Connect test leads between the test set and the time terminal (1&2) for the over voltage relay

  6. Inject voltage drop (120 v) in the voltage terminal (5&6)

  7. Increase the injection voltage slowly until the red indication on the over voltage relay appears

  8. Record this value, operating voltage (123v) and the time on the test sheet adjust it if needed

  9. Switch off the test set

Return Protection Relay and associated Equipment to Service

  1. Disconnect the test leads from terminal on the relay.

  2. Remove the test plug from the relay.

  3. Put the plug of connection for time-over current relay.
    If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  • Breaker trip coil.

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.

  1. Repeat the circuit breaker of the generator to the normal position.
