Introduction

This document outlines the equipment inspection I1, which must be performed on an annually bases as indicated by Y1 in the above title block.

The Control & Protection Burners consists of:

  • Release and Protection testing for Burners

  • N.G flame system

  • H.F.O flame system

  • Ignition system

  • Sol. valves (N.G, H.F.O, Gas Oil, Scavenging steam, Ignition, Atomizing steam)

  • Air dampers for each Burner

  • Lances for H.F.O Burners

  • Local gauge

For Testing & Protection System

Each protection loop have (sensor, switch, software logic, I/O electronic modules and final control element test)

Safety Precautions

  • A work permit must be issued and maintenance stuff should carry the suitable safety category.  

  • Safety fence with caution marks surrounding the work area is required.

  • Wear your safety and healthy equipment as safety helmet, safety shoes, ... etc.

  • Boiler must be out of service.

  • Don't allow to any material of fluids could cause a fire in the work area.

Tools and Equipment

  • Digital multi-meter, Current injector

  • Screw driver kit, manual tools

  • Oil spry cleaner & Teflon roll

  • Spanner, Air Blower

Work to be Carried Out

Preparation

  1. Obtain a copy of (I/O interface card, flame scanner, ignition flame, flame amplifier, sol.valves, dampers, cold junction, junction box ) maintenance manuals

  2. Obtain a copy of schematic loop wiring diagram for any protection signal, and data sheet.

Release & Protection Test

Ignition System

  1. Check igniter of each burner and make maintenance for it (clean detector electrode, clean spark electrode).

  2. Check transformer for each burner (measure resistance, check input signal).

  3. Check ignition flame (check amplifier signal, check relay, check signal from relay up to ovation screen).

  4. Tight all screw in junction box.

  5. Check sufficient air flow for cooling.

  6. Clean ignition gas box by air blower.

Natural Gas flame scanner

  1. Clean lens.

  2. Check socket.

  3. Check cable between flame scanner and junction box.

  4. Check power supply.

  5. Check flame amplifier (check signal voltage, check relay).

  6. Check signal from relay up to ovation screen.

  7. Check sufficient flow for cooling air.

  8. Clean natural gas flame box by air blower.

Heavy Fuel Oil Flame System

  1. Clean lens.

  2. Check socket.

  3. Check cable between flame scanner and junction box.

  4. Check power supply.

  5. Check flame amplifier (check signal voltage, check relay).

  6. Check signal from relay up to ovation screen.

  7. Check sufficient flow for cooling air.

  8. Clean heavy fuel oil flame box by air blower.

 

Solenoid Valves

  1. Clean moving parts for solenoid valves.

  2. Check "O" ring.

  3. Measure resistance for coil.

  4. Check wiring up to junction box.

  5. Check limit switches for valves.

  6. Check signal up to ovation screen and output command from ovation control.

  7. Check time for opening and closing valves.

Air Damper for Each Burner

  1. Check limit switches.

  2. Check wiring up to junction box.

  3. Adjust stroke.

  4. Check socket.

  5. Check positioner (measure power supply, check ma circuit ).

  6. Check signal up to ovation screen and output command from ovation control).

Lances

  1. Check limit switches.

  2. Check mechanical position.

  3. Check "O" ring.

  4. Measure resistance for coil.

  5. Check wiring up to junction box.

  6. Check signal up to ovation screen.

  7. Check limit switch ( measure resistance ).

  8. Check local gauge for right calibration.
