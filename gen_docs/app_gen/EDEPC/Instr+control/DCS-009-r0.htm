Work to be Carried Out

Steps to Prepare for Test

  1. Review technical reference material and manufacture’s manual.

  2. Open DPU cabinet.

  3. Using cabinet termination identify the location for the desired QAO card.

 

Start Test

  1. Using cabinet termination list for each card identify the two O/P wires for the 2nd channel.

  2. Using the screw driver to remove this two wires and isolate with tape.

  3. Connect the m A tester leads to the 1st channel.

  4. From CCR give command 0%, 25%, 50%, 75%, 100% which equals 4m A ,8m A , 12m A , 16m A and 20m A .

  5. Compare the command from CCR using radio with the mA reading.

  6. If the reading is OK or within the range ( + 0.01 mA ), repeat steps 1 to 5 for the another 3 channels.

  7. If any reading for any channel in CCR not correct replace the card.

Return Card to Service

  1. Remove the m a tester..

  2. Reconnect the O/P to terminal strip.

  3. Close the cabinet.

  4. Place a copy of test result in the field maintenance file.
