Introduction

This document outlines the testing procedure I1, which must be performed every 6 months as indicated by M6 in the above title block.

The first draft initiated by eng. Hesham El Sayed Mohamed El Assy, tested by eng. Mohamed Gad El Rab Hawad at 19 Feb 2003, reviewed by EDEPCo. quality control eng. Abdel Momem El-Emam, accepted by general director eng. Farouk Awad Aly, and approved by eng. Mostafa Alam Head Sector of Talkha power plant.

Query provides a point and click interface to construct and analyze the execution of the queries specifically designed for Industrial SQLTM Server. Query provides an efficient and simple method to develop simple and complex queries for your reporting and custom application requirements. It also is an excellent way to learn the proper syntax for developing SQL queries.

 Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should be carry the Suitable safety category,

  • Make Backup for soft ware before work.

Tools and Equipment

  • Original Software

  • CD Writer to Take Backup

  • Floppy Disks.

  • Paper And Pen.

Work to be carried out

  1. Start the program select

  2. The program windows appear

  3. Select the server (2INSQL OR 1 INSQL)

  4. From menu file chose the connect command

  5. Select the type of signals (Analog)

  6. Check the signal right or fault

  7. Repeat steps from 2 to 6 with (Discrete)

  8. Repeat steps from 2 to 6 with (string)

  9. Repeat steps from 2 to 6 with (Event)

  10. Repeat steps from 2 to 6 with (History value)

  11. Repeat steps from 2 to 6 with (Live value)

  12. Close the program.

Return the system to Normal operation

After That You Can Close This Program And Test Operation From Any Signals if like this ----à this means the program is ok
