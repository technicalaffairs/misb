Introduction

This document outlines the equipment inspection I1, to be performed on an annually bases as indicated by Y1 in the title block.

The chlorination producing unit function is to produce hypo-chloride Sodium by electrical analysis to sea water.

The Chlorination Production Unit consists of: five Electrolysers, two sea water pumps, two Filters, two degasifiers, two fans, two continuous dosing pumps, two shock dosing pumps, two storage tanks, two Flow control loops, one level control loop.

For testing the Chlorination Producing Unit perform the following:

  • Release and Protection testing for electrolysers, seawater pumps, filters, degasifiers, fans, continuous dosing pumps, shock dosing pumps.

  • Check the Flow/ Level control loops components (Level/Flow transmitters,  mA/pn. converters, pneumatic actuators (positioners), controllers , limit value cards and solenoid valves.

Safety Precautions

  • A work order must be issued; the maintenance staff should be qualified and carry the suitable safety ID category.

  • Wear safety and healthy equipment as safety helmet, safety shoes …etc

  • Safety fence with caution marks surrounding the work area is required.

  • The Chlorination Producing unit must be out of service.

Work to be Carried Out

Preparation

  1. Obtain a copy of ( Level/Flow transmitters, mA/pn. converters, positioners, controllers, limit value cards& solenoid valves ) maintenance manuals

  2.  Obtain a copy of relay control circuit diagram of Chlorination Producing unit   

Release and Protection Test

Seawater Pumps

  1. Check pressure switch of suction pressure of each pump w.r.t MPIS doc PS-001-r0 and check this alarm on alarm window.

  2. Check pressure switch of discharge pressure of pumps w.r.t MPIS doc PS-001-r0 and check this alarm on alarm window.

  3. Check limit value setting of seawater min flow protection.

Electrolysers

  1. Check PT100 Sensor  w.r.t MPIS doc RTD-001-r0 for inlet line and outlet line of each Electrolyser.

  2. Check RTD/ohm converter for inlet line and outlet line of each Electrolyser.

  3. Check limit value setting of temperature >max for inlet line and outlet line of each Electrolyser.

Filters

Check differential pressure of each filter according to approved MPIS.

Continuous Dosing Pumps / Shock Dosing Pumps

  1. Check pressure switch of suction pressure of each pump w.r.t MPIS doc PS-001-r0 and check this alarm on alarm window.

  2. Check pressure switch of discharge pressure of pumps w.r.t MPIS doc PS-001-r0 and check this alarm on alarm window.

  3. Check limit value setting of Hypo chloride Sodium min level protection.

  4. Check operation and shutdown timer for shock dosing pumps.

Fans

Check pressure switch of discharge pressure of fans w.r.t MPIS doc PS-001-r0

Closed Loop Circuit Test

Flow / Level Control Loops

  1. Check Flow/ Level transmitters w.r.t MPIS doc TRT-001-r0

  2. Check mA/pn. converters.

  3. Check pneumatic actuators (positioners).

  4. Check parameter of each controller.

  5. Check limit value setting of Flow/Level <min for each control loop.

  6. Check solenoid valves of each control loop.
