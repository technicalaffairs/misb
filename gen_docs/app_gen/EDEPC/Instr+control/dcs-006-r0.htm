Introduction

  • FUM 230

  • FUM 232

  • FUM 280

Work to be Carried Out

  1. Check the fixed alarm at control room by I&C engineer or by maintenance request from operation department.

  2. By DCS system information I&C engineer can define the king of problem and which cabinet and which FUM card has the problem

  3. Each FUM card has an indication lamp OFF at normal operation.

  4. If the indication lamp of FUM card flash light this means the problem in the field then check the field devices (switches, transmitters, RTD, thermometers ... etc)

  5. If the indication lamp of FUM card fixed light this means the problem in FUM module itself then change the FUM card
