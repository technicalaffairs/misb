The Distributed Processing Unit (DPU) consist of:

MSQ card, controller card, Part # (7381A29G05)

MSE card, memory card, Part # (7381A83G02)

MME card, memory card, Part # (7381A34G02)

MHC card, data high way controller card, Part # (7381A93G01)

QAW card, analog input card, (4-20mA), Part # (7379A31G04)

QAV card, analog input card, (-50,+50mV), Part # (7379A21G02)

QRT card, analog input card, (0-33mV), Part # (7379A62G02)

QAO card, analog output card, (4-20mA), Part # (2840A21G01)

QCI card, digital input card, Part # (7379A08G02)

QRO card, digital output card, Part # (2840A18G01)

QBE card, bus extender card, Part # (7379A84G01)

Multibus power supply, input 220V AC, output: + 12VDC, + 5VDC, PART # (1D5482G06).

Q-Line power supply, input 220V AC, output: + 13VDC, PART # (405A662G02).

8 cooling fan, 230 VAC, 14 watt, Part # (2450D25G03).

24 VDC transmitter power supply.

The cabinets receive the signals from sensors at the field (unit) then change it to ma and mv signals proportion with the actual values at field, then transmit it to displayed at monitors to process it according to the operation circumstances.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required.

  • Wear appropriate personal protective safety equipment as safety shoes and helmet

    • ... etc

  • Wear the ground strap.

  • Make backup for software before work.

  • Disconnect the cabinet from data highway if required according to station status without any effect on operation of plant.

Tools and Equipment

  • Screw driver, nut driver, air blower.

  • Spray video cleaner.

  • Ground strap.

  • Card extractor.

  • Digital voltmeter (DVM).

Work to be Carried Out

  1. Check the back up software for the cabinet.

  2. Check the operation of net work card function (data highway controller card (MHC card) by using (system highway errors) diagram.

  3. Clean all cabinet cards from dust by blower.

  4. Check the cabinet status by using (system status display) diagram.

  5. Check the multibus power supply of primary controller (+ 12V DC , 5V DC) .

  6. Check the transmitter power supply (24V DC) status by leds on the front of P.S.

  7. Check Q-line power supply (13V DC) status by using system status display diagram.

  8. Check the operation of cabinet cooling fan.

  9. Check the multibus power supply of Secondary controller (+12V DC , 5V DC) .

  10. Check both of primary and backup Controller data base by using configuration diagram on the operator station.
