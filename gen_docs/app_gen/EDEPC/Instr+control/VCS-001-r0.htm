Introduction

This document outlines the equipment inspection checklist I1, to be carried out every six months as indicated by M6 in the title block.

The Control & Protection of HP bypass & Live steam consists of:

Release and Protection testing for HP bypass & Live steam system:

a- HP-bypass (Hydraulic) .

b- HP-bypass (Electrical) .

c- Live steam & Warm up valve for live steam.

For testing protection system perform the following:

Each protection loop have (sensor, switch, software logic, I/O electronic modules, and final control element test)

Work to be Carried Out

Preparation

  1. Obtain a copy of (I/O interface card, HP-bypass, actuator) maintenance manuals.

  2. Obtain a copy of schematic loop wiring diagram for any protection & control signals.

Control & Protection Test

Release & Protection for Hydraulic HP- bypass & Injection valve

  1. Clean filter oil for servo valve.

  2. Check hydraulic pump pressure and insure no leakage.

  3. Check coil for emergency solenoid valve (measure resistance, check socket).

  4. Adjust stroke for valve and check potion mA

  5. Check wiring up to junction box to DCS, to solenoids, from pressure switches.

  6. Check signal up to ovation screen and output command from ovation controller.

  7. Check faults and fault lamps.

Closed Loop Circuit

  1. Check ( pressure transmitter, temperature element ) for control loop.

  2. Check normal operation during start up and return if required.

 

Release & Protection for Electrical HP- bypass & Injection Valve

  1. Check limit switch (measure resistance) .

  2. Check socket (clean socket, check wiring terminal).

  3. Check power unit (contactor, relay, operation).

  4. Check positioner ( measure power supply, check {mA circuit ).

  5. Adjust stroke.

  6. Check wiring up to junction box.

  7. Check signal up to ovation screen and output command from ovation controller.

Closed Loop Circuit

  1. Check ( pressure transmitter, flow transmitter, temperature element ) for control loop.

  2. Check normal operation during start up and return if required.

Release & Protection for Live Steam & Warm up Live Steam Valve

  1. Check limit switches (measure resistance)

  2. Check socket ( clean socket, check wiring terminal )

  3. Adjust stroke

  4. Check signal up to ovation screen and output command from ovation controller.
