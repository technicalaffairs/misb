Introduction

This document outlines the routine check I1 , which must performed on a daily bases as indicated by D1 in the title block.

(PCU): Process Control Unit

These cabinets contains the following type of cards:

MFP:

Multi function processor

NIS:

Network interface

NPM:

Network processor module

MFC:

Multi function controller

Each previous module card has an indication lamp at the top

  • At normal operation to be green light when the module card main or redundancy

  • At fault to be red light when the module card main or redundancy each previous module card have 8 small red indication lamb in the module card front.

The daily task for I & C engineers is monitoring DSC for Boiler to ensure on the normal operation

If there is an alarm you must open a work order and follow the steps stated at work to be carried out

Safety Precautions

  • A work permit must be issued, the qualified maintenance staff should carry the suitable safety category, 

  • Safety fence with caution marks surrounding the work area is required.

  • Wear your safety helmet, safety shoes, eye glass, nose mouth "dust cover" .... etc

  • Wear anti static hand probe to avoid electrical choke

  • The unit in service.

Tools and Equipment

  • Digital multi-meter

  • Hand held radio wave for communication

Work to be Carried Out

  1. Check the fixed alarm at control room by I&C engineer or by maintenance request from operation department.

  2. By DCS System information, I&C engineer will defined the kind of problem & which cabinet, and which module card cause the problem at Card its self or software or at the field

The module card (NPM & NIS)

  1. When (NPM & NIS) at main operation the 8 indication lamp to be count from 8 to 1 by flash red light.

  2. When (NPM & NIS) at redundancy the indication lamp to be (OFF) except indication lamp (7&8) to be fixed red light

The Module Card (MFP & MFC)

  1. When (MFP&MFC) at main operation the indication lamp to be (OFF) except the indication lamp (7&8) to be fixed red light.

  2. When (MFP & MFC) at redundancy the indication lamp to be (OFF) except the indication lamp (7)

  3. If the module card (MFP-NIS-NPM & MFC) have trouble the moduel have facility of (hard ware or soft ware) reset

  4. If the reset (hard ware or software) successful it is OK

  5. If the reset (hard ware or soft ware) not successful, this means the problem in moduel card it self the you have to change the moduel card
