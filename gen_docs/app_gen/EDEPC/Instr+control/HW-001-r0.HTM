Introduction  

This document outlines the testing procedure I1, which must be performed every 3 months as indicated by M3 in the above title block.

The first draft initiated by eng. Hesham El Sayed Mohamed El Aassy, tested by eng. Mohamed Gad El Rab Hawad at 26 March 2003, reviewed by EDEPCo. quality control eng. Abdel Momem El-Emam, accepted by general director eng. Farouk Awad Aly, and approved by eng. Mostafa Alam Head Sector of Talkha power plant.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should be carry the suitable safety category,

  • Make backup for soft ware before work.

  • Check any CD’s or floppy form any virus before opening in the computer.  

Tools and Equipment

  • Original Software

  • CD writer to make backup

  • Floppy Disks, Paper and Pen.

  • CD and floppy cleaner

  • Blower, Screw driver

  • Anti virus, Network cable

Work to be carried out

  1. The application software must be checked on a monthly bases.

  2. Schedule the operation of antivirus program and Live update the definitions from internet

  3. Schedule scan disk for hard disk surface media

  4. Schedule defragmentation for hard disk.

  5. Schedule the disk clean up.

  6. Check drive free space for partitions not less than 30%

  7. Check the operation of VGA card function.

  8. Check the operation of Sound card function

  9. Check the operation of modem card function

  10. Check the operation of Network card function.

  11. Clean the computer from dust.

  12. Clean the CD_ROM. using CD cleaner.

  13. Clean the floppy drive using floppy cleaner.

  14. Check the Office program.

  15. Check the ink level, reppon and the cartridge of the printer.

  16. Clean the Printer

  17. Check the self tests for the printer i.e. Print the printer configuration

  18. Check the color and black ink

  19. Check the path of paper.

Return the System to Normal Operation

  1. Test and print any pages from archiving.

  2. Test the computer operation i.e. all cards functions working well

  3. Test the operation of printers

  4. Test PC's connection to the network and to the server.
