Introduction

QAV is an analog input card (-50 to +50 mV) of six channels from thermocouple, and QRT is an analog input card (-50 to +50 mV) from RTD and of 4 channels.

This document outlines the testing procedures P1 for QAV, QRT carts, which must be carried out on an annually bases as indicated by Y1 in the above title block

The first draft initiated by eng. Ahmed Esmail Fouad & eng. Khalil Ahmed Awwad, QAV tested at 10 Nov. 2002, QRT tested at 05 April 2003 and revised by EDEPCo. quality control eng. Safety Precautions

  • A work permit must be issued and the qualified maintenance stuff should be carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear the safety helmet and shoes.

  • Unit must be shutdown.

Tools and Equipment

  • Safety helmet, shoes,

  • Hand held Radio, wireless communications means.

  • Screw Driver, Nut Driver,

  • Isolating tape, and test leads.

  • Temperature tester to inject Temp. or mV source according to thermocouple type.

  • mV source to inject Temp. according to RTD.

Work to be Carried Out

Steps to Prepare for test

  1. Review technical reference material and manufacture’s manual.

  2. Open DPU Cabinet.

  3. In case of QAV, use thermocouple in mV according to Temp. table (thermocouple type J)

  4. In case of QRT, use RTD in mV source to Temp. according to Temp. table (RTD)

  5. Using cabinet termination identify the location for the desired Card.

 

Start Test

  1. Using cabinet Termination list for each card Identify the input wires for the first channel

  2. Using the screw driver remove this two wires and isolate with tape.

  3. Connect the tester leads to the first channel .

  4. In case of QAV inject mV to the channel according to 20 C, 50 C, 75 C, 100 C, 150 C, 200 C, 250 C, and 300 C

  5. In case of QRT inject mV to channel according to the 20 C  , 100 C , 200 C , 300 C , 400 C , 500 C and 600 C

  6. Compare the mV injected using radio with the mV reading in the central control room (CCR) .

  7. If the reading is Ok or within the range (- + 0.01mV), repeat steps for the another channels .

  8. If any reading for any channel in CCR not correct replace the Card.

 

Return Card to Service

  1. Remove the jumper

  2. Reconnect the input wires to terminal strip.

  3. Close the cabinet.

  4. Place a copy of test result in the field maintenance file.
