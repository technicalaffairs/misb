Components

The safety system consist of two channels reading and three identical processor give three outputs, two outputs enough to trip action (two output of three system)
This system consists of :
One card power supply type APER with output 24 vdc & 125 vdc

  • Three cards type SFSR

  • Three static switches cards type ISRS

  • 12 soleniod VLV, 125 vdc

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipment as safety helmet, safety shoes ... etc

  • Unit in stand still

Tools and Equipment

  • Digital multimeter, screw driver

  • Wire connection

  • Contact cleaner

  • Semi flexible wire with 10 cm length to clean the by connections

    •

Work to be Carried Out
 There are two work conditions
Unit in stand still (S/D)

  1. Clean all cards

  2. Tightening

  3. Check the green lamp LED : BF

  4. If on : power supply is ok

  5. If not change the power supply card

  6. Check the two green lamp LED :QA (1.2) for eah card of SFSR

  7. If on : card is o.k

  8. If not : change the defective card

  9. Check the yellow lamp LED : E & the green lamp LED U for each ard type ISRS

  10. If o.k card is o.k if not : change the card

  11. Clean & tighten well the solenoid valves

  12. Check the solenoid valve by using digital multimeter check the DC reading resistance should be (550 Ω) ± 2%

Unit in Operating Mode
Lub Oil Sequence Tests

  1. Activate the auto test at operating mode from DCS to simulate the lub. Oil leakage

  2. Select for running lube oil main pump

  3. If test is done, test is o.k

  4. If not, change the pressure switch CBPS 10

  5. Select emergency oil pump

  6. If test is done test is ok

  7. If not change the pressure switch CBPS 12

  8. Select standby pump

  9. If test is done test is ok

  10. If not change the pressure switch CBPS 09

  11. Sequence test is end
