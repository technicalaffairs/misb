Introduction

This document outlines the equipment inspection I1, to be performed on an annually bases as indicated by Y1 in the title block.

The function of the VXI is to get and print the data of any unit operation within a certain time to serve fault analysis for example.

The VAX highway interface (VXI) Consists of:

MSP card,  controller card, part # (772B388G59) .

MUA card, interface card, part # (7381A06G01).

MHC card, data highway controller card, part # (7381A93G01)

MMI power supply,input:220VAC, output :+12VDC ,+ 5VDC), part # (4D33613G02).

2 Cooling fan, 230VAC, 14 watt, part # (2450D25G03) .

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

    •

  • Personal pratective equipment must be used ground strap ...etc.

  • The online maintenance work on the VXI cabinet must not affect the operation status of the plant.

  • Make backup for software before work if it is required.

  • Disconnect the cabinet from data highway if required according to station status without any effect on operation of plant.

Tools and Equipment

  • Suitable tool set, screw driver, nut driver, and air blower (Vacuum Cleaner)

  • Spray video cleaner

  • Ground strap, Card extractor

  • Digital voltmeter DVM model Fluk 87

    •

 

Work to be Carried Out

  1. Check the operation of network card function [data highway controller card (MHC)] using system highway errors diagram.

  2. Clean all cabinet card from dust by blower.

  3. Check the cabinet status by using (system status display) diagram.

  4. Check the power supply status by LEDS on front of power supply:

  5. If three LEDS for output voltage (-12,+12,+5VDC)à ON that is  means the power supply is ok .

  6. If any LED of three output voltage LEDSà  OFF that is means the power supply failed and must be replaced

  7. Check the operation of cabinet cooling fan for noise and temperature.
