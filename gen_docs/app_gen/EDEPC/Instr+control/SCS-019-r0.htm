Safety Precautions

  • A work permit must be issued the maintenance staff should be qualified and carry the suitable safety category

  • Wear the appropriate safety equipment as safety shoes, safety helmet

  • Safety fence with caution marks surrounding the work area is required.

  • Unit at shut down mode

  • Switch off all CB's on the (MCC)

Tools and Equipment

  • Screw driver, Jumpers

Work to be Carried Out

  1. Display "data" list 16, logic forcing and enter the signals L70L, L70R, L83JD2, L83PRES1, L83PRES2, L83TNROP and TNR

  2. Force logic signal L83PRES2 to be logic "1" check that "TNR"displayed corresponds to the value of start up preset constant TNKR7 given in the control spec. settings, adjust PROM constant TNKR7 if necessary.

  3. Force logic signal L83PRES2 back to "0" and L83PRES1 to be logic "1" check that "TNR" agrees with the value of load preset constant TNKR2 given the control spec. setting, if necessary adjust PROM constant TNKR2

  4. Unforce all logic signals

  5. Force logic signal L83JD2 and L83TNROP to be logic "1" make setpoint lower signal L70L a logic "1" and check that "TNR" goes to the minimum operating value TNKR4 as given in control spec. setting make setpoint lower signal a logic "0" and make setpoint raise signal L70R a logic "1" check that "TNR" goes to the maximum operating setpoint TNKR3 as given in the control as specified in the settings DWG, adjust PROM constants if necessary. Unforce all logic signals when check out is completed.

  6. Repeat para. 5 above for all rates given in spec - setting according to the attached table
