Components

The PLC system consist of:
Programmable logic controller C 100 consists of the following units:

  • One control unit type (5120 - 8)

  • One power supply type (5 F00 - 220 A) 24 VDC, Vmin = 20 V, Vmax = 28 V

  • One coupler type (5800)

  • One RS232 interface (58X1)

  • Two modules for analog inputs type (5212)

  • One module for logic inputs type (5211)

  • Two module for logic outputs type (5302)

  • One module for analog outputs 1/5v type (5315.V)

  • One module for analog outputs 4/20 mA type (5315.I)

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipment as safety helmet, safety shoes ... etc

  • Unit in stand still

Tools and Equipment

  • Multimeter, screw driver

  • Wire connection

  • Contact cleaner

  • Console of PLC : CEGELEC processing unit type C 100/5120

    •

Work to be Carried Out

  1. Unit in stand still

  2. Cleaning all cards

  3. Tightening

  4. Check the power supply o.k green lamp LED located in the power supply module (5f00)

  5. If green lamp LED is on the power supply module ok

  6. If not change the P.S. module

  7. Using the CEGELEC console

  8. Push the two (ON) push bottom simultaneously (of CEGELEC console)

  9. Choose the (debug) by enter F5

  10. Choose the data sequence by the cursor

  11. Choose the test sequence # 2(from manuel)

  12. Observe the data of sequence # 2

  13. If the numerical number associated to the data equal 5000 or 2000 : the module which under test from the I/O modules is failure change the defective module

  14. If reading is less than 2000 for all I/O modules the modules are o.k

  15. Close the work order
