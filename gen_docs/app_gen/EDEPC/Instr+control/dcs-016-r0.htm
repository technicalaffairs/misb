Components

The historical data reporter (HDR20) consists of:

MSP card, processor for printer, part #  (772B388G52).

MSL card, controller card, part # (7381A08G01).

MSE card, memory card, part # (7381A83G02).

MHC card, data high way controller card,  part # (7381A93G01).

MMI power supply, input: 220VAC, output: ±12VDC ,+5VDC , part # (4D33613G02).

Disk controller card, part #  (405A689G04).

Magnetic tape controller card , part #  (772B387G02).

6 cooling fan, 230 VAC, 14 WATT, part # (2450D25G03).

80MB hard disk, MAXFORT XT-1085, PART # (405A892H01).

This system can produce a detailed report of operation circumstances through a certain period of time.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required.

  • Wear appropriate personal protective safety equipment as safety shoes and helmet

    • ... etc

  • Wear ground strap.

  • Make backup for software before work.

  • Disconnect the cabinet from data highway if required according to station status without any effect on operation of plant.

 

Tools and Equipment

  • Screw driver , nut driver , blower.

  • Spray video cleaner.

  • Ground strap.

  • Card extractor.

  • Digital voltmeter (DVM) .

Work to be Carried Out

  1. Check the back up software for the cabinet.

  2. Check the cabinet status by using (system status display) diagram.

  3. Check the operation of net work card function (data highway controller card (MHC card) by using (system highway errors) diagram.

  4. Clean all cabinet cards from dust by blower.

  5. Check the operation of cabinet cooling fan.

  6. Check the MMI power supply of cabinet (± 12V DC , 5V DC) .

  7. Check the trend function by making any trend on the operator station.

  8. Check the free space on hard  disk by using (HDR disk control) diagram on the operator station.
