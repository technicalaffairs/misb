Introduction

This document outlines the routine testing procedures P1, which must performed on an annually bases as indicated by Y1 in the title block.

The DCS QBO Digital output card have 16 channel output to indicate alarm lamp on operator disk on MIMIC

Work to be Carried Out

Steps to Prepare for Test

  1. Review technical reference material and manufacture’s manual.

  2. Open DPU cabinet.

  3. Using cabinet termination identify the location for the desired QBO card.

 

 

Start Test

  1. Using cabinet termination list for each card identify the three I/P wires for the 1st channel.

  2. Using the screw driver to remove these two wires and isolate with tape.

  3. Connect the voltmeter leads to the 1st channel.

  4. Using radio from CCR make forced 1=on for this point according to the termination list.

  5. Watch the voltmeter reading to see the contact closed or opened.

  6. If the reading is OK, repeat steps 1 to 5 for the another 15 channels.

  7. If any reading for any channel in CCR not correct replace the card.

Return Card to Service

  1. Remove the voltmeter leads.

  2. Reconnect the O/P wires to terminal strip.

  3. Close the cabinet.

  4. Place a copy of test result in the field maintenance file.
