Introduction

This document outlines the equipment inspection I1, to be performed on an annually bases as indicated by Y1 in the title block.

The function of this engineer station is to review and update all the software files of DCS system

The engineer station consists of: 

MSP card, processor for CRT, Part # ( 772B388G57) .

MSP card, processor for printer, Part # (772B388G29)

MSP card,  processor for keyboard , Part # (772B388G56)

MDM card,  memory card, Part # (7381A81G01)

MHC card,  data highway controller card, Part # (7381A93G01)

Disk controller card,  Part # (405A689G03)

MMI power supply, input :220VAC,output:+12VDC,+ 5VDC, part # (4D33613G02)

Disk drive assembly W/floppy drive (80MB), part # (3D04153G08)

2 Cooling fan, 230VAC, 14 Watt, part # (2450D25G03)

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category (using ground strap)

    •

  • The online maintenance work on the engineer station cabinet must not affect  the operation status of the plant.

  • Make a backup of software before work if it is required.

  • Disconnect the cabinet from data highway if required according to station status without any effect on operation of plant.

Tools and Equipment

  • Suitable tool set, screw driver, nut driver, and air blower (vacuum cleaner)

  • Spray video cleaner.

  • Ground strap, Card extractor.

  • Digital voltmeter (DVM): Fluke 87.

Work to be Carried Out

  1. Check the back up software for the cabinet.

  2. Check the cabinet status by using (system status display) diagram.

  3. Check the operation of network card function (data highway controller card (MHC) by using (system highway errors diagram).

  4. Clean all cabinet cards from dust by blower.

  5. Check the operation of cabinet cooling fan for (noise – temp.).

  6. Check the power supply status by LED’S in front of power supply:
    - If three LEDS for output voltage (-12,+12,+5VDC)    à  ON that is means the power supply is OK

    - If any LED of three output voltage LEDS à   OFF that is means the power supply is failure
       and must be replaced 

  7. Check the operation of hard disk by using key switch on config mode.

  8. Check the performance of operator keyboard.

  9. Check the performance of engineer keyboard.

  10. Clean the printer.

  11. Check the operation of printer.

  12. Check the configuration program for printer.

  13. Check the two cooling fans operation, and check the sound & temperature.
