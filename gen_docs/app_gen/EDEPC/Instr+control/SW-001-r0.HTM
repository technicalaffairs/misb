Trend is a client application that allows you to query tags from multiple Industrial SQL Servers and plot them on a graphical display. Once you have added tags to a trend, you can manipulate the data in a trend graph in a variety of ways, including panning, zooming, and scaling. Trend graphs and their associated data can be printed so that the data can by analyzed away from the client machine. You can customize any trend by configuring display options, and set general options for use with all trends.

Work to be carried out

  1. Start the program trend

  2. The program windows appear without curve

  3. Select the server ( 2INSQL OR 1 INSQL )

  4. Select the type of signals ( Analog OR binary )

  5. You can make filter for any signals by using name or Address

  6. You can select the range of time ( 4/8/03 5:36:59 to 4/8/03 5:40:36 )

  7. The signal selected RC10 T201
