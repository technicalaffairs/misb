Introduction

This document outlines the testing procedure I1, which must be performed every 6 months as indicated by M6 in the above title block.

The first draft initiated by eng. Hesham El Sayed Mohamed El Assy, tested by eng. Mohamed Gad El Rab Hawad at 22 March 2003, reviewed by EDEPC quality control eng. Abdel Momem El-Emam, accepted by general director eng. Farouk Awad Aly, and approved by eng. Mostafa Alam Head Sector of Talkha power plant.

This  Program Display the Alarms at any time (Display event recorder ).and prepared by Mr. Liška from Zat Company.  This Auxiliary program used with Windows NT to Selection of file alg (Event Recorder).

Program SELEKT1 is determined for searching a record in file of alg type, which is created by product InTouch. Searching is performed in accordance with set parameters.

Program works independently without system InTouch and enables to search for files alg through network. For proper program function there is needed itself program selekt1.exe, section selekt in file win.ini, recommended configuration of file alg in InTouch. The program operates in environment Windows NT 4.0 and Windows 95.
