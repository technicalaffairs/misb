Introduction

Module is intended for the control of speed of the turbo set within its complete working range.
The speed control module Sc 2.3 (A12) function to control the speed of turbo set within its complete working range:
   - Turning speed equal 70 RPM,
   - Heating speed equal 1200 RPM,
   - Phasing speed equal 3000 RPM.
This document outlines the testing procedure P1, which performed every six months as indicated by M6 in the title block.

The first draft initiated by eng. Ashraf Farid, tested by  eng. Ashraf Farid at 20 May 2003, revised by quality control EDEPCo. eng. Abdl Menem Al-Emam, accepted by general director: eng. Aly Abd El-Whab, and approved by eng. Awad Mohamed Fathy; Head Sector of Talkha Power Station

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should be carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area, is required

  • Wear your safety helmet, safety shoes …. etc

  • Unit out of service.

Tools and Equipment

  • Digital Multimeter type Fluke 85

  • Voltage Injection set type Siemens Calibrator O5 – 7NH2011

  • Screw driver

Work to be Carried Out

  1. Connect jumper between point X20:90 and point X20:110 in rear control cabinet side no. JC02 (Imitation condition : limiter > limit)

  2. Disconnect point X20:108 in rear control cabinet side NO: JC02 (Imitation condition: tripe oil ok)

  3. Disconnect socket A31+54 type A2121B1 in front control cabinet side no. JC03 (Imitation condition: Digital speed)

  4. Inject voltage equal desired speed according to the attached curve on socket A31+54 type A2121B1(Point 8 and 7) in front control cabinet side no.  JC03 check the turbine speed ok at digital speed indicator in control room. (see table 1, and chart)

  5. Measure voltage on points A12+40:3 type A2110B1, A12+52:7 type A2703b in front control cabinet side no. JC01 to see that module cards tracing this inject voltage.

  6. If any measuring point not tracing (according the circuit diagram) then change the bad Card.

  7. Set push bottom (Hand / Automatic) on desired speed to automatic on control panel. figure 1

  8. Continue with increase injection volt slowly with observation departure and correct value must be zero when value actually of speed equal desired of speed the lamp of push Bottom H/A on desired speed will be light and system change to hand mode automatically.

Return Module in Service

  1. Remove Jumper between point X20:90 and point X20:110.

  2. Connect Signal (trip oil not ok) point X20:108.

  3. Remove injection Voltage set and connect socket A31+54 type A2121B1.
