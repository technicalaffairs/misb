Introduction

This document outlines the testing procedure P1, which performed every six months as indicated by M6 in the title block

Cards of Diamo (K), which is responsible for protection auxiliaries of unit (air fan, gas recirculation fan, condensate pump, feed water pump…..etc.). All of those cards have a special programming unit, which were putting in position (A), while the tested card putting in position (B)  see figure (1).

The first draft initiated by eng. Ramadan Helmy, tested at 04 May 2003, revised by quality control eng. Abdl Menem Al Emam at EDEPCo., accepted by: eng. Aly Abd Elwhab, and approved by eng. Awad Mohamed Fathy - Head Sector of talkha power station

Tools and Equipment

Portable automatic card tester type (ATU–PI ), made by Zat company.

The portable automatic DIAMO unit tester ATU – P1 used for testing and verifying the logic units DIAMO.

It comprises an assembly of input and output display unit converters and a Control board fitted with µp Z – 80 the memory part of the control board is Located on the front panel of the instrument and it is designed as a plug – in Interchangeable module.

The testing program for the respective unit type is stored inside this module.

The tested unit is plugged into four URS connectors fixed on the front panel of the tester.

The control is placed on the front panel and it consists of a press-button set and Of a seven-segment display.

Work to be Carried Out

  1. Put the reference programming  cards (k110 OR k810 OR  k530 OR  k540) in position (A) tester.

  2. Put the tested card in position (B) tester which we need to test.

  3. Connect the power supply to the tester and select auto position.

  4. Switch (ON) tester  

  5. The display list indicate the type of program tested card.

  6. Press (start) button to start the test

  7. The tester now is ready to run program of testing all function of the card and holding at any one of trouble occur and appear in digital monitor.

  8. If the display arrives to step number (54) then we will take procedures as follows:  

    1. Press (START) tester with press (ON) push bottom of tested card.

    2. Press push bottom (OFF) in tested card with press (START) of tester.

    3. Press both switch (ON) + (OFF) on tested card at the same time with press (START) in tester, wait up to the mark (END) appear in display

  9. If any holding coming during test steps, this means trouble inside element of tested card as follows
