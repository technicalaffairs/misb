Safety Precautions

  • Work permit must be issued the maintenance staff should be qualified and carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required.

  • Wear the personal protective equipment as (safety shoes, Helmet).

  • Turbine must be shut down &

  • ensure that all MCC CB off (racked out).

Tools and Equipment

  • Digital AVO. Meter

  • Frequency meter

    • , Remover card.

  • Speed tronic calibrator set model  (151A4925)

  • Special tool to adjust the variable resistance at the cards

Work to be Carried Out

Over Speed Test

  1. Connect calibrator set   cables to the connection A, B on the speedtronic page.

  2. Pull card SFPB at 1L

  3. Press and hold push button at front of card SSZB to make the speed control off .

  4. Increase speed until speed control just comes ON using calibrator set lamb on the card.

  5. Ensure the speed = 5620 r.p.m (110 % NHP)

  6. If it not correct the card must be changed.

  7. Repeat steps from 1 to 5 for channels 2, 3

Return Cards into Service

  1. Remove all Jumpers

  2. Remove the calibrator set

  3. Put cards in normal position
