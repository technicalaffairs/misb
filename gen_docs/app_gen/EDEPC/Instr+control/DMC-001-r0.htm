Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

    •

  • Personal protective equipment must be used safety shoes, safety helmet, ground strap ...etc.

    •

  • Unit must be shut down

Tools and Equipment

  • Suitable tool set

  • Safety helmet and safety shoes, ... etc.

  • Digital voltmeter Model Fluke 187

Work to be Carried Out

  1. Check o/p of power supply spiced to cards ( + 5 , + 15 + 28 ) VDC if o/p voltage not ok, calibrate this voltage by potentiometer within limit (± 2%)

  2. If o/p voltage not ok change power supply .

  3. Check o/p of power supply to relay +28 VDC if o/p voltage not ok, calibrate this voltage by potentiometer.

  4. If o/p voltage not ok change power supply.

  5. Turn off power supply special to cards ( + 5 , + 15 , + 28 ) VDC within ± 2%

  6. Turn off power supply special to relays ( + 28 VDC ).

  7. Remove relays.

  8. Maintain and test relays in laboratory for correct position of contact and coil resistance.

  9. Clean cards base using spray.

  10. Check cables on terminal board.

  11. Return relays to normal position.

  12. Refresh information through jumper for HMPK card.

  13. If problem occur download program from main computer (VAX – VMS ) to DMC (DAS, SGC, STC, BOP, MAC).
