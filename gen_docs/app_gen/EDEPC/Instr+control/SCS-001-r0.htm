Introduction

This document outlines the testing procedure for start up Control System for the gas turbine at the locations: Talkha GT, Hurgada, Sharm Al-Shaikh, and Port Said GT Power Stations.

The calibrated cards: SSKC - SGDD,

Components : SSKC, SGDD, VCE METER, Start up indicating light.

Safety Precautions

  • Turbine shutdown.

  • Work order must be issued.

  • Wear your safety helmet, shoes .... etc

Tools and Equipment

  • Digital voltmeter, 0-150 VDC   0-500 VAC, With freq. response of 10 Hz to 20 Khz, accuracy  0.01 V

  • Calibrator set cat no. IC 4988A100AA

Work to be Carried Out 

Fire VCE

  1. Check the Fire VCE value not lower than "Warm-up" VCE. Adjust "Warm-up" pot. full CCW to prevent interaction.

  2. Connect voltmeter to " VCE " pin 24 of SSKC (H07 R13) and Acom (G01R02)

  3. Depress "Fire " (PB-5) and read the value if different calibrator used.

  4. Adjust pot. R102 to set " Fire " VCE to value as given in the control spec.setting   

 

" Warm - up " VCE

  1. Depress " WU " (PB - 3) and check the correct value

  2. Adjust pot R101 to set " Warm-up " VCE value as given in the spec, control - cont sys settings

" Accelerate " VCE

  1. Depress " ACC " (PB - 1) and hold for 3 minutes. check to see that VCE increases to accelerate limit on 60 SEC time constant. (reaches limit in 3 minutes)

  2. After VCE has reached accelerate limit adjust pot R100 to set "Accel limit" VCE as given in the control spec. settings

Notes:  If " TCC " (PB-2) is depressed the one minute time constant is reduced

                " Accelerate " VCE level will be reached in approximately 20 seconds

" Maximum " VCE

  1. Depress " MAX " (PB-6)

  2. Set VCE to 10 volts. Adjust VCE meter using test voltmeter and " MAX " pushbutton, 10 volts

  3. Adjust pot R 106 to set " MAX " VCE to value as given in the control spec. setting

Minimum " VCE "

  1. Depress " Min " (PB - 4)

  2. Adjust pot R 104 to set " MIN " VCE to the value given in the control spec. - setting

Recheck of VCE Setting

  1. Jumper " NOTLCDA-4 " (SW6R34) to acom (GOIR 02) to put "0" on pin 30 of SSKC and remove VCE clamp

  2. Jumper to drop out the " Firing " SLEH element and put "0" on pin 28 of SSKC check value is " fire " VCE

  3. Jumper to pick up the " 28 FDX " SLEH element and put "0" on pin 18 of SSKC after 2 SEC check value is " Warm-up " VCE

  4. Jumper to pick up the "2WX" SLEH element and put "0" on pin 6 of SSKC check that VCE accelerates on a one minute time constant

  5. Rotate " MAN VCE " knob through its range verify that " speed control abnormal " light energizes with a slight clockwise movement from off. Return knob to off.

  6. Remove jumper to drop out the firing SLEH element and check that VCE goes to zero.

  7. Remove all jumpers and test equipment.
