Introduction

QCI and QSE are Digital Input Cards, each of 16 Channel, from switches. QCI indicates temp., pressure, diff. switches and QSE indicates the events time of switches working

This document outlines the testing procedures P1 for QCI and QSE carts, which must be carried out on an annually bases as indicated by Y1 in the above title block

Safety Precautions

  • A work permit must be issued and the qualified maintenance stuff should be carry the suitable safety category.

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • Wear the safety helmet and shoes.

  • Unit must be shutdown.

Tools and Equipment

  • Safety helmet, shoes,

  • Hand held Radio; wireless communications means.

  • Screw Driver, Nut Driver, and test leads.

  • Isolating tape.

Work to be Carried Out

Steps to Prepare for test

  1. Review technical reference material and manufacture’s manual.

  2. Open DPU Cabinet.

  3. Using cabinet Termination Identify the location for the desired Card.

 

 

Start Test

  1. Using cabinet Termination list for each card Identify the input wires for the first channel

  2. Using the screw driver remove this two wires and isolate by tape.

  3. Make Jumper to test the channel.

  4. Using radio to see if the contact closed in CCR ''Central Control Room''

  5. If the reading is Ok, repeat steps 1 to 5 for the another 15 channels.

  6. If any reading for any channel in CCR not correct replace the Card.

 

Return Card to Service

  1. Remove the jumper

  2. Reconnect the input wires to terminal strip.

  3. Close the cabinet.

  4. Place a copy of test result in the field maintenance file.
