Tools and Equipment

  • Safety fence caution marks

  • Digita ammeter

Work to be carried out 

  1. Make clean to control cubicle by vacuum cleaner

  2. Retighten terminals in control cubicles and connection box

  3. Power supply:-

  • Check the input/output voltage to

  • DC/DC converter (TC1)
    I/P 220 VDC (            ) & O/P 24 VDC

  • DC/AC inverter (TC1)
    I/P 220 VDC (            ) & O/P voltage 24 VAC (          )

Alarm system

  1. Switch off MCE (FA) for alarm system

  2. Racked out all electronic cards and clean it by vacuum cleaner and brush

  3. Heck jumpers on each card

  4. Racked in all cards

  5. Switch on MCB (FA) and make function test for alarm system

  6. Check the indication lamps and change the burned lamps

  7. Check the electrical coupling between two sumps by key switch position to(sump1+ sump2)

Level switches in the sump

  1. Check the level switches and clean it by rags and water

  2. Check the function for each level switch by increase the water level in the sump

  3. Observe the signal level in the control room in the event and alarm list

  4. The water levels in the sump are-

  • Water level too high (     ) alarm

  • Water level high (     ) alarm

  • Drain pump 2 on (     ) alarm

  • Drain pump1 on (      ) event

  • Main pump on (      ) event

  • Drain pump 1+2 off (      ) event

  • Drain pump 1+2 blocked (      ) event

Alarm detectors
Leakage in the stator housing

  1. Check MA in the device circuit and chec the value of resistance

  2. The normal value of MA is (7 MA) and resistance is (1.5 kohm)

  3. The alarm values are (30MA) & (330 ohm)

Bearing temperatures detector

  1. The normal resistance at temperature 20 c is (108 ohm)

  2. Record results in the maintenance file
