Introduction

This document outlines the equipment inspection I1, which must be performed every six months as indicated by M6 in the title block.

This test to inspect the eddy current transducer (five types) to measure shaft eccentricity,

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Safety and healthy equipment must be used (safety helmet, safety shoes, ...etc).

  • Unit should be out of service.

    •

Tools and Equipment

  • Digital Multimeter type Fluke 85

  • Screw driver

Work to be Carried Out

  1. Measure resistance between coaxial cable and shield (fig .1) the  measuring value should be equal to (0.8/1.2  Ώ ) and  also the measuring value between  surface / coaxial cable, surface / shield  should be tends to ∞ Ω.

  2. If the measuring value in step 1 is not the value (0.8/1.2  Ω, ∞ Ω  ) then the transducer is defected and should be changed.
