Introduction

This document outlines the testing procedures P1, which must be performed on an annually bases as indicated by Y1 in the title block.

COMPONENTS

Flame monitoring system function is to monitor the flame in combustion chambers of gas turbine version V94.2. There are two monitors on each combustion chamber.

Flame monitoring system consists of :

  • Scanning head type IRX2 (Detector)

  • Flame monitoring module type IRIS 3001

  • BFI scanning cable (BFI P/N:KW3).

  • Heat insulators, P/N: B512

  • Combustion chamber lenses.

Safety Precautions

  • A work permit must be issued, the qualified maintenance staff should carry the suitable safety ID category

  • Safety fence with caution marks surrounding the work area is required

  • Personal safety and healthy equipment must be used as safety shoes, and safety Helmet.

  • Unit to be in stand still or turning gear condition.

 

 

 

Tools and Equipments

  • Testing millimeter.

  • Contact cleaner.

  • Screw drivers .

  • Light source

Work to be Carried Out 

  1. Clean lenses on combustion chambers and the IRX2 scanning head lens.  

  2. Check sockets and plugs of  IRX2 scanning head.

  3. Check cables and ensure that the screen shield is connected..

  4. Switch power ON and check that early warning indication “VS” is ON. If “VS” is not ON, replace power supply.

  5. Apply light (ultra violet) in front of IRX2 lens and check that ”VS” is OFF and “RM” &” RF” are ON. If “RM” or “RF” is not ON, replace the scanning head.

  6. If both are ON, and “VS” is off during light check; System is OK.

  7. Close the work order
