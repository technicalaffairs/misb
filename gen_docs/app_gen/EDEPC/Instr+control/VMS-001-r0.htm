Introduction

This document outlines the equipment inspection I1, to be performed on an annually bases as indicated by Y1 in the above title block.

VMS consists of two parts:

  • Vibration and expansion monitoring system (4500 R)

  • Turbine Supervisory instrumentation

VMS monitoring system 4500R consists of  two parts:

  • Electronic cabinet (I/O module, power supply, NIC module, terminal block, local indication of expansion ).

  • CSI software server (on line configuration, MHM analysis, point analysis, RBM ware) this server is used for all units and each unit has local (4500 R) and DCS graphics and monitoring information for all reading .

Turbine supervisory instrumentation consists of:

      a)  Vibration sensors               b) Expansion sensors

Vibration sensors is two types (absolute vibrations, relative shaft vibration)

Expansion sensors is two types (casing expansion, relative expansion)

Absolute vibration on bearing no. (1, 2, 3, 4, 5, 6, 7)

Relative shaft vibration on bearing no. (1, 2, 3, 4, 7)

Casing expansion on bearing no. (1, 2)

Relative expansion on bearing no. (1,2 (thrust), 3, 4)

All reading gives information about machinery health and it is used to mechanical problem analysis .

Testing of VMS: Each VMS loop contains its own sensor /driver, data base configuration and I/O electronic modules

Safety Precautions

  • A work permit must be issued and maintenance stuff should carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear your safety and healthy equipment as safety helmet, safety shoes ... etc.

  • Turbine must be out of service.

Tools and Equipment

  • Digital multi-meter.

  • Screw driver kit, manual tools.

  • Oil spray cleaner & Teflon roll..

  • Fiber cable terminal tool

Work to be Carried Out

Preparation

  1. Obtain a copy of (4500R, expansion sensor, expansion driver, vibration sensor, vibration driver) maintenance manuals

  2. Obtain a copy of schematic loop wiring diagram for all sensors and data sheets.

VMS Test

4500R

  1. Check the system power supply.

  2. Check I/O interface card and NIC card.

  3. Check data base configuration.

  4. Check the connection between 4500R and DCS.

  5. Check the CSI online Host PC component, operating system and application soft ware.

Turbine Supervisory

Vibration System

Relative Shaft Vibration

  1. Check the gap between sensor (proximity) and the shaft according to the setting

  2. Check the driver voltage output according to the gap and manual setting

  3. Check the fixing of the sensor

  4. Check the cable between sensor and driver

  5. Check the junction box terminal

  6. Check the loop between sensor and 4500R,

  7. Check the reading in DCS monitor

 

 

Absolute Vibration

  1. Check the fixing of the sensor

  2. Check the cable between sensor and junction box

  3. Check the junction box terminal

  4. Check the loop between sensor and 4500R,

  5. Check the reading in DCS monitor

Expansion System

Relative Shaft Expansion

  1. Check the gap between sensor and the shaft according to the setting.

  2. Check the driver voltage output  according to the gap and manual setting.

  3. Check the fixing of the sensor.

  4. Check the cable between sensor and driver.

  5. Check the junction box terminal.

  6. Check the loop between sensor and 4500R.

  7. Check the reading in DCS monitor.

Casing Expansion

  1. Check the fixing of sensor lever.

  2. Check the driver voltage output according to manual setting.

  3. Check the fixing of the sensor.

  4. Check the cable between sensor and driver.

  5. Check the junction box terminal.

  6. Check the loop between sensor and 4500R.

  7. Check the reading in DCS monitor.
