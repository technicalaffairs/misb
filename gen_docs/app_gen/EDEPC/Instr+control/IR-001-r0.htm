 Introduction

This document out lines the testing procedures P1, to be performed every six months as indicated by M6 in the title block

These testing procedures for the following Instrumentation Recorders:

  1. One channel recorders type:  ZEPAREX 42 t0 45, ZPA Nova Paka, and

  2. Multi channel recorders types:
    KS 3870 for 6 Channels,  Transokomp 250 for 6 or 12 Channels,  Piotmaster 100 for 6 Channels,

    Their function to record information signal 24 hours round on paper roll by modification rear side connection it will be used to receive different kinds of signals.
    For example signal from RTD or thermocouple or signal from transmitter, this connection determines the type of the recorder.

  3. Alpha- log recorder multi channel recorders (six channels). type BA 048R/09/a6/07.01 no: 50074773, Endress+Hauser

    The display push buttons function as follows:

    Button (E) function displays main menu, Button (+) function selects chapter, button ( ®) ) selects parameter and button ( Θ ) exits from the recent menu & saving changes.

    Signal measuring 0-20 mA, 4- 20 mA, 0-10 v, thermocouple and RTD measuring also.

    The recorder is provided with digital screen optional to connected by computer system by using interface unit R 5485.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Safety and healthy equipment must be used as safety shoes, safety Helmet ..etc

    •

  • The recorder must be out of service.

  • Disconnect the power supply of the recorder.

  • Ensure that the recorder was calibrated in the laboratory.

Tools and Equipment

  • Digital Multimeter type Fluke 85

  • Voltage / Current injection set type Siemens Calibrator O5-7NH2011.

  • Contact cleaner, Spray type BW40, Cotton

Work to be Carried Out

  1. Clean the recorder and oiling it:

  2. Clean the electric boards by contact cleaner.

  3. Clean the bearing & shaft and gears by BW40.

  4. Put paper roll in the same position.

  5. Connect a power  supply 220 V ac.

  6. Select the recorder to input signal (4 – 20 ) mA for transmitter.

  7. Inject mA according to the curve between points 113 and 114 for each channel.

                 0%                      25%                        50%                        75%                100%

                4 mA                  8mA                        12 mA                     16 mA             20 mA

  8. If the test result satisfies this curve so that the channel is ok, if not ok adjust from zero potentiometer & span potentiometer and check again up to no different in the curve or change the amplifier card and check again.

  9. Repeat the steps (from 4 to 6) for input signal (0 – 10) V for thermocouple.

                 0%                      25%                        50%                        75%                100%

                 0 V                     2.5V                        5 V                         7.5 V               10 V

  10. Repeat the steps ( from 4 to 6 ) for resistance type PT100.

                 0 C                   200 C                    400 C                   600 C              850 C

                100                   75.48                     247.06                  313.65            390.38  

  11. In case of multi channels, repeat the steps (from 1 to 8) for second channel & third channel &forth channel & … etc.

Return Recorder to Service

  1. Open the door of the recorder.

  2. Move the locking lever to the intermediate position and withdraw it slightly together with the chassis.

  3. After that, the chart table is released, turn it slightly forwards and remove it by withdrawing upwards.

  4. Remove the locking cardboard insert the motor and the printer.
