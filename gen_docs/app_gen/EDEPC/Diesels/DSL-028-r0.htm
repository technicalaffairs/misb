This document lists the inspection checks, which must be performed every year as indicated by in the titel block

Safety Precautions

  • Work permit must be issued,

  • Diesel generator isolation

  • Shut down fuel valve

  • Switch off d.c supply

  • Switch off output diesel circuit breaker

  • Switch off input distribution board 380 v

  • Wear your safety helmet, gloves, goggles, shoes

Tools and Equipment

  • Test equipment level "dip sticks"

  •  Multimeter, and common sets keys

  • cleaning material "rags ,cloth "

  •  screw driver set

Work to be carried out 

  1. Check any oil leak

  2. Check operation of oil heater

  3. Check engine oil level

  4. Check hydrolic governor oil level

  5. Check any leaks in radiator

  6. Check for radiator air restriction

  7. Check hose and connection

  8. Check coolant level

  9. Check fan belt condition

  10. Check fan hub drive pulley and water pump

  11. Check motor operated louvers

  12. Check intake air system tightness

  13. Check air cleaner restriction piping and connection

  14. Check any fuel leaks

  15. Check fuel level

  16. Check fuel transfer pump

  17. Check exhaust system against leakage

  18. Check exhaust system for exhaust restriction

  19. Drain condensate trap

  20. Check battery charging system input and output voltage

  21. Check battery electrolyte level and specific gravity

  22. Check engine for unusual vibration also noise and heating

  23. Clean the main generator air inlet and outlet for restriction

  24. Clean the generator and check the switch gear start in automatic

  25. Clean the diesel engines using blower

  26. Remove any remaining oil and grease using solvent

  27. Perform operational loud test for the engine

  28. Check service tools availability
