Introduction

This document outlines the equipment inspection I1, which must be performed on weekly bases as indicated by W1 in the title block.

The CPX 928 is a single slot CPU for the C80-75 MFC that is programmed and configured by Windows based programming software to perform real time control of machines, processes, and material handling systems. It communicates with I/O and smart option modules over a rack mounted backplane using the VME C.1 Standard format.

Features

  • Single slot CPU with three serial ports.

  • Provides 6 Mbyte of battery–backed memory in the same slot (including 2 MB user memory).

  • Contains 256K of non–volatile user flash memory.

  • Supports BMA in release 7.92 and later.

  • Supports floating point calculations.

  • 12K inputs and outputs (any mix), and up to 8K analog I/O.

  • 0.4 microseconds per boolean function.

  • 96 MHz, 80486DX4 microprocessor.

  • Supports IC697 I/O.

  • Programmed by Windows based software products running on Windows NT or XP over Ethernet TCP/IP  or through an SNP port.

  • Configurable data and program memory.

  • Battery–backed calendar clock.

  • Three position operation mode switch.

  • Password controlled access.

  • Remote programmer key switch memory protection.

  • Seven status Leds.

  • Software configuration (No DIP switches or jumpers).

  • Reference information inside front door.

  • In–system up gradable firmware.

Operation

A process interface consisting of:

  • A circuit for distribution and monitoring of the process voltage

  • 16 testable logic inputs based on opto-isolators

  • An interface with the 2000 I/O backplane bus consisting of:

  • A circuit for plugging in with power on

  • An 2000 I/O bus slave interface 

  • Input-output registers.

Safety Precautions

  • A work permit must be issued

  • Maintenance staff should be well trained and qualified.

  • Caution marks must be hanged on the work place, safety rope around the work place.

  • Removing a Module

  • First, make sure rack power is turned off. Then, use a medium-blade, slotted screwdriver to loosen the top and bottom holding screws just enough to free the module from the rack. Note: Screws are held captive in the module, so they do not have to be removed from the module.

  • Grasp the board firmly at the top and bottom of the board cover with your thumbs on the front of the cover and your fingers on the plastic rack clips on the back of the cover.

  • Squeeze the rack clips on the back of the cover with your fingers to disengage the clips from the rack rail, and then pull the board firmly toward you until it clears the rack.
