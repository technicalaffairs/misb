Introduction                                  

This document outlines the checklist I1 for all types of pressure switches, which must be performed every year as indicated  by Y1 in the above title block.

Safety Precautions

  • A work permit must be issued, the maintenance staff should be qualified and carry the suitable safety category.

  • Safety fence with caution marks surrrounding the work area is required.

  • Wear safety helmet, shoes.

  • Close manual valve for indicator line

Tools and Equipments

  • Pressure tester (druck - dpi 610)

  • Ohmmeter or multimeter (Fluke - 180 series digital multimeter)

  • Safety driver

  • Set of key

Work to be carried out

Steps to prepare for test

  1. Review technical reference material and manufacture’s manual.

  2. Obtain copy of switch setting values record it in its MPRDS.

  3. Connect pressure calibrator to pressure switch

  4. Adjust open and close value

Calibrating procedure

  1. Inject the suitable pressure value reference to setting and record at MRDS, using ohmmeter, check contact if change in its situation.

  2. If there is any error, using screw driver adjust the switch and repeat step 1.

Return pressure switch to Service

  1. Return the switch  to service by removing the pressure tester from the switch vent.

  2. Close indicator vent.

  3. Open manual valve for switch line.

  4. Place a copy of test result in the field maintenance file.
