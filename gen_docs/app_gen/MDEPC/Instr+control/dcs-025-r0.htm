Introduction

This document outlines the equipment inspection I1, which must be performed on weekly bases as indicated by W1 in the title block.

The AS111 module is an analog output module providing 8 outputs (4-20 mA current) referenced with respect to the process voltage 0 V common.

The first draft is initiated by eng. Amal El-Shamy, tested by eng. Bahaa Abd El-Wahab, revised by MDEPC quality control eng. Adel El Naggar, accepted by general manager eng. Mohamed Elshreif, approved by eng. Ahmed Sawan head sector of Nubaria power station.

Features

Dimension

  • Height: 233 mm

  • Depth: 220 mm

  • Width: 1 slot = 25.4 mm

  • Weight: 0.380 kg

Operation

  • A process interface consisting of:

  • A circuit for distribution and monitoring of the process voltage

  • 16 testable logic inputs based on opto-isolators

  • An interface with the 2000 I/O backplane bus consisting of:

  • . A circuit for plugging in with power on

  • .An 2000 I/O bus slave interface 

  • Input-output registers.

 

Controls and indications on the front panel

  • A green LED on the front panel indicates if the module is working.

  • It remains on if the module is operating correctly and goes out in case of faulty operation ( fault detected in the hardware or configuration use of the module)

Input elements

  • The module has 16 logic input elements for acquisition of 16 free contacts.

  • Each element is based on an opto-isolator which provides galvanic isolation between the process and the internal voltages.

  • The module monitors the possibility of the inputs to be switched to 1 or 0. 

Process voltage

  • The process voltage is supplied by the backplane connector (distributed voltage), or by the process connector (external voltage).

  • A logic input based on an opto-isolator indicates the presence of the process voltage.

Protection

  • General protection on the process voltage +Vp by CTP with maximum tripping current at 20°C: 0.75 A.

Safety Precautions

  • A work permit must be issued

  • Maintenance staff should be well trained and qualified.

  • Caution marks must be hanged on the work place, safety rope around the work place.

Tools and Equipment

  • Complete tool set.  

Work to be carried out

  1. Check the supply voltage

  2. Check the interface leds
