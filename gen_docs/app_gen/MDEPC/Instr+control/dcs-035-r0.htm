Introduction

This document outlines the equipment inspection I1, which must be performed on weekly bases as indicated by W1 in the title block.

IR139 is a communication module for theCE2000. This module provides exchanges by MODBUS protocol. In this protocol, CE2000, Associated with C370 through IR139 board,   Can only be “master”

Each module channel can be connected to one”slave” subscriber (point to point) or to n Slaves (bus connection)

 For IR139-1 version, the basic module has 4 RS232/RS485 channels.

Dimension:

Height: 233 mm Width: 1 slot = 25.4 mm
Depth: 220 mm    

Connections :

The serial connections are located on the front panel.

They are not isolated, so IR139 module cannot be connected to devices without a floating 0V (the module and/or the device could be damaged).

An isolating module is recommended for outside connections (metal casing).

For RS232, the line length is limited to 15 m.

For RS485 with isolating module, this length is limited to 1 km.

 

Controls and Indications on The Front Panel

A green LED remains on for normal working. It goes out in the following cases:

  1. The module access through UT129 is not detected (UT129 missing or module hardware fault detected),

  2. The module detects an internal hardware fault,

  3. The module detects a user fault on a transmission line with one or several subscribers; in this case, normal working is resumed only when a correct exchange with the faulty subscriber(s) is detected

Protection

General protection on the process voltage +Vp by CTP with maximum tripping current at 20°C: 0.45 A.

  1. For each current input with power supplied by the sensor, protection by CTP with maximum tripping current at 20°C: 0.225 A.

  • Caution marks must be hanged on the work place, safety fence around the work place.
