  • Personnel protective equipment must be used as shoes and safety helmet

Preliminary Steps to Prepare for Relay testing

  1. Review technical reference materiel manufactures /  materiel

  2. Obtain copy  of relay setting from felid  maintenance file.

  3. Obtain outage for unit protected by this relay

  4. Inspect the secondary circuits of CT/VT miniature circuit breakers

  5. (MCB) and all cabling supplying the relay .

  6. Block any disturbing alarm or fault Recorders associated with this protection.

Preparation of Test

  1. Shorted terminal of CT in Socket (X101)(X201) in banal (CHB01)

  2. Connect the POWER Supply to the device (Secondary) Tester (OMICRON)

  3. Connect the secondary  injected tester (OMICRON) for inject the current (1.2.3) and Neuter (4) for two sides.

  4. Connect the timer leads

  5. Digs program feed the deice from laptop

  6. Event record from the device to lap top  

  7. Record the Pickup Value and drop out value and the time of Pickup Value
