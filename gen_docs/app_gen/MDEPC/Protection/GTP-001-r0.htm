Safety Precautions

  • Work order must be issued

  • Wear safety helmet

  • Ensure that the gas turbine is out of service, at cold state

Tools and Equipment

  • Digital Ammeter

  • Set of key

  • Contact cleaner

  • Filer

  • Lighting lamp

Work to be carried out

Steps to prepare

  1. Review technical reference materials and manufacture's manual

  2. Review all the wiring for all speed sensors

Testing procedure

  1. Measure the voltage of the card for speed sensor (24 VDC)

  2. Measure the gap between each speed sensor and the shaft (1.5mm)

  3. Review the terminals connection for these speed sensors in the junction box

  4. If there is a dust on the any speed sensor, then clean the contact

  5. If there is fault on any speed sensor, then change

  6. Testing the over speed protection circuit and check the setting value of this speed

Return speed protection circuit to service

  1. Check any tools can forget in the cabinet

  2. Close the cabinet of speed card
