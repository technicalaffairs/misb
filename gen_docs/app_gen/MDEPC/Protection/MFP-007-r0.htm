Introduction

Sepam series 40, with its current and voltage metering capabilities, offers high performing solutions for more demanding applications. Sepam series 40 units perform the following functions in addition to those performed by Sepam series 20:

Protection of closed ring networks or networks with parallel incomers, by directional protection directional earth fault protection suitable for all earthing systems: impudent, isolated or compensated neutral protection of networks with variable configurations requiring the switching of groups of settings and logic discrimination

All the necessary electrical measurements: phase and residual currents, phase-to neutral, phase-to-phase and residual voltage, frequency, power and energy,

Comprehensive network diagnosis assistance: 20 seconds of disturbance recording, detailed history of the last 200 alarms, storage of the last 5 tripping contexts

Adaptation of control functions by a logical equation editor

Customization of alarm messages to fit each application, and/or in the user’s language

Safety Precautions

  • A work permit must be issued.

  • The maintenance staff should be trained, qualified, and carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear your safety and healthy equipment as safety helmet, safety shoes ... etc.

  • Line isolation is required.

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection.

  • Feeder must be grounded.

Tools and Equipment

  • Safety helmet, safety shoes ... etc

  • ISA DRTS tester, Test leads.

  • PC laptop with necessary software {ISAWIZARD and ENER VISTA}

  • Tools bag.

Work to be Carried Out

Steps to prepare for relay testing

Preparation                                                          

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test.

  2. Be sure that the DC power of the relay is correct by applying a DC voltmeter across (FA1-A points (1,2) on the rear of the relay see figure no. 1.

Reading the Setting from the Relay using the PC

  1. Connect the personal computer to the relay.

  2. Run the Sepam program.

  3. Read the setting and safe it as URGENT FILE FOR SIPROTEC SEPAM M41 [name the feeder].

For 6.3KV Motor

Time Over Current (O/C)

  1. Be sure that ISA tester is connected to EARTH.

  2. Switch ON ISA tester.

  3. Connect the Personal Computer to the ISA tester.

  4. Insert test plug to XT3 see figure 1

  5. After ISA tester completely in service Connect Currents, Voltages and binary input leads between ISA tester and SEPAM M41.

  6. Insert the current terminals to XT13 points (1,2,3,4) see figure no.1

  7. Disable other functions.

  8. RUN ISAWIZARD software>>>>>GENERATION & DISTRIBUTION RELAYS>>>> Xpro advanced>>>>MANUAL MODE

  9. NOW you can test the relay.

    NOTE:  Don't forget to GENERATE PREFAULT CONDITION [I=0.001A,   V=0V].

  10. SET time over current = 0 [Ie instantaneous over current]

  11. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay.

  12. Set time over current to set value.

  13. Choose Shoot with current values [2*I,3*1,4*I and 5*I] to get these points on curve.

  14. STORE ALL TESTS

Now print the results and you have complete test results for this function.

Earth Fault (E/F)

  1. RUN ISAWIZARD software and choose GENERATION AND DISTRIBUTION RELAYS>>>>      X-PRO >>>>MANUAL MODE>>>>

  2. Insert the current terminals to 18, 19. See figure no.1

  3. Generate prefault condition [I=0.001A].

  4. Disable other functions.

  5. SET time earth fault =0 [i.e instantaneous] earth fault.

  6. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay.

  7. SET time earth fault to set value.

  8. Select fault and shoot current value equal to values [2*I,3*1,4*I and 5*I] to get these points on curve.

  9. Note that the injection should be on phase R only.

  10. STORE ALL TESTS.

  11. Stop ISAWIZARD program.

Negative Sequence 46                   

 

  1. RUN ISAWIZARD software and choose GENERATION AND DISTRIBUTION RELAYS>>>>      X-PRO >>>>MANUAL MODE>>>>

  2. Insert test plug to XT3 ... see figure 1

  3. After ISA tester completely in service connect current, and binary inputs leads between ISA tester and SEPAM M41

  4. Insert the current terminals to  XT3 points (1,2,3) ... see figure 1

  5. Generate prefault condition [I=0.001A].

  6. Disable other functions.

  7. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay.
    Note       Ia=3 x Io =3x I2    WHERE

    Ia: Phase current,   Io: Zero sequence current and I2: Negative sequence component

     

  8. Select fault and shoot current value equal to (I Setting*1.05) to determine the operating time.
    Note that the injection should be on phase R only.

  9. STORE ALL TESTS.

Return Relay and Associated Equipment to Service

  1. Enable any function which was disabled.

  2. Return relay to service by removing the test leads from the relay terminals and the test set.

  3. Make sure CT and VT secondary circuits are in normal condition.

  4. Remove the disturbance alarm LED`s on the relay which is caused by the test.

  5. Request feeder to be returned in service.

  6. Take pharos readings of all AC currents and voltages and compare them to those for normal load condition.

  7. Check that these reading agree with station meters (MW, MVAR, Amperes and power factor).

  8. Place a copy of the test result in the field maintenance file.
