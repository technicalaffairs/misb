Introduction

This document lists the testing procedures P1, which must be performed on an annually bases as indicated by Y1 in the title block

The SIPROTEC® 4 7UM62 is a numerical, multi-functional, protective and control device equipped with a powerful microprocessor. All tasks, such as the acquisition of the measured quantities, issuing of commands to circuit breakers and other primary power system equipment, are processed in a completely digital way.

Safety Precautions

  • A work permit must be issued.

  • The maintenance staff should be trained, qualified, and carry the suitable ID safety category.

  • Wear your safety and healthy equipment as safety helmet, safety shoes ... etc.

  • Safety fence with caution marks surrounding the work area is required.

  • Equipment isolation and grounding is required.

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for testing the protection relay.

Tools and Equipment

  • Safety Helmet, safety shoes ... etc

  • ISA DRTS tester, test leads.

  • PC laptop with necessary software {ISAWIZARD and ENER VISTA}

  • Tools bag.

Work to be Carried Out

Steps to prepare for relay testing

Preparation                                                          

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test.

  2. Be sure that the DC power of the relay is correct by applying a DC voltmeter across points (f1, f2 -- see figure 1) on the rear of the relay.

Reading the setting from the relay by using the PC

  1. Connecting the Personal Computer to the relay.

  2. Run the enervista program.

  3. Read the setting and save it as URGENT FILE FOR SIPROTEC 4 7UM62 [name the feeder].

FOR MAIN GENERATOR UNIT OF CTG

 

Differential protection test "stator earth fault"

  1. Connecting the Personal Computer to the ISA tester.

  2. Be sure that ISA tester is connected to EARTH.

  3. Switch ON ISA tester.

  4. After ISA tester completely in service Connect Currents, Voltages and binary input leads between ISA tester and SIPROTEC 4 7UM62. 

  5. RUN  ISAWIZARD software >>>>> GENERATION and DISTRIBUTION RELAYS >>>> X pro advanced   >>>> MANUAL MODE

  6. NOW you can test the relay.

  7. Insert the current terminals to Q1,Q3,Q5 (see figure 2).
    Note:  Do not forget to GENERATE  PREFAULT CONDITION [I=0.001A, V=0V].

  8. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay.

  9. Choose Shoot with current value equal to 1.05 of pick up current to get pickup time of the relay.

  10. Now print the results and you have complete test results for this function.

Earth Fault

  1. RUN  ISAWIZARD software and  chose GENERATION AND DISTRIBUTION RELAYS >>>> X-PRO >>>> MANUAL MODE >>>>

  2. Insert the current terminals to Q1,Q3,Q5  (see figure 2).

  3. GENERATE PREFAULT CONDITION [I = 0.001A].

  4. Disable other functions.

  5. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay.

  6. Select fault and shoot current value equal to ( I Setting*1.05 ) to determine the operating time.
    Note that: the injection should be on phase R only.

  7. STORE ALL TESTS

 Over Current

  1. RUN  ISAWIZARD software and choose GENERATION and DISTRIBUTION RELAYS >>>> X-PRO >>>>MANUAL MODE >>>>

  2. GENERATE  PREFAULT  CONDITION [I=0.001A, V=57V].

  3. Disable other functions.

  4. SELECT FAULT and INJECT CURRENT VALUE EQUAL TO (I Setting*1.01) to determine the operating time.

  5. Click Start.

  6. Repeat steps from 3-1, to 3-5 for all phases R, S, and T.
    Note that: you can inject on three phase one time.

  7. Choose STORE ALL TESTS.

Negative Sequence 46                  

  1. RUN ISAWIZARD software and choose GENERATION and DISTRIBUTION RELAYS >>>> X-PRO >>>> MANUAL MODE >>>>

  2. Insert the current terminals to Q1,Q3,Q5 (see figure 2).

  3. GENERATE PREFAULT CONDITION [ I = 0.001A ].

  4. Disable other functions.

  5. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay.
    NOTE:    Ia=3 x Io  =3 x  I2  WHERE:
    Ia: Phase current,  Io: Zero sequence current and I2: Negative sequence component.

  6. Select fault and shoot current value equal to (I Setting*1.05) to determine the operating time.
    Note that: The injection should be on phase R only.

  7. STORE ALL TESTS

Over / Under Voltage

  1. RUN  ISAWIZARD software and choose GENERATION and DISTRIBUTION RELAYS >>>> X-PRO >>>>MANUAL MODE>>>>

  2. Insert the voltage terminals to R15,R17,R18,R16 (see figure 3).

  3. GENERATE PREFAULT CONDITION [I=0.001A, V=57V].

  4. Disable other functions.

  5. SELECT FAULT and inject voltage value equal to (V setting) to determine the operating time for each stage.

  6. Choose Delta ramp with step ∆V 1V to get pickup voltage for each stage of the relay.
    Note that: You can inject on three phase one time.

  7. Chose STORE ALL TESTS.

Over / Under Frequency

  1. RUN  ISAWIZARD software and choose GENERATION and DISTRIBUTION RELAYS >>>> X-PRO >>>>MANUAL MODE>>>>

  2. Insert the voltage terminals to  R15,R17,R18,R16 (see figure 3).

  3. GENERATE PREFAULT CONDITION [I=0.001A, V=57V, 50 Hz].

  4. Disable other functions.

  5. Choose Delta ramp with step ∆F 0.5 Hz to get pickup frequency for each stage of the relay.
    Note that : You can inject on three phase one time.

  6. SELECT FAULT and inject voltage value equal to (V nominal) with frequency setting to determine the operating time for each stage.

  7. Chose STORE ALL TESTS.

 

 

Reverse Power

  1. RUN  ISAWIZARD software and chose GENERATION AND DISTRIBUTION RELAYS >>>> X-PRO >>>>MANUAL MODE>>>>

  2. GENERATE  PREFAULT  CODITION [I=0.001A, V=57V].

  3. Choose Delta ramp with step ∆I 0.001 Amp to get pickup current  for the relay.

  4. SELECT FAULT and INJECT CURRENT VALUE EQUAL to I Setting and angle 180ْ  and voltage value equal to nominal voltage determine the operating time.

  5. Click Start.

  6. Repeat steps from 3-1, to 3-5 for all phases R, S, and T.
    Note that: You can inject on three phase one time.

  7. Chose STORE ALL TESTS

Return relay and associated equipment to service

  1. Return relay to service by removing the test leads from the relay terminals and the test set.

  2. Make sure CT and VT secondary circuits are in normal condition.

  3. Remove the disturbance alarm LED's on the relay which is caused by the test.

  4. Request feeder to be returned in service.

  5. Take pharos readings of all AC currents and voltages and compare them to those for normal load condition.

  6. Check that these reading agree with station meters (MW, Mvar, Amperes and Power Factor).

  7. Place a copy of the test result in the field maintenance file.
