Introduction

The distance relays offer a comprehensive range of protection functions for application to many overhead line and underground cable circuits. the P442 model can provide single and three phase tripping.

All kinds of faults are detected including close-in three-phase, cross-country faults, evolving faults and high-resistance earth faults 

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure to be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued and

  • The qualified maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be well trained, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Safety and healthy equipment must be used (safety helmet, safety shoes ... etc).

  • Line isolation is required.

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection.

  • Transmission Line must be grounded.

Tools and Equipment

  • ISA DRTS tester

  • Test leads.

  • PC laptop with necessary software {ISA WIZARD and Micom P442}

  • Tools bag.

Work to be carried out

Steps to Prepare for Relay Testing

Note:  CT cores for BF and Protection Relay are different i.e. BF will not work during test.

Preparation

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test

  2. Be sure that the DC power of the relay is correct by applying a DC voltmeter across points (J1-) and (J2+) - see fig. 1 on the rear of the relay.

Reading the setting from the Relay using the PC

  1. Connect the personal computer to the relay.

  2. Run the Micom S1 program.

  3. Read the setting and safe it as URGENT FILE FOR P442 [name the feeder].

Distance Protection Test

  1. Connect the personal computer to the ISA tester

  2. Be sure that ISA tester is connected to EARTH.

  3. Switch ON ISA tester.

  4. After ISA tester completely in service connect currents, Voltages and binary input leads between ISA tester and test plug(X21-1 & X21-2)  see fig. 2 for Micom P442. 

  5. Make short between points (X21-1 13 & 14) and points (X21-1 15 & 16) see fig. 3

  6. RUN ISA WIZARD software and choose SPECIAL DISTANCE>>>> SPECIAL licensed >>>>ALSTOM >>>>P442 [>>>> means ENTER ORDER].

  7. From setting file. Plot the setting file in RIO file in ISA TESTER

  8. After that choose ZPRO >>>> the characteristic for Micom P442 will appear.

  9. Now you can choose two ways to test these relay.
    Note:  Don't forget to GENERATE PREFAULT CONDITION [I=0.001A, V=57V].

Now you can choose two ways to test these relay

  1. From TEST SELECTION chose automatic time zone (Z/T) and don’t forget to adjust the test sequence as follow [fault = all faults, Z min = zero, Z max = as the setting, STEP =0.5 OHM, Angle [degree] = 0, after that click OK the tester will operate and test  the R axis.

  2. Repeat step 1- BUT choose angle=90   the tester will test the X axis.

  3. Choose STORE ALL TESTS >>>>

  4. Go to RESULTS from main menu chose HEADER and apply all information about these feeder and ok

  5. Now print the results and you have complete test results for these relay.

  6. From TEST SELECTION choose Click and test and by curser click on the point which you need to test [generally choose all points on X and Y Axis]. 

  7. Repeat steps from 3 to 5.

  8. Now you have complete test results for these relay.

    9.

Other Function Test

Power Swing

  1. From main menu select TEST SELECTION select POWER SWING test and plot the setting.

  2. Click Start.

  3. Chose STORE ALL TESTS.

  4. Go to results from main menu choose header and apply all information about these feeder and ok.

  5. Now print the results and you have complete test results for these function.

Switch on to Fault ( SOTF )

  1. From main menu select test selection select sotf test and plot the setting from Rio file.

  2. Make prefault at zero current and voltage.

  3. Click ok.

  4. Inject current and voltage as fault case in any zone Z2 or Z3

  5. Click Start.

  6. Choose STORE ALL TESTS.

  7. From results will see that the time at all faults is equal to Z1 time.

Over Current (O/C)

  1. RUN ISA WIZARD software and choose GENERATION and DISTRIBUTION RELAYS>>>> X-PRO >>>>MANUAL MODE>>>>

  2. GENERATE PREFAULT CONDITION [I=0.001A, V=57V].

  3. Disable distance function.

  4. SELECT FAULT and INJECT CURRENT VALUE EQUAL to (I Setting*1.01) to determine the operating time.

  5. Click Start

  6. Repeat steps from 1 to 5 for all phases R, S, and T
    Note that  you can inject on three phase one time.

  7. Choose store all tests.

Earth Fault (E/F)

  1. RUN ISA WIZARD software and choose GENERATION AND DISTRIBUTION RELAYS>>>> X-PRO >>>>MANUAL MODE>>>>.

  2. GENERATE PREFAULT CONDITION [I=0.001A].

  3. Disable distance function.

  4. SELECT FAULT and INJECT CURRENT VALUE EQUAL to (I Setting*1.01) to determine the operating time.

  5. Click Start.

  6. Repeat steps from 1 to 5 for all phases R, S, and T.
    Note that: The injection should be on phase by phase not on three phases or on two phases.

  7. Choose store all tests

Return Relay and Associated Equipment to Service

  1. Enable any function which was disabled.

  2. Return relay to service by removing the test leads from the relay terminals and the test set.

  3. Make sure CT and VT secondary circuits are in normal condition.

  4. Remove the disturbance alarm LED's on the relay which is caused by the test.

  5. Request feeder to be returned in service.

  6. Take pharos readings of all AC currents and voltages and compare them to those for normal load condition.

  7. Check that these readings agree with station meters (MW, Mvar, Amperes and power factor).

  8. Place a copy of the test result in the field maintenance file.
