Introduction

This document lists the testing procedure P1, which must be performed on an annually bases as indicated by Y1 in the above title block

The  MBCZ16 Bus bar protection is a Modular, solid state, low impedance blast differential based on the circulating current principle, dual cratered of protection, provides continuous supervision of CT secondary circuits and constant monitoring of vital circuits extremely stable up to the max switch gear.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Issue work permit including the team work name.

  • Safety fence with caution marks surrounding the work area is required.

  • Outage arrangement is required.

  • Switches off  DC supply before inserting or with drawing any card from the relay .

  • Before with drawing MBCZ 16 modules from its case fit CT shorting to test bluge before inserted into test block.

Tools and Equipment

  • Test Module MBCZ 18  GEC .

  • Alarm Module MBCZ 17 GEC.

  • Bus wire Access MBCZ 19 GEC.

  • Connection of  test unit, Sverker 650 or Pulser

Work to be Carried Out

  1. Turn the key switch on the alarm unit to the PROTECTION OUT position.

  2. The test unit can be used to check the zone measuring units. To do this the covers must be removed from the test unit and the zone unit to be checked .

  3. The screened test lead should be plugged into the test unit using the end of the lead with the 3.5 mm jack plug. The 2mm plug on the other end of the test lead is then plugged into the test input on the phase of the zone to be tested.

  4. It is assumed for these tests that the bus bar is dead Note : Assume a matching ratio of 1.0 when testing units by this method.

  5. Set the selector switch on the front of the test unit to INJECTXI.

  6. Set the digital current switch to 00.

  7. Press the button marked INJECT and whilst holding it down increase the current unit the meter reads a current just below the setting of the supervision relay.

  8. Then increase slowly until the supervision operates to indicate a CT  fault . There is a fixed fifteen second delay built into this circuit .

  9. The minimum operating current can then be read from the meter as a direct percentage of rated current .

  10. For this test no bias will be produced and hence the setting will be somewhat less than the marked value . The effective setting will be in the order of 90% of the marked values.

  11. Set the MAN / AUTO switch to MAN.

  12. Set the CURRENT TO 80 % of the marked value of the selected setting and the relay should not indicate a trip. Increase the current to the marked setting value and the protection should operate .

  13. There should be a TRIP indication on the element being tested and a PROT. OPERATED indication on the alarm unit. The latter has a 15 second delay.

  14. Release the injection button and reset the indications. Check that when the inject button is now depressed the trip is instantaneous and that when the current I injected for five seconds no PROTN FAIL indication comes up .

  15. Repeat (14) on  the other two phases and then on each other zone in turn .

  16. Write the value of test in the test sheet.

  17. Decrease the value of injection current to become zero ( 0.0 ) .

  18. Turn the key switch ( MAN / AUTO ) on position ( AUTO ) as before .

  19. Reset all signals which occurred during the test .

  20. Insert the coupling test ( test lead ) between the unit test and measure test .

  21. Set the all cover of units which inserted .

  22. Set the switch in alarm unit ( MBCZ 17 ) on position ( PROT . IN ) .
