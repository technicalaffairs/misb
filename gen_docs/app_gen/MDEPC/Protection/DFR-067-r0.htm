Introduction

The T60 differential protection devices are designed for the fast and selective short circuit protection of  transformers , motors and generators and of other two , three or four phase winding arrangement.

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Adel Elnaggar, accepted by Eng Ali Zahran, general manager, approved by Eng Ahmed Sawwan head sector of Nubaria Power Station, and certified by Eng Mohammed Nageb Inspection and quality head sector.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Safety fence (rope) with caution marks surrounding the work area

  • Transformer isolation is required

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test deferential protection

Tools and Equipment

  • Omicron CMC 356  tester.

  • Test leads.

  • PC laptop with necessary software { Omicron CMC 356   and ENER VISTA}.

  • Tools bag.

Work to be carried out 

Steps to prepare for relay testing

Preparation                                                          

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test.

  2. Be sure that the DC power of the relay is correct by applying a D.C voltmeter across points (B5a, B6a—see page 4) on the rear of the relay.

Reading the setting from the relay by using the PC

  1. Connecting the Personal Computer to the relay.

  2. Run the enervista program.

Read the setting and safe it as URGENT FILE FOR T60 [name the feeder].

 

FOR MAIN TRANSFORMER OF CTG

Differential protection test

  1. Connecting the Personal Computer to the Omicron CMC 356   tester.

  2. Be sure that Omicron CMC 356    tester is connected to EARTH.

  3. Switch ON   Omicron CMC 356    tester.

  4. After Omicron CMC 356    tester completely in service Connect Currents leads between Omicron CMC 356    tester and test bluge for T60. 

  5. RUN  Omicron CMC 356    software>>>>>Test Universe Start Page>>>> Quick CMC>>>>Hardware Configuration >>>>>General >>>>> Details and select not used voltage and 3*32A,430VA@25A,25V rms and press OK .

  6. Press Test View .

  7. NOW you can test the relay.

  8. Insert the test plug in the terminal block BP14 or BP16(see page 4).

Note :  SET (Analog Output Table ) [I=0.1A,  f =50 HZ ].

  1. Choose Step/ ramp Auto Step with step ∆I 0.001 Amp and press F5 to get pickup current of  the relay.

  2. Press F10 to save the results to the report

  3. SET (Analog Output Table) current value equal to 1.05 of pick up current and press F5 to get Pickup time of the Relay

  4. Press F10 to save the results to the report

2nd and 5th  harmonic restraining test

  1. Repeat steps from A-1 with frequency 100Hz for 2nd harmonic test

  2. Repeat steps from A-1 with frequency 250Hz for 5th harmonic test.

    • Now print the results and you have complete test results for these relay.

EARTH FAULT (E/F)

  1. RUN  Omicron CMC 356    software>>>>>Test Universe Start Page>>>> Quick CMC>>>>Hardware Configuration >>>>>General >>>>> Details and select not used voltage and 3*32A,430VA@25A,25V rms and press OK

  2.  Insert the test plug in the terminal block BP15 (see page 4).

  3. SET (Analog Output Table) current value equal to [I=0.001A].

  4. Disable Differential function.

  5. Choose Step/ ramp Auto Step with step ∆I 0.001 Amp and press F5 to get pickup current of the relay.

  6. Press F10 to save the results to the report.

  7. SET (Analog Output Table) current value equal to 1.05 of pick up current and press F5 to get Pickup time of the Relay.

  8. Press F10 to save the results to the report.

  9. Note that: - the injection should be on phase R only.

  10. STORE ALL TESTS      

RESTRICTED EARTH FAULT (REF)

  1. Disable earth fault and differential protection functions

  2. Insert the test plug in the terminal block BP15 (see page 4)

  3. Choose Step/ ramp Auto Step with step ∆I 0.001 Amp and press F5 to get pickup current of the relay.

  4. Press F10 to save the results to the report.

  5. SET (Analog Output Table) current value equal to 1.05 of pick up current and press F5 to get Pickup time of the Relay.

  6. Press F10 to save the results to the report.

  7. Store all tests

Now you have complete test results for these relay

Return relay and associated equipment to service:-

  1. Enable any function which was disabled.

  2. Return relay to service by removing the test leads from the relay terminals and the test set.

  3. Make sure CT and V T secondary circuits are in normal condition.

  4. Remove the disturbance alarm LED`s on the relay which is caused by the test.

  5. Request feeder to be returned in service.

  6. Take phasors readings of all A C currents and voltages and compare them to those for normal load condition.

  7. Check that these reading agree with station meters (MW, Mvar, Amperes and power factor).

  8. Place a copy of the test result in the field maintenance file.
