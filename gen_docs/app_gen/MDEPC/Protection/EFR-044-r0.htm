Introduction

This document outlines the maintenance procedures P1, to be carried out on an annually base as indicated by Y1 in the title block.

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure to be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The MIF II Feeder Protection relay is a microprocessor-based relay designed for the following applications:

  • Main Protection for Medium and Low voltage feeders.

  • Main Protection for Medium size Motors.

  • Main Protection for Medium and Low power Transformers (less than 10MVA).

  • Earth Fault Protection for neutral earthed transformer

  • Differential Protection for Motors, if a CT differential connection is available.

  • Backup Protection for power Transformers (for any power level).

  • Backup Protection for Generators.

  • Backup Protection for Transmission lines.

  • Backup Protection for substation bus bars.

  • Thermal Image Protection/Supervision for Cables, Power Transformers, Grounding Resistors and Generation Elements

Safety Precautions

  • A work permit must be issued.

  • The maintenance staff should be trained, qualified, and carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear the safety and healthy equipment as safety shoes, safety helmet ... etc.

    •

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection.

  • The main and auxiliary transformers must be out of service.

Tools and Equipment

  • ISA DRTS tester.

  • Test leads.

  • PC laptop with necessary software {ISA WIZARD and ENER VISTA}

  • Tools bag

Work to be Carried Out 

Preparation

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test.

  2. Be sure that the DC power of the relay is correct by applying a DC voltmeter across points (A1,A2 see fig.1) on the rear of the relay.

Reading the Setting from the Relay using the PC

  1. Connect the personal computer to the relay.

  2. Run the enervista program.

  3. Read the setting and safe it as URGENT FILE FOR MIF II N [name the feeder].

For Main Transformer CTG

Earth Fault (E/F)

  1. RUN ISA WIZARD software and choose GENERATION AND DISTRIBUTION RELAYS >>>> XPRO >>>> MANUAL MODE>>>>

  2. Insert the test plug in the terminal block BP17 see figure 1.

  3. GENERATE PREFAULT CONDITION [I=0.001A].

  4. Disable other functions

  5. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay

  6. Select fault and shoot current value equal to (I Setting*1.05) to determine the operating time.

  7. Note that:  the injection should be on phase R only.

  8. STORE ALL TESTS

  9. Now you have complete test results for these relay.

Return Relay and Associated Equipment to Service

  1. Enable any function which was disabled.

  2. Return relay to service by removing the test leads from the relay terminals and the test set.

  3. Make sure CT and VT secondary circuits are in normal condition.

  4. Remove the disturbance alarm LED's on the relay which is caused by the test.

  5. Request feeder to be returned in service.
     

  6. Take pharos readings of all AC currents and voltages and compare them to those for normal load condition.

  7. Check that these readings agree with station meters (MW, Mvar, Amperes and power factor).

  8. Place a copy of the test result in the field maintenance file.
