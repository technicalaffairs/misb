Introduction

The distance protection device P437 is used for selective short circuit protection, ground fault protection, and overload protection in extra-high-voltage systems. The system can be operated as impedance-grounded or isolated-neutral systems.

All kinds of faults are detected including close-in three-phase; cross-country faults, evolving faults and high-resistance earth faults

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure to be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued.

  • The maintenance staff should be trained, qualified, and carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear your safety and healthy equipment as safety helmet, safety shoes ... etc.

  • Line isolation is required.

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection.

  • Transmission Line must be grounded.

Tools and Equipment

  • ISA DRTS tester.

  • Test leads.

  • PC laptop with necessary software {ISAWIZARD and Micom S1-13- P437}

  • Tools bag.

 

 

Work to be carried out

Note:  CT cores for BF and Protection Relay are different i.e. BF will not work during test.

 Preparation

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test.

  2. Be sure that the DC power of the relay is correct by applying DC voltmeter across points (25,26 see fig.1) on the rear of the relay

Reading the Setting from the Relay using PC

  1. Connect the personal computer to the relay

  2. Run the Micom S1 program, Read the setting and save it as URGENT FILE FOR P437 [name the feeder] 

Distance Protection Test

  1. Connect the personal computer to the ISA tester.

  2. Be sure that ISA tester is connected to EARTH.

  3. Switch ON ISA tester.

  4. After ISA tester completely in service connect Currents, Voltages and binary input leads between ISA tester and test plug (X21-1 & X21-2) see figure 2) for Micom P437. 

  5. Make short between points (X21-1 13 &14) and points (X21-1 15 & 6 ) see figure 3

  6. Run ISA Wizard software and design RIO file for P437 characteristic.

  7. NOW you can choose two ways to test this relay.

  8. Note: Don't forget to GENERATE PREFAULT CONDITION [I=0.001A, V=57V].

  9. From TEST SELECTION chose automatic time zone (Z/T) and don’t forget to adjust the test sequence as  follow [fault = all faults, Z min = zero, Z max = as the setting, STEP =0.5 OHM, Angle [degree] = 0, after that click OK the tester will operate and test the R axis.

  10. Repeat step 9, BUT choose angle = 90 the tester will test the X axis.

  11. Choose STORE ALL TESTS >>>>.

  12. Go to RESULTS from main menu choose HEADER and apply all information about this FEEDER and OK

  13. Now print the results and you have complete test results for this relay.

  14. From TEST SELECTION, choose Click and test and by curser click on the point which you need to test [generally choose all points on X and Y Axis]. 

  15. Repeat steps from 11 to 13.

  16. Now you have complete test results for this relay.

 

 

 

 

 

Other Function Test

Power Swing

  1. From main menu select TEST SELECTION select POWER SWING test and plot the setting.

  2. Click Start

  3. Chose STORE ALL TESTS

  4. Go to RESULTS from main menu choose HEADER and apply all information about this FEEDER and OK.

  5. Now print the results and you have complete test results for this function.

Switch on to Fault ( SOTF )

  1. From main menu select TEST SELECTION select SOTF test and plot the setting from Rio file.

  2. Make prefault at zero current and voltage

  3. Click ok.

  4. Inject current and voltage as fault case in any zone Z2 or Z3

  5. Click Start.

  6. Choose STORE ALL TESTS, from results will see that the time at all faults is equal to Z1 time.

  7. From results will see that the time at all faults is equal Z1 time.

Over Current (O/C)

  1. RUN ISA WIZARD software and choose GENERATION AND DISTRIBUTION RELAYS>>>> X-PRO >>>>MANUAL MODE>>>>

  2. GENERATE PREFAULT CONDITION [I=0.001A, V=57V].

  3. Disable distance functions.

  4. SELECT FAULT AND INJECT CURRENT VALUE EQUAL TO (I Setting*1.01) to determine the operating time

  5. Click Start.

  6. Repeat steps from 1 to 5 for all phases R, S, and T.

 Note that: You can inject on three phase one time.

  1. Choose STORE ALL TESTS.

 Earth Fault (E/F)

  1. Run  ISA WIZARD software and choose GENERATION AND DISTRIBUTION RELAYS>>>>X-PRO >>>>MANUAL MODE>>>>

  2. GENERATE PREFAULT CONDITION [I=0.001A]

  3.  Disable distance functions.

  4. SELECT FAULT and INJECT CURRENT VALUE EQUAL to (I Setting*1.01) to determine the operating time

  5. Click Start.

  6. Repeat steps from 1 to 5 for all phases R, S, and T.

    Note that: the injection should be on one single phase by phase not on three phases or on two phases.

  7. Choose store all testes

 Return Relay and Associated Equipment to Service

  1. Enable any function which was disabled.

  2. Return relay to service by removing the test leads from the relay terminals and the test set.

  3. Make sure CT and VT secondary circuits are in normal condition.

  4. Remove the disturbance alarm LED's on the relay which is caused by the test.

  5. Request feeder to be returned in service

  6. Take pharos readings of all AC currents and voltages and compare them to those for normal load condition.

  7. Check that these reading agree with station meters (MW, Mvar, Amperes and power factor).

  8. Place a copy of the test result in the field maintenance file.
