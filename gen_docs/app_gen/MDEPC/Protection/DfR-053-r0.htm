Introduction

This document outlines the testing procedures P1, which must be performed on an annually bases as indicated by Y1 in the above title block.

The 7SD610 numerical differential protection relay for HV cable between new power station 750 MW and HV switchgear protected zone is feinted at both ends, communication  link are designed for fiber optic between both relays.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal protective equipment must be used as safety shoes, and safety helmet.

  • Unit out of service.

  • Make Sure CT Secondary circuit are shorted.

Tools and Equipment

  • Secondary tester Amp (OMICRON)

  • Laptop computer with the software program omicron 

  • Laptop computer with the digs software program 

  • Special Link between the secondary test and the device

  • Special tools for measuring (Avometer clap screwdriver leads)  

Work to be carried out

 

Preparation

  1. Review technical reference materiel manufactures / materiel

  2. Obtain copy of relay salting from Felid maintenance file.

  3. Obtain outage for unit protected by this relay

  4. Inspect the secondary circuits of CT / PT miniature circuit breakers

  5. MCB and all cabling supplying the relay

  6. Block any disturbing alarm or fault recorders associated with this protection.

Testing Procedures

  1. Shorted terminal of CT in socket (X110) in panel (12) CHB01 (F19) in Power Station

  2. Connect the power supply to the device (Secondary) tester (OMICRON)

  3. Connation the secondary injected tester (OMICRON) for inject the current (1.2.3) and Neuter (4).

  4. Connect the timer leads.

  5. Determine the pickup value and drop out value and the time of pickup value.
