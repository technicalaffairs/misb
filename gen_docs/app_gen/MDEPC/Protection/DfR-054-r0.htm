Introduction

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The P632 differential protection devices are designed for the fast and selective short circuit protection of transformers, motors and generators and of other two, three or four phase winding arrangement.

Safety Precautions

  • A work permit must be issued

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area

    • is required.

  • Wear appropriate personal protective equipment must be used, safety shoes, helmet breathing mask, safety glasses

  • Feeder isolation is required.

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection.

  • Feeder must be grounded.

Tools and Equipment

  • ISA DRTS tester.

  • Test leads.

  • PC laptop with necessary software {ISAWIZARD and Micom S1}

  • Tools bag.

 

 

 

 

 

Work to be Carried Out

Note:  CT cores for BF and Protection Relay are different i.e. BF will not work during test.

Preparation

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test.

  2. Be sure that the DC power of the relay is correct by applying a DC voltmeter across points T20(25,26) on the rear of the relay .

Reading the Setting from the Relay using the PC

  1. Connect the personal computer to the relay.

  2. Run the MIcom S1 program.

  3. Read the setting and safe it as URGENT FILE FOR P632 [name the feeder].

  4. Connect the personal computer to the ISA tester.

  5. Be sure that ISA tester is connected to EARTH.

  6. Run ISA program.

  7. After ISA tester completely in service, connect currents leads between ISA tester and test plug (X87-1 points 22,24,26 and 28) for Micom P632. 

  8. Disconnect voltage terminal from terminal block XV01 points (9,10,11,12)

  9. Connect voltages leads between ISA tester and terminal block XV01 points (9,10)

  10. Make short between X87(points 13 &14) and between X87 (points 15 & 16) to get DC on terminal relay.

  11. RUN  ISAWIZARD software>>>>>GENERATION & DISTRIBUTION RELAYS>>>> Xpro advanced>>>>MANUAL MODE

  12. NOW you can test the relay.

Differential Protection Test

NOTE:  Don't forget to Generate Prefault Condition [I=0.001A, V=0V].

  1. Disable other function.

  2. Choose Delta ramp with step ∆I 0.05 Amp to get pickup current of the relay

  3. Choose Shoot with current value equal to 1.05 of pick up current to get pickup time of the relay.

  4. To test 2nd stage of differential, disable 1st stage of differential then shoot by setting current

  5. Stop  ISA wizard program.

  6. Stop all test and go to results from main menu choose header and apply all information about these feeder and OK

  7. Repeat the test, connect current leads between ISA tester and test plug (X87-1 points 2,4,6 and 8)

2nd and 5th Harmonic Restraining Test

  1. Repeat steps from 1 to 7 with frequency 100Hz for 2nd harmonic test the relay will not operate.

  2. Repeat steps from 1 to 7 with frequency 250Hz for 5th harmonic test the relay will not operate

  3. Now print the results and you have complete test results for these relay.

Now you have complete test results for these relay.

Thermal Overload Test

  1. Disable differential protection function

  2. Inject current from side b (220 kv side) test block (X87-1 points 2,4,6 and 8)

  3. Decrease the cooling and heating time constants to minimum value (1min).

  4. According to overload  setting curve, determine the value of the current which should be injected  to obtain corresponding pickup time

  5. Repeat step 4 for some points to verify the characteristic of thermal overload curve.

  6. Choose STORE ALL TESTS >>>>

  7. Go to RESULTS from main menu choose HEADER and apply all information about these FEEDER  and OK

Over Flux (V/F) Test

  1. Disable differential and thermal overload protection functions

  2. GENERATE PREFAULT CONDITION [I=0 A, V=57V F =50 Hz].

  3. According to V/F setting curve, determine the value of the voltage which should be injected to obtain corresponding pickup time.

  4. Repeat step 3 for some points to verify the characteristic of V/F curve.

  5. Choose STORE ALL TESTS >>>>

  6. Go to RESULTS from main menu chose HEADER and apply all information about these FEEDER and OK

Return Relay and Associated Equipment to Service

  1. Enable any function which was disabled.

  2. Return relay to service by removing the test leads from the relay terminals and the test set.

  3. Make sure CT and V T secondary circuits are in normal condition.

  4. Remove the disturbance alarm LED's on the relay which is caused by the test.

  5. Request feeder to be returned in service.

  6. Take pharos readings of all AC currents and voltages and compare them to those for normal load condition.

  7. Check that these reading agree with station meters (MW, Mvar, Amperes and power factor).

  8. Place a copy of the test result in the field maintenance file.
