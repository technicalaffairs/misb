Introduction

This document outlines the equipment inspection I1, which must be performed on a monthly bases as indicated by M1in the title block 

Technical Specification:

AC input : 380±10% V ac, 50Hz, 3PH
Admissible frequency fluctuations : ±5%

DC output : 135V

Rated intensity (Idc):- 1600A

Safety Precautions

  • Work order must be issued.

  • Switch off input & output supply.

Tools and Equipment

  • Suitable tool set

  • Digital voltmeter

  • Vacuum cleaner or blower brush

Work to be carried out

  1. Switch - off input and output supply

  2. Clean it from air dust and dirty

  3. Check the ventilation system

  4. Inspect over heating compartment of control board

  5. Check fuses, contacts and terminal titling

  6. Check performance of timers

  7. Inspect visually the thyristor, IGBT`S and diodes

  8. Switch on input and output supply

  9. Check input and output voltage

  10. Check the output current of battery charger and the battery current

  11. Check connection and disconnection in case of fault in power supply

  12. Check alarm & indication lamps

  13. Check pulses of transistors or the wave form on the output bridge diodes (if required)

  14. Retain the battery charger to operation mode and close the work order
