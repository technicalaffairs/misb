Introduction

This document outlines the maintenance procedures P1, which must be performed on yearly bases as indicated by Y1 in the title block.

The directional earth fault relay type IM3+P2g045 is used to detect earth faults on systems in with the neutral is earthed through a resistance. The IM3 relay is used as release for directional relay.

The first draft initiated by eng. Ahmed Galal, tested by eng. Mohamed Rabee, accepted by general manager eng. Hamdy Dorgam and certified by eng. Gaber Farahat WDEPC inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • The transformer must be was isolated.

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance.

Tools and Equipment

  • ISA test set type ART/3 or Omicron

  • Voltmeter, Ammeter, test plug, and test leads.

Work to be Carried Out

Relays Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

Test Steps

  1. Connect the current leads of a tester unit to the relay points according to the drawings and scheme.

  2. the 1st in testing. on element O/C.

  3. Inject current to each couple of points (1-2) on the element o/c till the red flag of the device displays.

  4. Record the results pick up value at 0.25A.

  5. Decrees the current until the relay is reset and measure the current at reset 0.21A.

  6. Record the results reset value at 0.21A.

  7. Time test in the relay is connect terminal timer to (3_4)on relay as connect terminal unit current to (1_2)on relay as inject the current to pick up relay and Record the results time .t=2.9sec.

  8. The 2nd in testing On element direction.

  9. Inject current to each couple of points (1-2) on the element direction till the red flag of the device displays. And inject voltage on point (3_4) 60v

  10. Record the results pick up value at 102mA.

  11. Decrees the current until the relay is reset and measure the current at reset 77mA.

  12. Record the results reset value at 77mA.

  13. Switch off secondary injection t test set.

  14. Return the relay to the panel.

  15. Return all connections to the normal position as before the test.

  16. Close the work permit
