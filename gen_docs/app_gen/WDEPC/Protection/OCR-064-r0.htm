Introduction

The MCGG is a microprocessor based over-current relay available in single, two, three or four pole versions manufactured by GEC ALSTHOM. Each pole is provided with a choice of four inverse time characteristics and three definite time characteristics as well as a trip test position. An instantaneous element is also available on each phase. The MCGG 22 is the single phase version.

This procedure details the steps required to completely check the proper operation and calibration of the MCGG22 relay. It is recommended this procedure be carried out on a yearly bases.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear safety and healthy equipments as safety helmet, safety shoes ... ets

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current

    injection.

  • Switch off the DC power supply before inserting or withdrawing the relay

    Outage arrangement must be done

Tools and Equipment

  •  Single phase Secondary Injection Test Set MTM2.

  • Test plug, Test Leads.

  • Multi-meter.

Work to be Carried Out

Relays Preparation

  1. Review the manual and drawings.

  2.  Carefully follow these test procedure steps

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

  8.  Verify the test results match the relay setting - If not consult technical services.

  9.  Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolating the relay current circuits for test.

  10. Look at field maintenance file and compare the position of the setting switches on relay with those shown in the setting sheet

Testing Procedure(s)

Power Supply Test

  1. Press the RESET button and check that all LED’s are illuminated. This confirms that all measuring boards are operating satisfactorily.

Current Pick-Up and Reset Check Test

  1. Connect test leads between the current terminals on the relay as shown in drawing and the current terminals on the test set.

  2. Slowly increase the current until the green LED lights. The relay must operate with a current between +/- 10% of the setting value.

  3. Record this value on the test sheet.

  4. Slowly decrease the current until the green LED reset.

  5. Record this value on the test sheet.

Time Setting Test

  1. Inject current into the relay and increase it to reach (5 Is) . Keep this value and switch off the test set.

  2. Connect the relay time delayed output contacts to the timer contacts of the test set.

  3. Switch on the test set and inject current. The timer will stop after delay time(30sec).

  4. Record this value on the test sheet.

  5. Remove the leads from the time delayed contact and the test set.

Instantaneous Setting Test

  1. Connect the instantaneous trip out put contacts to the timer contacts on the test set. This is to stop the current as soon as the relay picks up.
      
    CAUTION: Don't keep this current on the relay for more than a few seconds as it may exceed the rating and damage the relay.

  2. Decrease the setting of Is to 0.05 In.

  3. Increase current quickly to reach (30 Is). Keep this value and switch off the test set.

  4. Connect the relay trip output contacts to the timer contacts of the test set.

  5. Switch on the test set and inject current. The timer will stop instantaneous (50ms).

  6. Record this value on the test sheet.

Trip Test

  1. Request the operator to switch on the main circuit breaker.

  2. Operate the output auxiliary contact by injection to trip the circuit breaker.

  3. Check that the circuit breaker tripped.

Return the Relay and Power Equipment to Service

  1. Verify the test result match the relay setting. If not consult technical services.

  2. Test trip the circuit breaker associated with the relay.

  3. Make a final check that all switches on the relay are set according to the setting sheet.

  4. Make sure the CT secondary circuit is in normal condition

  5. Remove the CT shorts which were installed in the Power Supply

  6. Return the relay and power equipment to service.
