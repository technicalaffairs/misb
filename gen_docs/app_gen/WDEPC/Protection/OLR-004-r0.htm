Introduction

This document out lines the testing procedure P1, to be performed on an annually bases as indicated Y1 in the title block

These procedure details the steps required to completely check the proper operation and calibration of RSF 5 relay. The first preliminary steps to prepare for relay testing describes the activates required prior to testing.

The first draft is initiated & tested by eng. Mostafa Hassan, accepted by general director eng. Farag Abo Yousef, approved by eng. Abd El-Mohsen Abdel-Ghafar, and certified by eng. Mona Ahmed Elsharkawy: WDEPC engineering Consultant "A" and MPIS coordinator.

Safety Precaution 

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category

  • Wear your safety helmet & safety shoes.

  • Safety fence with caution marks surrounding the work area is required.

  • The equipment isolation is required.

Tools and Equipment

  • ISA test set (secondary injection voltage)

  • Clamp ampere, Avometer

 

Work to be carried out

 

Preparation

  1. Obtain a copy of relay setting from field maintenance file and compare it with setting on the relay

  2. Obtain outage for transformer or equipment  protected by this relay.

  3. Review technical reference and manufacture manual

 

Test Steps

  1. Connect the current test set terminals to point (1-2) for phase R

  2. Inject the current for phase R then increase the current gradually until the red flag for phase R appears then record IR (pick - up)

  3. Decrease the current gradually until the red flag disappear then record IR (reset)

 

Return Relay and Associated Equipment to Service 

  1. Return relay to service by removing the plug test

  2. Make sure CT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  • Breaker trip coils

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.

  • Make sure CT secondary circuits are in normal condition

  1. Return the work permit.
