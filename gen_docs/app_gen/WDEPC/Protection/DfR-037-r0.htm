Introduction

The three phase differential Relay type R30 is primarily intended to protect the following system components

  • Transformers.

  • Generators

The Relay is connected directly to main current Transformer of each feeder of the protected item.  

The first draft is imitated and tested by eng. Mostafa Hasas, accepted by general director eng. Farag Abo Yousef, and approved by eng. Abd ElMohsen Abd El-Ghafar: headsector of Kafr El Dawar PS

This document revised by eng. Mona Ahmed Elsharkawy: WDEPC inspection & quality control 's  head sector and MPIS coordinator.

Safety Precaution 

  • A work permit must be issued and the maintenance staff should carry the suitable safety category

  • Wear your safety helmet & safety shoes.

  • Safety fence with caution marks surrounding the work area is required.

  • The equipment isolation is required

  • Make sure that CT secondary are shorted before isolate the current circuit for test current injection

Tools and Equipment

  • ISA test set (secondary injection voltage)

Work to be carried out

Preparation

  1. Obtain a copy of relay setting from field maintenance file and compare it with setting on the relay

  2.  Obtain outage for transformer protected by this relay.

  3. Check that the two bus disconnecting switches are opened and tagged from all sides H.V, MV and  LV sides.

  4. Inspect the CT secondary circuits and all cabling supplying the relay including matching CT

  5. Use the station drawings to isolate the following

  • Breakers trip coils.

  • Breaker failure scheme associated with the Relay.

  • Disturbance alarms and fault recorder associated with the Re.

  1. This procedure is used to perform offload relay testing.

  2. Check there is no AC current from CT's by using a clip on ammeter

 

Request Power Transformer to be returned in Service

  1. Take crossed readings of R30 current sand voltages from all sides (HV and LV) sides and compare them to these for normal load condition

  2. Place a copy of the test Result in the field maintenance file.

 

Test Steps

  1. Inject the current to points ( 1,19 ) for phase R for H.V side

  2. Increase the injected current gradually till the red flags of the injected phases appear then record the pick up value .

  3. Inject current greater than pick up then decrease injected current gradually until the red flag disappear then record the reset value .

  4. Inject the current to points ( 3,19 ) for phase R for L.V side

  5. Increase the injected current gradually till the red flags of the injected phases appear then record the pick up value .

  6. Inject current greater than pick up then decrease injected current gradually until the red flag disappear then record the reset value

  7. Repeat the pervious steps from 1to 6 phase S but the injected current to point s ( 4,19) H.V   side (6,19) L.V side

  8. Repeat the pervious steps from 1to 6 phase T but the injected current to points ( 7,19)

 

Return Relay and Associated Equipment to Service 

  1. Check the return of relay- wires and trip-flags to their original places and tighten them carefully.
  2. Be sure that the relay is completely ready for service any time.
  3. Return the work permission.
