Introduction

This document outlines the maintenance procedures P1 on motors 3 ph 6 kv AC section, which must be performed on yearly bases as indicated by Y1 in the title block.

This is electronic current relay provided with electromechanical out put element, adjustment by steps of the operation and time delay values with high resolution

This device to protect all types of motors (6 kv or 380 v) against (thermal over load - negative sequence current - short circuit - earth fault)

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment

The first draft initiated by eng. Jehan Abdel Monsif, tested by eng. Mohamed Rabee, accepted by general manager eng. Hamdy Dorgam of Damanhour power station and certified by eng. Gaber Farahat WDEPC inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is

  • Safety fence with caution marks surrounding the work area is required

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Line isolation is required

Tools and Equipment

  • ISA test 3 phase

  • Test plug handle

  • Voltmeter, clamp Ammeter, test leads

Work to be Carried Out

Testing the relay and protection outputs

  1. Carefully folow these test procedure stes.

  2. Verify the test result match the relay setting. If not cinsult technical services.

  3. Test trip the circuit breaker associated with the relay.

 

Testing Procedure (s)
 Preparations

  1. Disconnect the circuit breaker for motor under test.

  2. Look at field maintenance file and compare the position of the setting switches on relay with those on the setting sheet.

  3. Set the earthing link on the relay to line side position.

Testing the relay

Short circuit test

  1. Setting for test
         * Make lead for parameter  I1    > on stat ON
         * Make lead for parameter  I2 > on stat OF
    Note :
         * I1 > ------ referring to protection against short circuit
         * I1 >------- referring to protection against negative

  2. Connect the test lead between the current terminal (1A-2A) for phase "R"

  3. Increase the injection current fastly until the red indication on short circuit on the relay appears I1>

  4. Record this value in the test sheet

  5. Repeat the above test from step (1 to 6) but inject current between terminal (3A-2A) for phase "S"

Negative phase sequence test

  1. Setting for test
         * Make lead for parameter I1> on stst ON
         * Make lead for parameter I2 > on stat ON
         * Connect the test lead between the current terminal (1A-2A) for phase "R"

  2. Adjust tester on suitable range for test

  3. Increase the injection current slowly until the read indication on negative phase sequence current on the relay appears I2>

  4. Record this value in the test sheet

  5. Repeat the above test from step (1 to 4) but inject current between terminal (2A-3A)for phase "S"

For testing the time

  1. Rise the current at the pick up of the relay

  2. Turn off the tester

  3. Connect test leads between the timer leads (1B-3B) (the relay contact)

  4. Turn on the tester

  5. The timer will stop once the relay has picked-up

  6. Record this value in the test sheet and compeer with curve in the catalog

Thermal test

  1. Setting for test
         * Make lead for parameter    I1> on stat ON
         * Make lead for parameter    I2> on stat OFF
            Connect the test lead between the current terminal (1A-2A)for phase "R"

  2. Adjust tester on suitable range for test

  3. Increase the injection current until "5A" (equal to 500% from the base current value fore relay)

  4. Turn off the tester

  5. Connect test leads between the timer leads (1B-3B) (the relay contact)

  6. Turn on the tester

  7. The timer will stop once the relay has picked-up

  8. Record this value in the test sheet and compeer with curve in the catalog

  9. Disconnect the tester

  10. Record this value in the test sheet

  11. Repeat the above test from step (1to10) but inject current between terminal (2A-3A) for phase "S"

Earth fault test

  1. Setting for test
         * Make lead for parameter     I1 > on stat ON
         * Make lead for parameter     I2 > on stat ON

  2. Connect the test lead between the current terminal (1A-2A) for phase "R"

  3. Adjust tester on suitable range for test

  4. Increase the injection current slowly until the red indication on earth fault on the relay appears E/F 4. record this value in the test sheet

Earth fault test

  1. Put the circuit breaker on the tester position

  2. Request the operator switch "ON"

  3. Injecting the pick-up current in to the relay for check any element

  4. Check that the circuit breaker tripped

Return relay and associated equipment to service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set i.e
         * Disconnect the test leads of the current terminals
         * Disconnect the test leads between the trip contact

    2.

  2. Make sure CT secondary circuits are in normal condition

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
         * Breaker trip coils
         * Breaker failure scheme associated this relay
         * Disturbance alarm and fault recorder associated with this relay
