Introduction

These procedure details the steps required to completely check the proper operation and calibration of AT31 relay.

It is recommended this procedure be carried out an yearly bases the first preliminary steps to prepare for relay testing describes the activates required prior to testing.

The first draft is imitated  and tested by eng. Mostafa Hasas, accepted by general director eng. Farag Abo Yousef, approved by eng. Abd El-Mohsen Abd El-Ghafar.

This document revised by eng. Mona Ahmed Elsharkawy: WDEP inspection & quality control's head sector and MPIS coordinator.

Safety Precaution 

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Wear your safety helmet & safety shoes.

  • Safety fence like ropes with caution marks surrounding the work area .

  • The equipment isolation is required

Tools and Equipment

  • ISA test set (secondary injection voltage)

  • Clamp ampere or Avometer

 

Work to be carried out

 

Preparation

  1. Obtain a copy of relay setting from field maintenance file and compare it with setting on the relay

  2. Obtain outage for transformer or equipment  protected by this relay.

  3. Review technical reference and manufacture manual

 

Test Steps

  1. Connect the current test set terminals to point (7-10) for phase R

  2. Connect the timer test set terminal to point (23-24)

  3. Inject the current for phase R then increase the current gradually until the red flag for phase R appears then record IR (pick - up) and record the time

  4. Decrease the current gradually until the red flag disappear then record IR (reset)

  5. For find the time for trip increase the current until the timer red flag appear them record the time for trip

  6. For phase S repeat steps from (1) to (5) connect the current test set to point (8-11)

  7. For phase T repeat steps from (1) to (5) connect the current test set to point (9-12)

 

Return Relay and Associated Equipment to Service 

  1. Return relay to service by removing the plug test

  2. Make sure CT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  • Breaker trip coils

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.

  • Make sure CT secondary circuits are in normal condition

  1. Return the work permit.
