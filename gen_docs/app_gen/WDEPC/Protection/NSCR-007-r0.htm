Introduction

This document outlines the maintenance procedures P1, which must be performed on yearly bases as indicated by Y1 in the title block.

The relay circuit, tuned to the supply frequency, rejects the harmonics produced by current transformer saturation. A slight time delay on operation helps to provide stability on heavy external faults and is obtained by allowing the auto-transformer to saturate above the relay setting. This limits the current supplied, and the attracted armature unit operates only on the slower part of its time-current.

The first draft initiated by eng. Ahmad Galal, tested by eng. Mohamed Rabee, accepted by general manager eng. Hamdy Dorgam of Damanhour power station and certified by eng. Gaber Farahat WDEPC inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • The transformer must be was isolated.

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance

Tools and Equipment

  • ISA test set type ART/3 or Omicron.

  • Voltmeter, Ammeter, test plug, and test leads.

Work to be Carried Out

Relays Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

Test Steps

  1. Connect the current leads of a tester unit to the relay points according to the drawings and scheme.

  2. The 1st in testing.

  3. Inject current to each couple of points (9-10) to (R_S).in terminal (B50_x1) till the red flag of the device displays.

  4. Record the results pick up value at 0.85A.

  5. Decrease the current until the relay is reset and measure the current at reset 0.61A.

  6. Record the results reset value at 0.61A

  7. Inject current to each couple of points (9-11) to (R_T) in terminal (B50_x1) till the red flag of the device displays.

  8. Record the results pick up value at 0.85A.

  9. Decrease the current until the relay is reset and measure the current at reset 0.61A.

  10. Record the results reset value at 0.61A

  11. Inject current to each couple of points (10-11) to (T_S) in terminal (B50_x1) till the red flag of the device displays.

  12. Record the results pick up value at 0.85A.

  13. Decrease the current until the relay is reset and measure the current at reset 0.61A.

  14. Record the results reset value at 0.61A

  15. time test in the relay is connect terminal timer to{ B50_x5  (19_20)}in relay as connect terminal unit current to{B50_x1 (9_10)}in relay as inject the current to pick up relay and Record the results time .at k=1

  16. and repeat time test to(R_T) & (S_T).

  17. The 2nd  testing

  18. Inject current to each couple of points (9-10) to (R_S).in terminal (B50_x1) till the red flag of the device displays.

  19. Record the results pick up value at 1A.

  20. Decrease the current until the relay is reset and measure the current at reset 0.9A.

  21. Record the results reset value at 0.9A

  22. Inject current to each couple of points (9-11) to (R_T) in terminal (B50_x1) till the red flag of the device displays.

  23. Record the results pick up value at 1A.

  24. Decrease the current until the relay is reset and measure the current at reset 0.9A.

  25. Record the results reset value at 1A

  26. Inject current to each couple of points (10-11) to (T_S) in terminal (B50_x1) till the red flag of the device displays.

  27. Record the results pick up value at 1A.

  28. Decrease the current until the relay is reset and measure the current at reset 0.9A.

  29. Record the results reset value at 0.9A

  30. Time test in the relay is connect terminal timer to{ B50_x5  (17_18)}in relay as connect terminal unit current to{B50_x1 (9_10)}in relay as inject the current to pick up relay and Record the results time at k=1 and repeat time test to(R_T) & (S_T).

  31. Switch off secondary injection t test set.

  32. Return the relay to the panel.

  33. Return all connections to the normal position as before the test.

  34. Close the work permit
