Introduction

This procedure details the steps required to completely check. he proper operation of the SEL 321 distance relay It is recommended this procedure be carried out on a yearly bases

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment

The first draft initiated by eng Hoda Mohamed Ahmed & Heba Abdel Meniem at Al Dekhaila 220kv substation, approved at 25 May 2005.

This document was been updated by eng Ad El Naseer Mostafaa at Sidi Krir PS, West Delta Electricity Production Company.

Safety Precautions              

  • A work permit must be issued and

  • The maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be well trained, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Safety and healthy equipment must be used (safety helmet , safety shoes ... etc).

  • Ensure that the CT secondary are correctly short circuited

  • Disconnect the DC supply and Insert the test socket.

Tools and Equipment

  • TZ 2 test set: this test set consists of 2 units supply timer and volt injection unit and current injection unit

  • Multimeters

  • Test plug (MMLG 0)

  • Test leads

  • Screwdriver

Work to be Carried Out

 

Preliminary Steps to Prepare for Relay Testing

  1. This procedure is used to perform off load relay test

  2. Obtain copy of relay setting from field maintenance file

  3. Obtain outage for line protected by this relay

  4. Check that the bus and line disconnect switches are opened and tagged

  5. Inspect the CT secondary circuits the VT miniature circuit breakers (MCB`S) and all cabling supplying the relay

  6. Use station drawing to isolate:

  •  Disturbance alarms and fault recorder associated with this relay

  •  Auto - reclose associated with this relay

Testing the Relay, Relay scheme and Protection Outputs

  1. Carefully follow these test procedure steps

  2. Verify the test results match the relay setting if not consult technical services

  3. Test trip all breakers associated with the relay

Prior to starting testing procedure(s) ensure the following stepos

  1. Look at EEHC panel nameplate and check that EEHC system designation nameplate agrees with the line that has been taken out of service for maintenance

  2. Check there is no AC current from CT`s by using a clip on ammeter

  3. Check there is no AC voltage from VT`s by using a voltmeter

Preparation

  1. Ensure that the CT secondaries are correctly short circuited on the case side of test plug

  2. These connections must be made before the test plug is inserted

  3. Connect the current terminals IA,IB,IC and IN and the voltage terminals VA,VB,VC and VN on the test plug to the corresponding current and voltage terminals on the supply unit test set

  4. Insert the test plug

  5. A djust the test unit switches as

  • Put the angle switch at - 90º

  • Ampere switch on 3A

  • The injection direction switch on normal position in case of forward zones Z1, Z2, Z4 and on reverse position in case of reverse zone Z3

  • Put the injection type switch on I + V p

Testing the Phase to Ground and Phase to Phase Measuring Elements

  1. Select phase to ground fault A-N by using relay selector switch on supply unit test set

  2. Connect the outgoing voltage from the supply unit

  3. Switch on the injection switch

  4. Start the test by pressing the switch on the control unit momentarily

  5. Observe that the red indications A,G and Z1 on the relay will light up

  6. Change the value of the potential tap up or down unit the relay operates in second step( The red indication Z2 on the relay will light up with the lest indications)

  7. Record this pickup points on test sheet

Repeat the above Test for Phases B-N, C-N, A-B, B-C and C-A

  1. The above test but this time calculate potential corresponding to the relay setting of the second zone for phase to ground and phase to phase fault set this value on the test set and then change it up or down until the relay operates in third step

  2. Switch off switch

Measuring the Operating Times

  1. Set potential switch to 50% of the zone 1 reach

  2. Select phase to phase fault A-B by using relay selector switch on supply unit test set

  3. Connect test leads between timer contact on the relay to the stop terminal of the timer

  4. Reset the interval timer

  5. Switch on

  6. Start the test by pressing switch on

  7. The timer will stop once the relay has picked up

  8. Record the zone 1 tripping time on test sheet

  • Repeat tests for zone 2 and zone 3 operating times

  • Repeat test for phase to ground operating times

Testing the Relay, Relay Scheme and Protection Outputs

  1. Verify the test results match the relay setting if not consult technical services

  2. Perform checks of the acceleration of the relay breaker failure and reclosure schemes separate technical procedures should be used to test each of these systems

  3. Test trip all breakers associated with the relay.

Return Relay and Associated Equipment to Service

  1. Return Relay to Service y Withdrawing The Test Plug Handle From The Relay.

  2. If Any Equipment Associated with the Relay has been Isolated Without The Test Plug, It should be return to service by the following restoration steps

  • Disturbance alarms and fault recorder associated with this relay

  •  Auto - reclosure associated with this relay

Request Power Line to Returned to Service

  1. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those for normal load conditions

  2. Check that reading agree with station meters (MW/MVAR/Power Factor/Amperes/Voltages)

  3.  Place A copy of the test result in the field maintenance file
