Introduction

The Generator stator earth fault protection F11X, independent of the effect of higher harmonics, is used as the turns or earth fault protection of generators stator operating in block with transformer

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence  with caution marks surrounding the work area is required

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • The voltage transformer must be out of services

  • Generator out of services, C.P. open and tagged isolated

Tools and Equipment

  • Secondary injection test set

  • Test leads, clamp ammeter, voltmeter

  • Complete tool set

 

 

 

 

 

 

Work to be Carried Out

 

Earth Fault Test

  1. Disconnected the generator circuit breaker

  2. Remove the input voltage terminals (1&4)

  3. Connect test leads between voltage terminal (1&4)

  4. Increase the injected voltage slowly until it makes its signals.

  5. Record this value on the test sheet (adjust it if needed)

  6. Switch off the set

Time Test

  1. Connect the time delayed contacts on the relay to timer terminals on the test set

  2. Start the test by injecting the pick up current into the relay

  3. Record the value of time on the test sheet (adjust it if needed)

  4. The timer will stop once the relay has pick up

Return Relay to Service

  1. Disconnect the test leads from the terminal board and connect the relay terminals (1,4) to generator terminal

  2. Disconnect the test leads from timer terminal on the relay and connect the relay time delay terminal to the normal connection condition

  3. If any equipment associated with the relay has been isolated, it should be returned to connect the breaker trip coil terminal
