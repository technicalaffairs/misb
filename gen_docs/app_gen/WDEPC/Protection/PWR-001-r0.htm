Introduction

This document outlines the maintenance procedures P1, which must be performed on yearly bases as indicated by Y1 in the title block.

The DL400, is a pilot wire relay for the protection of overhead lines and cables. the protection detects phase faults in unearthed systems and phase and earth faults in solidly or resistance earthed systems. the zone of protection may also include a power transformer.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initiated by eng. Ahmad Galal, tested by eng. Mohamed Rabee, accepted by general manager eng. Hamdy Dorgam of Damanhour power station and certified by eng. Gaber farahat WDEPC inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • The line must be disconnect from two sides

Tools and Equipment

  • ISA test 3 phase.

  • M. ammeter, Test Leads .

  • Secondary injection test set .

Work to be Carried Out

Relays Preparation

  1. Look at field maintenance file and compare the setting on relay with those shown on the test sheet.

  2. Set the earth link on the relay to line side position.

 

Test Steps

O/C terminal

  1. Connect test leads between  the current terminals ( 3– 6) for phase ( R ).

  2. Increase the current slowly until the relay operates.

  3. Record this value in the test sheet.

  4. Connect test leads between  the current terminals ( 4– 6) for phase S and repeat the previous steps.

  5. Connect test leads between  the current terminals ( 5– 6) for phase T and repeat the previous steps.

  6. For testing the time repeat the previous steps of the time.

  7. Connect test leads between  the current terminals ( 3– 6) for phase R and repeat the previous steps.

  8. Disconnect the terminal 9 is (+VDC)

  9. Connect test leads between  the time terminals ( 9– 10) for phase  and repeat the previous steps for phase S & phase T.

87PW Terminal

  1. Connect the test leads between the current terminals ( 1 – 2 ).

  2. Increase the current slowly until the relay operates.

  3. Record this value in the test sheet.

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set i.e.
    - Disconnect the test leads of the current terminals.
    - Disconnect the test leads between the trip contact.

  2. Make sure CT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
    - Breaker trip coils.
    - Breaker failure scheme associated this relay.
    - Disturbance alarm and fault recorder associated with this relay.
