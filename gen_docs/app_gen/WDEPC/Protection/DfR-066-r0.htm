Introduction

Three phase differential Relay type IRD1-T2 is primarily intended to protect the transformers. The Relay is connected directly to main current Transformer of each feeder of the protected item.

Gomaa Ghater:  head sector of Kafr el Dawar Power Station, and certified by eng. Gaber Mohmed Frahat : WDEPC inspection and quality control general director and MPIS coordinator.

Safety Precautions

  • The equipment isolation is required.

  • A work permit must be issued

  • The maintenance staff should carry the suitable ID safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

Tools and Equipment

  • Secondary current injection test set ( ISA Tester )

  • AVO meter

  • Clamp ampere.

Work to be Carried Out

Steps to Prepare for Relay Testing

  1. Obtain outage for transformer protected by this relay.

  2. Obtain copy of Relay setting from field maintenance file.

  3. Check that the  bus disconnecting switches are opened.

  4. Inspect the CT secondary circuits and all cables supplying the relay including matching CT'S.

 

 

Preparation

  1.  This procedure is used to perform offload relay testing.

  2. Check there is no AC current from CT's by using a clip on ammeter.

Test Steps

  1. Inject the current to points (  A3,A4  ) for phase R for H.V side

  2. Increase the injected current gradually till the red flags of the injected phases appear then record the pick up value .

  3. inject current greater than pick up value then decrease injected current gradually until the red flag disappear then record the reset value .

  4. inject the current to point (B3,B4) for phase R for L.V side .

  5. increase the injected current gradually till the red flag of the injected phases appear then record the pick up value .

  6. inject current greater than pick up value then decrease injected current gradually until the red flag disappear then record the reset value .

  7. Repeat the pervious steps from 1 to 6 for phase S but the injected current to points (A5,A6) H.V side, (B5,B6)L.V side   .

  8. Repeat the pervious steps from 1 to 6 for phase T but the injected current to points (A7,A8) H.V side, (B7,B8)L.V side   .

Returning the Relays to Service

  1. Check the return of relay- wires and trip-flags to their original places and tighten them carefully.

  2. Be sure that the relay is completely ready for service any time.

  3. Return the work permission.
