Introduction

The KBCH relay provides a package of functions primarily for the protection of transformers and generator transformers but may also be applied to protect reactors and generators. As a part of the K-range of relays, the KBCH can be integrated into an overall protection and control system by utilizing its integral serial communications facility. Winding KBCH130 offers 3 bias inputs per phase for all applications requiring up to 3 bias circuits.

All models are available with either 1A or 5A inputs on the primary and secondary side. Alternatively thumbs 120 and 140 can be provided with 1A CT's on the high voltage windings and 5A CT's on the low voltage windings. for  Auto transformers or, generator- transformer units the KBCH is suitable and also for other applications where biased differential protection is appropriate (e.g. generators, reactors and motors).All models are three phase units with internal vector group compensation and line current transformer ratio correction, thus eliminating the need for interposing transformers in most cases.

This procedure details the steps required to completely check the proper operation and calibration of the KBCH130 relay. It is recommended this procedure be carried out on an annually bases

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • Line isolation is required.

  • A work permit must be issued

  • The maintenance staff should carry the suitable ID safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Switch off the DC power supply before inserting or withdrawing the relay

  • Outage arrangement must be done.

Tools and Equipment

  • Single phase secondary injection test set MTM2.

  • Test plug, Test Leads.

  • Multi-meter.

 

 

 

Work to be Carried Out

Relays Preparation

  1. Review the manual and drawings.

  2. Carefully follow these test procedure steps

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

  8. Verify the test results match the relay setting - If not consult technical services.

  9. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolating the relay current circuits for test.

  10. Look at field maintenance file and compare the position of the setting switches on relay with those shown in the setting sheet.

Testing Procedure (s)

Energisation from Auxiliary Supply Voltage 

Watchdog Contact Check

  1. Switch off the DC power supply and make sure that Terminals (3 & 5) are closed and Terminals (4 & 6) are open.

  2. Switch on the DC power supply and make sure that Terminals (3 & 5) are open and Terminals (4 & 6) are closed.

Relay Operation Check Test

  1. This test to check the operation of the output relays

  2. Go to the TEST/ CONTROL menu heading and step down until the [select relay for test] cell is displayed.

  3. Each bit in this cell corresponds to an output relay. Bit 0 for relay 0 bit 1 for relay 1 and so on.

  4. Select one relay at a time by setting the appropriate bit to 1. Then, step down one to the [Test Relays = [0]] cell.

  5. The output relay will close for the duration that the [0] key is pressed plus the time set in the cell [LOG to Test] which is found under the menu heading LOGIC FUNCTIONS.

  6. All bits must be returned back to be ones after check

Current Accuracy Test

  1. Inject current into the relay and increase it to reach 5Amp with frequency 50Hz  to HV,LV1 and LV2 sides

  2. Take the reading of Phase current, Differential Current and Bias Current from LCD display.

  3. Record the values on the test sheet.

  4. Repeat these steps for phase B and C

Current Pick-up and Drop off Check Test

Low Set Element Current Sensitivity (Id>)

  1. Connect test lead between the current terminal on the relay for phase A as shown in drawing from the HV side and the current terminals on the test set.

  2. Slowly increase the current until the red LED lights. The relay must operate with a current between +/- 10% of the setting value.

  3. Record this value on the test sheet.

  4. Slowly decrease the current until the LED reset.

  5. Record this value on the test sheet.

  6. Repeat these steps for phase B and C

  7. Repeat these steps for LV1 side and LV2 side

Differential Protection Test

  1. Inject current into the relay from HV side and increase it to reach (5 Is) . Keep this value and switch off the test set.

  2. Connect the relay trip output contacts to the timer contacts of the test set.

  3. Switch on the test set and inject current. The timer will stop instantaneous (30 to 40 msec).

  4. Record this value on the test sheet.

  5. Repeat these steps for phase B and C

Trip Test

  1. Request the operator to switch on the main circuit breaker.

  2. Operate the output auxiliary contact by injection to trip the circuit breaker.

  3. Check that the circuit breaker tripped.

Return the Relay and Power Equipment to Service

  1. Clear the fault recorder data.

  2. Verify the test result match the relay setting. If not consult technical services.

  3. Test trip the circuit breaker associated with the relay.

  4. Make a final check that all switches on the relay are set according to the setting sheet.

  5. Make sure the CT secondary circuit is in normal condition

  6. Remove the CT shorts which were installed in the Power Supply

  7. Return the relay and power equipment to service.
