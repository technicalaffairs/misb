Introduction

This document outlines the maintenance procedures P1, which must be performed on yearly bases as indicated by Y1 in the title block.

The RXBA  4 operates for single . two and three phase fuse failure in the supervised circuuits and for an interruption in any of the pilot_wires of their associeated fuses.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category.

  • Maintenance staff should be well trained qualified or directly supervised by some one who is .

  • Safety fence with caution marks surrounding the work area is required.

  • Personal protective equipment must be used (safety shoes, helmet).

  • Switch off the DC power supply before inserting or withdrawing the relay.

Tools and Equipment

  • Secondary injection test set

  • Test leads

  • Test plug handle type rtxb 18

Work to be Carried Out

  1. Isart the test handle in test switch.

  2. Connect the relay testing set to terminal 5A phase(R) and 8A (neutral) of the test handle.

  3. Incress the voltage until the relay in the (R_PHASE)pick up and measure the voltage at pick up.

  4. Switch off secandary injechiont test set.

  5. Repeat the mesurment with relay testing est connected terminal 6A phase(S) and 8A (neutral) at PT 1.

  6. Repeat the mesurment with relay testing est connected terminal 7A phase(T) and 8A (neutral) at PT 1.

  7. Record the operation voltage value.

  8. Switch off secondary injection test set.

  9. connect the relay testing set to terminal 9A phase (R) AND 12A (neutral) the test handle at  P.T 2.

  10. Repeat the measurement with relay testing est connected terminal 10A phase (S) and 12A (neutral)   at   P.T 2.

  11. Repeat the measurement with relay testing est connected terminal 11A phase (T) and 12A (neutral)   at   P.T 2.

  12. Switch off secondary injection test set.

  13. Record the operation voltage value.

  14. Pull out the test handle.

  15. Return the relay to the panel.

  16. Return all connections to the normal position as before the test.

  17. Close the work permit.
