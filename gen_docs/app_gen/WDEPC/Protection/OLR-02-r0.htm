Introduction

These procedure details the steps required to completely check the proper operation and calibration of A 32 relay.
It is recommended this procedure be carried out an yearly bases the first preliminary steps to prepare for relay testing describes the activates required prior to testing.
The first draft is imitated  and tested by Eng. Mostafa Hasas,  accepted by general director Eng. Farag Abo Yousef, Approved by Eng. Abd ElMohsen Abd ElKhafar.
This document revised by Eng.Mona Ahmed Elsharkawy  inspection & quality control 's  head sector and MPIS coordinator in WDEP    

Safety Precaution 

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category
  • Wear your safety helmet & safety shoes.
  • Safety fence like ropes with caution marks surrounding the work area .
  • The equipment isolation is required

Tools and Equipment

  • ISA test set (secondary injection voltage)
  • Clamp ampere or Avometer

Work to be Carried out        

Preparation    

  1. Review technical reference and manufacture manual

  2. Obtain a copy of relay setting from field maintenance file and compare it with setting on the relay

  3. Obtain outage for transformer or equipment  protected by this relay.

Test Steps

  1. Connect the current test set terminals to point (1-6) for phase R

  2. Inject the current for phase R then increase the current gradually until the red flag for phase R appears then record IR (pick - up)

  3. Decrease the current gradually until the red flag disappear then record IR (reset)

  4. For phase S connect the current test set to point (2-7) then repeat steps from (1) to (3)

  5. For phase T connect the current test set to point (3-8) then repeat steps from (1) to (3)
