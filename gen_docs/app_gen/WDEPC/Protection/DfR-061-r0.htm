Introduction

This document outlines the maintenance procedures P1 to completely check the proper operation and calibration of the relay, these procedures must be performed on yearly bases as indicated by Y1 in the title block.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initiated by eng. Rasha Soliman, tested by eng  Mohamed Rabee, accepted by general director eng. Hamdy Dorgam, approved by eng. Mohamed Abou Aiasha head sector of Damanhour power station and and certified by eng. Gaber Farahat WDEPC Inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required

  • Personal protective equipment must be used as safety shoes, and safety helmet...etc.

  • Line isolation is required.

Tools and Equipment

  • Secondary injection test set .

  • Clamp ammeter, voltmeter, test leads

  • Complete tool set, test plug

Work to be carried out

  1. Disconnect the circuit breaker for generator.

  2. Connect test leads between the test set and the current terminal (1&15) for phase (R) in the for the Differential relay.

  3. Connect test leads  between the test set and the time terminal (31&32) for phase (R) in the Differential relay. 

  4. Increase the injected current slowly until the red indication on the Differential element phase  (R) on the relay appears.

  5. Record this value (of the current and the time) on the test sheet ( adjust it if needed ).

  6. Switch off the test set.

  7. Repeat the above test from step ( 2 ) to ( 6 ) but inject current between terminal ( 2 &15) for phase ( S ) and terminal ( 3&15 )for phase ( T ) and time terminal (31&32) for phase (S) and terminal (31&32) for phase (T).

Return relay to service:

  1. Disconnect the test leads from timer terminal on the relay .

  2. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  3.      * Breaker trip coil
         * Disturbance alarm and fault recorder associated with this relay.

  4. Repeat  the circuit breaker of the generator to the normal position.
