Introduction

This document outlines the maintenance procedures P1 to completely check the proper operation and calibration of the relay, this procedures must be performed on yearly bases as indicated by Y1 in the title block.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initiated by eng. Rasha Soliman, tested by eng  Mohamed Rabee, accepted by general director eng. Hamdy Dorgam, approved by eng. Mohamed Abou Aiasha head sector of Damanhour power station and and certified by eng. Gaber Farahat WDEPC Inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Line isolation is required.

  • The voltage transformer must be out of service.

Tools and Equipment

  • Secondary injection test set .

  • Voltmeter, clamp ammeter.

  • Test leads, complete tool set.

Work to be carried out

Over current test

  1. Disconnect the circuit breaker .

  2. Remove the OC relay set for phase ( R )

  3. Connect test leads  between the current terminal ( 1&4 ) on the OC relay set

  4. Increase the injected current slowly until the red indication on over-current set appears

  5. Record this value on the test sheet ( adjust it if needed )

  6. Switch off the test set

  7. Repeat the above test from step ( 2 ) to ( 6 ) for phase ( S ) but the terminals (2 -5)

  8. Repeat the above test from step ( 2 ) to ( 6 ) for phase ( T ) but the terminals (3 -6)

Time test     

  1. Rise the current at the pick up of the relay

  2. Turn off the tester

  3. Connect test leads between the timer leads (15 -16) (the relay contact)

  4. Turn on the tester

  5. Record this value in the test sheet

  6. Repeat the steps of timer for phases S and T

 Return relay to service

  1. Disconnect the test leads from the terminal on the OC relay

  2. Put the OC relay set on this place.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration: steps:
             * Breaker trip coil.
             * Breaker failure scheme associated with this relay.
             * Disturbance alarm and fault recorder associated with this relay.
