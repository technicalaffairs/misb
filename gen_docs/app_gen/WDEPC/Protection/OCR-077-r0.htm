Introduction

This document outlines the maintenance procedures P1, which must be performed on yearly bases as indicated by Y1 in the title block.

This relay will function as back up for the transformer differential relay. They have the independent time log characteristic and are provided with an instantaneous function.

The first draft initiated by eng. Ahmad Galal, tested by eng. Mohamed Rabee, accepted by general manager eng. Hamdy Dorgam of Damanhour power station and certified by eng. Gaber Farahat WDEPC inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • The transformer must be was isolated.

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance

Tools and Equipment

  • ISA test set type ART/3.

  • Voltmeter, Ammeter, test plug, and test leads

Work to be Carried Out

Relays Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

Test Steps

  1. Connect the current leads of a tester unit to the relay points according to the drawings and scheme.

  2. Inject current to each couple of points (17-20) on the terminal (B50_X1) should be to PH (R)on the till the red flag of the device displays.

  3. Record the results pick up value at 9A.

  4. Decrease the current until the relay is reset and measure the current at reset 7.5A.

  5. Record the results reset value at 7.5A.

  6. time test in the relay is connect terminal timer to (7_8)on the terminal (B50_X1)  as connect terminal unit current to (17_20)on as inject the current to pick up relay and Record the results time .t=2sec.

  7. Inject current to each couple of points (18-20) on the terminal (B50_X1) shued be to PH (S )on the till the red flag of the device displays.

  8. Record the results pick up value at 9A.

  9. Decrease the current until the relay is reset and measure the current at reset 7.5A.

  10. Record the results reset value at 7.5A.

  11. time test in the relay is connect terminal timer to (7_8)on the terminal (B50_X1)  as connect terminal unit current to (18_20)on as inject the current to pick up relay and Record the results time .t=2sec.

  12. Inject current to each couple of points (19-20) on the terminal (B50_X1) should be to PH (T) on the till the red flag of the device displays.

  13. Record the results pick up value at 9A.

  14. Decrees the current until the relay is reset and measure the current at reset 7.5A.

  15. Record the results reset value at 7.5A.

  16. time test in the relay is connect terminal timer to (7_8)on the terminal (B50_X1)  as connect terminal unit current to (19_20)on as inject the current to pick up relay and Record the results time .t=2sec.

  17. Switch off secondary injection t test set.

  18. Return the relay to the panel.

  19. Return all connections to the normal position as before the test.

  20. Close the work permit
