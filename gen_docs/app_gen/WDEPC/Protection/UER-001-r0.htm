Introduction

This document outlines the maintenance procedures P1, which must be performed on yearly bases as indicated by Y1 in the title block.

The relay comprises a directional current relay RXPE40 which measures the voltage over on resistor unit, which means that the relay has its max sensitivity when the current Irn the the protection leads the voltage reference vector Urn 75%, this means that the directional relay is most sensitive for reactive power feed to the generator.

The first draft initiated by eng. Ahmad Galal, tested by eng. Mohamed Rabee, accepted by general manager eng. Hamdy Dorgam of Damanhour power station and certified by eng. Gaber farahat WDEPC inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • The transformer must be was isolated.

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance.

Tools and Equipment

  • ISA test set type ART/3 or Omicron.

  • Voltmeter, Ammeter, test plug, and test leads.

Work to be Carried Out

Relays Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

 

Test Steps

  1. Connect the current leads of a tester unit to the relay points according to the drawings and scheme.

  2. Insert the test handle in test switch.

  3. The 1st in testing. On element O/C.

  4. Inject current to each couple of points (3A-4A)  in test plug on the element o/c till the red flag of the device displays.

  5. Record the results pick up value at 5.5A.

  6. Decrease the current until the relay is reset and measure the current at reset 5.3A.

  7. Record the results reset value at 5.3A.

  8. The 2sd in testing. On element DIR.

  9. Inject current to each couple of points (3A- 4A)  in test plug on the element direction till the red flag of the device displays. And inject voltage on point (9A_10A) 100v.

  10. Increase current until to pick up value at 2.9A    

  11. Record the results pick up value at 3.9A.

  12. Decrease the current until the relay is reset and measure the current at reset 2.2A.

  13. Record the results reset value at 2.2A.

  14. The 3rd in testing. On element U/V.

  15. Inject voltage to each couple of points (9A-10A)  in test plug on the element u/v till the red flag of the device displays.

  16. Increase voltage 100v.   

  17. Decrease voltage until to pick up value at 90v.   

  18. Increase voltage until to reset value at 93v.

  19. Record the results reset value at 93v.   

  20. Switch off secondary injection t test set.

  21. Return the relay to the panel.

  22. Return all connections to the normal position as before the test.

  23. Close the work permit.
