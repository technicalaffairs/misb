Introduction

The GSCT 12X1 relay with an internal phase angle can be used as earth fault protection in non compensated networks, but it can be also as a reverse power under certain circumstance

Since this relay reacts to the real mean value of A.C. components (voltage & current) with the ferro dynamic element a torque can be produced only if both current & voltage contain the same basic or harmonic frequency

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • The voltage transformer must be out of services

Tools and Equipment

  • Secondary injection test set

  • Test leads, clamp ammeter, voltmeter

  • Complete tool set

  • Make sure that CT secondary circuits are shorted before isolation the current circuits

  • The voltage transformer must be out of services

 

 

Work to be Carried Out

  1. Disconnected the circuit breaker before working

  2. Remove the input current terminals number (1,2)

  3. Remove the input voltage terminals number (4,5)

  4. Connect test leads between the current terminals (1&2) and the voltage terminal (4&5)

  5. Inject constant voltage of 100 V and increase the current slowly until the red indication on the relay appears

  6. Record this value on the test sheet (adjust it if needed)

  7. Switch off the set

Time Test

  1. Connect the time delayed contacts on the relay to timer terminals on the test set

  2. Start the test by injecting the pick up current into the relay

  3. Record the value of time on the test sheet (adjust it if needed)

  4. The timer will stop once the relay has pick up

Trip Test

  1. Put the circuit breaker on the test position

  2. Request the operator to switch on the circuit breaker

  3. Injecting the pick up current into the relay

  4. Check that the circuit breaker tripped

Return Relay to Service

  1. Disconnect the test leads from the terminal board and return the relay terminals (8,9) to normal condition

  2. Disconnect the test leads from timer terminal on the relay and connect the relay time contact to normal condition

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps
    Breaker trip coil
