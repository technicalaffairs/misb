Introduction

These procedure details the steps required to completely check the proper operation and calibration of MRS1 relay . it is recommended this procedure be carried out every four years bases the first preliminary steps to prepare for relay testing describes the activates required prior to testing .

The first draft initiated by eng. Moustafa Osman ,tested and revised by eng.Ali Ghater , accepted by eng. Farag Abo Youseff :general manager, approved by eng. Gomaa Ghater:  head sector of Kafr el Dawar Power Station, and certified by eng. Gaber Mohmed Frahat : WDEPC inspection and quality control general director and MPIS coordinator.

Safety Precautions

  • The equipment isolation is required.

  • A work permit must be issued

  • The maintenance staff should carry the suitable ID safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

Tools and Equipment

  • Secondary current injection test set ( ISA Tester )

  • AVO meter

  • Clamp ampere.

Work to be Carried Out

Preparation

  1. Review technical reference and manufacture manual.

  2. Obtain a copy of relay setting from field maintenance file and compare it with setting on the relay.

  3. Obtain outage for equipment protected by this relay.

Testing Procedures

  1. Connect the current test set terminals to point (B3-B4) for phase R.

  2. Connect the timer test set terminal to point (D1-E1).

  3. Inject the current for phase R then increase the current gradually until the red flag for Negative sequence  appears then record I2 (pick - up) and record the time.

  4.  Decrease the current gradually until the red flag disappear then record I2 (reset).

  5. Repeat steps 1-3 for phase S & T .

Returning the Relays to Service

  1. Return relay to service by removing the plug test

  2. Make sure CT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service.

  4. Return the work permit .
