Introduction

This document outlines the maintenance procedures P1, which must be performed on yearly bases as indicated by Y1 in the title block.

The RADHA relay is of the high impedance restraint type(with a relatively high series impedance in the differential circuit) and has been designed with filter circuits which will attencate the dc component. tests and practical experience show that RADHA with recommended setting does not operate undue for external short-circuit sever for the extreme transient CT saturation.

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initiated by eng. Ahmad Galal, tested by eng. Mohamed Rabee, accepted by general manager eng. Hamdy Dorgam of Damanhour power station and certified by eng. Gaber Farahat WDEPC inspection head sector and MPIS coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required

  • Wear safety and healthy equipments as safety helmet, safety shoes, ... etc

  • Ensure isolation of the equipment which the protection relay will be tested.

  • Make sure that CT secondary circuits are shorted before isolating the current for the injection.

Tools and Equipment

  • Ohmmeter, bridge type

  • Single-phase secondary test.

  • Test plug handle tape rtxh 18.

  • Test plug, test leads

  • Test cabinet.

 

Work to be Carried Out

  1. Start the test handle in test switch.

  2. Connect the relay testing set to terminal 3 L1 phase (R) and 6 (neutral) of the test handle.

  3. Increase the voltage until the relay in the R_PHASE pick up and measure the voltage at pick up.

  4. Switch off secondary test.

  5. Repeat the measurement with relay testing set connect 4_6 L2 phase (S).

  6. Record the operating (differential voltage).

  7. Switch off secondary injection test set.

  8. Repeat the measurement with relay testing set connected terminal 5_6 L3 phase (T).

  9. Record the operating (differential voltage).

  10. Switch off secondary test.

  11. Pull out the test handle.

  12. Return the relay to the panel.

  13. Return all connections to the normal position as before the test.

  14. Place a copy of the test result in the field maintenance file.

  15. Close the work permit.
