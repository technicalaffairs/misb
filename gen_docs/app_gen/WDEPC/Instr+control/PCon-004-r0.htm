Introduction

This document outlines the equipment inspection I1 for unit 5, which must be performed every six months as indicated by M6 in the title block.

Work to be Carried Out

Boiler Circulating Pump Body Temperature

  1. Check the TSHH 101A "temperature switch very high", from thermocouple TE 101A, range: 0 to 100 °C, high threshold 60°C MZ value 0.60, LO (14) = 1(trip + start forbidden)

  2. Check the TSH 101 B "temperature switch high", from thermocouple TE 101B, Range: 0 to 100°C, high treshold 55C°, MZ value 0.55, LO (22) = 1(alarm)

  3. Check the TDSH 102 "differential temp switch (pump suction & pump body", from thermocouple TE 102 A & TE 102C, range: 0 to 400 °C, High threshold 56 °C, MZ value 0.56, LO 17 = (1)(start forbidden), values in micro Z cub by using micro Z consol

  4. Check the FLAPDSLL 001 ""pressure differential switch very low", from PDL 001, range :0 to 3 bar MZ value 0.58 bar, LO (33) = 0(trip)

  5. Check FLAPDSL 001, from PDT 001, range: 0 to 3 bar, MZ value 1.12 bar trip, LO (12) = 1(alarm)

  6. Check PDT 001 "pressure differential transmitter (upper drum, lower drum", differential pressure between suction manifold and lower drum

    0% 0 bar 4 mA
    20% 0.60 bar 7.2 mA
    40% 1.20 bar 10.4mA
    60% 1.80 bar 13.6mA
    80% 2.4 bar 16.8mA
    100% 3 bar 20mA

  7. Ensure that the two hand valves (HVC 001, HVC002) open to obtain pressure equal to (1.9 bar 1.8 bar) at start up.

Signals appear at MZ cub

  • Lo .33 PDSLL 001 A

  • Lo .12 PDSL 001

  • Lo. 034 PDSLL 002 A

  • Lo .13 PDSL 002
