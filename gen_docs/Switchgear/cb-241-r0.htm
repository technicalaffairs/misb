Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category.

  • Maintenance staff should be well trained, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear the personal protective equipment must used as safety shoes, safety helmet ...etc

  • CT must be out of service and isolated from the network

Tools and Equipment

  • Cable set 15m (49 ft)

  • 2*15m (49 ft) 95mm (current cables )

  • 2*15m (49 ft) 2.5mm (sensing cables)

  • The tester MOM 690

  • Power supply                                                                                                   

Work to be Carried Out

  1. Measuring the resistance of a breaker

  2. Make certain the line is de-energized on both side of the breaker .ground the breaker on one side and make certain it is closed

  3. Ground the microhmmeter.

  4. Make certain the microhmmeters ON/ OFF while making connections.

  5. Connect the current cables to the DC+and COM terminals and the sensing cables to the sensing inputs to both sides of the breaker, making sure that the polarities match properly.

    IMPORTANT :
    the sensing cables must be connected inside the current cables.otherwise the test data will be incorrect. See fig.

  6. Switch on theMOM690 .

  7. Select “AUTO”or “MAN” with the <func> button.

  8. Set output current to zero to start the measurement. cables must be connected inside the current cables.

  9. Increase the current to the desired value (600A for example ).

  10. Read the resistance value.

Measuring Resistance at Busbar Joints

  1. Ensure the line is de-energized and the test object is grounded.

  2. Ground the microhmmeter.

  3. Make certain microhmmeters ON/OFF switch is OFF while making connections

  4. Connect the microhmmeters current cables to the test object . do not connect the sensing cables . measurement will be done manually using an external portable voltmeter

  5. Switch on the MOM690

  6. Select  “MAN”  with the  < FUNC > button.

  7. Set output current to zero to start the measurement

  8. Increase the current to the desired value (100A for example ).

  9. Using an external voltmeter , measure the voltage drop across each contact element within section of the busbar being tested . the voltmeter must be set to DC .

  10. Calculate  the actual  resistance.

Example :

If the voltage drop is 0.0067V at a current of 100A the resistance will be 0.0067/100 Ώ i.e67 μΩ
