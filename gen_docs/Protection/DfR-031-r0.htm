Safety Precautions

  • A work permit must be issued, the qualified maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding work area is required.

  • Wear your safety equipment as safety helmet, safety shoes ... etc.

  • Isolate the equipment which its protection relay under test

Tools and Equipment

  • Current injection device with timer.

  • Test plug, test leads

  • Test cabinet.

Work to be Carried Out

  1. Check  the operation of the auxiliary and instantaneous over current unit manually to see that they operate smoothly with out noticeable friction or binds .

  2. Check the contact gap and wipe which should agree with values given under section on  servicing .

  3. The relay have three winding (W1.W2.W3).

  4. Inject the current gradually for each winding.

  5. Record the value of the tripping current for each winding.

  6. Compare the result with the reference value.

  7. Repeat the test for each phase.

Return Relay into Service

  1. Return the relay to the panel.

  2. Withdraw the test plug handle from the relay.

  3. Return all connections to the normal position as before the test.

  4. Place a copy of the test result in the field maintenance file.

  5. Close the work permit.
