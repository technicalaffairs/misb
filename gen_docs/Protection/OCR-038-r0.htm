Introduction

The METI 31+ MCGG 82 relay is designed specially for directional control of the time delayed element of the over current relay type MCGG, and manufactured by GEC.

This procedure details the steps required to completely check the proper operation and calibration of the METI 31+MCGG 82 relay.

It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Safety and healthy equipment must be used (safety Helmet, safety shoes, ...etc).

  • The transformer must be was isolated.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for current injection

Tools and Equipment

  • Test set type MTM2

  • Voltmeter, Ammeter

  • Test leads, Test plug type MMLGO1

 

Work to be Carried Out

Testing the relay and protection outputs

  1. Carefully follow these test procedure steps.

  2. Verify the test result match the relay setting. If not consult technical services.

  3. Test trip the circuit breaker associated with the relay.

Preparations

  1. Make sure CT secondary circuits are shorted at the incoming blocks before isolating the relay current circuits for test

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting sheet

  3. Set the earthing link on the relay to line side position

Over Current Test

  1. Connect test leads between the current terminals no. (22,28) on the test plug. to the current terminals on the test set for phase "R"

  2. Slowly increase the injection current until reach to the operating value and then the yellow "led" for phase R light. The relay must operate with a current between ± 10% of the setting value

  3. Record this value "I pick - up" on the test sheet

  4. Increase the injected current with a value 25% larger than "I pick - up" operating value

  5. Slowly decrease the current until the relay resets

  6. Record this value "I reset" on the test sheet

  7. Switch off the test set

  8. Connect leads between time delayed trip out put contact terminal No.`s (29,30) on the relay to the timer (normally open) on the tester

  9. Inject current with a value 25% larger than the operating value "I operate"

  10. Ensure that the relay operates

  11. Switch off the tester set switch on and make reset for the timer on the tester

  12. Switch on the tester unit. Take the time value and must agrees with the setting value on the relay

  13. Record this value on the test sheet

  14. Repeat the above test from step No 1 to 13 for testing phases S,T but inject current between terminals as follow. For phase S terminals no. (24,28) for phase T terminals no. (26,28) on the test plug

Directional Test

  1. This relay set has a selection of relay characteristic angles (-60º,45º,0º,+30º,+45º)

  2. Ensure that the polarities of all connection are correct according to station drawing

  3. Set the angle on the relay as in setting

  4. Connect test leads on the voltage terminal no. (6) on the test plug for the directional element relay for phase R and tester unit phase R also

  5. Connect leads between the current terminal no. (22,28) on the test plug and tester unit phase R also

  6. Switch on the test set

  7. Inject 100 ac volt from the test set

  8. Use phase rotation indicator to be sure phase rotation R, S and T on the relay.

  9. Inject current into the relay until it operates.

  10. Adjust the phase shift angle between the injected current and voltage to a value equal setting angle.

  11. Observe the output element (yellow led light) on the over current element relay at this moment picking up

  12. Increase the angle on the test set until the relay output element of over current resets

  13. Record this angle value on the test sheet at this moment

  14. Continuous to increase the angle until the relay picks up again record this angle value on the test sheet. At this moment

  15. Repeat the above test for phase S, and T with taking into consideration the connection must be according to drawing

  16. The angles obtained during the above tests shoild be within +6º of the boundary angles quoted in the table 1

Minimum Operating Voltage

  1. Repeat the above test but must start the applied voltage from zero and then slowly increase its value until relay picks up at this moment record this value (voltage) in the test sheet for phase R

  2. Repeat the above step for the other phases (S&T)       

Trip test

  1. Request the operator to switch on the main circuit breaker

  2. Operate the output auxiliary contact by injection to trip the circuit breaker

  3. Check that the circuit breaker tripped.
