Introduction

The protection relay for transformer TPU-2000R is a multifunction relay manufactured by ABB and this relay is digital type. Differential & earth fault functions are utilized

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Block is out of service.

  • Make sure all block test switches are out.

  • Ensure all CT path elements are secure after any test disturbance

Tools and Equipment

  • Doble F6150 test set

  • Laptop with win ECP relay software downloaded

  • Voltmeter test plug and test leads

Work to be Carried Out

Relays preparation

  1. Review the manual and drawings

  2. Prepare the previous or commissioning results of the relay testing

  3. Check the wires of the relay by hand

  4. Check the AC source is available to the tester

  5. Check the existence of the DC sources in panels

  6. Insert the test plug into current test switches and connects it to the tester according to the drawings and scheme

  7. Check all the leads used in tests

Differential tests

Testing the 87 T differential unit minimum pickup

  1. Make the test connections as shown in figure 1 for the 87T phase pairs to be tested.

  2.  Set the primary and secondary currents to 0.07 (W1) & 0.1 (W2) amperes RMS.

  3. Set primary and secondary current source angles to 0 degrees

  4. Apply the currents.

  5. Slowly raise the current from 0.07 (0.1) amperes RMS until the rlay trips. The contact monitor should indicate a closed contact. The "Differential" target along with the phase target should light. This should occur when the secondary current reaches 0.12 (0.24) amperes RMS ± 3 %

Testing the 87 T differential unit with adjustable percent slope setting

  1. Make the test connections as shown in figure 1 for the 87T phase pairs to be tested.

  2. Set the primary and secondary currents to 0.73 (1.5) amperes RMS.

  3. Set the secondary current source angles to 180 degrees

  4. Apply the currents.

  5. Slowly raise the secondary current RMS until the relay trips. The contact monitor should indicate a closed contact. The "Differential" target along with the phase target should light. This should occur when the secondary current reaches 0.73 (1) amperes RMS ± 3 %

Testing the 87 T differential unit with harmonic restraint

  1. Make the test connections as shown in figure 1 for the 87T HARM phase pairs to be tested (both sources synchronized and in parallel)

  2. Set current source to 0.40 (0.80) amperes RMS at 0 degrees 50 hz. Set current source 2 to 0.1 (1) amperes RMS at 0 degrees 100 hz

  3. Apply the currents. The TPU-2000R display should read "Trip restrained" Slowly lower only current source 2 from 0.1 (1.5) amperes RMS until the relay trips. The contact monitor should indicate a closed current source 2 is 0.06 (1.24) amperes RMS

Testing the 87 H differential unit

  1. Make the test connections as shown in figure 1 from the 87 H phase pairs to be tested.

  2. Set the current source 1 to 2.50 (4.8) amperes RMS at 50 hz.

  3. Set current source 2 to 2.50 (4.8) A at 100 Hz Suddenly apply the current for 1 second. The relay should trip. The contact monitor should indicate a closed contact. The "Differential" target along with the phase target light. Go to the fault records and verify that the 87 H unit initiated the last trip.

Restoring the relay into service

  1. Return relay to service by removing the test plugs, and switch on test switches.

  2. Check relay configuration and logic
