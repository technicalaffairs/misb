Introduction

The protection relay for Auxiliary generator GPU-2000R is a multifunction relay manufactured by ABB and this relay is a digital type. differential & phase time over current & phase instantaneous Voltage controlled time & Loss of excitation & over voltage & volts / Hertz & distance / impedance & stator ground fault inadvertent energize & phase unbalance / negative sequence & over frequency fault function are utilized

Safety Precautions

  • A work permit must be issued,

  • The maintenance staff should be trained, qualified, and carry the suitable safety ID category

  • Safety fence with caution marks surrounding the work area is required

  • Block out of service is required

  • Make sure all block test switches are out

  • Ensure all CT path elements are secure after any test disturbance

Tools and Equipment

  • Double F6150 test set

  • Laptop with ECP relay software downloaded

  • Voltmeter, test plug and test leads

Work to be Carried Out

Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester

  5. Check the existence of the DC sources in panels

  6. Insert the test plug into current & voltage test switches and connect it to the tester according to the drawing and scheme

  7. Check all the leads used in tests

Testing the 87M Current Differential Function

  1. Make the connections as shown in figure 1, for the neutral phase A current transformer.

  2. Set the timer to start upon application of current and to stop when the contact monitor senses a closed contact.

  3. Apply the minimum current (0.2)A.

  4. The "pickup" LED on the relay should light and the contact should indicate a closed contact.

  5. The "Differential" LED should also light.

  6. The time for a differential trip should be approximately two (2) cycles

  7. Repeat the above tests for the neutral phase B and the neutral phase C current transformers.

  8. Test the load side of the differential circuit,

  9. Repeat the above tests for the load phase A current transformer using the connections as shown in figure 1

  10. Repeat the tests using the load phase B and the load phase C current transformer.

Testing the 51 P phase time Over Current Function

  1. Make the connection as shown in figure1, for the phase to be tested.

  2. Apply a balance voltage of 53 volts RMS phase to neutral.

  3. Set a single phase of current to 6 amperes RMS.

  4. Set the timer to start upon application of current and to stop when the contact monitor senses a closed contact.

  5. Apply the current to the relay. The "pickup" LED should be lit and the unit should trip in 4.00 seconds ± 7% The "Over current / Impedance" LED should be lit

  6. Repeat the test for the remaining phases listed, as shown in figure1.

Testing the (50P phase instantaneous &51VC voltage controlled time) over Current Function

  1. Make the connections as shown in figure1, for the phase to be tested.

  2. Apply a balance 3-phase voltage of 40 phase voltage of volts RMS phase to neutral.

  3. Set a single phase of current to 6 amperes RMS.

  4. Slowly increase the current and to stop when the contact monitor senses a closed contact

  5. The "pickup" LED should be lit and the "over current" LED should be flashing

Testing the 40 loss of excitation function

  1. Make the connections as shown in figure 1, for the phase to be tested

  2. Apply a balance 3-phase voltage of 57 phase to neutral volts RMS and set a 3-phase of current to 3 ampere RMS with a phase angle of 90" leading

  3. Slowly increase the current and to stop when the contact monitor senses a closed contact

  4. The "pickup" LED should be lit and the "loss of excitation" LED should be flashing

Testing the 67P Phase Directional over current Function

  1. Make the connections as shown in figure1, for the voltage phase to be tested.

  2. Apply a balanced 3-phase voltage of 53 phase to neutral volts RMS and set a single phase of current to 10 ampere RMS in phase with its voltage.

  3. Set the timer to start upon application of current and to stop when the contact monitor senses a closed contact

  4. Apply the current to the relay.

  5.  The "pickup" LED should be lit and the "Over current" LED should be flashing.

  6. The unit should trip in 1.93 seconds ± 7%. The "over current" LED should be it. Change the angle of the tested phase current by 180°.

  7. Again apply the current to the relay.

  8. Verify that the relay does not trip.

  9. Repeat the test for the remaining phases listed, as shown in figure1

Testing the 59 Over Voltage Function

  1. Make the connections as shown in figure2, page 16, for the phase to be tested.

  2. Apply a single-phase voltage of 57 volts RMS to the relay

  3. Slowly increase only the tested phase's voltage from 76 volts RMS.

  4. The "pickup" LED is lit. This should occur when the voltage reaches 75 ± 3% volts RMS. The relay should trip in 1 seconds later.

  5. The contact monitor should indicate a closed contact and the "voltage - volts / Hertz" LED should be lit.

  6. Repeat the above test with a balanced 3-phase voltage

  7. Examine the Operation Records. All of the tests are indicated by a "59 trip" event

Testing the 24 volts/hertz function

  1. Make the connections as shown in figure 2, for the phase to be tested

  2. Apply a balance voltage of 57 volts RMS phase to neutral

  3. Slowly increase the voltage and to stop when the contact monitor senses a closed contact

  4. The "pickup" LED should be lit and the "voltage-volts/hertz" LED should be flashing

Testing the 21-1a distance / impedance function

  1. Make the connections as shown in figure 1, for the phase to be tested

  2. Apply a balance voltage of 10 volts RMS phase to neutral

  3. Slowly increase the current and to stop when the contact monitor senses a closed contact

  4. The "pickup" LED should be lit and the "over current / impedance" LED should be flashing

Testing the 59G stator ground fault function

  1. Make the connections as shown in figure 5, for the phase to be tested

  2. Apply a balance voltage of 57 volts RMS phase to neutral

  3. Apply a volts RMS at 50 hertz representing the Vº

  4. Slowly increase the voltage Vº and to stop when the contact monitor senses a closed contact

  5. The "pickup" LED should be lit and the "stator ground" LED should be flashing

Testing the 27G stator ground 3rd harm under voltage fault function

  1. Make the connections as shown in figure 5, for the phase to be tested

  2. Apply a balance voltage of 57 volts RMS phase to neutral

  3. Apply a volts RMS at 150 hertz representing the Vº

  4. Set a volts Vo.to 4 volt

  5. Slowly decrease the voltage Vo and to stop when the contact monitor senses a closed contact

  6. The "pickup" LED should be lit

Testing the 50IE inadvertent energization function

  1. Make the connections as shown in figure 1, for the phase to be tested

  2. Apply a balance three- phase voltage of 57 volts RMS phase to neutral

  3. Apply a balance three-phase current of 3 amperes RMs at 50 HZ

  4. Change the frequency setting to 46 Hz to the voltage and to stop when the contact monitor senses a closed contact

  5. The "pickup" LED should be lit and the "reverse power" and the "over current / impedance" LED should be flashing

Testing the 46 phase unbalance / negative sequence function

  1. Make the connections as shown in figure 2, for the phase to be tested

  2. Apply a balance three - phase voltage of 57 volts RMS phase to neutral

  3. Set a single - phase current to 0.4 amperes RMS

  4. Slowly increase the current and to stop when the contact monitor senses a closed contact

  5. The "pickup" LED should be lit and the "phase unbalance" LED should be flashing

Testing the 81 over frequency function

  1. Make the connections as shown in figure 2, for the phase to be tested

  2. Apply a balance three - phase voltage of 57 volts RMS phase to neutral at frequency of 50.0Hz

  3. Slowly increase the frequency and to stop when the contact monitor senses a closed contact

  4. The "pickup" LED should be lit and the "frequency" LED should be flashing

Restoring the Relay into Service

  1. Return  relay to service by removing the test plugs, and switch on test switches.

  2. Check relay configuration and logic.
