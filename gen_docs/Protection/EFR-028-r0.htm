Introduction

This document out lines the testing procedure I1, which must be performed every six months as indicated by M6 in the title block.

This procedure details the steps required to completely check the proper operation and testing of the relay

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet...etc.

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Switch off DC power supply before changing setting of the relay.

Tools and Equipment

  • Secondary injection test set,

  • Clamp ammeter, voltmeter, test leads,

  • Complete tool set

 

Work to be Carried out               

Earth Fault Relay Test

  1. Disconnect the circuit breaker for feeder

  2. Remove the input current terminal number (8 &11) from terminal board ( A )

  3. Connect test leads between the current terminal (8&11)

  4. Increase the injected current slowly until the red indication on the instantaneous current (Earth) relay appears

  5. Record this value on the test sheet ( adjust it if needed )

Trip Test

  1. Put the circuit breaker on the test position,

  2. Request the operator to switch on the circuit breaker,

  3. Injecting the pick-up current into the relay,

  4. Check that the circuit breaker tripped

Return Relay to Service

  1. Disconnect the test leads from the terminals board (A),

  2. Connect the input current terminal (8&11) in terminals board (A),

  3. Disconnect the test leads from timer terminal on the relay,

  4. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps

  5. Breaker trip coil,

  6. Breaker failure scheme associated this relay,

  7. Disturbance alarm and fault recorder associated with this relay
