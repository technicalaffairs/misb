Introduction       

This document outlines the testing procedure P1, which must be performed on an annually bases as indicated by Y1 in the above title block .      

The type IFC relays are single - phase time over current relays with long - time characteristics - this relay types include an induction disk time over current unit.

The type IFC 57AD over current manufactured by G.E this procedure details steps required to completely check the proper operation and calibration the IFC 57AD relay .

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • Use safety and healthy equipment as safety helmet and safety shoes ... etc

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuit for current injection

Tools and Equipment

  • Test plug handle, ammeter.

  • Secondary injection test set model SR – 51A - E.

Works to be Carried Out

Preparation

  1. Make Sure CT secondary circuit are opened at the incoming terminal blocks before isolating the relay voltage circuit for test

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting sheet .

Testing the Relay

  1. Connect test leads between the current terminals number (5 and 6) on the relay to the current terminals on the test set as shown in fig. (1).

  2. Slowly increase the current until the contacts close the relay must operate with a current between +10% of the setting value.

  3. Record this value on the test sheet.

  4. Increase the injected current with a value 25% greater than the operating value.

  5. Slowly decrease the current until the relay resets.

  6. Record this value on the test sheet.

 Time Test

  1. Connect the time delayed contacts numbers ( 1 and 2 ) on the relay to timer terminals on the test set .

  2. Start the test by injecting current into the relay. the timer will stop once the relay has picked up .

  3. Record this value of time on the test sheet .

  4. Repeat the above test steps for other phases .

The Steps for Putting the Relay Into Operation

  1. Disconnect the test leads from the terminals ( 5 – 6 ) .

  2. Disconnect the test leads from the trip contact ( 1 – 2 ) .

  3. Make sure that the CT secondary circuits are in normal condition.
