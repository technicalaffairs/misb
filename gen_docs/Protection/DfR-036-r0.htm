Introduction

This document outlines the equipment inspection I1, to be carried out on an annually bases as indicated by Y1 in above title block.

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initiated by eng, Mohamed Osman and tested by eng Mahmoud Yousaf reversed by EDEPC quality control  eng. Adel Abdel  Qader El-Naggar at accepted by general director eng. Sawsan El Lethy.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet...etc.

  • Line isolation is required.

Tools and Equipment

  • Secondary injection test set,

  • Clamp ammeter, voltmeter, test leads,

  • Complete tool set, Test plug

Work to be Carried out               

  1.  Disconnect the circuit breaker for generator.

  2. Removed the plug of connection for Differential relay.

  3. Put the test plug of  connection in Differential relay.
     

  4. Connect test leads between the test set and the current terminal (5&6) for phase ( R ) in the test plug  for the Differential relay.

  5. Connect test leads  between the test set and the time terminal (1&2) for phase(R) in the test plug for the Differential relay.

  6. Increase the injected current slowly until the red indication on the Differential element phase  ( R ) on the relay appears.

  7. Record this value (of the current and the time) on the test sheet ( adjust it if needed ).

  8. Switch off the test set.

  9. Repeat the above test from step ( 4 ) to ( 7 ) but inject current between terminal ( 5  &6) for phase ( S ) and terminal ( 5&6 ) for phase ( T ) and time terminal (1&2) for phase (S) and terminal (1&2) for phase (T).

Return Relay to Service

  1. Disconnect the test leads from timer terminal on the relay.

  2. Remove the test plug from the relay.

  3. Put the plug of connection for High-Speed Differential relay.

  4.  If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  • Breaker trip coil

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.

  1. Repeat  the circuit breaker of the generator to the normal position.
