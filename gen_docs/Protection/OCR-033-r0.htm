Introduction

This document outlines the testing procedure P1, which must be performed every six months as indicated by M6 in the title block

This procedure details the steps required to completely check the proper operation and calibration of the relay. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initiated by eng  Sawsan El-Lethy, tested by eng. Wael Taha, reversed by EDEPC quality control eng. Adel Abdel Kader El-Naggar, accepted by general director eng. Mohamed Goda El Said, certified by eng. Aly Mostafa Al-Lethy EDEPC inspection & quality control head sector.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Line isolation is required.

  • The voltage transformer must be out of service.

Tools and Equipment

  • Secondary injection test set.

  • Voltmeter, clamp ammeter.

  • Test leads, complete tool set.

    •

Work to be Carried Out

 

Over current test

  1. Disconnect the circuit breaker for feeder,

  2. Removed the OC relay set for phase ( R ),

  3. Connect test leads  between the current terminal ( 1&2 ) on the OC relay set,

  4. Increase the injected current slowly until the red indication on over-current set appears,

  5. Record this value on the test sheet ( adjust it if needed ),

  6. Switch off the test set,

  7. Repeat the above test from step ( 2 ) to ( 6 ) for phase ( S ) and phase ( T ).

Time test     

  1. Connect the instantaneous trip contact numbers ( 7&8 ) on the relay to timer terminal on the test set, figure 1

  2. Inject the pick-up current value into the relay for phase ( R ),

  3. Record the time (adjust it if needed),

  4. Connect the time delayed contact numbers ( 2&3 ) on the relay to timer terminals on the test set,

  5. Injecting the pick-up current value into the relay for phase ( R ),

  6. Record the value of time on the test sheet ( adjust it if needed ),

  7. The timer will stop once the relay has picked-up,

  8. Repeat the above test from step ( 1&4 ) for phases ( S ) & ( T ).

Trip test

  1. Put the circuit breaker on the test position,

  2. Request the operator to switch on the circuit breaker,

  3. Injecting the pick-up current into the relay,

  4. Check that the circuit breaker tripped.

 Return relay to service

  1. Disconnect the test leads from the terminal on the OC relay

  2. Put the OC relay set on this place.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration: steps:
             - Breaker trip coil.
             - Breaker failure scheme associated with this relay.
             - Disturbance alarm and fault recorder associated with this relay.
