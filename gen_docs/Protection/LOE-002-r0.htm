Introduction

This document out lines the testing procedure P1, which must performed on an annually bases as indicated by Y1 in the above title block.

The CEH51A  relay is a single phase off set mho relay and its used to detect a loss of excitation on synchronous machines. It is recommended that loss of excitation protection be considered for all synchronous generators .

Safety Precautions

  • A work permit must be issued and the maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • The generator must be out of service

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance.

Tools and Equipment

  •  Secondary injection test set with phase shifter 

  •  Multimeter,   Test lead   

Work to be carried out

  1. Make sure CT secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test

  2. Obtain copy of relay setting from  maintenance file.

  3. Connect test leads between the current terminals (5 - 6) and (3 - 4) on the relay fig (1).

  4. Connect test leads between the voltage terminals  (7 - 8) on the relay  fig (1)

  5. Connect test leads between the trip terminals (1 -2)on the relay fig (1)

  6. Increase the voltage to 120 VAC

  7. Increase the current slowly until the contact just closes .

  8. Turn phase shifter and determine the two angle between which the contacts of the relay are closed

  9. Record this value in the test sheet.

Return Relay and associated Equipment to Service

  1. Disconnect the test leads from the current terminals ( 5-6 ) and ( 3-4)

  2. Disconnect the test leads between trip contact terminals  (1 - 2)

  3. Disconnect the test leads between  voltage  terminals (7 - 8)

  4. Check that the CT secondary connection return to its normal position as before the test.

  5. If any equipment associated with the relay had been isolated it should be returned to the service.
