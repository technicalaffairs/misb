Introduction

SHNB 102 MICROMHO is a static distance protection scheme, specially designed for comprehensive, ultra high speed protection of HV and EHV power transmission systems. Three zones of protection are included, each employing separate measuring elements covering each type of phase and earth fault. When used with a suitable signaling channel, MICROMHO will ensure very rapid clearance of any type of fault within the complete transmission line section.

A selection of alternative tripping schemes, in-built as standard, provides great versatility of operation, simply by the flick of a switch. MICROMHO is the ideal choice for the first main protection system complemented by both the OPTIMHO relay, described in publication R4056 and the QUADRAMHO relay, described in publication R5580. This relay has been designed for system voltages up to 220kV and as a second main or back-up protection for higher voltage systems.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Line isolation is required.

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance.

    •

Tools and Equipment

  • ISA test set type ART/3 or Omicron

  • Voltmeter,  Ammeter

  • Test plug test leads

Work to be carried out

Relays Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

Test Steps

  1. Connect the current and voltage leads of a tester unit to the relay points according to the drawings and scheme.

  2. Inject current and voltage  to each couple of points till the red flag of the device displays.

  3. Repeat the steps for each pair of points according to the test sheet

  4. Record the results.

    5.

Testing the Relay and Protection Outputs

  1. Carefully follow these test procedure steps 

  2. Verify the test result match the relay setting. If not consult technical services.

  3. Test trip the circuit breaker associated with the relay.

Trip Test

  1. Request the operator to switch on the main circuit breaker.

  2. Operate the output auxiliary contact by injection to trip the circuit breaker.

  3. Check that the circuit breaker tripped.

Return the Relay to Service

  1. Return the relay to service after check that green led light, 2nd red led not light

    2.

  2. Return the line to service after check that every thing is normal
