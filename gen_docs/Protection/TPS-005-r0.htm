Introduction

The protection relay for transformer is manufactured by ABB and this relay is digital type.

The first draft initiated by eng. Abdel Raheem Salem Mohamed Ezz, tested by eng. Khaled El Etriby at 3 April 2004, reversed by EDEPC quality control eng. Adel Abdel Kader El-Naggar, accepted by general director eng. Atef Farh, and certified by eng. Aly Mostafa Al-Lethy EDEPC inspection & quality control head sector.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Transformer must be out of service.

  • Make sure CT secondary circuits are shorted.

  • Voltage transformer miniature circuit breakers (m.c.b) is opened

Tools and Equipment

  • Siemens test set two units:

  • Current unit type 7VP48, voltage unit type 7VP49.

  • Voltmeter, ammeter, test leads, and screw driver .

Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing 

  1. Review technical reference material manufactures or manual.

  2. Obtain copy of relay setting from field maintenance file.

  3. Obtain outage for unit protected by this relay

  4. Check that the bus and unit disconnect switches are opened and tagged.

  5. Inspect the secondary circuits of CT, PT miniature circuit breakers.

  6. MCB and all cabling supplying the relay.

  7. Block any disturbance alarms or fault recorders associated with this protection.

Preparation of Test

  1. Place the two cases one on the top of other (the voltage unit on the top of current unit)

  2. Connect the two units together by the cable connector 7 XV6201 between test plug "X2" in current unit and test plug "X1" in voltage unit.

  3. Connect the power supply cable 7XV6200 (3*380 V) between the plug "X1" in the current unit and the source (careful the positive sequence).

  4. Connect an ammeter with the socket "17 ,18" in current unit and voltmeter with the socket "5,6" in voltage unit.

Test Procedure for transformer Protection Relay RET 521

  1. To inter test mode

  2. Select the (test – test mode – operation – on), then the relay is in test mode and the yellow lamp is flashing .

Differential Protection

Block the other function earth fault and restricted earth fault which feed from the same current transformer as differential protection ,select the (test-test mode-block function E/F is on &RE/F is on).

HV SIDE

  1. Connect the current leads from (L1&N) on the tester to (L1&N) on the relay.

  2. Increase the current to the relay until the protection function operate ,record this value (pick up value ) and make sure the trip is O.K.

  3. Decrease the current to the relay until the protection function released ,record this value (drop out value ) 

  4. Repeat the previous steps (1,2&3)for phases L2 & L3 .

  5. Connect the current leads (  L1,L2 , L3 & N ) on the tester to the relay .

  6. inject the three phases current and Increase it until the protection function operate ,record this value (pick up value ) and make sure the trip is O.K.

  7. Connect the timer leads from the tester to free contact on the relay.

  8. Inject current twice the pick up value and determine the operate time.

  9. Close the CBs where the isolators are opened and inject the current to the relay to trip the CBs.

LV SIDE

  1. Repeat the previous steps from 1-8 for LV side .

  2. Remove the block function   earth fault and restricted earth fault which feed from the same current transformer as differential protection ,select the (test-test mode-block function E/F is off &RE/F is off). Then the relay is ready.

Restricted Earth Fault Protection (High Voltage Side)

  1. Block the other function earth fault and diff. protection, which feed from the same current transformer as restricted earth fault protection, select the (test-test mode-block function E/F is on & diff. is on).

  2. Connect the current leads from (L1&N) on the tester to  the relay terminals which is connected to the C.T. on H.V. neutral side  .

  3. Increase the current to the relay until the protection function operate ,record this value (pick up value ) and make sure the trip is O.K.

  4. Decrease the current to the relay until the protection function released ,record this value (drop out value ) 

  5. Connect the timer leads from the tester to free contact on the relay.

  6. Inject current twice the pick up value and determine  the operate time .

  7. Close the CB's where the isolators is opened and inject the current to the relay to  trip  the C.Bs.

  8. Remove block of other functions earth fault and diff. prot. which feed from the summate , record this value (pick up value ) and make sure the trip is OK

  9. Decrease the current to the relay until the protection function released ,record this value (drop out value ) 

  10. Connect the timer leads from the tester to free contact on the relay.

  11. Inject current twice the pick up value and determine  the operate time .

  12. Close the CB's where the isolators is opened and inject the current to the relay to trip the CB's.

  13. Remove block of other functions restricted earth fault and diff. prot. which feed from the same current transformer as earth fault protection ,select the (test-test mode-block function RE/F is off & diff. is off). then the relay is ready.

Over current Time Protection

  1. Block the other function restricted  earth fault and diff. prot. which feed from the same current transformer as   over current protection ,select the (test-test mode-block function R E/F is on & diff. is on).

  2. Connect the current leads from (L1, L2, L3, N) on the tester to (L1, L2, L3, N) on the relay.

  3. Select (L1&N) on the tester to inject the current for phase L1, increase the current phase L1 to the relay until the protection function operate ,record this value (pick up value ) and make sure the trip is O.K.

  4. Decrease the current to the relay until the protection function released; record this value (drop out value).

  5. Connect the timer leads from the tester to free contact on the relay.

  6. Inject current twice the pick up value and determine the operate time.

  7. Close the CBs where the isolators is opened and inject the current to the relay to  trip  the CBs.

  8. Repeat the previous steps (2&3) for phases L2 & L3.

  9. Remove the block function differential protection and restricted earth fault which feed from the same current transformer as over current time protection ,select the (test-test mode-block function Diff. is off &RE/F is off). Then the relay is ready.

Restricted Earth Fault Protection ( Low Voltage Side)

  1. Block the other function earth fault and diff. prot. which feed from the same current transformer as restricted earth fault protection ,select the (test-test mode-block function E/F is on & diff. is on).

  2. Connect the current leads from (L1&N) on the tester to  the relay terminals which is connected to the C.T. on L.V. neutral side  .

  3. Increase the current to the relay until the protection function operate, record this value (pick up value) and make sure the trip is O.K.

  4. Decrease the current to the relay until the protection function released; record this value (drop out value).

  5. Connect the timer leads from the tester to free contact on the relay.

  6. Inject current twice the pick up value and determine the operate time.

  7. Close the CB's where the isolators is opened and inject the current to the relay to trip  the CBs.

  8. Remove block of other functions earth fault and diff. protection, which feed from the same current transformer as restricted earth fault protection, select the (test-test mode-block function E/F is off & diff. is off). then the relay is ready 

Earth Fault Protection (Low Voltage Side)

  1. Block the other function restricted  earth fault and diff. prot. which feed from the same current transformer as   earth fault protection ,select the (test-test mode-block function R E/F is on & diff. is on).

  2. Connect the current leads from (L1&N) on the tester to  the relay terminals which is connected to the CT on LV neutral side.

  3. Increase the current to the relay until the protection function operate, record this value (pick up value ) and make sure the trip is OK.

  4. Decrease the current to the relay until the protection function released, record this value (drop out value ) 

  5. Connect the timer leads from the tester to free contact on the relay.

  6. Inject current twice the pick up value and determine the operate time.

  7. Close the CB's where the isolators are opened and inject the current to the relay to trip the CB's.

  8. remove block of other functions restricted earth fault and diff. protection, which feed from the same current transformer as earth fault protection, select the (test-test mode-block function RE/F is off & diff. is off).

  9. To exit from test mode select (test – test mode – operation – off ) the yellow lamp disappeared, then the relay is ready for operation .

  10. Directional over current prot. 66 kv side

  11. Block the differential protection ,select the (test- test mode-block function diff. is on).

HV SIDE

  1. Disconnect the m.c.b for PT secondary circuits.

  2. Connect the voltage leads from (L1,L2,L3&N) on the tester to (L1,L2,L3&N) on the relay.

  3.  Connect the current leads from (L1&N) on the tester to (L1&N) on the relay.

  4. Inject 3-ph voltages 100 v.

  5. Increase the current to the relay to about 0.5A.

  6. Make the current lag the voltage by 90˚.

  7. Increase the current to the relay until the protection function operate, record this value (pick up value) and make sure the trip is OK.

  8. Decrease the current to the relay until the protection function released, record this value (drop out value).

  9. For p.u. value move the angle until the relay is released and move to back again untie the relay p.u. record this interring value of c/c and move in this direction until the relay is released and move back until the relay is p.u. record this value the other side of c/c .

  10. Repeat the previous steps (3,4,5,6,7&8) for phases L2 & L3 .

  11. Connect the timer leads from the tester to free contact on the relay.

  12. Inject current twice the pick up value and determine the operate time.

  13. Close the CBs where the isolators is opened and inject the current to the relay to  trip  the CBs.
