Introduction

This document out lines the testing procedure I1, which must be performed every six months as indicated by M6 in the title block.

This procedure details the steps required to completely check the proper operation and testing of the relay.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Switch off DC power supply before changing setting of the relay.

Tools and Equipment

  • Secondary injection test set,

  • Clamp ammeter, voltmeter, Test leads,

  • Complete tool set

    •

 

Work to be Carried out

Earth Fault Relay Test

  1. Disconnect the circuit breaker for feeder

  2. Remove the earth fault over current relay 

  3. Connect test leads between the current terminal (1&2) on the earth fault over current relay set

  4. Increase the injected current slowly until the red indication on the  earth fault over current relay set appears

  5. Record this value on the test sheet ( adjust it if needed )

Time Test

  1. Connect the trip terminal contacts No. ( 5&6 ) on the relay to timer terminal on the test set ( FIG 1) ,

  2. Injecting the pick-up current into the relay ,

  3. Record the value of time on the test sheet ( adjust it if needed )

Return Relay to Service

  1. Disconnect the test leads from the terminal on the earth fault over current relay 

  2. Put the earth fault over current relay set on this places

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:

  • Breaker trip coil,

  • Breaker failure scheme associated this relay,

  • Disturbance alarm and fault recorder associated with this relay
