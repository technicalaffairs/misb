Introduction

The 7UM511 is a numerical machine protection unit from the "Numerical Machine Protection seriesUM51" and provides a practical combination of the most important generator protection functions.

It provides comprehensive and safe protection for smaller generators such as emergency diesel generator sets, run of the river hydro plants, or private power stations Furthermore, and the scope of functions of the relay allows using it as network splitting device additionally, the unit forms the basis for the protection of larger generators. By adding other units of the 7UM51 series, all protection requirements encountered for the smallest to the largest machines can be met.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • The unit out of service and work permit required.

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance.

    •

Tools and Equipment

  • ISA test set type ART/3 or Omicron

  • Voltmeter, Ammeter

  • Test plug test leads

 

Work to be carried out

    Relays Preparation

    1. Review the manual and drawings

    2. Prepare the previous or commissioning results of the relay testing.

    3. Check the wires of the relay by hand.

    4. Check the AC source is available to the tester.

    5. Check the existence of the DC sources in panels.

    6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

    7. Check all the leads used in tests.

Test Steps

  1. Connect the current leads of a tester unit to the relay points according to the drawings and scheme.

  2. Inject current and voltage to each  protection function  according to scheme till the red flag of the device displays.

  3. Repeat the steps for each pair of points according to the scheme and test sheet.

  4. Record the results.

Testing the Relay and Protection Outputs

  1. Carefully follow these test procedure steps 

  2. Verify the test result match the relay setting. If not consult technical services.

  3. Test trip the circuit breaker associated with the relay.

Trip Test

  1. Request the operator to switch on the main circuit breaker.

  2. Operate the output auxiliary contact by injection to trip the circuit breaker.

  3. Check that the circuit breaker tripped.
