Introduction

This document out lines the testing procedure I1.

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.
 The first draft initiated by eng Sawsan  El_Lethy, tested by eng Wael Taha, revised by quality control  eng. Adel Abd El_Kader El_Naggar at EDEP Co., accepted by general director eng. Mohamed Goda Elsaid

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Wear safety and healthy equipments as safety helmet, safety shoes ..etc

  • The voltage transformer must be out of service

Tools and Equipment

  • Secondary injection test set
  • Test leads, Clamp ammeter, voltmeter
  • Complete tool set

Works to be Carried Out

 Over Current Test

  1. Disconnect the circuit breaker for feeder

  2. Removed the plug of connection for Earth fault relay.

  3. Remove the input current terminal number (8,9,10&11)from terminal board ( A )

  4. Connect test leads between the current terminal (8&11) for phase ( R )

  5. Increase the injected current slowly until the red indication on over-current element phase ( R ) on the relay appears

  6. Record this value on the test sheet ( adjust it if needed )

  7. Switch off the test set

  8. Repeat the above test from step ( 4 ) to ( 7 ) but inject current between terminal    ( 9&11 ) for phase ( S ) and terminal ( 10&11 )for phase ( T ) on the terminal board ( A )

Time Test

  1. Connect the time delayed contact numbers ( 1&2 ) on the relay to timer terminals on the test set ,

  2. Start the test by injecting the pick-up current into the relay for phase ( R )

  3. Record the value of time on the test  sheet ( adjust it if needed )

  4. The timer will stop once the relay has picked-up

  5. Repeat the above test from step ( 1&4 ) for phase ( S ) & ( T )

Trip Test

  1. Put the circuit breaker on the test  position ,

  2. Request the operator to switch on the circuit breaker ,

  3. Injecting the pick-up current into the relay ,

  4. Check that the circuit breaker tripped

Return Relay to Service

  1. Disconnect the test leads from the terminals board ( A ).

  2. Put the plug of connection for Earth fault relay.

  3. Disconnect the test leads from timer terminal on the relay.

  4. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps.
         - Breaker trip coil,
         - Breaker failure scheme associated this relay,
         - Disturbance alarm and fault recorder associated with this relay
