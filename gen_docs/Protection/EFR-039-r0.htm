Introduction

These procedure details the steps required to completely checking th proper operation and testing of the earth fault relay type (RSF 5)

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

    •

  • Line isolation and work permit required

  • Reviewing the relay test sheet to know th required settings

Tools and Equipment

  • ABB test set type SVERKER 650

  • Clip - on ammeter type KEW snap 2002

  • Test leads 

Work to be Carried Out

Preparation

  1. This procedure should be used for performing off load line

  2. Look at the MRB panel nameplate and check this MRB system designation nameplate agrees with the line that has been taken out of service for maintenance

  3. Perform one of the following measuring checks to verify that there is not current following through this relay and the testing will occur on the correct relay

  4. Using clip - on ammeter checks that there is not current flowing in the RSF 5 relay circuit

  5. Check all connections leads and wiring to the relay are in a good condition

  6. attach all connections between the relay and the test set

  7. Supply A.C power to the SVERKER 650 test set

Testing the function operation

  1. Connect the test leads between the test set and the earth fault relay

  2. Connect the D.C supply (b10) of the line under test

  3. Measure th A.C injection current using clip-on ammeter type KEW. SNAP 2002

  4. Increase the current injected until the relay operates

  5. Make reset to the relay

  6. Tripping test of the line circuit breaker performed by

  •  Connect the C.B of the line under test

  • Adjust the current tripping value on the test set

  • Turn on the test set switch until the relay operates and the C.B tripped

  1. Make reset to the relay

Return the relay to service

  1. Return relay to service by removing the test leads from the relay terminal and the test set

  2. Make sure that the relay terminals are in normal condition

  3. Make sure that any equipment associated with the relay that isolated during tests restored to service

  4. Make sure that there is no signals appeared on the signals panel

Power line to b returned in service

  1. Take crossed reading of all AC currents and voltages and compare them to those for normal load condition

  2. Keep a copy of the test results in the field maintenance file
