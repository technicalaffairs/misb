Introduction

    The MCGG 62 is single, two, three or four pole over current relay, with separate measuring boards for each phase or earth fault input. Each pole of the relay provides a choice of four IDMT characteristics and three definite time characteristics. Additionally, each pole is provided with an instantaneous high-set element which can be disabled if not required.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category.

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • Safety and Healthy equipment must be used (safety Helmet, safety shoes, ...etc).

    •

  • Line isolation is required.

  • Make sure CT Secondary circuits are shorted.

  • Ensure all CT path elements are secure after any test disturbance.

Tools and Equipment

  • ISA test set type ART/3

  • Voltmeter, Ammeter, Test leads

                        Work to be carried out

Relays Preparation

  1. Review the manual and drawings.

  2. Prepare the previous or commissioning results of the relay testing.

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the trip leads of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

Test Steps

  1. Connect the current and voltage leads of a tester unit to the relay points according to the drawings and scheme.

  2. Obtain copy of relay's setting specification from field maintenance file

  3. Look at field maintenance file and compare the data in liquid crystal display LCD of relay by show the operational parameters and setting with setting sheet and data.

  4. Check that the cables are in good condition and that all wire connections are tight and clean.

  5. Block protection signals to  Breaker trip coils and transfer trip scheme's

  6. Block any disturbance alarms or fault recorders associated with this protection

  7. If not required for end to end tests, open the relay power line carrier (PLC) links

  8. Turn off the associated auto-enclosure scheme.

  9. Go to the switchgear and inspect the CT''s, the VT miniature circuit breakers (MCB's), and all cabling supplying the relay:

  10. Perform function test on relay scheme to ensure all associated auxiliary relays are working properly.

  11. Close CB and inject current more than pickup value the CB should be trip and red the led light.

  12. Test trip all breakers associated with this protection scheme.

  13. Inject current  to each couple of points till the red flag of the device displays.

  14. Repeat the steps for each pair of points according to the test sheet.

  15. Record the results.

  16. Verify the test results match the relay setting specifications previously obtained from field  maintenance rile. If not consult Technical  Services/Central protection

Testing the Relay and Protection Outputs

  1. Carefully follow these test procedure steps 

  2. Verify the test result match the relay setting. If not consult technical services.

  3. Test trip the circuit breaker associated with the relay.

Trip Test

  1. Request the operator to switch on the main circuit breaker.

  2. Operate the output auxiliary contact by injection to trip the circuit breaker.

  3. Check that the circuit breaker tripped.
