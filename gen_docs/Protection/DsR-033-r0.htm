Introduction

This document outlines the testing procedures P1, to be performed on an annually bases as indicated by Y1 in the title block.

The Micromho static distance protection scheme is designed for high speed phase and earth fault protection for high voltage and EHV over head transmission lines. Three zones of protection are provided, each of individual Settings, zones 1 and 2 having polarized mho characteristics and zone 3 having offset mho characteristic.

Safety Precautions

  • Work permit must be issued the maintenance staff should be qualified and carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required.

  • Wear the personal protective equipment as (safety shoes, Helmet).

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Switch off DC power supply before inserting or withdrawing any card from the relay.

Tools and Equipment

  •  XS92a test set ( 2 units )

  • Test Plugs 1V and 2V, Test leads, Screwdriver

Work to be Carried Out

 

Preliminary Steps to Prepare for Relay Testing

  1. This procedure is used to perform off load relay test.

  2. Review technical reference material: Manufacturer's manual.

  3. Obtain copy of relay setting from field maintenance file.

  4. Obtain outage for line protected by this relay.

  5. Check that the bus and line disconnect switches are opened and tagged.

  6. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB'S), and all cabling supplying the relay.

  7. Use station drawing to isolate:

    • Breaker trip coils and transfer trip scheme.

    • Breaker failure scheme associated with this relay.

    • Disturbance alarms and fault recorder associated with this relay.

    • Auto-reclose associated with this relay.

Testing the Relay and Protection Outputs

  1. Carefully follow these test procedure steps.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Test trip all breakers associated with the relay.

Prior to Starting Testing Procedure Ensure the Following Steps

  1. Look at the EEHC panel nameplate and check that this EEHC system designation nameplate agrees with the line that has been taken out of service for maintenance.

  2. Check there is no AC current from CT's by using a clip on ammeter.

  3. Check there is no AC voltage from VT's by using a voltmeter.

Preparation

  1. Connect the XS92a test set.

  2. Connect test plug 1V as shown in figure (a) and test plug 2V as shown in figure (b).

  3. Ensure that the CT secondary is correctly short circuited on the case side of test plug 1V

  4. These connections must be made before the test plug is inserted.

  5. Connect the current terminals IA, IB, IC and IN on the test plug to the corresponding current terminals on the test set.

  6. Connect the voltage terminals VA, VB, VC and VN on the test plug to the corresponding voltage terminals on the test set.

  7. Remove relay isolating plugs 1V and 2V.

  8. Insert the test plug 1V and 2V in the relay.

Measuring the Operating Times

  1.  The operating Time of a distance relay should be measured using the tripping contact (free contact).

  2.  Connected The free contact to (+) and ( __\__ ) in the front panel of test set XS 92a .

  3.  Enter the all parameters of zone by ZL = 0.5 ZI

  4. When the trip occurs the value is automatically stopped by free contact then read the value of the time of zone I in the terminal

  5. Repeat the above step to measuring the time of zone II , III .

Testing the Phase to phase Measuring Elements

  1. Repeat all Phase to ground tests but with different fault codes as follows:-
                    Fault code  Phase R-S               121           Forward direction
                                       Phase S-T              122           Forward direction
                                       Phase T-R              123           Forward direction

    In the reverse direction  (021), (022), (023) (Block)

Test Trip all Breakers Associated with the Relay

  1. Reconnect Breaker trip coils.

  2. Close only the circuit breakers associated with the relay.

  3. Adjust the injection parameters to make trip.

  4. Start injection and make sure all breakers are tripped.

Restore the Relay to Service

  1. Unplug the test set XS92a from the power source.

  2. Remove the test leads from the relay and the test equipment.

  3. Remove the test plugs 1V and 2V from the relay.

  4. Return the relay to the permanent settings as indicated in the maintenance file.

  5. This Completes the Test Procedure.

Return Relay and Associated Equipment to Service

  1. Return relay to service by inserting the relay plugs1V and 2V to the relay.

  2. If any equipment associated with the relay has been isolated without the test plug, it should be returned to service by the following restoration steps:

  • Breaker trip coils and transfer trip scheme.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

  • Auto-reclosure associated with this relay.
