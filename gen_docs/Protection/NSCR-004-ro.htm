Introduction

Negative sequence protection relay trips when N,S,C12 equal to setting value. It is consists of 3 coils for three phases A,B,C. Maintenance and testing done annually

The first draft initiated by eng. Abd ElHafez Farghaly, tested by eng. Hamdy Hsaneen, accepted by eng. Sleem Ahmed Aly; general manager, approved by eng. Abd El Kerim Abu Zeid Ismail Head Sector of High Dam Hydraulic Power Station.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Isolate DC supply to relay

  • Switch off test - switches for DZW : YPOB for line

  • Switch off test - switches for NECATIVE SEQUANCE protection

Tools and Equipment

  • Test set type K 501

  • Ammeter

  • Voltmeter

  • Timer

  • Test leads

 

 

Work to be Carried Out

  1. Prepare test set to inject current suitable for setting relays value (from 1 ma: 500ma)

  2. Raise current gradually until relay PTO TRIP

  3. Make sure Aux. relay 4Pn energize

  4. Record the measurement values

  5. Raise current gradually until relay PTO trip

  6. Make sure Aux. relay 4Pn energize

  7. Measure delay time

  8. Record the measurement values

  9. Switch on DC supply to relays

  10. Switch on C.B

  11. Perform actual test for switching off C.B when PTO protection relays trips

  12. Make sure all alarms shown in DCS

  13. Return relay and associated equipment to service

  14. Make sure CT secondary circuits are in normal condition
