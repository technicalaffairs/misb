Safety Precautions

  • A work permit must be issued, the qualified maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding work area is required.

  • Wear your safety equipment as safety helmet, safety shoes ... etc.

  • Equipment isolation which its protection relay under test.

Tools and Equipment

  • Current injection device with timer.

  • Test plug, test leads

  • Test cabinet.

Work to be Carried Out

  1. Enter the tested relay in the tested cabinet.

  2. I1 and I2 refers to restrain circuit, I0 refers to  the differential current.

  3. The direction of current through the restraint circuit should be reversed.

  4. The target  operation should be checked by passing 85 percent of rated current through the contact circuit.

  5. The relay have three winding (W1.W2.W3)

  6. Inject the current gradually for each winding.

  7. Record the result of tripping current value for each winding.

  8. Compare result reference.

  9. Repeat the test for each phase ( see figure1 ).

Return Relay into Service

  1. Return the relay to the panel.

  2. Withdraw the test plug handle from the relay.

  3. Return all connections to the normal position as before the test.

  4. Place a copy of the test result in the field maintenance file.

  5. Close the work permit.
