Introduction

The RMX911 is intended primarily for use with the time over current relays types IKT 943 and the IDMT relays IKC911. It is designed to determine the direction of earth fault in both solidly Grounded and ungrounded power system.

The first draft is initiated, tested by eng. Hany Bassuony Abdel Razek, accepted by general manager eng. Gamal Abdel Wahab head sector of inspection & quality control

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Switch off DC power supply before changing setting of the relay.

Tools and Equipment

  • XS92a test set

  • Test leads, Screwdriver

Work to be Carried out

Preliminary Steps to Prepare for Relay Testing

  1. This procedure is used to perform off load relay test.

  2. The test is done for ( 50/51 & 67G ) as a one relay.

  3. Review technical reference material: Manufacturer's manual.

  4. Obtain copy of relay setting from field maintenance file.

  5. Obtain outage for line protected by this relay.

  6. Check that the bus and line disconnect switches are opened and tagged.

  7. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers MCB'S, and all cabling supplying the relay.

  8. Use station drawing to isolate:

  • Breaker trip coils.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection Outputs

  1. Carefully follow these test procedure steps.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Test trip all breakers associated with the relay.

Prior to Starting Testing Procedure(s) Ensure the Following Steps

  1. Look at the EEHC panel nameplate and check that this EEHC system designation nameplate agrees with the line that has been taken out of service for maintenance.

  2. Check there is no AC current from CT's by using a clip on ammeter.

  3. Check there is no AC voltage from VT's by using a voltmeter.

Preparation

  1. Connect the xs92a test set with the relay as shown in Figure 1.

  2. Ensure that the CT secondary is correctly short circuited and voltage teminals supplying the relay is opened.

  3. These connections must be made before starting test.

  Test Trip all Breakers associated with the Relay

  1. Reconnect Breaker trip coils.

  2. Close only the circuit breakers associated with the relay.

  3. Adjust the injection parameters to make trip.

  4. Start injection and make sure all breakers are tripped.

Restore the Relay to Service

  1. Unplug the test set XS92a from the power source.

  2. Remove the test leads from the relay and the test equipment.

  3. Return the relay to the permanent settings as indicated in the maintenance file.

  4. This Completes the Test Procedure.

Return Relay and associated Equipment to Service

  1. Return relay to service by withdrawing the test plug handle from the relay.

  2. Reconnect any equipment associated with the relay has been isolated:

  • Breaker trip coils.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.
