Introduction

The RANZA type fault locator is an essential complement to protective relays of transmission lines, since it measures with great accuracy and indicate the distance to the fault.

The RANZA works in co-operation with the line protection and receives the required phase – selection and starting information from this. However,it can also be provided with built – in phase selection equipment and then only requires starting information from the line protection.

The first draft initiated, tested by eng. Hany Bassuony Abdel Razek, accepted by UEEPC general manager eng. Gamal Abdel Wahab inspection & quality control head sector, and approved by Dr. Tharwat sabry.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Safety and Healthy equipment must be used as safety Helmet, safety shoes, ...etc.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Line isolation is required.

Tools and Equipment

  • XS92a test set

  • Test plug RTXH18

  • Test leads

Work to be Carried Out

Preliminary steps to Prepare for Relay Test

  1. Press the black button 3 times.

  2. Now the printer will print the setting.

  3. Calculate the following:

ZL1 = P1 + JP2

ZL0 = P3 + JP4

Zs    = P5 + JP6

Ko   =  ( ZL0   -  ZL1 ) / 3 ZL1

Test Procedure

  1. Insert the test plug RTXH18

  2. Connect the circuit as shown in Fig. 1

  3. Adjust the test set xs92a as follow:

FC = 111

SQ =  1F1

A = 2

TF = 3000

Mod  ZZ

ZL = more than half of ZL of the line

PHI = the angle of the line

K0 = from the above calculation

ZS =  from the above calculation

DZL = 0.01

Start

  1. Now the injection is started and the impedance will decrease until it reaches a value equal 0.5 ZL at this moment make start to fault locator by shorting the test leads which connected to 1 and 2.

  2. Press ◙ to stop the injection.

  3. Now the fault locator is started.

  4. Wait until the LCD of the fault locator lights with the percentage of the line length where the fault occurred,  it is 50.

  5. The printer now will print the sheet which reports the fault loop and faulted phase and the fault location as a percentage of the line length.

  6. Repeat the above test steps 3 to 7 for fault codes 112 , 113, 121, 122, 123 .

            Note

FC =111 

for phase R to  Earth

FC =112

for phase S to  Earth

FC =113

for phase T to  Earth

FC =121

for phase R to  S

FC =122

for phase S to  T

FC =123

for phase T to  R

  1. After finishing the test switch off the tester and withdraw the test plug.

  2. Make reset to the fault locator.
