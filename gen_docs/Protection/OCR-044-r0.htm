Introduction

This document outlines the testing procedures P1, which must performed on an annually bases as indicated by Y1 in the titled block

The RSZ3Ena relay is an electronic independent over current time lag relay. This procedure details the steps required to completely check the proper operation and calibration of the RSZ3Ena relay. This document is divided into 3 sections.

The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Wear safety and healthy equipments as safety helmet, safety shoes ...etc.

  • Line isolation is required.

  • Make sure CT Secondary circuits are shorted before isolating the relay current circuits for current injection.

 

 

 

Tools and Equipment

  • Secondary injection test set (Siemens type Clock)

  • Test leads, clamp meter type Hoky, voltmeter

  • Complete tool set.

Works to be Carried Out

Testing the Relay, and Protection Output

  1. Verify the test results match the relay setting, if not consult technical services.

  2. Test trip the breaker associated with the relay.

Preparation

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolating the relay current circuit for test.

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting sheet.

  3. Label the wires

  4. Remove the input current terminals number 1,2,3,4,5.6.7 and 8

  5. Connect test leads between the current terminal number 1and 4 on the relay to the current terminals on the test set

  6. Slowly increase the current until the setting and wait the time until the red indication on the over current element

  7. Repeat steps 5,6 for 2,5 and 3,6 for phase S and T

  8. Connect test leads between the earth terminals number 7.8 on the relay to current terminals on the test set

  9. Slowly increase the current time until the setting and wait the time until the red indication on the earth fault element

  10. Record this value on the test sheet.

  11. Increase the injected current with a value 25% greater than the operating value.

  12. Slowly decrease the current until the relay resets.

  13. Record this value on the test sheet.

 

 

Time test

  1. Inject current higher than the setting value so the relay picks up firmly.

  2. Keep the injected current and switch off the test set.

  3. Connect the time delayed contacts number 3 and 4 on the relay to timer terminals on the test set.

  4. Switch on the test set.

  5. Start the test by injecting current into the relay. The timer will stop once the relay has picked up.

  6. Record this value of time on the test sheet.

  7. Repeat the above test from step 1 to step 6 for other phases, but inject current at terminal 8 and 11 for phase S and terminals 9 and 12 for phase T.

Trip Test

  1. Request the Operator to switch on the main circuit breaker.

  2. Operate the output-delayed- contact by hand to trip the circuit breaker.

  3. Check that the circuit breaker trip.
