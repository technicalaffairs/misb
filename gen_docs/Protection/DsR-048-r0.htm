      Introduction

This procedure details the steps required to completely checl the proper operation of the micromho distance relay. It is recommended this procedure be carried out on a yearly bases

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment

The first draft initiated by Mostafa Ibraheem and tested by eng Abd Elnaseer Mostafa ,  accepted by general manager eng Shoaqi Mahmod, approved by eng Mohamed Mahdy Rezk.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Safety and Healthy equipment must be used (safety Helmet, safety shoes, ...etc).

    •

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for current injection

    •

Tools and Equipment

  • TZ 2 test set this test set consists of 2 units supply, timer and volt injection unit and current injection unit
  • Multimeters
  • Test plug (MMLG 01)
  • Test leads
  • Screwdriver

Work to be Carried Out

Preliminary steps to prepare for relay testing
  1. This procedure is used to perform off load relay test
  2. Obtain copy of relay setting from field maintenance file
  3. Obtain outage for line protected by this relay
  4. Check that the bus and line disconnect switches are opened and tagged
  5. Inspect the CT secondary circuits, the VT miniature circuit breakers (MCB`s) and all cabling supplying the relay
  6. Use station drawing to isolate
  7. Disturbance alarms and fault recorder associated with this relay
  8. Auto - re-close associated with this relay

Testing the relay, relay scheme, and protection outputs

  1. Carefully follow these test procedure steps
  2. Verify the test results match the relay setting, if not consult technical services
  3. Test trip all breakers associated with the relay

Prior to starting testing procedure (s) ensure the following steps

  1. Look at the EEA panel nameplate and check that this EEA system designation nameplate agrees with the line that has been taken out of service for maintenance
  2. Check there is no AC current from CT`s by using A clip on ammeter
  3. Check there is no AC voltage from VT`s by using A voltmeter

Preparation

  1. Ensure that the CT secondary are correctly short circuited on the case side of test plug
  2. These connections must be made before the test plug is inserted
  3. Connect the current terminals IA, IB, IC and IN and the voltage terminals VA, VB, VC, and VN on the test plug to the corresponding current and voltage terminals on the supply unit test set
  4. Insert the test plug
  5. Adjust the test unit switches as
  6. Put the angle switch at -90º
  7. Ampere switch on 3A
  8. The injection direction switch on normal position in case of forward zones Z1, Z2, Z4 and on reverse position in case of reverse zone Z3
  9. Put the injection type switch on I + Vp

Testing the phase to ground and phase to phase measuring elements
  1. Select phase to ground fault A-N by using relay selector switch on supply unit test set
  2. Connect the outgoing voltage from the supply unit
  3. Switch on the injection switch
  4. Start the test by pressing the switch on the control unit momentarily
  5. Observe that the red indications A,N and Z1 on the relay will light up
  6. Change the value of the potential tap up or down until the relay operates
    In second step (the red indication Z2 on the relay will light up with the last indication)
  7. Record this pickup points on test sheet

Repeat the above test for phases B-N, C-N, A-B, B-C and C-A
  1. Repeat the above test but this time calculate potential corresponding to the relay setting of the second zone for phase to ground and phase to phase fault set this value on the test set and then change it up or down until the relay operates in third step
  2. Switch off switch

Measuring the operating times

  1. Set potential switch to 50 % of the zone 1 reach
  2. Select phase to phase fault A-B by using relay selector switch on supply unit test set
  3. Connect test leads between timer contact on the relay to the stop terminal of the timer
  4. Reset the interval timer
  5. Switch on
  6. Start the test by pressing switch on
  7. The timer will stop once the relay has picked up
  8. Record the zone 1 tripping time on test sheet
  9. Repeat tests for zone 2 and zone 3 operating times
  10. Repeat tests for phase to ground operating times

Switch on to fault (sotf)

  1. Inject current only and disconnect the voltage terminal
  2. The indication led for (sotf) light up

Voltage transformer supervision

  1. Inject current and volt
  2. Disconnect voltage terminal
  3. The indication led for (V - fail) light up

Testing the relay, relay scheme, and protection outputs

  1. Verify the test results match the relay setting if not consult technical services
  2. Perform check of the acceleration of the relay, breaker failure and reclosure schemes, separate technical procedures should be used to test each of these systems
  3. Test trip all breakers associated with the relay

Return relay and associated equipment to service

  1. Return relay to service by withdrawing the test plug handle from the relay
  2. If any equipment associated with the relay has been isolated without the test plug it should be returned to service by the following restoration steps
  3. Disturbance alarms and fault recorder associated with this relay
  4. Auto - reclosure associated with this relay

Request power line be returned to service

  1. Take crossed readings of all AC currents and voltages supplied to the relay and compare then to those for normal load conditions
  2. Check that readings agree with station meters
    (MW / MVAR / power factor / amperes / voltages)
  3. Place a copy of the test result in the field maintenance files
