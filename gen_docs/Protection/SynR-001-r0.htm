Introduction

There are eight up to isolated inputs to the relay that can be arranged to operate up to alternative ladder inputs, which perform functions assigned to them in the ladder diagram, as determined by the setting of the INPUT MASKS and so making maximum use of the available control inputs. Software filtering is applied to eliminate the adverse effects of induced ac signals in the external wiring eight programmable output relays are provided on all models.

These relays can be arranged to operate in response to any, or all, of up to 16 alternative ladder outputs, whose function is assigned in the ladder diagram, by suitably setting the OUTPUT MASKS. In addition there is a watchdog relay for external indication of equipment failure/healthy status. Measurements are only available in relays including check synchronism features (KAVS and KAVR). These relays have two voltage transformers for measuring line and bus bar voltages.

Safety Precautions 

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Line isolation must be done.

Tools and Equipment

  • ISA test set type ART/3 or Omicron

  • Voltmeter, and Ammeter

  • Test plug test leads

    •

 

 

Work to be carried out

Relays Preparation

  1. Review the manual and drawings

  2. Prepare the previous or commissioning results of the relay testing

  3. Check the wires of the relay by hand.

  4. Check the AC source is available to the tester.

  5. Check the existence of the DC sources in panels.

  6. Disconnect the outputs   of the relay and connect one of them to the timer of the tester.

  7. Check all the leads used in tests.

    8.

Test Steps

  1. Connect the voltage  leads of a tester unit to the relay points according to the drawings and scheme

  2.  Inject voltage to each couple of points.

  3. Repeat the steps for each pair of points according to the test sheet.

  4. Record the results.

    5.

Testing the Relay and Protection Outputs

  1. Carefully follow these test procedure steps 

  2. Verify the test result match the relay setting. If not consult technical services.

Technical Data

The KAVS is in tended to provide the check synchronizing or dead line

 charge or dead bus charge interlocks required for some auto-reclose applications. If also provides split system detection and system synchronizing interlocks for reconnection following a system split

The slip frequency settings for check synchronizing and system synchronizing are independent of the phase angle and line settings, which are separately adjustable for each function.

Under voltage block and /or differential voltage block can be applied independently to boh check synchronizing and system synchronizing
