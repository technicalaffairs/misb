Introduction

This document lists the testing procedure P1, which must be performed on an annually bases as indicated by Y1 in the above title block

The type IFC 57AD over current manufactured by G.E this procedure details steps required to completely check the proper operation and calibration the IFC 57AD relay.

The first draft initiated by eng. Fayza El-Sharkawy, tested by eng. Shirin Hamdy & eng. Azza Salama, Revised by Quality Control Eng. Adel El-Nagar at EDEP Co., and accepted by General Director Eng. Mahmoud Saker and approved by eng. Moustfa Alam Head Sector of Talkha Power Plant.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuit for current injection.

  • Transformer must be switched off and isolated.

Tools and Equipment

  • Ammeter.

  • Test Leads.

  • Secondary Injection test set.

Work to be Carried Out

Preparation

  1. Ensure that the C.T. secondary circuits are shorted at the incoming terminal blocks before isolating the relay current circuit for test.

  2. Look at field maintenance file and compare position of the setting switches on relay.

  3. Test trip the breaker associated with the relay.

  4. Verify the results match the relay setting, if not consult technical services.

  5. Carefully follow all test procedure steps.

Current Test

  1. Connect test leads between the current terminals number (5 and 6) on the relay to the current terminals on the test set.

  2. Slowly increase the current until the contacts close the relay must operate with a current between + 10% of the setting value.

  3. Record this value on the test sheet.

  4. Increase the injected current with a value 25% greater than the operating value.

  5. Slowly decrease the current until the relay resets.

  6. Record this value on the test sheet.

Time Test

  1. Connect the time delayed contacts numbers (1 and 2) on the relay to timer terminals on the test set

  2. Start the test by injecting current into the relay. The timer will stop once the relay has picked up.

  3. Record this value of time on the test sheet.

  4. Repeat the above test steps for other phases.
