Work to be Carried Out

Temperature Amplifier (IL - 1L ) STKD

  1. Place all control thermocouple switches in (REJECT) position.

  2. Make jumper between point ( 4C , 4D ) and between ( 5C , 5D ) at calibrator set

  3. Set thermocouple selector switch to read average exhaust temperature usually position (12)

  4. Adjust MV to obtain 700oF at temperature indicator (26TR) check that millivolt value is 24.69 + 0.2 MV

  5. Connect a digital voltmeter to read temperature amplifier output (TX) of STKD at (8H) , Acom. on calibrator set by putting AVO.

  6. Adjust (R87){2} “offset” to obtain TX output of 0.00 +  0.01volt

  7. Increase MV to obtain 1200oF at temperature indicator (26TR) check that millivolt value is 36.49 + 0.2 MV

  8. Adjust “GAIN” (R83){1} to obtain TX output of 5.00 +  0.01volt

Temperature Amplifier (IL - 1K ) SOTH

For Channel (A)

  1. Make jumper between point ( 4C , 6D ) and between ( 5C , 7D ) at calibrator set

  2. Connect jumper between ( + ) OTA and ( + ) CK and jumper between CAL (OTA) and (-) CK  on patch board .

  3. Set thermocouple selector switch to read average exhaust temperature usually position (12)

  4. Put avometer at (12I , G) at calibrator set

  5. Adjust MV to obtain 700 oF at temperature indicator (26TR)

  6. It should be read zero volt if it is not, adjust it from (R73){1}”offset”

  7. Increase MV to obtain 1200 oF at temperature indicator (26TR)

  8. It shout be read 5 volt if it is not, adjust it from (R70){2}”spn”

For Channel (B)

  1. Make jumper between point ( 4C , 8D ) and between ( 5C , 9D ) at calibrator set

  2. Connect jumper between ( + ) OTB and ( + ) CK and jumper between CAL (OTB) and (-) CK on patch board .

  3. Put avometer at (13I , G) at calibrator set

  4. Adjust MV to obtain 700oF at temperature indicator (26TR)

  5. It should be read zero volt if it is not, adjust it from (R73){6}”offset”

  6. Increase MV to obtain 1200oF at temperature indicator (26TR)

  7. It shout be read 5 volt if it is not, adjust it from (R70){5}”spn”

Over Temperature ALARM (SOTG)

For Channel (A)

  1. Connect jumper between ( + ) OTA and ( + ) CK and jumper between CAL (OTA) and (-) CK  on patch board .

  2. Make jumper between point ( 4C , 6D ) and between ( 5C , 7D ) at calibrator set

  3. Adjust MV to obtain 1000oF at temperature indicator (26TR)

  4. Temp. alarm at card SOTG must be light

  5. If it is not adjust card must be changed

Over Temperature TRIP (SOTH)

For Channel (A)

  1. Connect jumper between ( + ) OTA and ( + ) CK and jumper between CAL (OTA) and (-) CK  on patch board .

  2. Make jumper between point ( 4C , 6D ) and between ( 5C , 7D ) at calibrator set

  3. Raise MV to obtain 1028 oF at temperature indicator (26TR). Until trip lamp at card SOTH just comes on

  4. If it is not adjusted, adjust it from (R72) Resistance No.{3}OFT  (on card SOTH)
