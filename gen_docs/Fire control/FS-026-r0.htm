Introduction

This document out lines the testing procedures P1, to be performed on an annually bases as indicated by Y1 in the title block.

The fire alarm system was designed following the supplied specifications. It consists of one main panel located in the steam turbine building with 39 zones, including 4 zones in the machine shop/spare parts building, 3 zones in the circulating pump house, 1 zone in the combustible fuels storage building and 20 zones in the turbine building. In addition, 11 zones are supplied to be connected to by others

Sensor energized when smoke occur in any zone which cause current increasing through wires to fire alarm panel  ( simplex ) with appearing alarm in smoke zone

Safety Precautions

  • A work permit must be issued

  • Personal protective equipment must be used safety shoes, safety helmet, ...etc.

Tools and Equipment

  • Suitable tool set

  • Safety helmet and safety shoes, ... etc.

    •

  • Air drier & contact cleaner

  • Digital voltmeter model Fluk 187

  • Smoke source, Power supply 24 VDC

Work to be Carried Out

  1. Remove sensor.

  2. Remove base special to sensor.

Testing Procedures

  1. Clean base.

  2. Clean sensor.

  3. Clean connection terminals to base.

  4. Clean connection terminals to sensor.

  5. Connect the power supply 24VDC on base and check the sensor and the LED light (flash).

  6. Test the sensor by smoke source, check LED (on) that is mean the system is ok, if not  change the sensor and check again the system.

  7. Measure power on connection terminals to base to check if the voltage is within the limitation of 24 VDC.

Return System into Service

  1. Assemble base in the same location.

  2. Assemble sensors in the same location.

  3. Test sensors and observe alarms on main fire alarm panel (Simplex).

  4. Make reset to fire alarm panel after alarm appearance.

  5. Test fire alarm system manually and observe alarm on fire alarm panel (Simplex).

  6. Make reset to fire alarm panel (Simplex) after alarms appearance.
