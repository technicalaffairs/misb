Introduction

This document outlines the testing procedures P1, to be performed on an annually bases as indicated by Y1 in the title block.

Fire Fighting sys. With using water are used in main transformers, aux. trans,  turbine home and oil tank for comb. Cycle units.

When fire occur and the temp. equal (194°C) the fire sensor change position from open contact to close contact there for due to connect  voltage ( 24 VDC ) from main control panel ( system 3 ) to solenoid valve on fire sys. So that the water open on source of fire .

Work to be Carried Out

  1. Remove sensor.

  2. Test the sensor in the laboratory as follows:
    a)     Clean sensors.
    b) Check the continuity by using DVM (ohm) should be open.
    c) Check the sensor by heat source and observe the contact is closed if not change the sensor.

  3. Measure power on connection terminals to sensor to check if the voltage is within the limitation of 24 Vdc

  4. Assemble sensor in the same location.

  5. Test sensor in the same location.

  6. Be sure the solenoid valve energized.

  7. Observe deluge system working automatic.

  8. Observe alarms on main fire alarm panel (system 3 & simplex ), the water is come and check the all nozzle is ok.

  9. Make reset to fire alarm panel after alarm appearance.

  10. Test fire alarm system manually.

  11. Observe deluge system working automatic as before.

  12. Observe alarms on main fire alarm panel  (system 3 & simplex ).

  13. Make reset to fire alarm panel (system 3 & simplex ) after alarms appearance.
