Introduction

This document lists the equipment inspection. which must be performed on a monthly bases as indicated by M1  in the title block for inverter (u.p.s) with the following data:
   - AC input 380v, 50hz, 3ph
   - AC output 380v, 23A 50hz, 3ph
   - Static By bass 380V, 23A, 50 HZ, 3PH

The first draft initiated by : Eng. Abdel Aziz  EL Bagoury, tested by eng. Doaa Anis, revised by quality control Eeng. Adel  Abdel EL Kader EL Nagar at EDEPCo., accepted by general director eng. Mahmoud Saker and approved by eng. Mostafa Alam Head Sector of Talkha PS 2x210 MW

Safety Precautions

  • A work permit must be issued.

Tools and Equipment

  • Suitable tool Set

  • Digital voltmeter

Work to be Carried Out

  1. Check for correct operation of  DC room air condition.

  2. Check ventilation fan of (u.p.s) .

  3. Read AC input voltage, AC output voltage, AC input current, AC output current, and output rectifier voltage DC and record.

  4. Check all abnormal cases inside ( U.P.S ).

  5. Check over heating compartment of rectifier cards, inverter cards, card fram, static switch card and display unit.

  6. Inspect visually fuses, contactors, and capacitors.

  7. Clean the floor of the DC room.
