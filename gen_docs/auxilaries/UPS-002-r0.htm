Introduction

This document lists the equipment inspection, which must be performed on an annualy bases as indicated by Y1 in the title block for Inverter:

DC input: 100 - 140V 312.5A

AC output: 220V, 113.6, 50 HZ, I PH

Bypass: 220V, 113.6 A, 50 HZ, I PH


The first draft initiated by Eng Metwaly Mahmoud El_Sayed El_Hendawy and revised by Operation and maintenance's General Director: Eng Mohamed Reda Abd El-Azez

Safety Precautions

  • Work order must be issued

  • Transfer UPS to by pass.

 Tools and Equipment

  • Suitable Tool Set

  • Digital Voltmeter

  • Vacuum Cleaner or Blower

  • Brush

Work to be Carried Out

  1. Transfer inverter (UPS) to by pass.

  2. Switch off (UPS) input and output supply.

  3. Clean from air dust and dirty.

  4. Check ventilation system and replace fans bearings.

  5. Inspect over heating compartment of control board static board.

  6. Check fuses, contacts and terminal tightening.

  7. Inspect visually for Scr's and Diodes.

  8. Inspect visually for capacitors.

  9. Check (UPS) ventilation area.

  10. Put (UPS) in service and check input and output voltage.

  11. Check indication lamps.
