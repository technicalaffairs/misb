Introduction

The combined over current and earth-fault SPAJ 140 C relay is a digital relay is designed specially for selective short-circuit and earth-fault protection with definite time operation or the calculated operate time at inverse time operation. These procedure details the steps required to completely check the proper operation and calibration of the SPAJ 140C relay. It is recommended this procedure be carried or a yearly bases. This work be carried out section of this document is divided into three section.

The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test.

The second section preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test. The document is formatted the way minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • Line isolation and a work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Make sure that CT secondary circuits are shorted before isolating the current circuit for test current injection

  • Safety fence (rope) with caution marks surrounding the work area,

Tools and Equipment

  • Manta test set (powertec)

  • Volt meter

  • Ammeter

  • Test leads

 

 

Works to be Carried Out

Steps to Prepare for Relay Testing 

  1. Review technical reference material and manufacturer's manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay

  3. Obtain outage for line protected by this relay

  4. Check that the circuit breaker is opened and tagged.

  5. Inspect the CT secondary circuits, the VT miniature circuit breakers (MCB`s) and all cabling supplying the relaying.

  6. Use the station drawings to isolate the following
                * Breaker trip coils
                * Breaker failure scheme associated with the relay and disturbance alarms

Testing the Relay

  1. Carefully follow this test procedure

  2. Verify the test results match the relay setting, if not consult technical services, read the paragraph of relay settings .

  3. Trip the circuit breaker associated with the relay   

Preparation

  1. This procedure should be used for performing off load line

  2. Look the EEC panel name plate and check this EEC system designation name plate agrees with the line that has been taken out of service for maintenance

  3. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test

  4. Using a clip on ammeter checks there is no current flowing in the SPAJ relay circuits. Wire going to the rear terminals number 1,2,3 and 4 should have no current flowing in them.
    Note , Always check the continuity of the ammeter before using it in a (possible) live current circuit.
     

  5. Check all external connections to the relay are in a good condition and check terminal block wires are tight.

  6. Compare the settings being recorded on the display of the relay with those on the EEC test sheet, located in the field maintenance file. Check that all the appropriate settings have been correctly applied to the SPAJ relay.  

Reading the Settings From the Relay

To verify the relay setting values, press the reset button of the relay for 0.5 seconds and repeat until the yellow LED lights for the desired feature. The three green digits to the right of the display show the value recorded in the relay for the selected setting. The following chart shows the available features.

    Yellow Indicator                                                                  Value on display

          I > In                                                                   Low set over current setting 
          Tx > {s}                                                             Operate time for low set over current
          I >> / In                                                              High set over current
          t >> {s}                                                              Operate time for high set over current
          Io >/ In                                                                Low set earth fault setting
          to > /In                                                                Operate time for low set earth fault
          Io >> In                                                              High set earth fault
          To > /InIs                                                            Operate time for high set earth fault

 Over Current Test

  1. Be sure that the Dc power of the relay is correct by applying a Dc voltmeter across points 61 and 62 on the rear of the relay. 

  2. Connect the current test leads between the current terminals on the relay 1 and 2 if using the 5 Amp mode or 1 and 3 if using the 1 Amp mode. For phase R, connect to current terminals A and B on the test set with the current mode switch set to LOW.

  3. Select the position of fault mode on test set.
            a . Set the left switch on A-B
            B . Set the right switch on 0-0

  4. Connect A.C power to the test set

  5. Select the current button in modify mode

  6. Inject current to the relay by selecting the current button in the modify section and select the static/dynamic button and choose the static mode ( static mode means we get light at static)

  7. Increase slowly the injected current by pressing the FAULT button and move the modify switch at the same time until you reach to the operating value and then the yellow " LEDs" labeled " L1,>/ In" light The relay must operate with a current between ± 10% of the setting value. The operate current is shown on the display of the test set.

  8. Record this value (I operate) on the test sheet

  9. Switch off the test set

  10. Connect leads between the time delayed trip output contact terminals 65 and 66 on the relay to the time (normally open) terminals on the tester. Choose the upper if the trip out put contact is dry or lower if the contact is wet.

  11. Switch on the test set.

  12. repeat steps 6 and 7 to inject a current 25% larger than the operating value, Ioper

  13. Select the yellow button to the time mode. This will light when the relay operates

  14. Slowly decrease the injected current by selecting the static button and fault button in the control section of the test set. Move the modify switch anti-clockwise until the yellow button goes dark.

  15. Press the current button in the modify mode to record the value of Ioper from the display of the test set

  16. Repeat steps 6 and 7 to inject a current 25% larger than the operating value, Ioper 

  17. Select the yellow button to the time mode

  18. Select the dynamic mode by pressing the static/dynamic button. The dynamic light will then be lit

  19. Press the Fault button. After a time delay, the relay will operate and we will get that time displayed on the display of the test set. We also get the red trip indicator light at O/P Trip on the relay .

  20. Record this value, Trip Time on the Test Sheet .

  21. Switch off the test to reset the LED's on the relay .

  22. To test phase S, move the leads from terminals 1&2(1&3) on the relay to terminals 4&5 (4&6)

  23. Repeat the above test from steps 6 to 20for phase S. In this case indicator L2,1>In will light .

  24. Switch off the test set.

  25. To test Phase T, move the leads from terminals 4&5(4&6) on the relay to terminals 6&8 (7&9)

  26. Repeat the above test from steps 6 to 20 for Phase T. In this case indicator L2,1>In will light.

  27. Switch off the test set .    

 

Earth Fault Test

28.   Move the leads from terminals 7&8 (7&9) on the relay to terminals 25&26 (26&27) to test 
        the earth fault feature on the relay .

29.   Repeat the above test from steps 6 to 20 for the earth fault feature. In this case indicator L2,
        1>In will light.

Return Relay and Associated Equipment to Service

30.   Return relay to service by removing the test leads from the relay terminals block and the test

31.  Set. make sure CT and VT secondary circuits are in normal condition

32.   If any equipment associated with the relay has been isolated, it should be returned to service

33.   By the following restoration steps

34.   Breaker trip coils

35.   Breaker failure scheme associated this relay.

36.   Disturbance alarm associated with this relay.

37.   Request feeder to be returned in service

38.   Take phasor readings of all A.C currents and voltages and compare them to those for normal
        load condition

39.  Check that these reading agree with station meters (MW, Mvar, Amperes and power factor)

40.  Place a copy of the test result in the field maintenance file
