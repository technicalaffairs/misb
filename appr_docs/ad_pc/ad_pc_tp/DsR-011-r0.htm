Introduction

7S130 is a six system solid state distance protection has three of them use star measuring quantities and the other use delta measuring quantities. The six assuring systems could set for distance independent of each other. The operating time is half cycle.                              

Independent is setting of reach along the resistance and reactance. Has quadrilateral characteristic for tripping.

Safety Precaution

  • Line isolation and work permit required
  • Isolate dc supply to relay before any adjustment or maintenance on the relay
  • Switching off the auto recloser unit.

Tools and equipment

  • Tester (7vp 48, 7vp49)
  • Ammeter
  • Voltmeter

Work to be Carried Out

  1. Review technical reference material manufactures
  2. Obtain outage for line protected by this relay
  3. Obtain copy of the relay setting
  4. Check that isolating switch and C.B are opened and tagged
  5. Inspect the secondary circuit of CT , VT
  6. Isolate the breaker failure scheme associated with this protn
  7. Block any disturbance alarm

 Preparation of tester

  1. Place the two cases the voltage unit on the top and current unit in the bottom.

  2. Connect the two units together by the cable connector by the cable connector 7xv6201 between test plug x2 in current unit and test plug x1current in the voltage unit

  3. Connect the power supply cable 7xv6200 (AC supply) between the plug x1 in the Current unit and the source.

  4. Connect the cable 7xv6200-1 between the plug x3 in the current unit and the test plug 7xv61 on the relay.

  5. Connect the ammeter with the terminal 17, 18 in current unit and voltmeter on Terminal 5,6 in voltage unit.

  6. The selector switch si7 must be in position 2 for built in timer

  7. The selector switch s10 for current must be adjusted decoding to the rated of the Secondary current transformer.

  8. Turn the test switch (test /service) on the relay to position (test)

  9. Insert the plug in test socket module relay

  10. Switch on the main switch s1 on the current unit and also switch s13 for the voltage unit.

  11. Check the phase sequence, it must be positive sequence, and that happened when H13 signaling lamp 13 green and if the signaling lamp H14 is red then the sequence is negative

  12. Adjust the selector switch s5 to (zero) angle

  13. Adjust the switch (S15 –1x) of the current phase selection at L on current unit and adjust the selector switch ( s6- ux5-6 ) of voltage phase selection at (L1-N) on the voltage unit

  14. Select the fault type (IL1, IN) (UL 1, un) by pressing the push button switch s2 ,s5 on the current unit and s1,s4 on the voltage unit

  15. Increase the voltage from the variable transformer knob (course and fine) to be more than the calculation value of voltage.

  16. Press on the push button s8 to start the injection and adjust the current at IA from the single phase variable transformer T1 for current IL 1

Direction check

  1. Simulate a fault type fault
  2. Adjust the voltage at the normal value for the three phases
  3. Adjust the current at approximately 0.5 In
  4. The green LEDs on the six measuring module 7TL 52 must be light if the direction is correct
  5. Change the direction from the voltage test set or from the plug on the module 7TL 51. The EDS' must be off.

Under impedance start

  3. Decrease the voltage value to be lower than 48v of this phase and increase the current until to reach the pick up value J>/Jn, & check the LED’s

First distance zone
  1. Set timer plug t2, t3, t4 to ∞
  2. Prepare the tester for R-E fault as mentioned
  3. Adjust the angle to 90,increase the current more than starting current and the voltage to 48v ,decrease the voltage until to reach the tripping, compare this value by the value you calculated from the table
  4. Adjust the angle to zero and repeat step ( 3 )
  5. Repeat the steps ( 3 & 4 ) to the other phases

Second distance zone

  1. Insert the timer plug t2
  2. Repeat the above steps ( In first zone ) ( 3 & 4 ) for all phases

  3. Repute the following in to service

  4. Disturbance alarm and fault recorder
  5. The associated auto- relosure scheme
  6. The associated breaker failure scheme
  7. Return the test switch (TEST/SERVICE) on the relay to position “SERVICE”

Request Power Line to be Returned to Service 

  1. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those for normal load conditions

  2. Check that these reading agree with station meters: (MW, MVAR, power factor, Amperes, voltages)

  3. Place a copy of the test results in the field maintenance fill.
