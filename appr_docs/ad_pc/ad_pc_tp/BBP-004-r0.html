 Introduction

This procedure, details the steps required to completely checking the proper operation and calibration of the bus bar differential relay type 6SS13, SIEMENS make. It is recommended this procedure be carried out on yearly bases.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The 7SS13 is a solid state bas bar relay manufactured by SIEMENS it is intended for very high speed and selective protection for high voltage and extra high voltage networks in 50-60 HZ this relay schemes detects all types of faults in systems with solidly earthed or insulated neutral.

The 7SS13 relay particularly suitable for applications in systems with high short circuit capacities and critical stability conditions because of it is extremely short tripping time and high reliability.

It was thus possible to give the bas bar relay system 7SS13 maximum degree of safety against spurious operation of external short circuits. While ensuring in the event of internal short circuits that a tripping signal is emitted within the very short time of less than half cycle. The relay 7SS13 has to operate under severely aggravated conditions.

The properties of the relay even under extremely difficult condition namely:

  • High degree of stability even under falsifying influences owing to CT 's saturation.

  • Short tripping time in the event of an internal short circuit,

  • Correct behavior and proper response to relocation of faults,

  • Ease of commissioning and setting.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Wear the safety and healthy equipment.

  • Make sure line is isolated.

  • Make sure CT secondary circuits are shorted before isolating the current circuits for test current injection.

  • Switch off DC power supply before inserting or with drawing the relay.

  • Check that is no current from CT 's by using clip on ammeter.

  • Block any disturbance alarms or fault recorder associated with this relay test.

  • Isolate the breaker failure scheme associated with this relay.

Tools and Equipment

  • Secondary injection test type 7VP49 SIEMENS tester

  • Clip on ammeter

  • Digital voltmeter, Test leads

Work to be Carried Out

Steps to Prepare for Relay Testing

  1. Review technical reference material: manufacture’s manual

  2. Obtain a copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Inspect the CT secondary circuits and all cabling supplying the relay.

  4. Use the station drawing to isolate:

  • Breaker failure scheme associated with the relay.

  • Disturbance alarms and fault recorder associated with this relay

Testing the Relay and Protection Outputs

  1. Carefully follow these test procedure steps.

  2. Verify the test results match the relay setting. If not consult technical services.

  3. Test trip all breakers associated with the relay.

  4. Connect one or more circuit breakers to the bus and initiate a trip from the 7SS13 Relay.

  5. Repeat to include all breakers

Testing Procedures Preparations

  1. Obtain authorization from the control center

  2. Switch off the DC supply

  3. Draw out the output tripping card 7TR54 from the relay 7SS13

  4. Isolate the relay 7SS13 from the current circuits by shorting between the input current terminals with the output current terminals for the relay on the terminal block for all feeders one by one.

  5. After making the short circuit in the previous step, open the link between the input and output side’s of the current terminal block to isolate the relay.

  6. Switch on the DC supply.

  7. Measure the operating voltage (V0) on BU1, BU2 and measure the stabilizing voltage (VS) on BU3, BU4 on the card 7TD10 the reading must be zero volt

  8. After isolating the relay install a short on the output terminals for the relay on terminal blocks for all feeder one by one

Note: If the sectionalizing disconnect switch was opened, the feeders on bus section one will light in the led on A1 module. However if the sectionalizing disconnect switch is closed the feeders on both section 1, 2 will light the led on module A2.

Checking the Supervision, Start and Trip Values

For Feeder 1

  1. Make sure that the isolators switch for this feeder must be connected the isolated bus.

  2. Put the test switch of the relay to the position (O) test position.

  3. Inject current on the input terminals for the relay on phase R-N.

  4. Slowly increase the current until the supervision led (lower one) will light up.

  5. Record this value on test sheet .

  6. Increase the current slowly until the start led (the middle one) lights up.

  7. Record this value on the test sheet.

  8. Continue Increasing current slowly until the trip led (highest one) lights up

  9. Record this value on the test sheet

  10. Decrease the current until it becomes zero.

  11. Repeat all the above steps from 1 to 10 but for phases S-N, T-N.

  12. Record all these results on the test sheet.

  13. Repeat the above steps from 1 to 10 for phase {R-S, S-T, T-R}

Testing Other Feeders

Repeat The above test From Step 1 To 13 for all other feeders

Trip Testing

  1. Obtain permission from the dispatching center and perform a trip test on each feeder one at the time.

  2. Isolate the bus section, which test the feeder on it.

  3. Connect this bus isolator to free bus.

  4. Connect the circuit breaker for this feeder to the isolated bus.

  5. Connect all the other feeder to the other bus.

  6. Don’t forget that the bus coupler should be opened before testing.

  7. Switch off the DC supply for the B.B DIFF relay 7SS13.

  8. Insert the special output-tripping card 7TR54 for this feeder.

  9. Select any phase {for ex. R-N}

  10. Slowly inject the current until the circuit breaker trip.

  11. Make sure that the circuit had tripped.

  12. Repeat the above step from 1 to 11 for all other feeders.

  13. Repeat all the above steps with the other bus through the other isolator after isolating the second bus.

Return Relay and Associated Equipment to Service

  1. Remove the test lead from the terminal block for the relay.

  2. Remove the shorting between input current terminals and the output current terminals.

  3. Close the links between the input and output sides of the current terminal block.

  4. Use the station drawing to return to the service the following equipment.

  • Beaker failure scheme associated with this relay.

  • Disturbance alarm and fault recorder associated with the relay

Request Power Line to be returned to Service

  1. Take crossed reading of all A currents and voltages supplied to the relay and compare them to those for normal load conditions.

  2. Check that readings agree with station meters ( MW, MVAR, P.F, A, V)

  3. Place a copy of the test result in the field maintenance file.
