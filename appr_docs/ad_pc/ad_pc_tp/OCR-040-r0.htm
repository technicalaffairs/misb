Introduction

    It is a protection unit designed for use in three phase networks with the neutral earthed directly or through an impedance

    The operating principle involved consists of detecting over-current contingent on phase to phase faults or earth faults

    Each protection unit consists of an enclosure housing the following component

  • One RMA 422 current measuring relay for detecting earth fault

  • two RMA 420 current measuring relay for detecting phase to phase fault

  • RT 210 definite time - lage relay

    Over current measuring relays are designed on the principle of an electromagnetic balance the balance arm is maintained in equilibrium by two opposing forces

    The first of these forces is the electromagnetic attraction between two high-permeability magnetic cores in the field of an induction coil through which the current be monitored passes.

    The opposing force is produced by a restoring spring

    Definite time lag relay consist of a rotary electromagnet acting on an adjustable time delay device which produces rapid contact movement at the end of the relay period

Work to be carried out

Testing the Relay, and Protection Output

  1. Carefully follow all test procedure steps from 2 to 7
  2. Verify the test result match the relay setting. If not consult technical services
  3. Test trip the circuit breaker associated with the relay
    4.

Preparation

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolating the relay current circuits for test
  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting sheet
    3.

Current Test

  1. Label the wires before removing its from terminal block of the relay

  2. Remove the input current terminals number 1,2,3,4,5 and 6 from the relay

  3. Connect test leads between the current terminals number 1 and 2 on the relay to the current terminals on the test set

  4. Slowly increase the current until the red indication on over-current element of phase R on the relay appears The relay must operate with a current between ± 5% of the setting value

  5. Record this value on the test sheet

  6. Increase the injected current with a value 25% greater than the operating value

  7. Slowly decrease the current until the relay resets

  8. Record this value on the test sheet

    9.

Time Test

  1. Inject current higher than the setting value so the relay picks up firmly

  2. Keep the injected current and switch off the test set

  3. Connect the time delayed contacts number 14 and 26 on the relay to timer terminals on the test set

  4. Switch on the test set

  5. Start the test by injecting current into the relay. The timer will stop once the relay has picked up

  6. Record this value of time on the test sheet

  7. Repeat the above test from step 1 to step 6 for other phase, but inject current at terminal 3 and 4 for phase S and terminals 5 and 6 for phase T

    8.

Trip Test

  1. Request the Operator to switch on the main circuit breaker.

  2. Operate the output-delayed- contact by hand to trip the circuit breaker.

  3. Check that the circuit breaker trip.
