Introduction

The KBCH 120 is a fully numerical design relays offer biased differential current, current, restricted earth fault and over fluxing protections primarily for the protection of tow or three winding transformer units-power transformers, auto transformers or generator.

All models are three phase units with internal vector group compensation and line current transformer ratio corrections, thus eliminating the need for interposing transformer in most cases

Safety Precautions

  • Line isolation is required.

  • A work permit must be issued

  • The maintenance staff should carry the suitable ID safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Outage arrangement must be done.

Tools and Equipment

  • Secondary injection test set type CTR 233

  • Test leads

  • Multi meter

 

 

 

 

 

 

Work to be Carried Out

Current setting test (Id>)

  1. Connect test leads between the current terminals on the relay (21,22) phase a(primary side) and the current terminals on the test set type CTR 233 (phase a)

  2. Slowly increase the current from 0 amp and note the pick-up value at which the relay operates as indicates by the red light emitting diode (TRIP LED) on the front plate

  3. Record the operating (differential) current on the test sheet.

  4. Reduce the current slowly and note the drop-off value at which it resets check that the pick-up and drop-off are within the range shown in table1
                  1.1 (Id>)
        Is  = -------------
               (CT Ratio cor)
      Current level
    Pick - up 0.9 x Is to 1.1 x Is
    Drop - off 0.9xpick-up to 1.0xpick-up

                                                 Table 1
     

    Id> is low set setting which will be found in the cell (Id>)under the setting menu heading. CT ratio cor is the CT ratio correction which is used to compensate for a mismatch in currents due to the line side current transformer ratios, there is one ratio correction factor for the HV side, which is found in the cell (HV ratio cor) and one for the LVI side found in the cell (LVI ratio cor) both if these are found under the setting menu headings. The appropriate CT ratio factor should be used to calculate the current to inject depending upon.

  5. Repeat the above test for each of the remaining phases on the HV side, and for all phases on the LVI side these are listed in table 2

Check the operating time

  1. Repeat step no. 1 in current setting test

  2. Connect the relay time delayed out put contacts number (30-32) to the timer contacts of the test set type CTR 233

  3. Prepare test set type CTR 233 for measuring time delay & inject
    (I normal = 0.5 I setting, I fault = 2 I setting) into the A phase and note the time displayed when the timer stop

  4. Check the operating time for relay recorded by the timer is within the range 30ms to 40 ms

  5. Repeat the same steps no 1,2,3,4 for remaining phases b,c on the HV side & phases on the LV side as listed in table 2 above

 

Circuit breaker trip test

  1. Set up the circuit breaker for trip test

  2. Repeat step No 1 in current setting test

  3. Inject current into the relay until the circuit breaker trips at the time belayed setting

  4. Close the circuit breaker again

  5. Repeat the same steps 1,2,3 for remaining phase B&C on the HV side & phases on the LV side as listed in table 2 above

Return the relay and power equipment to service

  1. Verify the test result match the relay setting if not consult technical service

  2. Make a final check that all switches on the relay are set according to the setting sheet make sure the CT secondary circuit is in normal condition

  3. Remove the CT shorts which were installed in the power supply

  4. Return the relay and power equipment to service by removing test leads from the relay terminals block and test set
