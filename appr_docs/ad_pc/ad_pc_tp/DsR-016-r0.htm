Introduction

This document is Test procedure for Relay SEL 321 by using XS92a set tester.

The relay SEL 321 continues protection elements, many logic schemes, programmable I/O and logic ,event recorder fault location, Metering ,and other junctions.

Outputs from all of the protection elements, logic schemes, etc…. Are available programmable logic ( SEL Logic the control equations ) for tripping, alarming ,keying, communications, and any other deceived functions.

The relay SEL321 includes three communications ports and or front panel user – interface for entering settings reviewing operations, checking metering, and other functions.

Safety Precautions

  • Check that the bus and line disconnected switches are opened and Tagged

  • Make sure C.T secondary are shorted.

  • V.T miniature circuit breaks ( M.C. B’s) is isolated ( opened)

  • Isolate D.C supply to relay, when doing any adjustments or maintenance or the relay SEL 321

  • Isolate Breaker failure ( B.F) scheme associated with relay,

  • Disturbance alarm or fault recorder associated with relay.

  • Isolate breaker trip coils and transfer trip scheme.

  • Auto – recluse , associated must be isolate with relay SEL 321 ( Test plug isolate the all above points

Tools and Equipment

  • Test set type XS92a.

  • Ammeter.

  • Test leads

  • Timer

Work to be Carried Out


Steps To Prepare For Relay Testing

  1. Review technical reference materiel manufacture’s manual

  2. Obtain copy of relay SEL 321 setting from field maintenance file

  3. Obtain outage for line protected by this relay SEL 321 opened and tagged.

  4. Check that the bus and line disconnected switches are opened and tagged.

  5. Block protection signaling to : breaker trip coils and transfer trip schemes

  6. Go to the switchgear and inspect the C.T ‘S and V.T miniature circuit breakers ( MCB’s) , and all cabling supplying the relay.

  7. Check that the cables are in good condition and the all wire connections are tight and clean.

  8. Connect the set test XS92 a to the relay SEL 321 according to the figure 

Testing Procedures 

Zone Reaches 

Zone I            Phase To Earth 

  1. Restore DC supply to relay SEL 321 scheme.
  2. Prepare the set test xs92 or following:

                       MOD                    ZZ ( distance)

                         ZL                      UP to setting Z1

                       PZL                      Angle of setting line ( 0-360 º )

                        K0                           0.7

                        FC                      Fault code forward 1/ 1 ( phase R to earth)

                        SQ                         1F1

                         A                            2

                        TF                       Time fault Zone (250 m sec.)

                      DZL                      Step parameters and range limits ( 0.1)

                    Start (-)                   Injection of testing signals.

Notes:

  1. We entered the above parameters according to the Flowchart of set test XS92a 

  2. When the relay gives tripping signal press ◙ to stop injection.

  3. Read the values of testing in hand terminals of set test XS92a. And compare this value with the value of setting in test sheet of relay SEL 321.

  4. Press the reset button on the relay front panel and check that the led indicator reset and the alarm contacts open

  5. Repeat the above test with the phases ( S ) by using FC ( fault code)( 112) & phase (T) by using FC (113) . Make sure relay SEL 321 gives trip signaling with all phases R,S,T in forward direction.

  6. Repeat the above test, but in the reverse direction by using FC fault code (011),( 012) ,(013), make sure that the relay SEL 321 is blocked in the reverse direction .

Phase To Phase
  1. Entered the all above parameters, but in fault code (FC) as following: -

                            Phase    R- S                      121   Forward direction ( Trip)

                            Phase    S – T                     122   Forward direction ( Trip)

                            Phase   T – R                      123   Forward direction ( Trip)
In the reverse direction 021,022,023 (block)

Zone II 

  1. Repeat the above test for zone I but the value of ZL up to Z2(setting ) and time fault ( TF) Up to time of zone (2)

  2. Make sure that the relay SEL 321 gives trip signaling in the values of setting and in the forward direction and the relay gives block in reverse direction.

Zone III

  1. Repeat the above test for zone II but the value of ZL up to Z3(setting) and time fault ( TF) up to time of zone (3).

  2. Make sure that the relay SEL 321 gives trip signaling in the values of setting and in the forward direction but in reverse direction the relay blocks.

Zone IV

  1. Repeat the above test for zone II but the value of ZL up to Z4(setting) and time fault ( TF) up to time of zone (4)

  2. Make sure that the relay SEL 321 gives trip signaling in the values of setting and in the forward direction but in reverse direction the relay SEL 321 blocked.

Measuring The Operating Times

  1. The operating time of distance relay SEL 321 should be measured by using tripping contact ( free contact)

  2. Connected the free contact to " + "  and  "__\__ " in the front panel of set test XS92 a.

  3. Enter the all parameter of zone I by ZL = 0.5 Z1 (setting)

  4. When the trip occurs the injection automatically stopped by free contact of relay SEL 321. Then read the value of time zone I in the hand terminal of set test XS92a.

  5. Repeat the above step to measuring the operating time of zone II , III, IV

                                Time of Z1                              At ZL = 0.5 Z1

                                Time of Z2                             At ZL = 0.5 (Z1+Z2)

                                Time of Z3                             At ZL = 0.5 (Z2 + Z3)

                                Time of Z4                             At ZL = 0.5 (Z3 + Z4)

Note
How to find " free contact " for relay SEL 321 ?

  1. Select any contact in the relay SEL 321 and measured DC ( Not Found)

  2. Measured (ohm) buzzer by using avometer for this free contact selected (Not found).

  3. Make trip by distance relay SEL 321 and measured ohm (buzzer) for this free contact (measured of buzzer) with trip signaling. 

  4. Connect this free contact selected to set test XS92 a in two point “ + “ &" __\__ " in the front panel of set test

End To End Test For Relay SEL 321 

  1. Repeat the test procedures of zone II and make sure that the trip occur at time of zone II (T2)

  2. Make brigade ON receive signal.

  3. Make sure that the relay SEL 321 gives trip signaling of zone II at time of zone I (T1)

Switch Into Fault ( SOIF) Test 

  1. Repeat the test of zone IV (SQ = 1F1) make sure that the relay SEL 321 gives trip signaling in time of zone IV (T4)

  2. Repeat the above test of zone IV (SQ = OF1) in set test XS92a make sure that the relay SEL 321 gives trip signaling in time of zone I ( T1)

Return Protection Scheme And Associated Equipment To service

  1. Return relay SEL 321 to service by withdrawing test plug handle from the relay.

  2. Make sure C.T secondary are opened.

  3. Return relay to service by restoring supply from voltage transformer (V.T miniature circuit breakers (MCB’s ) is closed.

  4. Place the associated barker failure ( B.F) protection scheme back in service.

  5. Restore the previously blocked disturbance alarms and fault recorders associated with this protection.

  6. Return the relay power line carrier (PLC) links to their normal position.

  7. Turn ON the associated auto – recluse scheme.

  8. Turn ON the power supply DC to relay SEL 321

  9. Take crossed readings of all AC currents and voltages supplied to the relay SEL 321 and compare them to those for normal load conditions.

  10. Check that these receding agree with station meters MW, MVAR, Power Factor, Amperes, and voltages.

  11. Place a copy of the test results in the field of maintenance file and test sheet
