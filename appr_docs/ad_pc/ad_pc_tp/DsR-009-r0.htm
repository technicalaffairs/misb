Introduction

These procedure details the steps required to completely check the proper operation and calibration of the L3WYS and L3WYaS relay. It is recommended this procedure be carried out on yearly bases.

The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area,

  • Line isolation is required.

  • Make sure that CT secondary circuits are shorted before isolating the current circuit for test current injection.

Tools and Equipment

  • ABB Relay Test set Type QZW 415.

  • Phase rotation meter.

  • Small slot screwdriver.

  • Test leads, 5 Amp, Clip-on ammeter, Timer.

Work to be Carried Out

Steps to Prepare for Relay Testing

  1. Review technical reference material and manufacturer's manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Obtain outage for line protected by this relay.

  4. Check that the line and bus disconnect switches are opened and tagged.

  5. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB's) and all cabling supplying the relaying.

Testing the Relay
  1. Carefully follow this test procedure.
  2. Verify the test results match the relay setting, if not consult technical services.
  3. Trip the circuit breaker associated with the relay.

Checking the Under Impedance Starting Elements

  1. Referring to the calibration curves fig 2 and the values calculated on the test sheet determine the %Z value corresponding to the relay setting of the starting element.

  2. Turn the %Z switches on the test set to agree with calculated impedance of starting element.

  3. Select single phase to ground faults by turning switch "g" to position 2.

  4. Select the phase fault by Turing switch "f" to the position R.

  5. Push down and turn the voltage control switch "n" to the on position "I"

  6. Press the push button "nl" and observe that the auxiliary relay PA/R, PE, PtaW and tripping contactor PD will pick up.


    7. The red trip indicator lamp on the test set will light up when the tripping contactor PD picked up.

  7. Change the %Z which set on the test set by turning up or down the %z switches step by step

  8. Press the push - button nl every step until PA/ R does not pick up.

  9. Record this value on test sheet 1.

  10. Repeat the above test for other phases S and T by turning switch "F" to S and T position respectively.

  11. Repeat the above steps for phase to phase fault by turning switch "g" to position I)

Checking the Pick up Value for Measuring Element for Phase to Ground Fault

  1. Referring to the calibration curves fig 3 and the values calculated on the test sheet determine the %z value corresponding to the relay setting of the 1 st time step.

  2. Turn the %z switches on the test set to agree with calculated zone 1 setting.

  3. Select single phase to ground faults by turning "g" to position 2

  4. Select the phase fault by turning switch "f' to the positions R, S, and T respectively.

  5. Push down and turn the voltage control switch "n" to the on position " I ".

  6. Press push button (nl) and observe when " PA/R, PEW and tripping contactor PD " pick up (the red trip indicator lamp on the test set will light when the tripping contactor operates).

  7. Change the %Z which set on the test set by turning up the %z switches step by step until PS 11 pick up with PA/R , PE , and the tripping contactor PD.        Note :   PTaW does not pick up.

  8. Repeat the above test 3 times to determine the average pick up point.

  9. Record the pick up point for this phase to ground on test sheet 1.

  10. Repeat the above test for the other phases S, T

  11. Repeat the above test once again but this time increase %z setting on the tester until the relay operates in the 2nd time step. The contactor PS 11 will pick up with the above flag and PTaW does not pick up. Change the %Z which set on the test set by turning up the %z switches step by step until PS III pick up with PA/R, PE, and the tripping contactor PD

  12. Repeat the above test once again but this time increase %z setting on the tester until the relay operate in the third time step. The contactor PS III will pick up with the above flag and PTaW does not pick up. Change the %Z which set on the test set by turning up the %Z switches step by step until the relay operate in last time step, the contactor PD will pick up by the fourth time contact of timer.

Checking Tripping Times

    General: originally the operating times of the L3 relay components were checked using Multi Amp. Timer connected to the QZW 415 tester.

The Following Procedure Gives the Steps for using this Timer

  1. Toggle switch " d " on the QZW 415 to the " see " position to select time measurement.

  2. Use two test leads to connect the digital timer " stop " inputs to terminals 12 & 13 (Y1) located in the top right comer of the QZW tester

  3. Use two test leads to connect the digital timer " Start " inputs to the terminals 3 & 4 located in the bottom right comer of QZW tester just above switch n..

  4. Connect an AC supply to the digital timer circuit.

  5. Press the reset button for the digital timer display.

  6. Time tests for phase to ground faults

  7. Repeat the test of checking the Pick up Value for Measuring Element for Phase to Ground Fault from steps 1 to 5 as given above to measure the Zone I operation time for each phase to ground unit.

Note: Set the line impedance %Z switches to the value which equals half of the normal operation of the zone I Impedance.

  1. To perform the time test press and turn switch nI in the closed position.

  2. Press down and turn voltage control switch n to the on position (I) the time will start and then stop once the PD trip relay has picked up.

  3. Record the time which appear on the timer display on test sheet 1.

  4. Repeat the above test for the other two phases

  5. Repeat the above test but change %Z setting on the QZW 415 tester to a value = (%Z1) + (%Z2 - % Z1) /2 until the relay operate in the 2 nd Zone time.

  6. Repeat the above test but change %Z setting on the QZW 415 tester to a value = (%Z2) + (%Z3 - % Z2) /2 until the relay operate in the third Zone time.

  7. Repeat the above test but change %Z setting on the QZW 415 tester to a value = (%Z3) + (%Z4 - % Z3) /2 until the relay operate in the starting Zone time.

  8. Remove the DC isolating plug from the relay to de energize the DC circuits.

  9. Time tests for the ph - ph unit

  10. Select phase to phase faults by turning switch " g " to position " I ", and perform time tests for the phase to phase unit by repeating the above steps.

  11. Remove the DC isolating plug from the relay to de energize the d.c circuits.

  12. Switch off the testers.

  13. Remove all the test lead between the relay and the tester.

 

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set.

  2. Make sure CT and VT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
               - Breaker trip coils.
               - Breaker failure scheme associated this relay.
               - Disturbance alarm and fault recorder associated with this relay.

Request Power Line to be returned in Service

  1. Take crossed readings of all AC currents and voltages and compare them to those for normal load condition

  2. Check that these reading agree with station meters (MW, MVAR, Amperes and Power factor)

  3. Place a copy of the test result in the field maintenance file.
