Introduction

    This procedure details the steps required to completely check the proper operation and calibration of the RADSS-B differential relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

This document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • Work order must be issued.

Tools and Equipment

  • Test plug Handle type RTXH 18.

  • Two Currents Source.

  • Two multi-meter.

  • Diode (Rectifier).

  • Two adjustable resistor 0 - 3.3 A, 200 W.

Work to be Carried Out

Steps to Prepare for Relay Testing

  1. Review technical reference material manufacturer's manual.

  2. Obtain outage for transformer protected by this relay.

  3. Check that the disconnect switches from all transformer sides are opened and tagged.

  4. Inspect the secondary circuits of CT, and all cabling supplying the relay.

  5. Before inserting the test plug into the relay check from the station drawing that the test plug will isolate:

  • Breaker trip coils.
  • Breaker failure scheme associated with this relay.
  • Fault recorder, event recorder and disturbance alarm associated with this relay.
  • If the test does not isolate one of the pervious steps you have to isolate it.

Testing the Relay and the Relay Outputs

  1. Carefully follow this test procedure steps.

  2. Verify the test result match the relay setting if not consult the technical services.

  3. Test trip all breakers associated with the relay.

Preparations

  1. This procedure is used to perform off load test.

  2. Look at the EEHC panel nameplate and check that EEHC system designation nameplate agrees with the transformer that has been taken out of service.

  3. Using a clip-on ammeter check there is no current flowing in the terminal no.3, 4, 5, 6, 7, and 8 on the rear side of the test switch.

Checking the Restrained Operating Current (Isr)

  1. Insert the test plug RTXH 18.

  2. Connect the current source to the terminals 3 and 12 and check with the relay set to 0.25, 0.35 and 0. 5 0 times In (by slowly increasing the current), that the relay will operate at current equal to the set current +/- 10%

  3. Relay operation can be checked by observing the DC voltage between terminals 17 and 16 on test plug or light emitting diode I> on the RXSGAI UNIT should light

  4. RTXH 18 and the light emitting diode I> on the RXSGAI UNIT should light.

  5. Repeat step 1 to step 3 for terminals 6 with 12, 4 with 13, 7 with 13, 5 with 14 and 8 with 14.

Return Relay to Service and Associated Equipment

  1. Switch of the Test equipment.

  2. Withdraw the test plug from the relay.

  3. If any equipment associated with the relay has been isolated without the test plug it should be return to service by the following restoration steps:

  • Fault recorder event recorder and disturbance alarm.

  • Breaker failure scheme.

Request Power Transformer to be Returned to Service

  1. Take crossed reading of all AC current and voltage supplied to the relay, and compare it with the load conditions that reading should agrees the station meters reading (Mw/MVAR/power factor).

  2. Place a copy of the test result in the field maintenance
