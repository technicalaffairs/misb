Introduction

    The SU91 relay is a solid state relay. The relay detects the fault on the busbar, and trips all the breakers connected to the bus. The relay requires current inputs from all circuits connected to the bus. All the current transformers used in busbar relay should have the same ratio and they should be on the full winding with a minimum knee point. The secondary circuits of the current transformer are connected together (in parallel) in CT Junction box the busbar differential protection is based on differential voltage measurement. 

    The output of the SU91 busbar differential relay initiates two LOR Lokout relay (86B) the output of these two lokout relay are used to trip and block all circuit breakers connected to the bus and to initiate the station annunciator and sequence events recorder. The trip and block contacts of the lockout relay are provided with test switches to remove the output circuits during testing.

    This procedure details the steps required to completely check the proper operation and calibration of the SU91 relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 2 sections. The first steps to prepare for relay testing describes the activities required prior to testing, could be performed days before the actual test. The second section is the actual test. 

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category

  • Safety fence with caution marks surrounding the work area is required.

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted ,VT secondary circuits are open and Busbar isolation .

Tools and Equipment

  • Secondary injection test set

Work to be carried out

Steps to Prepare for Relay Testing

  1. Review technical reference material Manufacturer's manual.

  2. Obtain copy of relay setting from field maintenance file.

  3. Obtain outage for the bus bar, which is protected by this relay.

  4. Check that the bus disconnect switches are opened and tagged.

  5. Inspect the CT secondary circuits, the VT Miniature circuit Breakers (MCB'S), and all cabling supplying the relay.

  6. Isolate the following

  • All Breakers Trip coils.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

Test Procedure(s)

  1. From DC drawing open switches (output circuits) which will block the trips for all breakers and initiate the breaker failure.

  2. From substation drawing AC drawing first short circuit the CT's circuits and open test switch, which fed the SU91 relay by AC current and Insure that the relay is completed isolate from secondary circuits of current transformers.

  3. Manual test (see fig. 1)

  4. Place the keys in module MSZ 81(S3) in position (1).

  5. Place the phase selector switch (S2) in MSZ91 in position R

  6. Push the trip push button (S4) in module MSZ81.

  7. The relay will operate and start led will illuminate for phase(R) in unit UT 91 and also trip red led will illuminate in unit UZ92.

  8. Push the reset button (S1) in MSZ81 to reset the relay

  9. Repeat steps from step 2 to step 5 with selector switch (S2) in MSZ91in (S) and (T) position respectively.

  10. Return the key (S3) in MSZ 81 to position (0)

Injection Test By Using Secondary Injection Test Set

  1. Place the key (S1) in module MSZ91 in test position.

  2. Determine the zero point voltage of the AC voltage test set.

  3. Connect the terminal of the AC voltage of the test set to the two terminals test [2, 3] in module MSZ91 (Note: Ensure that the zero voltage point of test set is connected to [0, 9] earthed point in module MSZ91).

Test the Supervision Voltage Unit (UT91)

  1. Select the phase faulted by turning switch (S2) in NSZ91 to phase R.

  2. Increase the AC voltage from 0 volt to the setting value in UT91 say 10 volts, UT91 will operates by Red LED and LED will illuminate.

  3. Decrease the AC voltage to 0 volts.

  4. Make reset for UT91 unit.

  5. Repeat steps from step 1 to step 4 with selector switch (S2) in module MSZ91 in module MSZ91 in (S) and (T) position respectively.

  6. Record the above results in test sheet.

Test the High Impedance Unit [UZ92]

  1. Switch off the DC/DC converter to 0 position.

  2. Withdraw the supervision unit UT 92.

  3. Switch on the DC/DC converter to (I) position.

  4. Select the phase faulted by turning switch (S2) in MSZ91 to phase (R).

  5. Increase the AC voltage from 0 volt to the setting value in UZ 92 say (8ov), the relay will operate and red led will illuminate.

  6. Decrease the AC voltage to 0 volt.

  7. Make reset for this unit, from white reset bottom on UZ92 Module.

  8. Repeat steps from step 4 to step 7 with selector switch (S2) in MSZ91 in (S) and (T) position respectively.

  9. Record this results in test sheet.

  10. Switch off the DC/DC converter to (0) position.

  11. Insert the super vision unit UT 92.

  12. Switch on the DC/DC converter to (1) position.

  13. Switch off the AC test set.

  14. Withdraw the test terminals from the test unit MSZ91.

  15. Return the key (S1) in MSZ91 to service position.

  16. Return the keys (KS) and (S2) on MSZ81 module to service position.

Return Relay and Associated Equipment to Service

  1. Return the AC test switch fed the SU91 by AC current to its normal position.

  2. Return the output test switches (output circuits ) to normal position and any equipment associated with the relay and isolated should be returned to service by the following restoration steps

  • All breakers trip coil and transfer trip scheme.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.
