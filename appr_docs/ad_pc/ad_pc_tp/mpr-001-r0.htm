Introduction

This document outlines the testing procedure P1, which must be carried on an annually bases as indicated by Y1 in the above title block,

The micro-processor controlled relays of the MCX912 provides a large number of protection functions, This relay can be used to replace several conventional relays.

The first draft  initiated by eng. Fayza El-Sharkawy, tested by eng. Shirin Hamdy & eng. Azza Salama at 29 April 2003, revised by Quality Control eng. Adel El-Nagar at EDEP Co., accepted by General Director eng. Mahmoud Saker, and approved by eng. Moustfa Alam Head Sector of Talkha Power Station.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area is required

  • Make sure that the the pump mechanically and electrically isolated.

  • Switch off DC power supply before inserting or withdrawing any card from the relay.

  • Switch off the AC power supply of the motor.

Tools and Equipment

  • Secondary injection test.

  • Test leads, and test plug handle.

Work to be Carried Out

Preparation

  1. Make Sure CT secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

Testing the Relay

  1. Connect the test leads between  the current terminals ( A2 – A3  ) phase R. (figure 1)

  2. Connect the test leads between the trip contact  ( A4 – A5 )

Negative Sequence Test      (Phase R)

  1. Increase the current slowly until the relay trips

  2. Record the current value and the time value at which the  relay trips  in the test sheet  .

Earth – Fault Test       (Phase S)

  1. Connect the test leads between the current terminal ( A6 – A7 ) phase S. (figure 1)

  2. Increase the current very  slow until the relay trips  .

  3. Record the current and the time values in the test sheet  .

Over Current Test       (Phase R)

  1. Change the negative sequence time to 10 sec.

  2. Increase the current until the relay trips

  3. Record the current and the time values in the test sheet  .

Short Circuit Test         (Phase R)

  1. Increase the current to the S.C setting until the relay trips

  2. Record the current and the time values in the test sheet  .

Thermal Over Load Test

  1. Increase the current to a large value until the relay trips

  2. Record both current and the time values in the test sheet  .

Repeat the steps no. 4, 5 & 9 to 15 for phase T,

Note: Terminals should be ( A10 - A11 ) at step 2 of Testing the Relay

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set i.e.
    -     Disconnect the test leads from the terminals ( A2 - A3 ) , ( A6 - A7 ) , ( A10 - A11 )
    - Disconnect the test leads between the trip contact ( A4 - A5 ). 

  2. Make sure CT and PT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
           - Breaker trip coils.
           - Breaker failure scheme associated this relay.
           - Disturbance alarm and fault recorder associated with this relay.
