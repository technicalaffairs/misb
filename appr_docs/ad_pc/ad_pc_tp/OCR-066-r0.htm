Introduction

The KCGG 142 is a numerical 3 plase over current and earth fault relay with 3 stages of phases and earth fault protection I>/I0>, I>>/10>> & I>>>/I0>>>, which can be used for 3 phase differential protection or restricted earth fault (ref) protection

The output relay that is to trip the circuit breakers must be allocated in the relay masks t > a ; for t > b and t > c

The first draft initiated and tested by eng. Khaled Saad Ahmed, accepted by eng. Fatma Elzhraa Mohamed: general manager, approved by eng. Abdel Maneam Rabeay: head sector of El Nakra Power Station, and certified by eng. Mohamed Kamel Ahmed: UE zone quality control general director and MPIS coordinator.

Safety Precautions

  • Line isolation is required.

  • A work permit must be issued

  • The maintenance staff should carry the suitable ID safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Outage arrangement must be done.

Tools and Equipment

  • Secondary injection test set type CTR 233

  • Test leads

  • Multi-meter

  4. Decrease the current from the tester type ctR 233 until the relay rest and the contact change from RL 0 ( I → 0)

  5. Repeat the same steps 1,2,3,4 for phases B, C on terminal (23,24) for phase B, and (25,26) for phase C

Check the operating time

  1. Repeat step no. 1 in current setting test.

Circuit breaker trip test

  1. Set up the circuit breaker for trip test.

  2. Repeat step No 1 in current setting test.

  3. Inject current into the relay until the circuit breaker trips at the time belayed setting.

  4. Close the circuit breaker again.

  5. Repeat the same steps 1,2,3 for remaining phase B&C on terminal (23,24) for phase B and (25,26) for phase C

Return the Relay and Power Equipment to Service

  1. Verify the test result match the relay setting if not consult technical service

  2. Test trip the circuit breaker associated with the relay

  4. Make sure the CT secondary circuit is in normal condition

  5. Remove the CT shorts which were installed in the power supply
