Introduction

Series [PT - 40] maximum current relays are used in protection circuits as elements responding to an increase of current. Relays having letter (T) in the designation [PT - 40-T] are intended for operation in the tropics. The relay of a tropical design differs form the general commercial type in that it is made of special materials.

The first draft initialed by eng. Mohamed Mostafa, and tested at 01 Feb. 2002, at Samalout 500, ME

Safety Precautions

  • Line isolation and work permit required.
  • Make sure CT secondary circuits are shorted  and VT secondary voltage circuits are open.

Tools and Equipment

  • Test unit [secondary injection current] (YΠ3 -1)
  • Clip on ammeter.
  • Test leads.
  • Voltmeter.

Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical reference material: Manufacturer's manual.
  2. Obtain copy of relay setting from filed maintenance file and compare it with the setting on the relay.
  3. Obtain outage for line protected by this relay.
  4. Check that the bus and line disconnect switches is opened tagged.
  5. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB'S), and all cabling supplying the relay.
  6. Use the station drawing to isolate the following.
  • Breaker trip coils.
  • Breaker failure scheme associated with this relay
  • Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection Output

  1. Carefully follow these test procedures.
  2. Verify the test results match the relay setting, if not consult technical services.
  3. Test trip the breaker associated with the relay.

Prior to Starting Testing Procedure (S) Ensure the Following Steps

  1. This procedure is used to perform off load relay testing.
  2. Check that the panel nameplate agrees with the EEC designation of the line that has been taken out of service for maintenance.
  3. Check there is no AC current from CT by using clip-on ammeter.
  4. Check there is no AC voltage from VT by using voltmeter.
  5. Check all external connection to the relay are in good condition and check that the terminal block wires are tight.

Testing Procedure
  • Checking the pick-up current [Iop.] and reset value [Ir.] :
  • Remove the normal current blocks and put the block test as the following :

  • Use the test unit [YII3-1] for injection current.
  • Clean the relay contact and measure it by vometer (Ω).
  • Connect the relay contact (N.O) on vometer position (Ω) or terminal contact of test unit (YII3-1).
  • Contact the terminal current out put (A,B) for the (YII3-1)on the terminal of test block [ph.A-O].
  • Injection the current from (auto.) and increase it until the current relay picked-up and the contact is closed and read the valve of current from the test unit, is [Iop.], decrease the current from the (auto) until the relay is reset [de-energized] and the contact is open.
  • Read this value and its (Ir.).
  • Recovered the value of [Iop. & Ir.] in test sheet.
  • With the same steps we. Inject current on [ph.B-O] & [ph.C-O].
  • We measure the op. Current [Iop.]& the rest current [Ir.] in each case.
  • Record [Iop.& Ir.] in test sheet.

Note

  • In case two current relay connection series and there is one has lower value from the other, we make bridge on the coil terminal after that we testing at lower value relay.
  • From possible in stead of inject current on the test block. We inject the current on the coil terminal directly on the relay [one after one].

Checking the Line Trip by this Protection

  • Connect the trip and signal terminal.
  • Leave the (B.F.) terminal is OFF.
  • Connect the (CB) of the line and the isolator is OFF.
  • Connect the strap of the over current protection.
  • Connect the strap (H) for the auto reclosure, (3 ph.).
  • Inject the current on (ph.A-O) until the relay operate (picked-up) and the contact is closed and trip the line and disconnect the (DC) of C.B
  • Connect the (B.F) terminal.
  • Put the normal block in it place.
  • Check all the terminal is good fixed.
  • Finish the permission.
  • Connect the line and measure the current by clip on meter and record it.
  • Measure the voltage by voltmeter.
  • Put the test sheet in the field maintenance file.
