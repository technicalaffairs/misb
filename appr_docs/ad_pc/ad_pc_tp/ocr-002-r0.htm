Introduction

RC91 is a single phase static over current relay with an inverse line characteristic the RC91 module is included in LZ 96 relay configuration (A36), this relay provides back-up directional ground over current (DGOC) protection (Device No.67G), this relay has a power supply module (NF 92) ,test unit (XX91).  

The first draft initialed by Eng. Mohamed Mostafa, and tested at 01/01/02, Samalout 500, ME

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Wear safety and healthy equipments as safety helmet, safety shoes ..etc

  • Line isolation must be done.

  • Make sure CT secondary circuits are shorted before the relay current circuits for test current injection.

Tools and Equipment

  • Test unit XS92a [secondary injection unit].

  • Voltmeter, Clip on ammeter, Test leads.

Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical reference material: Manufacturer's manual.
  2. Obtain copy of relay setting from filed maintenance file and compare it with the setting on the relay.
  3. Obtain outage for line protected by this relay.
  4. Check that the bus and line disconnect switches is opened tagged.
  5. Inspect the CT secondary circuits, the VT miniature breakers (MCB' S), and all cabling supplying the relay.
  6. Use the station drawing to isolate the following.
  • Breaker trip coils.
  • Breaker failure scheme associated with this relay
  • Disturbance alarms and fault recorder associated with this relay.

Testing the relay and protection output

  1. Carefully follow these test procedure steps .

  2. Verify the test results match the relays if not consults technical services.

  3. Test trip the breaker associated with the relay.

Prior to starting testing procedures ensure the following steps

  1. This procedure is used to perform off load relay testing.

  2. Check that the panel name plate agrees with the EEC designation of the line that has been taken out of service for maintenance.

  3. Check there is no AC current from CT by using clip-on  ammeter

  4. Check there is no AC voltage from VT by using voltmeter.

  5. Check all external connections to the relay are in good condition and check that the terminal block wires are tight.

 Testing Procedure

  1. Prepare the test unit [XS92 a with YX91] as in figure (1 , 2) and preparing steps for test unit .

  2. Disconnect the DC - DC. converter (N F92).

  3. Put the test plug (YX91) in test unit (XX91) for this relay (RC91).

  4. Connect the D C. D C. converter (N F92).

  5. Make XS92a  ON.
