Introduction

The MCAG is a static restricted earth fault relay available single and triple pole manufactured by GEC ALSTHOM. This relay is used for protected the power transformers and generators. This is a current operated having seven equally spaced setting. The setting ranges are given below:

0.025                    0.1 A
0.05                      0.2 A
0.1                        0.4 A
0.2                        0.8 A
0.25                      1.0 A
0.5                        2.0 A
1.0                        4.0 A

The relay may be used for any type of high impedance circulating current protection
When used as a high impedance relay a stabilizing resistor must be connected in series with the current circuit

Safety Precautions

  • Make CT secondary circuits are shorted before isolating the relay for test

  • Switch on the MCB for DC supply

  • The equipment under test must be out of service

Tools and Equipment

  •  Secondary injection test

  • Multi finger test plug type MMLB 01 for use with block type MMLG is fitted

  • Miniature split plug type MMLB 03 to fit relay plug bridge

  • Three calibrated multi meters 0-10A ac, 0-400 V ac

Works to be Carried Out

Terminals of the relay are normally as below, but reference should always be made to the relevant diagram

Relay type AC current input Normally open contacts
Single pole 27-28 1,3 and 2,4
Triple pole R   23,24 1,3 and 2,4
Triple pole S   25,26 1,3 and 2,4
Triple pole T   27,28 1,3 and 2,4

The contacts are normally connected in parallel for the three phases but aversion of the relay having contacts brought out separately is available

Commissioning test

  • If the relay is wired through an MMLG test block it is recommended that all secondary injection tests should be carried out using this block.
  • Ensure that the main system current transformer are shorted before isolating the relay from the current transformer in preparation for secondary injection test.
  • When type MMLG test block facilities are installed it is important that the sockets in the type MMLB 01 test plug, which correspond to the current transformer secondary winding, are linked before the test plug is inserted into the test block. Similarly, a MMLB 02 single finger test plug must be terminated with an ammeter before it is inserted to monitor CT secondary currents.

Relay CT shorting switches

With the relay removed from its case, check electrically that the CT shorting switch is closed

Secondary injection testing

  1. Connect the circuit as shown in figure (1) and ensure that the current transformer primary is open circuit and that if any earthing connection are fitted
  2. Increase the current until the relay just operates
  3. Record the operating current of the relay which measured by ammeter A2
  4. Record the total current which measured by ammeter A1
  5. This test should be repeated for each pole of the relay
