Introduction

This procedure details the steps required to completely check the proper operation and calibration of the RAZFE  distance relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The first draft initialed by eng. Saher Farok Mohamed, tested at 22 Aug.1996  at Metro by eng. Saher Farok Mohamed, approved by general director of  Protection rng. Maher Gerges.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required,

  • Line isolation is required.

  • Make sure CT secondary are shorted before isolating the relay current circuits for test current injection.

    •

    • Wear your safety and healthy equipment as safety helmet, safety shoes,  etc.

  • Ensure that the line isolation must be done.

Tools and Equipment

  • ASEA test set type TURH

  • Voltmeter

  • Clip-on ammeter

  • Test leads Timer

  • test plug type RTXH 18

  • 440/ 220/110V Auxiliary transformer (if local supply is only 220 v or380v)

Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical reference material manufacturer's manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay..

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Inspect the Secondary circuits Of CT, VT Miniature Circuit breakers ( MCB, S ) and all cabling supplying the relay.

Before inserting the test plug handle into the relay, check from the station drawing that the test plug will isolate:

  • Breaker trip coils and transfer trip scheme.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms or fault recorder associated with this relay.

  • Auto-reclosure associated with this relay.

If the test plug does not isolate one of the last four steps you have to isolate it .

Testing The relay, relay scheme, and protection outputs

  1. Carefully follow this test procedure steps.

  2. Verify the test results match the relay setting - If not consult Technical services.

  3. Perform checks of the acceleration of the relay, breaker failure and reclosure, Separate technical procedures should be used to test each of these systems

  4. Test trip all breakers associated with the relay.

  5. Return relay and associated equipment to service

  6. Return relay to service by withdrawing the test plug handle from the relay

  7. If any equipment (associated) with the relay has been isolated without the test plug it should be returned to service by the following restoration steps:

  • breaker failure scheme protection

  • disturbance alarms and fault recorders

  • acceleration scheme.

This Procedure is used to Perform off Load Relay Tests

  1. Look at the EEHC panel nameplate and check that this EECH system designation name plate agrees with the line - that has been taken out of service for maintenance.

  2. Using a clip-on ammeter, check there is no current flowing in the RAZFE relay circuits wires going to the rear of the test switch RTXP 18 in the distance relay terminal numbers 3141516nl 8/9/10 should have no current.

  3. Using AC voltmeter, check there is no voltage on the terminal numbers 11/12/13/14 on the rear of the test switch RTXP 18 which came from voltage transformer.

Preparations

  1. Prepare the relay test set for testing. Place the two cases one on top of the other with the main case at the bottom Connect the two cases together by using the RTXG connector on the top case. Lock the connector.

  2. Connect terminals RSTN of the test set to 110 v three-phase with neutral, 50/60 HZ, with known phase sequence, but allow the main switch H to remain open, position 0, until further notice.

  3. If only 220 V or 380 V three-phase Local supply is available a separate auxiliary transformer can be used to obtain 110V to the test set. Fig.1 shows the reconnection possibilities of this transformer.

  1. Connect terminal RI, SI, TI, NI, and RU, SU, TU, NU of the test set to the corresponding test terminals as shown in Fig. 2. short- circuit terminals 4-6-8-10 of the test plug handle RTXH 18.

                                        Fig. 2:  test setup with TURH.

  1. Insert the test- plug handle in the test switch of the distance relay and check that all tripping circuits are open. Keep the dc supply to the terminals of the relay. ( see the connection diagrams for the supplied distance relay

  2. Set the switches:

  • Q in position 1, i.e. first quadrant in the impedance diagram.

  • M in position I, until further notice.

  • P in position I, i.e. voltage transformer ratio equal to 1.

  1. Set the product of the setting of the current selector switches S1 and S2, approximately equal to the rated current of the distance relay, i.e.
    SI x S2 = 1 x 1 = 1 for rated current 1 A
    SI x S2 = 1 x 2 = 2 for rated current 2 A
    SI x S2 = 1 x 5 = 5 for rated current 5 A

  2. Turn the main switch H to position 1.

  3. With the switch AF in position l, the light -emitting diode, located above AF, is lit if all terminals R, S, T and N are supplied with voltage. the light-emitting diode remains dark if one or several phases are missing supply voltage.

  4. The phase sequence in the a.c. supply is checked with the switch AF in position 2, the light-emitting diode is lit if the phase sequence is R, S, T, the diode is dark if the phase sequence is reversed.

  5. Set the switch H to position 0.

Power Swing Blocking Relay Measuring Element

  1. The characteristic of the power swing relay measuring element is shown in Fig. 8

  2. Check on printed board assembly B11 that the power swing relay programming switch K4 has a plug in position 1 or 2. If the plug is in position 3 the power swing relay is decommissioned and only p-flag will arrive when testing according to below.

  3. Set switch F in position RST.

  4. If the distance relay is provided with auxiliary relays for cooperation with communication links according to the permissive tripping scheme principle, the relay should give signal for “ Communication send" during the tests above.

  5. Set switch GR and potentiometer FR at their highest values and switch Gx and potentiometer Fx at position 0, i.e. the voltage and the current will be in phase 0°.

  6. Turn switch H to position I and reduce the setting of GR until the distance relay operates, which can be checked by observing the operation of the tripping or "power swing" indications of the relay.

  7. Increase GR one step and reset the relay and the indicators.

  8. Reduce slowly the setting of FR until the "power swing" operates and wait about 2 seconds. (The tripping indication shall now operate.)

  9. Turn switch H to position 0. Read off the values of GR and FR and calculate the operating impedance which should be equal to the 100%-line limit in Fig. 8 according to the formulas XM  & RM are  given in the above.

  10. Increase the set value of the resistance in the test set to approximately 110 %.

  11. Reduce the setting of GR or FR below the operating value obtained in step No. (4) of the relay and observe the tripping indication of the distance relay. This indication should appear approximately 2 sec. after the reduction of the setting. Check that the signal for power swing blocking is obtained during the above interval of 2 sec. as well as "power swing" c blocking indication (P).

  12. Keep GR and FR at a value below the operating value of the relay and set switch M to position 0 (full voltage)'. Reset the relay and the indicators.

  13. Turn switch M to position 1 (set values of GR and FR) and observe the tripping indication. This indication should appear instantaneously but no signial or indication for power swing should be obtained.

Time Measurement

  1. Make sure that the relays which may have been removed from the distance relay in conjunction with previous measurement are put back to their respective positions.

  2. The steady-state fault conditions for different types of faults can be simulated with the aid of relay test set and the total operating time of the relay for different steps can then be checked with a separate timer.

  3. Please note that the test set does not reproduce the dynamic phenomena.

The test at zones (2) and (3) can be carried out in the following way

  1. Insert the test-plug handle RTXH 18 in the distance relay. Connect the contact H 2 of the test set to obtain simultaneous start of the timer and of the distance relay when turning the main switch H to position 1. the tripping pulse from the relay is connected to stop the timer

  2. The connection is made according to Fig. 2 which applies to a synchronous timer type 51 of AEG' s make. When the timer is of the design that can be stopped by D.C supply the stop circuit for the timer is connected directly to the test terminal in the trip circuit of the distance relay.

  3. On the other hand, the stop circuit of the timer must be insulated, an extra auxiliary relay with known operating time should be provided. the trip circuit of the distance relay should operate this extra relay.

  4. Set the switch F of the test set in position e.g. TR and switch Q in position 1

  5. Set the switches S1, S2, GX , GR and potentiometer FX and FR so that the "desired impedance" is applied to the relay.

  6. Reset indicators and timer.

  7. When the switch H is turned in position 1, the desired impedance is applied to the distance relay and the starting elements initiate the time-lag relay to operate. When the set time of each time-lag relays is up they extend the reach of the distance relay. The distance relay gives a tripping impulse when its reach is larger than the "impedance" applied by the test set.

  8. In this way the operating time of the distance relay can easily be checked for desired type of fault and fault position.
    If the time measurement is to be utilized for the matching of the settings of the time-lag relays to the selectivity plan, it is suggested that the measurement should be carried out as follows:

  9. Select the applied " impedance" so that it is approximately equal to, but larger than, the reach set for the first or second step and with phase angle equal to 90° . this is obtained with switch GR and potentiometer FR in 0 position.

  10. Perform the time measurement test as stated above.

Return relay and associated equipment to service

  1. Return relay to service by withdrawing the test plug handle from the relay If any equipment (associated) with the relay has been isolated without the test plug it should be returned to service by the following restoration steps

     - Breaker failure scheme protection
     - Disturbance alarms and fault recorders.
     - Acceleration scheme.

Request power line be returned to service

  1. Take crossed readings of all AC currents and voltages supplied to the relay

  2. Compare them to those for normal load conditions.

  3. Check that readings agree with station meters MW/MVAR/Power Factor/Amperes /Voltages 

  4. Place a copy of the test results in the field maintenance file
