Introduction

This procedure, details the steps required to completely checking the proper operation and calibration of the LZ96 relay. It is recommended this procedure be carried out on yearly bases. This work be carried out section of this document is divided into 3 sections.

The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required,

  • Line isolation must be done.

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection

Tools and Equipment

  • TZ/2 test set (voltage unit & current unit).

  • Multi-meter,

  • Clip-on ammeter, Test leads.

Work to be Carried Out

Major activates in Testing the Relay and Protection Out puts

Preparation at the Line

  1. Connect the test equipment with the relay.

  2. Prepare the test set.

  3. Test the starting unit.

  4. Test Zone reaches.

  5. Measure the operating times.

  6. Check the tripping and signaling circuits.

  7. Verify the test results match the relay setting. If not, consult technical services.

Preparation of the line

  1. Obtain outage for line protected by this relay because this procedure should be used for performing off-loads not in service relay tests.

  2. Look at the relay panel nameplate and check that the designation nameplate agrees with the line that has been taken out of service for maintenance.

  3. Check that the CB’s and isolator switches for that line are opened and tagged.

  4. Using a clip-on ammeter, check there is no current.

  5. Using voltmeter check there is no AC voltage supply the LZ96 circuits. Wires going to the rear terminals number 5, 6, 7 and 8 should have no volt (Check the station drawing).

Connection of Test Equipment with the LZ96 Relay

  1. Short the CT secondary circuits then isolated from the LZ96 relay.

  2. Isolate the VT secondary circuit from the LZ96 relay.

  3. After checking the station drawing and depending on the particular field site the following steps may be required:

    • Isolate the breaker failure scheme associated with this relay.

    • Isolate the disturbance alarms or fault recorders associated with this relay.

    • Isolate the acceleration scheme terminals from this relay.

  4. Turn of the associated auto-reclosure scheme

  5. Connect the current terminals of the test (current unit) to the terminals 1, 2, 3 and 4 of the LZ96 relay and short the terminal 01, 02, 03 and 04 (check the station drawing).

  6. Connect the voltage terminals of the test set (voltage unit) to the terminals 5, 6, 7 and 8 the LZ96 relay (Check the station drawing).

Preparation of the Test set Units

  1. Interconnect the power supply between the two test set units.

  2. Connect three- phase 220/380 volt AC supply to the connector end to the current test unit, then check the phase rotation. (green light means rotation is RST)

  3. Turn both of the large current and voltage adjustment knobs to their minimum position.

  4. On the current test set unit, turn the test selector knob to (v+1) position.

  5. On the voltage test box the timer test mode switch should be switched to (v+1) position.

  6. Turn the tow phase adjustment knobs on the voltage box to select the desired phase angle of the current w.r.t the voltage according to the phase angle setting on 1w91 of LZ96 relay.

  7. Select the position of the current direction knob on the current test current unit depending on the position of the switch +) on the unit 1w91 of LZ96 relay.

  8. Energize the voltage unit and the current unit of the test set by pushing the white “on” pushbutton for each one.

 

Testing the Starting unit

  1. Preset both the voltage and current units for the type of fault.

  2. Put the test start switch, located on the voltage unit to start position.

  3. Inject a fixed fault current value more than the rated current (1.5 In or 2 In).

  4. Raise the fault voltage to normal level and then reduce the voltage until the relay operates with the maximum time step.

  5. Record the test current value, the fault voltage and the phase angle.

Testing Zone Reaches

  1. The switch positions are identical to those given in step 3 and 4 for testing the starting unit in the forward direction.

  2. To check the operating point for the LZ96 relay, the fault voltage should be raised to the normal level with a fixed current flow and then the voltage reduced until the desired Zone of the relay operates.

  3. Record the test current value, the fault voltage value and phase angle

  4. Calculate the operating point of this zone and this calculated value should match the zone setting ZI.

    Where : ZI             ZM             MI             100/NI
    I = Zone index, I = 1, 2, 3
    NI = %value of zone (I) set on the unit EM92
    Zm = As in table 4.1
    MI = the factor of zone (I) set on the unit EM92
    M = 1,10.100

  5. Change the phase angle and repeat steps 1to 4 above calculate other operating points.

  6. Repeat steps 1 to 4 for the other zones noting the fault voltage values at which the timing unit just does not enable the measurement go the next zone, i.e the timing lamp of the next zone does not light up.

Measuring the Operating Times

  1. To start the timer the selector switch on the current box should be selected to indicate “volt box” on the voltage box the timer start switch can be selected for F,v, which will start the timer once the fault voltage is applied to the relay.

  2. To stop the timer connect a tripping contact from the LZ96 relay to the (C2) stop binding posts of the current test set if the tripping contact is dry contact. But if the contact has a voltage once the relay picks up should be connected to the test set via (20 Voltage input plug).

  3. It is only necessary to measure the operating times for one type of fault.

  4. For each measurement the injected impedance must be between two neighboring reach settings.

  5. The reset switch on the current test set will need to be pushed to prepare for thnext timing sequence.

Checking the Tripping and Signaling Circuits

  1. Reconnect the alarm signaling circuit

  2. Be sure that all isolator switches are opened and then, close the associated CB’s Z

  3. Inject the LZ96 (first zone reach) to indicate a tripping signals the. CB’s at the same time checking the signaling circuit is performed.

Return Relay to Service

  1. Switch off the test set then, remove the test leads between the relay and the test set.

  2. Connect the VT secondary circuits to the relay.

  3. Connect the CT secondary circuits to the relay then open the shorting links.

  4. The associated equipment with the relay should be returned to service by the following sequence:
       - Breaker failure scheme protection.
       - Disturbance alarms and fault recorders.
       - Acceleration scheme.
       - Auto-reclosure scheme.

  5. Request Power Line be Returned to Service

  6. Take crossed readings on all A.C currents and voltages supplied to the relay and compare them to those for normal load conditions.

  7. Check that the readings agree with the line meters: (MW/MVAR/ power factor/Amperes/voltage)

  8. Place a copy of test results in the field maintenance file.
