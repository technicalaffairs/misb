Introduction

    The MCGG is a microprocessor based over-current relay available in single, two, three or four pole versions manufactured by GEC ALSTHOM. Each pole is provided with a choice of four inverse time characteristics and three definite time characteristics as well as a trip test position. An instantaneous element is also available on each phase. The MCGG 22 is the single phase version.

    This procedure details the steps required to completely check the proper operation and calibration of the MCGG22 relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • Line isolation and work permit required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Switch off the DC power supply before inserting or withdrawing the relay

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category,

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Outage arrangement must be done.

Tools and Equipment

  • Secondary Injection Test Set.

  • Test Leads.

  • Multi-meter.

Work to be Carried Out

Testing the Relay, and Protection Output

  1. Carefully follow these test procedure steps

  2. Verify the test results match the relay setting - If not consult technical services.

  3. Test trip all breakers associated with the relay.

Preparation

  1. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolating the relay current circuits for test.

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown in the setting sheet.

  3. Switch on the MCB for the DC supply and check that terminal 13 on the relay case is positive with respect to terminal 14

. Testing Procedure (s)

Power Supply Test

  1. Release the RESET button and check that the LED’s reset.

  2. Press the RESET button and check that all LED’s are illuminated. This confirms that all measuring boards are operating satisfactorily.

    3.

Trip Test

  1. Set the curve selection switches to position 111 (test mode) for the pole of the relay, which is to be tested.

  2. Check that the LED’s associated with this pole of the relay flash at a frequency of the approximately once per second.

  3. Press and hold the RESET button. After approximately 6 seconds the time delayed and instantaneous output elements associated with the pole under test should trip, and at the same time, the LED’s for instantaneous and time delayed elements should illuminate continuously on this pole.

  4. Release the RESET button, the output elements should reset.

  5. Change the curve selection switches from lll (test mode) to the position shown in the setting sheet.

    6.

Current Setting Test

  1. Connect test leads between the current terminals on the relay (27, 28) and the current terminals on the test set.

  2. Slowly increase the current until the green LED lights. The relay must operate with a current between +/- 10% of the setting value.

  3. Record this value on the test sheet.

  4. Increase the injected current to a value 25% greater than the operate value.

  5. Slowly decrease the current until the relay and the green LED, both, reset.

  6. Record this value on the test sheet.

    7.

Time Setting Test

  1. Inject current into relay until the relay picks up. Keep this value and switch off the test set.

  2. Connect the relay time delayed output contacts (No. 1, 2 or 5, 7) to the timer contacts of the test set.

  3. Switch on the test set and inject current. The timer will stop once the relay has picked up.

  4. Record this value on the test sheet.

  5. Remove the leads from the time delayed contact and the test set

    6.

Instantaneous Setting Test

  1. Connect the instantaneous trip out put contacts (No. 15, 16 or 8, 9) to the timer contacts on the test set. This is to stop the current as soon as the relay picks up.

  2. CAUTION: Do not keep this current on the relay for more than a few seconds as it may exceed the rating and damage the relay

  3. Increase current quickly until the instantaneous element picks up.

  4. Record the pick up value on the test sheet. It must be within +/- 10% of the setting value.

Circuit Breaker Trip Test

  1. Set up the circuit breaker for a trip test.

  2. Inject current into the relay until the circuit breaker trips at the time delayed setting.

  3. Close the circuit breaker again.

  4. Remove the wire from the relay time delayed contact terminals No. 5 or 1.

  5. Inject a value of current greater than the instantaneous setting suddenly into relay until the circuit breaker trips at the instantaneous setting.

  6. Re-install the wire which was removed

  7. Remove the shorts from the CT secondary circuits.

    8.

Return the Relay and Power Equipment to Service

  1. Verify the test result match the relay setting. If not consult technical services.

  2. Test trip the circuit breaker associated with the relay.

  3. Make a final check that all switches on the relay are set according to the setting sheet.

  4. Make sure the CT secondary circuit is in normal condition

  5. Remove the CT shorts which were installed in the Power Supply

  6. Return the relay and power equipment to service.

    7.
