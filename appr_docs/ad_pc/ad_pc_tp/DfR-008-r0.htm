Introduction

Type KBCH 120 a fully digital relays, offer biased diff current, restricted earth fault and over fluxing protection for 2 winding power transformer, auto transformers or generators-TR units. The relay included a fault recorder, event recorder, disturbance recorder, measurements.
The first draft initiated by eng Mohamed Helmy El_Qady, tested at El EZZ 220kv substation, West Delta Zone

Safety Precautions

  • Work permit must be issued, the qualified maintenance staff must carry the suitable ID safety category,

  • Safety fence with caution marks must be surrounding the work area,

  • Wear appropriate safety and healthy protective equipment must be used as safety shoes, helmet .. etc.

  • Isolate the transformer from all sides.

Tools and Equipment

  • Secondary injection (Sverker 650)

  • Multi meter 

  • Test plug type MMLBO with test black type MMLGO

  • Two pole switch

  • Two diodes (2.5A)

  • Programmable synchronized, variable frequency current source capable of producing up to 55% 5th harmonic super imposed on the fundamental  

Work to be Carried Out
Steps To Prepare For Relay Testing

  1. Review technical reference material manufacture manuactfer`s manual 

  2. Obtain outage for transformer protected by the relay

  3. Inspect the secondary circuits of CT and all cabling supplying the relay

  4. Before inserting the test plug into the relay check from the station drawing that the test plug will isolate 
              - Breaker trip coils
              - Breaker fail  scheme associated with this relay
              - Fault recorder, event recorder and disturbance alarm associated with this relay
              - If the test plug not isolate one of the previous steps you have to isolate it

Testing The Relay and The Relay Outputs

  1. Measurement cheek
  2. Diff prot n (high set & low set)
  3. Operating time (high & lowest)
  4. Restricted earth fault prot n, operating time   
  5. Phase compensation
  6. Bias characteristic  
  7. Magnetizing inrush restraint
  8. Over flux prot n (Alarm-trip)
  9. Check trip for C-B by each prot n function in the relay    

Measurement checks

  1. Inject a known value into each phase input
  2. With the C.T ratios in the cells (HV. CT ratio) and (LV1 CT ratio) in the setting menu set to the values of the line CT" them the displayed measured values will be in the equivalent primary quantities

HV and LV1 winding Measurement checks
Connect the CT input to the relay as shown below:

  1. Set all the bits in the cell (S1 Fn- Links) to 0 in the setting menu. this will disables all the protection elements so that the relay will not trip
  2. Display the cell (MSl Ia HV) in the measurement menu.
  3. Inject rated current and insure that the displayed value lies within + 10% of the injected value
  4. Display (MS1 Ic LV1) and check that the value lies in this range (lies within  + 10% of the injected value)
  5. Check that the cells (MS1 Ia diff) & (MS1 Ib diff) & (MS1 Ic diff) displayed correct values of diff current it should be (injected current x 2 + 10 %)
  6. Check that the cells (MS1 Ia bias) & (MS1 Ib bias) & (MS1 Ic bias) displayed correct values of the bias current, in this case it should  be (injected  + 10 %)

Frequency Measurement Check

  1. Inject a current of known frequently to terminal (21-22) of the relay. (The frequency of injected current must be in the range (15-65 Hz)
  2. Display the cell (MSIF) in the measurement menu
  3. Check that the displayed value lies in the range of  (injected frequency + 2 %)

Restricted earth fault
There are two restricted earth fault elements for this relay model, one on the high voltage side of the transformer and one on the low voltage

Restricted earth Fault Protn

  1. Set all the bit to 0 except bit 3 for the cell (S1 fn. Links) in the setting menu. This ensure that only the REF protn on the high voltage side of the transformer is enabled
  2. Inject a current through terminals (27-28) slowly increase the current and note the pick-up Value at which the relay operate
  3. Reduce the current and check that the pick-up and drop-off values are:
                            * Pick-up ----------- (0.9 x Is -------- 1.1 x Is)
                            * Drop-off -----------(0.9 x pick-up --------1.0 x pickup)

R.E.F. Element HV Side Operating Time

  1. Connect the test set into the terminal (27-28)and connect the relay contacts for this protn function to both trip the test set and to stop a timer starts
  2. Inject 5 x Is A and check that the operating time is within the range (20 m.sec ---- 30 m.sec)

R.E.F. Current Sensitivity LV1 side (I0 > LV1)  

  1. Set all the bits to 0 except bit 4 which should be set to 1 for the cell (S1 Fn - link) in the setting menu this ensure that only the REF protn on the low voltage side is enabled
  2. Inject a current through the terminals (83-84) slowly increase the current and note the pick-up value
  3. Reduce the current and not the drop-off volume
  4. The values expected for the pick-up and drop off is within:-
                              * Pick-up             0.9 x Is     to 1.1 x Is
                              * Drop-off            0.9 x pick-up     to 1.0 x pick-up        

R.E.F. Element LV Side Operating Time

  1. Connect the test set into the terminal (83-84)and connect the relay contacts for this protn function to both trip the test set and to stop a timer starts
  2. Inject 5 x Is and check that the operating time is within the range (20 m.sec ---- 30 m.sec)

Phase Compensation
This test will verify that the relays internal phase compensation is functioning correctly

  1. Inject a current to phase A HV (21-22)and low voltage LV1 (77-78) winding the injected currents are anti phase (180º phase shift)
  2. The phase compensation for both these winding should be set to the same value which will result no diff current if the magnitudes of the injected currents are equal
  3. The diff currents can be monitored using the measurements menu check that it is zero current
  4. Then go to the setting menu and change the cell (SI LV1 vector cor) to the corresponding (opposite) setting listed in table (1) go to the measurement menu and check the displayed values of diff currents and these values corresponds to the listed values in the table 1
  5. It is not necessary to perform checks on all the possible combinations of setting but it is recommended that the tests should be carried out with those setting that are to be used in the relay application
  6. After finishing put (S1 LV1 vector car) to the original settings
