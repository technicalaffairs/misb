Introduction

    The SEL 49 is a digital protective relay, which can be used as either as a digital fault locator on transmission line or to provide transmission line protection. It has (6) six supervised mho units to start the fault locator. The SEL-49 Provides the Fault distance in Kilo meters (Km) and stores fault data in memory.

    The SEL-49 is connected to measure the three (3) phases currents and phase voltage. There is also an input for the residual current from a parallel transmission line to compensate sate for zero sequence mutual coupling from the parallel circuit.

    This procedure details the steps required to completely check the proper operation and calibration of the fault locator SEL-49 relay. It is recommended this procedure be carried out on an annually bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued

  • The maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Permission between S/S and NECC to disconnect the line.

  • Make sure CT secondary are shorted at current test switch.

  • Make sure VT secondary are opened at voltage test switch.

  • Isolate DC supply to SEL 49 when maintenance on the SEL-49 and SEL data.

Tools and Equipment

  • ABB tester XS92a for relay injection Or equivalent

  • Test leads, Small slot screwdriver.

  • Fluke 77 multi-meter for measuring.

  • Clip on ammeter.

Work to be carried out

Preliminary Steps to Prepare for Relay Testing

  1. Obtain and Review Manual Instruction for DFL-SEL.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Ensure the no. of panel name and which contain DFL - SE L-49 for the line which shall be tested.

  4. This procedure should be used for performing off load line.

Preparations

  1. From As Built drawing determine the No of test switch TS-2 for current terminal test switch 5®IA, 9®IB, 13®IC, and connect them together with the neutral (3, 7, 11).

  2. From this wiring use clip on ammeter to sure there is no current flowing through clip on ammeter Relaying Zero current this sure that the line breakers are switched off.

  3. From As Built drawing determine the No of test switch TS-1 for voltage terminal 1, 3, 5 and 7 (VA, VB, VC and VN) respectively use fluke 77 for measuring AC voltage.

  4. It will reading zero voltage this sure that the line breakers are switched off.
