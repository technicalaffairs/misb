Introduction

This document outlines the testing procedure P1, which performed every six months as indicated by M6 in the above title block.

This procedure details the steps required to completely check the proper operation and calibration of the distance relays types 7SA511, 7SA513, using OMICRON CMC156  Test Set.

This Testing Procedure Contains

Zone & Time reaches.

Directionality of relay

CB Trip & Alarm and Signaling

S.O.F Test.

Power Swing Test.

End to End Test:  (send & receive).

Back up ( O/C  & E.F ) Test.

Safety Precautions

  • A work permit must be issued and the qualified testing staff should carry the suitable safety categories.

  • Safety fence with caution marks surrounding the work area is required.

  • Wear Safety and healthy equipment as safety helmet, safety shoes ... etc.

  • Circuit breaker must be fully isolated.

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted before isolating the current circuits for test current injection.

  • Switch off DC power supply before inserting or withdrawing the Distance Relay under test as 7SA511, 7SA513, MICOM P411 or P442

Tools and Equipment

  • OMICRON test set type CMC156

  • Test leads.

  • Clip on Ammeter, Voltmeter

 

 

Work to be Carried Out

Preparations

The preparation for testing requires four essential tasks can be accomplished in the office and two tasks ( items 5 & 6 ) requires the engineer to be on site

  1. Choosing the correct OMICRON test module.

  2. Entering test object parameters – details and settings of the relay.

  3. Hardware configuration – details of the wiring connections and configuring binary inputs ... etc.

  4. Defining the report structure ( How will the results be formatted in the report ? ).

  5. Running the test.

  6. Saving and printing the report.

Preliminary Steps to Prepare for Relay Testing

  1. Review technical references material Manufacturer’s manual for distance relay 7SA511/7SA513/ MICOM P411 or P442

  2. Review technical references material Manufacturer’s manual for OMICRON test set .

  3. Obtain copy of relay setting from field protection file and compare it with the setting on the relay.    

  4. Obtain outage for line protected by this relay.

  5. Check that the bus and line disconnect switches are open and tagged.

  6. Inspect the CT secondary circuits, the VT (MCB’S), and all cables supplying the relay 7SA511/ 7SA513/ MICOM P411 or P442.

  7. Use the station drawings to isolate the following items:

  • Breaker trip coils.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

Testing the Relay

  1. Carefully follow all test procedure steps.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Test trip the breaker associated with the relay

Prior to Starting Testing

  1. This procedure is used to perform off load relay testing.

  2. Look at the control panel nameplate and check that this control system designation agrees with the line that has been taken out of service for maintenance.

  3. Check there is no AC current from CT’s by using a clip on ammeter.

  4. Check there is no AC voltage from VT’s by using a voltmeter.

Manually or Automatic Stepping the Current

Manually Test

  1. Select the current triple of the CMC 156

  2. Select to step all thee current or 1 phase ( as you like )

  3. Specify step size as 0.1 A

  4. Click the up button until the O/C relay pick up.

Automatic Test

  1. Select the current triple of the CMC 156

  2. Select to step all three current or 1 phase ( as you like ).

  3. Specify step size as 0.1 A.

  4. Select “ AUTO STEP “ Check Box.

  5. Click the up button once.

  6. Note: the O/C relay pick up.

Prepare the Relay for Service

  1. Rest all function on the distance relay by pressing RESET on the keyboard of the relay.  

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminals block and the test set.

  2. Make sure CT and VT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated,
    it should be returned to service by the following restoration steps:

  • Breaker trip coils.

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.

Request Power Line be Returned to Service

  1. Take crossed reading of all AC currents and voltages supplied to the relay 7SA511, 7SA513, MICOM P411 or P442 and compare them to those for normal load condition.

  2. Check that readings agree with station meters (MW/MVAR/Power factor/Ampere/Voltages).

  3. Place a copy of the test results in the field maintenance file.
