Introduction

The type IM3X relay can be used as un retarded over current or under current relays for either protective or indicating duties. In conjunction with a separately mounted time-lag relay it can be used for over current protection with a fixed time-lag, by virtue of its low consumption, even when the capacity of the CT is limited

This procedure details the steps required to completely check the proper operation and calibration of the directional earth fault over current relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

 Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area,

  • Line isolation and work permit required

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection

Tools and Equipment

  • HARALD test set

  • Voltmeter, Ammeter, AND Test leads 

Work to be Carried Out

Steps to prepare for relay testing

  1. Review technical reference material and manufacturer's manual

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay

  3. Obtain outage for line protected by this relay

  4. Check that the circuit breaker is opened and tagged

  5. Inspect the CT secondary circuits, the VT miniature circuit breakers (MCB's) and all cabling supplying the relaying

  6. Use the station drawings to isolate the following:
         * Breaker trip coils
         * Breaker failure scheme associated with the relay and disturbance alarms

Testing the relay

  1. Carefully follow this test procedure

  2. Verify the test results match the relay setting

  3. Trip the circuit breaker associated with the relay

Preparation

  1. This procedure shoud be used for performing off load line

  2. Look the EEC panel name plate and check this EEC system designation name plate agrees with the line that has been taken out of service for maintenance 

  3. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test

Reading the setting from the relay by using the PC

  1. By using the MMI (main machine interface) of the relay read the setting

Earth fault protection test

  1. Be sure that the DC power of the relay is correct by applying a DC voltmeter across points 4 and 6 on the rear of the relay

  2. Connect the current test leads between the current terminals on the rear of the current relay 1.2 of the (IM3X) first stage

  3. Connect AC power to the test set

  4. Inject current to the relay

  5. Increase slowly the current until reach to the operating value and then the red "LED's" light the operate current is shown on the display of the using ammeter

  6. Record this value (I operate) on the test sheet

  7. Connect leads between the time delayed trip output contact terminal 13 and 14 on the rely to the time terminals on the tester

  8. Switch on the timer of the test set

  9. The value of the trip time display on the test set

  10. Record this value (t" operate) on the test sheet

  11. Increase the current injection to 25% larger than the operating value (I operate)

  12. Slowly decrease the injection current until the relay reset tack the value of this current (I reset)

  13. Record this value (I reset) on the test sheet

  14. Repeat steps from 2 to 13 to test phase R (2) second stage

Return relay and associated equipment to service

  1. Return relay to service by removing the test leads from the relay terminals and the test set

  2. Make sure CT and V T secondary circuits are in normal condition

  3. Remove the disturbance alarm LED`s on the relay which is caused by the test

  4. Request feeder to be returned in service

  5. Take phasor readings of all A C currents and voltages and compare them to those for normal load condition

  6. Check that these reading agree with station meters (MW, Mvar, Amperes and power factor)

  7. Place a copy of the test result in the field maintenance file
