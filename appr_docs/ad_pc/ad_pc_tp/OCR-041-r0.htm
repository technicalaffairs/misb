Introduction

Rapdk3 is a 3ph. directional over current protection for cables & overhead lines, it may include a measuring unit type RXPDK 21H or PXPDK 22H

Rapdk1 is a single phase directional over current protection, it may also include a measuring unit type PXPDK 21H or PXPDK 22H.

PAPDK4 is a directional (3ph. O.C. and E.F.) over current protection relay.

RXPDK 21H it is a measuring unit with a voltage polarized, and characteristic angle settable:- 120º to + 120º, it has two stages:

  1. 1st stage (low set): it is a directional stage from 0.074 to 3.26A (1Anolinal) or 0.37 to 16.3A (5A nominal)

Low set stage programmable for the following time characteristics:
Normally inverse
Very inverse
Extremely inverse
Long time inverse
Definite time delay settable (from 50 ms to 8.1s)

  1. 2 nd. stage (High set): it is a non-directional stage from 0.1 to 40A (1A nominal)  or
         0.5 to 200 A (5A nominal)

RTPDK 21H measuring unit is a static microprocessor based relay, it consists mainly of: two input transformers (one for current and another for voltage), filter circuit, microprocessor, MMI, Leds for (indication of relay in service, start, trip of lowest stage and trip of highest stage)

RXPDK 22H is an uni-or bidirectional time - over current relay with voltage polarization and voltage enabling, or no directional time - over current relay with under voltage enabling

Two current variants with setting ranges 3.7 --- 163 mA and 15 --- 650 mA, the characteristic angle is settable to 0º 90º separate built-in over or under voltage protection function, two binary i/ps to reset indications and to change the characteristic angle

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

    •

  • Line isolation must be done.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for current injection

Tools and Equipment

  • Test set type 3/TRE

  • Test leads, Digital multi-meter

Work to be Carried Out

Testing the Relay and Protection O/P as Follows

  1. Carefully follow these test procedure steps

  2. Verify the test results match the relay setting

  3. Test trip the CB associated with the relay

Preparation

  1. Make sure CT secondary circuits are shorted at the incoming blocks before isolating the relay current circuits for test

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting setting sheet

  3. Preparing the test set 3/TRE

  • Connect the supply to the plug of the power supply in the current box

  • Connect the cables between the current box and voltage box of the tester

  • Connect the current & voltage circuits of the relay to the tester

  •  Be sure that the i/p 3 ph. Currents at points 3,5,7 and the o/p at points 4,6,8 respectively at the test block

Testing the Relay

  1. When testing the directional current function, when the phase angle of the polarizing voltage relative to the injection current is equal to the set characteristic angle of the relay, the operating value is equal to the setting value

     
              Operating condition :     I. Cos (φ - a) = set operating value

  1. Where a is the relay characteristic angle

  2. When testing the directional relay RAPDK 21H/22H, a polarizing voltage ≥ 5 volt, must be applied to each individual phase or earth fault

Testing Earth Fault Element

  1. Apply 5 volt at least at points (14,15) at test block

  2. Inject a current above the set current value of the relay at points (9,10) at test block

  3. Change the phase angle between the voltage and current to get a trip

  4. Record the current value at different phase angles inside the operation zone of the relay

Testing Over Current element

  1. Connect the 3ph. voltage at the points (11,12,13) and 3ph. currents (i/p at points (3,5,7) and the o/p at (4,6,8) at the test block)

  2. Apply at least 5 volt to each phase

  3. Inject a current for phase R at points (3,4) at test block above the set current value of the relay

  4. Change the phase angle between the voltage and current to get trip

  5. Record the current value at different phase angles inside the operation zone of the relay

  6. Repeat steps from (b ---- e) for phase S and phase T.
