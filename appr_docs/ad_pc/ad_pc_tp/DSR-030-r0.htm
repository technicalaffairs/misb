Introduction

The REL100 digital line protection scheme is designed for simultaneous phase - phase and phase - earth loop impedance measurement with individual numerical measuring elements for each type of fault and each distance zone ensures fast and reliable fault detection

Following functions are available optionally independent one on another:

  • 16 programmable signaling output relays

  • Two additional distance measuring zones with programmable directionality

  • Phase selective single phase tripping

  • Power swing detecting element with programmable effect on different zones

  • Fuse failure supervision function

  • Synchro check and dead line check function

  • Supervision of protected power line

  • Stub protection

  • Weak and infeed trip and echo logic

  • Directional or non directional earth fault over current protection

  • Accurate fault locator based on well proven measuring algorithm

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

 Safety Precautions

  • Line isolation and a work permit must be issued

  • The qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection

Tools and Equipment

  • FREJA test set and Test plug handle type RTXP 24

  • Voltmeter, Ammeter, and Test leads 

Work to be Carried Out

Steps to prepare for relay testing

  1. Review technical reference material and manufacturer's manual

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay

  3. Obtain outage for line protected by this relay

  4. Check that the circuit breaker is opened and tagged

  5. Inspect the CT secondary circuits, the VT miniature circuit breakers (mcb.s) and all cabling supplying the relaying

  6. Use the station drawings to isolate the following:
         * Breaker trip coils
         * Breaker failure scheme associated with the relay and disturbance alarms

Testing the relay

  1. Carefully follow this test procedure

  2. Verify the test results match the relay setting

  3. Trip the circuit breaker associated with the relay

Preparation

  1. This procedure should be used for performing off load line

  2. Look at the EEC panel name plate and check this EEC system designation name plate agrees with the line that has been taken out of service for maintenance 

  3. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test

Reading the setting from the relay by using the PC

  1. By using the keys on the MMI (Man Machine Interface) of the relay read the setting

Distance protection test

  1. Be sure that the DC power of the relay is correct by applying a DC voltmeter across points 134 :402 (B12) and 134 :402 (B12) on the rear of the relay.

  2. Connect the current test leads on the current terminals of the test plug handle 13A , 15A , 17A , 19A ,  and the voltage test leads on the voltage terminals of the test plug handle 21A , 22A , 23A , 24A and the trap leads on terminals 9A(trip R), 10A (trip S), 11A (trip T) of the test plug handle.

  3. Connect AC power to the test set

  4. Checking of impedance zones:
    a)  Set the fault impedance (Z) and impedance angle Z φ for the zone under test to within ± 10% of the zone setting
    b)  Supply the relay with healthy conditions (depress key < W > W - Healthy) for at least two seconds
    c)  Apply the fault condition (depress key < S > S - faulty / auto open)
    d)  Check that the correct trip outputs and external signals are obtained for the type of voltage generated and the trip
    time is in accordance with the zone timer under test.
    e)  If the relay dose not operate after the set time of the zone timer, turn (off) the generators (depress key < Q > Q -
    open) and check that the zone timer is turned on and test point is within the range.
    f)  Repeat steps ( a ) to ( d ) for other test points.
    g) Repeat steps ( a ) to ( f ) for other zone.
    h) Repeat steps ( a ) to ( g ) for other types of faults.

Return relay and associated equipment to service

  1. Return relay to service by removing the test leads from the relay terminals and the test set

  2. Make sure CT and V T secondary circuits are in normal condition

  3. Remove the disturbance alarm LED`s on the relay which is caused by the test

  4. Request feeder to be returned in service

  5. Take phasor readings of all AC currents and voltages and compare them to those for normal load condition

  6. Check that these reading agree with station meters (MW, Mvar, Amperes and power factor)

  7. Place a copy of the test result in the field maintenance file
