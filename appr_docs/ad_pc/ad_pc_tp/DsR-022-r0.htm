Introduction

This document outlines the testing procedures for the distance relay type LZ96 (analog) by using XS 92 a test set. but these document for distance relay type LZ 96a (digital) by using XS 92 a set test. 

The first draft initiated by eng Mohsen Abd el Hameed Soliman  El-Suez 500 and tested at 27/5/02

Safety Precautions

  • Check that the bus and line disconnect switches are opened and Tagged.
  • Make sure CT secondary are shorted.
  • Make sure VT miniature circuit breaks (M.C.B's) is isolated. (opened).
  • Isolate DC supply to relay.  
  • Isolate:
           * Breaks failure (BF) scheme associated with relay.
             * Breaker trip coil and transfer trip scheme 
              * Auto-recluse associated with the Relay 

Tools and Equipment
  • Test set XS92 a (secondary injection)
  • Test plug and test leads
  • Voltammeter, Climb ammeters.
  • Phase rotation meter

Work to be Carried Out
  1. Review technical reference material manufacturer's manual
  2. Obtain copy of relay LZ 96 a setting from field maintenance file.
  3. Obtain outage for line protected by this relay LZ 96 a.
  4. Check that the bus and line disconnect switches are opened and togged
  5. Block protection signals to  Breaker trip coils and transfer trip schemes.
  6. Go to the switchgear and inspect the CT 's, and VT Miniature circuit breaker (M.C.B's) and cabling supplying the relay LZ 96 a.
  7. Check that the cables are in a good condition and the all wire connections are tight and clean.

Testing procedure:

  1. Connect test plug of set test XS 92 a to the relay LZ 96 a  

  2. Restore D.C supply to LZ 96 a relay scheme.

  3. Prepare the set test XS 92a as following:

Zone reaches                 Zone ( I )                                                             Phase / earth

MOD (ZZ)                     Distance

ZL                                  up to setting Z1

PZL                               Angle of setting line protected (0….360˚)

K0                                 (0.7)

Fc                                   fault code forward or refers 111 (phase R to E)

         SQ                                  1F1

         A                                     2

        TF                                    Time fault zone (250) ms

        DZL                                 Step parameters and range limits (0.1)

       Start(-)                              injection of testing signals

Note :

  1. We entered the all above parameters according to the flow charts of set Test XS92a 

  2. When the relay LZ96 a  gives tripping signal press ◙ to stop injection

  3. Read the values in hand terminals of set test XS 92a, and compare this value with the value of setting in test sheet of relay LZ96 a

  4. Press the reset button an the front panel of the relay and check that the indicator ON MODE and VALUE windows in the front panel of relay are reset. 

  5. Repeat the above test with the phase (S) by using fault code FC (112), phase (T) by using fault code FC (113).Make sure the relay LZ 96 a gives trip signaling with all phases.

  6. Repeat the above test, but in the refers direction by using fault code FC (011) & (012) & (013)Make sure that the relay LZ 96 a is blocked in refers direction

Phase To Phase
  1. Enter all above parameters, but the deference that in (FC) 
                Phase R-S              121                Forward direction (trip) 
                Phase S-T              122                Forward direction (trip)
                Phase T-R              123                Forward direction (trip) 

                     In the refers direction  (021), (022), (023) (Block)

Zone II

  1. Repeat the above test for zone I , But change the value of ZL up to Z2 , and TF up to time of Z2 .
  2. Make sure that the relay LZ 96 a in forward direction (trip) & in reverse direction (Block)

Zone III
  1. Repeat the above test for zone II , But change the value of ZL up to Z3 , and TF up to time of Z3 .
  2. Make sure that the relay LZ 96 a in forward direction (trip) & in reverse direction (Block)

Zone IV
  1. Repeat the above test for zone III , But change the value of ZL up to Z4 , and TF up to time of Z4
  2. Make sure that the relay LZ 96 a in forward direction (trip) & in reverse direction at ½ Z4 Make (Trip)

Measuring The Operating Times
  1. The operating Time of a distance relay LZ 96 a should be measured using the trip contact (free contact). Connected The free contact to (+) and ( __\__ ) in the front panel of set test XS 92a .
  2. Enter the all parameters of ZL at ZL = 0.5 ZI
  3. When the trip occur the injection automatically stopped free contact. Then read the value of time zone( I ) in the hand  terminal
  4. Repeat the above step to measuring the time of zone II , III , IV

Time of Z1 at ZL = 0.5 Z1

Time of Z2 at ZL =  0.5( Z1+Z2 )

Time of Z3 at  ZL = 0.5( Z2+Z3 )

Time of Z4 at ZL =0.5( Z3+Z4 )

Note: How to Find "Free Contact " For Relay LZ 96 a ?

  1. Select any contact and measured DC (Not Found)

  2. Measured (Buzzer) by avometer (Not Found)

  3. Make trip by using, XS 92 a (then these contact-measured buzzes by avometer).

  4. Connect this free contact to set test XS92 a.

End To End Test For Relay LZ 96 a

  1. Repeat the test of zone II and make sure that the trip occur at time of  Z2.

  2. Make bridge on receive signal.

  3. Make sure that the relay LZ96 a trip zone II at time of zone I

Switch into Fault (SOIF) Test:

  1. Repeat the test of zone IV but ( SQ = 0F1) in set test XS92 a

  2. Make sure that the zone IV trip at time of zone I.

Return Protection  Scheme And Associated Equipment to Service

  1. Return relay LZ 96 a to service by with drawing the test plug handle From the relay.

  2. Make sure CT secondary are opened.

  3. Return Relay to service by restoring supply Form :(VT) miniature circuit breaks (MCBS) is closed

  4. Place the associated Breaker failure (B.F) protection Scheme in service.

  5. Restore the previously blocked disturbance alarms and fault recorders associated with this protection .

  6. Return the relay power line carries (PLC) links to their normal position.

  7. Turn on the associated auto-reclosing scheme.

  8. Turn on the powers supply DC to relay LZ 96 a.

  9. Take crossed readings of all AC currents and voltages and power in the relay and compare them to those for normal load conditions.

  10. Check that these readings agree with station Meters; MW, MVAR, Power factor, amperes, voltages.

  11. Place a copy of the test results in the field maintenance test sheet (report) .
