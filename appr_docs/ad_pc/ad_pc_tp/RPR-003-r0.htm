Introduction

This document lists the testing procedure P1, which must be performed on an annually bases as indicated by Y1 in the above title block

These relays are designed for operation at rated system voltage to detect an abnormal power flow in a given direction. for balanced load conditions only one relay is required per terminal .

The first draft  initiated by eng. Fayza El-Sharkawy, tested by eng. Shirin Hamdy & eng. Azza Salama at 23 April 2003, revised by EDEPCo., Quality Control eng. Adel El-Nagar, accepted by General Director eng. Mahmoud Saker, and approved by eng. Moustfa Alam Head Sector of Talkha Power Station.

Safety Precautions

  • A work permit must be issued and the qualified test staff should carry the suitable safely category

    • .

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • The generator must be out of service.

Tools and Equipment

  • Test Plug handle, Clamp ammeter.

  • Secondary injection test set OR Double-F2200 test set

  • Test Leads.

    •

Work to be Carried Out

Preparation

  1. Make Sure C.T secondary circuit are opened at the incoming terminal blocks before isolating the relay current circuit for test

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the test sheet.

Testing the Relay    Figure 1

  1. Connect the test leads between  current terminals ( 5 - 6 ).

  2. Connect the test leads between the Voltage terminals ( 7 - 8 ).

  3. Connect the test leads between the trip terminals ( 1 - 2 )

  4. Increase the voltage to 120 volt.

  5. Increase the current slowly until the relay picks up.

  6. Record the value of  the current as the start current.

  7. Increase the current to the value 5 x Istart then 10 x Istart.

  8. Record the time, which the relay trips at every current value.

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set i.e.
    -     Disconnect the test leads from the current terminals ( 5 - 6 ) , voltage terminals ( 7 - 8 )
    - Disconnect the test leads between the trip contact (1- 2 )

  2. Make sure CT and PT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
    - Breaker trip coils, - Breaker failure scheme associated this relay.
    - Disturbance alarm and fault recorder associated with this relay.
