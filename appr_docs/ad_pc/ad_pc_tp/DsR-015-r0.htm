Introduction

The document  is test procedure for distance relay 7SL32 ( Sienems ) by using XS92a set test

Safety Precautions

  • Check that the bus and line disconnect switches are opened and tagged.

  • Make sure CT secondary circuits are shorted  

  • VT Miniature circuit breaker (MCB`S) is isolated (opened)

  • Isolate DC supply to relay. When doing any Adjustments or Maintenance on the 7SL32 relay

  • Isolate 
                * Breaker failure (B.F) Scheme associated with relay 
                * Disturbance alarm or fault recorder associated with relay
                * Breaker trip coils and transfer trip scheme
                * Auto - reclosing associated with the relay 7SL32
                 ( Test plug isolate the above all points ) 

Tools and Equipment

  • Test set type (XS92a)

  • Ammeter.

  •  Test leads.

  • Timer

Work to be carried out

Steps to prepare for relay testing

  1. Review technical reference material: Manufacturer's manual

  2. Obtain copy of relay 7SL32 setting form field maintenance file 

  3. Obtain outage for line protected by this relay 7SL32.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Block protection signals to : breaker trip coils and transfer trip schemes.

  6. Go to the switchgear and inspect the CT`s and VT miniature circuit breaker ( MCB`S), and all cabling supplying the relay

  7. Check that the cables are in good condition and the all wire connections are Ti , ht and clean


Testing Procedures

 Zone Reaches

ZONE I  (phase to earth )

  1. Restore DC supply to 7SL32 relay scheme

  2. Prepare the set test XS92a as following
                                        MOD     ZZ            (Distance)
                                        ZL           up to setting Z1
                                        PZL         angle of setting time protected (0 ....... 3 ¡ 0 )
                                        KO          0.7
                                        FC           fault code forwarde or reverse & (lll) phase R to earth 
                                        SQ          1F1
                                        A             2
                                        TF           Time fault zone (250) m.sec
                                        DZL        step parameters and range limits (0.1)
                                        Start (-)   injection of testing signals  

Note 

  1.  We entered the all above parameters according to the flowchart of set test XS92a

  2. When the relay gives tripping signal press ◙ to stop injection

  3. Read the values in hand terminals of set test XS92a and compare this values in hand terminals of set test with the values of setting test sheet of relay 7SL32

  4. Press the reset button an the relay front panel and check that the led indicator resets and the alarm contacts open

  5. Repeat the above test with the phases (S) by using FC ( fault code ) (112) & phase (T) by using FC (fault code) (113). Make sure the relay 7SL32 give trip signaling with all phases

  6. Repeat the above test, but in the reverse direction by using (FC) fault code (012) & (011) & (013). Make sure that the relay 7 SL 32 is blocked in reverse direction

 Phase / Phase 

  1.  Entered the all above parameter, but the difference that in (FC) 
     Fault code   phase  R- S         121    Forward direction (trip)
                        phase  S- T         122    Forward direction (trip)
                        phase  T- R         123    Forward direction (trip) 
  2. In the reverse direction (021) & (022) & (033) (Block)

Zone II

  1. Repeat the above test for zone I, but change the value entered ZL up to Z2, and TF up to time of  Z2

  2. Make sure the relay 7SL32 in forward direction (trip) & in reverse direction (block)

Zone III

  1. Repeat the above test for zone II, but change the value of ZL up to Z3, and TF up to time of Z3

  2. Make sure the relay 7SL32 in forward direction (trip) and reverse direction (block)

Zone IV

  1. Repeat the above test for zone III, but change the value of  ZL up to Z4, and TF up to time of  Z4

  2. Make sure that the relay 7SL32 in forward direction (trip) and reverse direction (block)

 

Measuring the operating times

  1. The operating time of a distance relay 7SL32 should be Measured using the trip contact (free contact). Connected the free contact to "+" and "  __\__  " in the front panel of set test XS92a 

  2. Enter the all parameters of zone I by ZL = 0.5 Z1

  3. When the trip occur the injection automatically stopped by free contact. Then read the value of time zone I in the hand terminal 

  4. Repeat the above step to measuring the time of zone II & III & IV

Note :  How to find " free contact " for relay 7SL32 ? 

  1. Select any contact and measured DC (Not found)

  2. Measured (buzzer) by avometer (Not found)

  3. Made trip (this contact measured buzzer by Avometer )

  4. Connect this free contact to set test XS92a 

End to End Test For Relay 7 SL 32

  1. Relay the test of zone II and make sure that the trip occur at time of Z2 

  2. Make bridge ON receive signal

  3. Make sure that the relay 7ZL32 trip zone II at time of zone I

 Switch into fault ( SOIF ) test

  1. Repeat the test of zone IV. but ( SQ = 0 F1) in set test XS92a

  2. Make sure that zone IV trip at time of zone I

Return Protection Scheme and Associated Equipment to Service

  1. Return relay to services by with drawing the test plug handle from the relay

  2. Make sure C.T secondary are opened

  3. Return relay to service by restoring supply from  voltage transformer (V.T) miniature circuit breaker MCB`S is closed

  4. Place the associated breaker failure (B.F) protection scheme back in service

  5. Restore the previously blocked disturbance alarrns and fault recorder associated with this protection

  6. Return the relay power line carrier (PLC) links to their normal position

  7. Turn ON the associated auto - reclosure scheme

  8. Turn ON the power supply DC to relay 7SL32

  9. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those for normal load conditions.

  10. Check that these readings agree with station meters:-
                                MW , MVAR , power factor , Amperes , voltages

  11. Place a copy of the test results in the field maintenance file, and test sheet
