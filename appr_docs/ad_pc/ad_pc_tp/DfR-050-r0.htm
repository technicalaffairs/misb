Introduction

This document outlines the testing procedures P1, which must be performed every six months as indicated by M6 in the above title block.

This procedure, details the steps required to completely check the proper operation and calibration of the relay ( D21Se3 ). 

The first draft initiated & tested by eng. Mahmoud El-Attar, accepted by eng. Saad El-Attar: protection general manager, revised and certified by eng. Mohamed Omran: development & quality surveillance general manager and MPIS Canal zone coordinator, and approved by eng. Al-Saied Abd El-Azez Alnagy: head sector of protection, measuring & testes.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • This procedure should be performed for off load lines, so line isolation is required.

  • Isolate the relay DC supply before drawing out any relay card.

  • Isolate the relay associated breaker trip coils, breaker failure system, disturbance alarm and fault recorder systems

Tools and Equipment

  • Secondary Injection test set ( Sverker ).

  • Test leads, Avometer   

  • AC supply

Work to be carried out

 

Relay Preparation

  1. Review technical reference material: Manufacturer's manual.

  2. Obtain copy of relay settings from field maintenance file and compare it with the relay settings .

  3. Obtain outage for line protected by this relay .( work permit must be issued )

  4. Check that the bus and line disconnect switches are opened and tagged .

  5. Make sure CT's Secondary circuits are shorted before isolating the relay current circuits for current injection.

  6. Isolate the relay DC supply .

  7. Check that all cables supplying the relay and all wire connections are in good conditions.

Tester Preparation

 Preparing the tester unit (Sverker ) as followings:

  1. Connect the power supply to the tester unit (Sverker).

  2. Connect the connecting terminals between the relay and Sverker.

Parameters Preparation

     Review the test certificate for required settings.

relay Testing

 The test is accomplished in five stages

  1. Finding relay operating current value for phase to earth faults on 220 kv side.

  2. Finding relay operating current value for phase to phase faults on 220 kv side.

  3. Finding relay operating current value for phase to earth faults on 11 kv side.

  4. Finding relay operating current value for phase to phase faults on 11 kv side.

  5. Finding relay operating time .

Finding relay operating current on 220kv side: ( PH - E )

  1. Connect the relay phase ( R ) to tester.

  2. Slowly increase the injected current until the relay operate ( indicated by the flag ).

  3. Record the operating current ( Iop ) in test sheet.

  4. Check that the relay trip contact is closed.

  5. Check the relay Alarm contact and breaker failure contact.

  6. Repeat the steps ( 2 : 5 ) with phase ( S ) And with phase ( T ).

Finding relay operating current on 220kv side: ( PH - PH )

  1. Connect the relay phases ( R , S ) to tester.

  2. Slowly increase the injected current until the relay operate ( indicated by the flag ).

  3. Record the operating current ( Iop ) in test sheet.

  4. Check that the relay trip contact is closed .

  5. Check the relay Alarm contact and breaker failure contact.

  6. Repeat the steps ( 2 : 5 ) with phases ( S , T ) And with phases ( T , R ).

Finding Relay operating current on 11kv side: ( PH - E )

  1. Connect the relay phase ( R ) to tester.

  2. Slowly increase the injected current until the relay operate ( indicated by the flag ).

  3. Record the operating current ( Iop ) in test sheet.

  4. Check that the relay trip contact is closed.

  5. Check the relay Alarm contact and breaker failure contact.

  6. Repeat the steps ( 2 : 5 ) with phase ( S ) And with phase ( T ).

Finding Relay operating current on 11kv side: ( PH - PH )

  1. Connect the relay phases ( R , S ) to tester.

  2. Slowly increase the injected current until the relay operate ( indicated by the flag ).

  3. Record the operating current ( Iop ) in test sheet.

  4. Check that the relay trip contact is closed.

  5. Check the relay Alarm contact and breaker failure contact .

  6. Repeat the steps ( 2 : 5 ) with phases ( S , T ) and with phases ( T , R ).

Finding Relay operating Time

  1. The operating time should be measured using the trip contact ( free contact ).

  2. Inject testing current until the phase ( R ) relay operate, then switch off the test set.

  3. Connect the timed contact of the phase ( R ) relay to the timer stop terminals of the test set.

  4. Switch on the tester with timer ( ON ) and measure the phase ( R ) relay operating time.

  5. Record the operating time in test sheet.

Note: How to find " free contact" on relay ( D21Se3 ):

  1. Select any free contact and measure it's DC voltage ( must have not DC voltage ).

  2. check that it is normal open contact using  avometer buzzer.

  3. check that it is close with trip, using tester Sverker and avometer buzzer.

Return Protection Scheme and associated Equipment to service

  1. Return relay ( D21Se3 ) to service by removing the test leads from the relay terminals .

  2. Make sure CT secondary are not short circuited .

  3. Place the associated barker failure ( B.F ) protection scheme back in service.

  4. Restore the previously blocked disturbance alarms and fault recorders associated with this protection.

  5. Turn ON the power supply DC to relay ( D21Se3 ).

  6. Take crossed readings of all AC currents and voltages supplied to the relay ( D21Se3 ) and compare them to those for normal load conditions.

  7. Check that these readings agree with station meters MW, MVAR, Power Factor, Amperes, and Voltages.

  8. Place a copy of the test results in the field maintenance file.
