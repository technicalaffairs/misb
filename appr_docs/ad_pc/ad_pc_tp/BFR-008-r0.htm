Introduction

The 132kv back up devices is designed for disconnecting of short circuits if any circuit breaker of the feeder in a failure refuses to operate.

The back up device provides quicker tripping of fault in comparison with back up protections
The back up device meets requirement of selectivity with correct fixation of feeders to bus bars the back up device provides the tripping of a certain bus bar to which the feeder under fault belongs  

The first draft initialed by eng. Mohamed Mostafa Abdel Gawad, tested Samalout 500kv, approved by eng. Ahmed Saied Abdel Azez: Middle Egypt general director of quality control & development  & MPIS coordinator.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Back up protection isolation and work permit required

  • Make sure CT secondary circuits is in normal condition and also PT terminals

Tools and Equipment

  • Injection secondary tester ( yп3-1)

  • Voltmeter, Clip ammeter

  • Test leads, megger

Work to be Carried Out

Preliminary steps to prepare for relay testing

  1. Review technical reference material: Manufacturers manual

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay

  3. Obtain outage for line protected by this relay

  4. Check that the bus and line disconnect switches are opened and tagged

  5. Inspect CT secondary circuits, the VT miniature circuit breakers (MCB`S) and all cabling supplying the relay.

     

  6. Use the station drawing to isolate the following:
       - Breaker trip coils
       - Breaker failure scheme associated with this relay.
       - Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection output

  1. Carefully follow the test procedure.

  2. Verify the results match the relay setting if not consult technical services

  3. Test trip the breaker associated with the relay

Prior to Starting Testing Procedure (s) ensure the Following Steps

  1. This procedure is used to perform off load relay testing

  2. Check that the panel nameplate agrees with the EEHC designation of the line that has been taken out of service for maintenance

  3. Check there is no AC current from CT by using clip - on ammeter

  4. Check there is no AC voltage from VT by using voltmeter

  5. Check all external connections to the relay are in good condition and check that the terminal block wires are tight
