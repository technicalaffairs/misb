Introduction

    The RSZ3F2 relay is an electro-mechanical over-current manufactured by (EAW). This procedure details the steps required to completely check the proper operation and calibration of the RSZ3F2 relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

  • Line isolation must be done.

Tools and Equipment

  • Secondary Injection Test Set

  • Ammeter

  • Test leads

Work to be carried out

Testing the Relay, and Protection Output

  1. Carefully follow all test procedure steps from page 2 to page 7.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Test trip the breaker associated with the relay.

Preparation

  1. Make sure CT secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test.

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting sheet.

Test Procedure (s)
Current Test

  1. Label the wires before removing its from terminal block of the relay.

  2. Remove the input current terminals number 7,8 and 9 from the relay.

  3. Connect test leads between the current terminals number 7 and 10 on the relay to the current terminals on the test set.

  4. Slowly increase the current until the red indication on over-current element of phase R on the relay appears. The relay must operate with a current between ± 10% of the setting value.

  5. Record this value on the test sheet.

  6. Increase the injected current with a value 25% greater than the operating value.

  7. Slowly decrease the current until the relay resets.

  8. Record this value on the test sheet.

Time test

  1. Inject current higher than the setting value so the relay picks up firmly.

  2. Keep the injected current and switch off the test set.

  3. Connect the time delayed contacts number 3 and 4 on the relay to timer terminals on the test set.

  4. Switch on the test set.

  5. Start the test by injecting current into the relay. The timer will stop once the relay has picked up.

  6. Record this value of time on the test sheet.

  7. Repeat the above test from step 1 to step 6 for other phases, but inject current at terminal 8 and 11 for phase S and terminals 9 and 12 for phase T.

Trip Test

  1. Request the Operator to switch on the main circuit breaker.

  2. Operate the output-delayed- contact by hand to trip the circuit breaker.

  3. Check that the circuit breaker trip.

Return the Relay to Service

  1. Verify the test results match the relay setting, if not consult the technical services.

  2. Test trip the circuit breaker associated with the relay.
