Introduction

MCTI 39 is normally used as a back-up protection for the circuit breaker when the circuit breaker fails disconnect when other protection operates.

This instrument is called " circuit breaker failure protection " 

This relay is a three phase device with a common output element for all three phases This protective relays manufactured by GEC, ALSTHOM this relays although generally of robust construction require careful treatment prior to installation and a wise selection of sit.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence with caution marks surrounding the work area is required.

Tools and Equipment

  • Variac

  • DC voltmeter to check auxiliary voltage

  • Double pole switch

  • Secondary injection test set 

Works to be Carried Out

The operating current, as a proportion of relay rated current, is set by a plug bridge and 1x to 2x multiplier. Thus the current setting is continuously variable between 1x the lowest tap setting and 2x the highest tap setting, eg to achieve of 0.72 x rated current on a relay with plug bridge taps of 0.2 x, 0.4 x,0.8 x and 1.6 x rated current, select the 0.4 tap and set the multiplier potentiometer to 1.8 x 0.4 x 1.8 = 0.72 rated current

DC auxiliary supply check (Vx)

In practice on breaker fail schemes the d.c supply is not applied to the relay until the main protection has operated. It is there for necessary to link across the protective relay contacts or across the appropriate contacts of the trip relay to apply the auxiliary d.c supply to the relay terminals

Current setting test

  1. Connect test leads between the current terminals on the relay (23-24) and the current terminals on the test set this for phase A
  2. Select plug tap to 0.2 and multiplier to 1.0
  3. Inject current and slowly increase the current until the relay operates. Note the value of the current and recorded
  4. Decrease the current slowly, note the value of current when the relay drops off and record the value of current
  5. Select the plug tap to 1.6 and multiplier to 2.0, repeat the tests and noting current values
  6. Repeat tests injecting in B and C Phases

Operate / Reset timing and Contact Operation

Operate Time

  1. Inject current at 2 X setting into "A" phase via a double pole switch, one pole to initiate a timer on closing

  2. Connect relay output contact to stop timer on closing

  3. Record operating time, operating time to be less than 12 m. sec

  4. Repeat test on B and C phases

Reset time

  1. Inject current at 2x setting into "A" phase via a double pole switch one pole to initiate a timer. on opening

  2. Connect relay output contact to stop timer on opening

  3. Record reset time, this should be less than 12 m. sec

  4. Repeat test on B and C phases

Return the Relay to Service

  1. Make a final check that all switches on the relay are set according to the setting

  2. Make sure the CT secondary circuits is in normal condition

  3. Return the relay to service
