Introduction

This procedure detail the steps required to completely check the proper operation and calibration of the SD34a relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

The high-speed distance relay SD34a is electromechanical three-phase distance relay, which is intended primarily for protection of medium and high voltage networks.

Over current starting relays are built into the SD34a relay, it also possible to connect additional under -impedance starting relay.

The high speed distance relay SD34a operates with a pure stepped resistance time characteristic, consisting of 4 steps, indeed the relay has 3 impedance steps and a part from the non-adjustable high-speed, step, three adjustable time steps, the last of which can function either directionally or independently of direction. Checking the Over Current Starting Unit

Over Current for Phase (R)

  1. Connect between two tow current terminals ( IR, IS, IT, or IM ) with the same position of the set current selector switch (b3) on the test set (AEG) to the current terminals (1 4 ) on the distance relay

  2. Slowly increase the current until the relay picks up.

  3. Record this value on the test sheet. As a pick up current

  4. Increase the injected current to value 25% greater than the operating value.

  5. Slowly decrease the current until the relay resets.

  6. Record this value on the test sheet. As reset current

  7. Inject current higher than the setting point so that the relay picks up firmly.

  8. Keep this value and switch off the test.

  9. Connect the tripping terminals (19 - 20) on the distance relay to the timer terminals on the test set.

  10. Switch on the test set.

  11. Start the test by injecting current into the relay the will stop once the relay has picked up.

  12. Record this value of time on the test sheet.
