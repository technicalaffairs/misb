Introduction

    The static switched distance protection SD35A with over current starting is intended for the selective protection of overhead lines or cables in medium high voltage systems the system can be operated with the neutral either isolated resonant-earthed via arc-suppression coils or low-resistance earthed.

    The static distance protection SD35A consists of several plug-in sup assemblies built into either a single-tier "19" Intermas sub-rack or a surface - mounting case. When built into a subrack connection is via interface connector and when built into a surface - mounting case, via terminal strip on the case. This construction from function modules with neatly arranged setting elements makes for ease of handling during setting and testing.

    These procedure details the steps required to completely check the proper operation and calibration of the SD35A relay. It is recommended this procedure be carried out on yearly bases. This word be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

    The first draft  initiated  and tested by eng. Soso Abd El Azeez, at 16 April 1998, in Talkha Lab.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category,

  • Maintenance staff should be well trained, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Safety and healthy equipment must be used (safety helmet , safety shoes ... etc).

  •  Line isolation and work permit required.

  •  Make sure CT secondary circuits are shorted for isolating the current circuits for test current injection

Tools and Equipment
  • AEG relay test set 74-2.
  • Voltmeter.
  • Clip-on ammeter.
  • Test leads
Work to be carried out

Preliminary Steps to Prepare for Relay Testing

  • Review technical reference manufacturer manual.
  • Obtain copy of relay setting from field file and compare it with the setting on the relay.
  • Make sure the bus and line disconnect switches, are open and tagged there is no AC current and there is no AC voltage.
  • Cheek that CT and VT secondary circuit are in normal condition.

Testing Procedure(s)

  1. Connect current terminals on the test set R, S, T, and N to the current terminal on the relay.
  2. Connect the voltage terminals on the test set: U, V, W and mp to the voltage terminals on the relay after disconnect it from the relay.
  3. Connect the main input cord of AEG74.2 test set to 220VA.C source.
  4. Energize the test set 74-2 by pushing b1 down.

Testing the Over Current Starting Unit

  1. Put b11, b12 at (0) angle.
  2. Put the fault selector switch b20 on (Rm) position.
  3. Put the b1 on (ON) position.
  4. Push the red push-button f b3
  5. Increase the injected current slowly by rotating (m1) until the relay picks up and (AR) red indication lights up.
  6. Record this value (I pick-up) on the test sheet and compare with setting value (I/IN).
  7. Decrease the current by rotating (m1) in the reverse direction until the relay resets.
  8. Record this value (I reset) in the test sheet.
  9. Calculate the value of rest factor K RESET = should be 0.9
  10. Push the button b2.
  11. Reduce injected current to zero and switch of the current by b2.
  12. Repeat the above test for other phases (Sm, Tm)
  13. Record this values in the test sheet.

Testing the Phase to Earth Impedance Characteristic

  1. Put b11, b12 at (0°) angle (Cos 0° = 0 & Sin 0° =1).
  2. Turn the switches: b15, b16 and b17 to agree with zone one setting.
  3. Turn b10 on lamp position.
  4. Put b8 in yellow line and b10 in blue line.
  5. Inject current by rotating (m1) greater than (I/IN)
  6. Turn b20 on Rm. Position.
  7. Start the test by pushing down b3 (red) and the first step time will be recorded on the screen of the timer, the red indications AR, T and ( Aus ) light up on the relay.
  8. Chang the % value of zone on e impedance by turning b21 up or down until the relay operating second step the red indications T2 on the relay will light up and the second step time will be record (I on the screen of the timer.
  9. Recorded this value of (ZI/Z2) on test sheet.
  10. Repeat the above with second zone impedance setting on the test set.
  11. Change b21 up until the relay operates in the third step.
  12. The red indication T3 on the relay will be light.
  13. Record this value of k (Z2/Z3) on test sheet.
  14. Push the push-button b2.
  15. Repeat the above test for phases Sm, Tm.
  16. After test every phase it must be set b13 on L.L position) and push down b3 to sure the direction of the rely.
  17. Repeat the above test angle 90° for all phases (Cos 90° = 1, Sin 90°=0)

Performance and Tripping Tests

  1. Press pushbuttons PR, PS/M, PT the starting signals are simulated and the corresponding led's will light.
  2. Press push buttons PA simultaneously with press push-buttons PR, PS/M, PT the trip signal will be formed.

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set.
  2. Check that CT and VT secondary circuits are in normal condition.
  3. If any equipment associated with the relay has been isolated it should be returned to service.

Returned Power Line to Service

  1. Check that reading agree with station meters (MW/MVAR/power Factor/Amp/Voltage)
  2. Place a copy of the test results in the field maintenance file.
