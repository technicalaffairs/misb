Introduction

    The RD110 relay is an electromechanical three-phase distance relay, which is used to protect many 33, 66 and 132KV lines at EEA substations. The relay has three zones of impedance protection each with a different timing step. The relay has one directional element and one measuring element, which work in conjunction with all zones. The relay measures phase to phase and phase to ground faults. It is manufactured by EAW.

    This procedure details the steps required to completely check the proper operation and calibration of the RD110 relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet ... etc.

  • Ensure that Line isolation is done.

  • Make sure CT secondary circuits are shorted and VT secondary circuits are opened before injection.

Tools and Equipment

  • Test set type PE1

  • Voltmeter

  • Clip - on ammeter

  • Phase Rotation meter

  • Test leads, Ammeter (5 AMP)

  • Secondary Injection Test Set.

 

Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical reference material: Manufacturer’s manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB’S), and all cabling supplying the relay.

  6. Use the station drawing to isolate the following:

  • Breaker trip coils.

  • Breaker failure scheme associated with this relay.
  • Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection Output

  1. Carefully follow these test procedure steps from page 2 to Page 7.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Test trip the breaker associated with the relay.

Prior to Starting Testing Procedure (s) Ensure the Following Steps

  1. This procedure is used to perform off load relay testing.

  2. Check that the Panel Nameplate agrees with the EEA designation of the line that has been taken out of service for maintenance.

  3. Check there is no AC current from CT by using clip-on ammeter.

  4. Check there is no AC voltage from VT by using voltmeter.

  5. Check all external connections to the relay are in good condition and check that the terminal block wires are tight.

Preparations

  1. Make sure secondary circuits are shorted before isolating the current circuits of the relay.

  2. Using voltmeter, check that there is DC voltage on the relay, (positive on terminal 25 and negative on 26).

  3. Switch off the MCB for a.c. voltage from voltage transformer.

  4. Label the wires before removing from terminals of the relay.

  5. Remove the wires from the current terminals (1, 2, 3, 4) on distance to terminals (6, 7, and 8) of the relay.

  6. Connect between the current terminals (1, 2, 3, 4) on distance relay to terminals (IR, TS, IT, Io) on the PE1 test set, respectively.

  7. Connect between the voltage terminals (5, 6, 7, 8) on the distance relay to terminals (UR, US, UT, Uo) on the PE1 test set, respectively.

  8. Connect the trip terminal (23, 24 or 21, 22) on the distance relay to terminal (Aus) on the PE1 test set.

  9. Put an ammeter between the two terminals labeled (I) and also put a short between terminals (OFFNER) on the test set.

  10. Set the current selector switch on the test set as shown in figure 1 to value according to rated current relay. (1 Amp = 10Z; 5 Amps = 2Z)

  11. Set the timer switch on the test set as shown in fig 1 to vertical position (on).

  12. Set the (W ) switches on the test set to zero position.

  13. Set the line angle selector switch on the test set to a value according to the line angle.

  14. Set the switch (GRUNDLAST) to (EIN) position.

  15. Set the switch (FEALERART) to (MK) position.

  16. Set the directional selector switch as shown in fig.1 to (BEZOG) position (forward direction)

  17. Set the main switch to (Aus) the position.

  18. The test set PE1 can now be energized by plugging the (Main Input) cord into a 220 Vac wall outlet.

  19. Set the main switch to (EIN) position.

Testing Procedure (s)

Checking the Pick-Up Value of the Phase to Phase Measuring Elements

Zone 1 Test

  1. Select phase to phase fault by turning the test set phase selector switch to (RS) position as shown in figure 1.

  2. Turn the (W ) switches to agree with the zone one setting.

  3. Start the test by pushing down the push button (EIN) (number 8 as shown in figure 1).

  4. The timer will stop once the relay has picked up which indicates that the relay has operated and tripped in first zone time T1.

  5. Change the value of zone one impedance which is set on the test set by moving the (W ) switches up or down until the relay operates in the second step.

  6. Record the value on test sheet 1.


    7. Note: If rk is used, it must be checked by the same method from step 1 to step 5. (at Zone 1 Test Section).

Zone 2 Test

  1. Set the (W ) switches to agree with the zone two setting.

  2. Start the test by pushing down the push button (EIN). The timer will stop once the relay has picked up which indicates that the relay has operated and tripped in second zone T2.

  3. Change the value of zone two impedance which is set on the test set by turning the (W ) switches up or down until the relay operates in the third step.

  4. Record this value on test sheet 1.

 

Zone 3 Test

  1. Set the (W ) switches to agree with the zone three setting.

  2. Start the test by pushing down the push button (EIN). The timer will stop once the relay has picked up which indicates that the relay has operated and tripped in third zone T3.

  3. Change the value of zone three impedance which is set on the test set by turning the (W ) switches up or down until the relay operates in the fourth step.

  4. Record this value on test sheet 1.

Phase to Phase Test for Other Phases (ST and TR)

Repeat the above tests (Zones 1, 2, 3) by changing the phase selector switch to ST and TR, respectively.

Checking the Pick-Up Value of the Phase to Ground Measuring Elements

Repeat the above tests (Zones 1, 2, 3) in addition to the previous step “Phase to Phase Test for Other Phases (ST and TR)” by changing the phase selector switch to R0, S0 and T0, respectively.

Reverse Direction Test

  1. Turn the directional selector switch to the vertical position (backward reach).

  2. Start the test by pushing down the push button (EIN).

  3. Observe that “ Qf ” flag will appear on the relay. This indicates that the fault is in the backward direction.

Measuring the Operating Times

  1. Set the impedance on the test set to a value which equals half the normal pick up value in the First time step for phase to phase fault. This will allow a good operation of zone one of the relay.

  2. Select phase to phase fault position.

  3. Reset the timer on the test set.

  4. Start the test by pushing down the push button (EIN). The timer will stop once the relay has picked up.

  5. Record the zone one tripping time from the timer on test sheet (1).

  6. Repeat the above test from step 1 to step 5 to measure the zone two tripping time once again but this time set the impedance on the test set to a value equal to (Z1 + Z2)/2.

  7. Repeat the above test from step 1 to step 5 to measure the zone three tripping time once again but this time set the impedance on the test set to a value equal to (Z2 + Z3)/2.

  8. Set the main switch of the test set to the (Aus) position.

  9. Disconnect the PE1 test set from the power supply and Remove the secondary connections between the relay and the test set.

 

 

Checking the Over-current Starting Units

Over-current Starting For Phase R

  1. Use secondary Injection test set to inject current between terminals 1 and 4 on the relay.

  2. Slowly increase the current until the relay picks up.

  3. Record this value on the test sheet.

  4. Increase the injected current to a value 25% greater than the operating value.

  5. Slowly decrease the current until the relay resets.

  6. Record this value on the test sheet.

  7. Inject current higher than the setting point so the relay picks up firmly.

  8. Keep this value and switch off the test set.

  9. Connect the tripping terminals (23, 24 or 21, 22) on the distance relay to the timer terminals on the test set.

  10. Switch on the test set.

  11. Start the test by injecting current into the relay. The timer will stop once the relay has picked up.

  12. Record this value of time on the test sheet.

Over-current Starting for Phases S and T

  1. Repeat the above test from step 1 to step 12 but inject current between terminals 2 and 4 for phase S and terminals 3 and 4 for phase T. This completes the test.

  2. Remove the test leads between the relay and test set.

  3. Return the wires, which were removed in step 5 Page 3.

Trip Test

  1. Switch on the MCB, which was switched off in step 3 Page 3.

  2. Request the operator to switch on the main CB of the line.

  3. Operate the output auxiliary contact E1 by hand to trip the circuit breaker.

  4. Check that the circuit breaker tripped.

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test lead connections from the relay.

  2. Any equipment associated with the relay has been isolated should be returned to service by the following steps :

  • Breaker trip coils.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

  1. Check that CT and VT secondary circuits are in normal condition.

Request Power line be returned to Service

  1. Take crossed readings of all AC. currents and voltages supplied to the relay and compare them to those for normal load conditions.

  2. Check that readings agree with station meters: (MW/ MVAR/Power factor/Amperes/voltages)

  3. Place a copy of the test result in the field maintenance file
