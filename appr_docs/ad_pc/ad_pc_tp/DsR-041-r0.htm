Introduction

    Static distance protection relays for the protection of high voltage transmission lines and under ground cables. Test set equipment type ZFB contained three portable metal cases: Supply unit, Control unit, Fault impedance.

    This procedure details the steps required to completely check the proper operation and calibration of the Optimho relay relay. It is recommended this procedure be carried out on an annually bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

    The first draft initiated and tested at 6 November1997by eng. Alaa Eldeen Mohamed, accepted by general director eng. Afaf Elbadoy, and approved by eng .Mahmod Hasan .

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet .

  • Ensure that Line isolation is done.

  • Make sure CT secondary circuits are shorted and VT secondary circuits are opened before injection.

Tools and Equipment

  • ZFB test set. (This test set consists of 4 units: supply unit, control unit, fault impedance unit and CT unit).

  • Timer

  • Test Plugs 1V and 2V

  • Multi-meters

  • Secondary injection test set

  • Test leads

  • The ZFB test set is fully described in publication R5228.

  • Screwdriver . Test box with parallel plug

Work to be carried out 

Preliminary Steps to Prepare for Relay Testing

  1. This procedure is used to perform off load relay test.

  2. Review technical reference material: Manufacturer's manual.

  3. Obtain copy of relay setting from field maintenance file.

  4. Obtain outage for line protected by this relay.

  5. Check that the bus and line disconnect switches are opened and tagged.

  6. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB'S), and all cabling supplying the relay.

  7. Use station drawing to isolate

     

    • Breaker trip coils and transfer trip scheme.

    • Breaker failure scheme associated with this relay.
    • Disturbance alarms and fault recorder associated with this relay.
    • Auto-reclosure associated with this relay.

  8. Look at the EEA panel nameplate and check that this EEA system designation nameplate agrees with the line that has been taken out of service for maintenance.

  9. Check there is no AC current from CT's by using a clip on ammeter.

  10. Check there is no AC voltage from VT's by using a voltmeter.

Testing the Relay and Protection Outputs

  1. Carefully follow all test procedure steps from page 2 to page 6.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Perform checks of the acceleration of the relay, breaker failure and reclosure schemes, separate technical procedures should be used to test each of these systems.

  4. Test trip all breakers associated with the relay.

Preparation

  1. Connect the ZFB test set as shown in fig. 1

  2. From station drawing determine:

    • The input and output current terminals.

    • The voltage terminals.

  3. According to step 2:

    • Connect the current terminals IA, IB, IC and IN on the test plug to the corresponding current terminals on the supply unit test set.

    • Ensure that the CT secondary circuit are correctly short circuited on the case side of test plug.

    • These connections must be made before the test plug is inserted

    • Connect the voltage terminals VA, VB, VC and VN on the test plug to the corresponding voltage terminals on the supply unit test set.

  4. Remove relay isolating plugs 1V on Card RFC04102001E and 2V on Card RVH02401000B

  5. Insert the test plug in the relay.

  6. Set the main on / off switch, S1, on the supply unit to off position.

  7. The test set type ZFB can now be energized by plugging the (Main Input) Card into 380 Vac supplies source.

Testing the Phase to Phase Measuring Elements

  1. Calculate from line impedance Z, the value for phase to phase impedance corresponding to zone one setting.

    ZL-L= 2* Z

  2. A table of the range of fault impedance available from the test set along with the available fault angles appears inside the lid of the fault impedance unit. Select a value of fault impedance close to ZL-L from the table and it should have:

    • A phase angle as near as possible to the relay characteristic angle.

    • Fault impedance just above the calculated value.

     

      Note:The corresponding X (choke tap) and R (res. tap) from the table.

     

  3. On the fault impedance test set unit, connect the resistance and reactance taps which correspond to the fault impedance.

  4. Select the source impedance required by connecting the flexible link from the terminal marked COMMON to the appropriate ohm tap of the source impedance. If it is required to test the relay under maximum voltage conditions the source impedance should be short-circuited by linking the COMMON to the 0 terminal.

  5. Divide the fault impedance selected by the calculated value.
    % potential = Fault impedance selected / calculated value

  6. Set the % potential switches S5 and S6 on the control unit to 100%.

  7. Set the current reverse switch S3 on the control unit to the normal position.

  8. Select phase to phase fault A-B by using the relay selector switch on the supply unit test set.

  9. Connect the outgoing voltage lead from the supply unit to the terminal marked Ph/F.

  10. Switch on switch Sl on the supply unit.

  11. Start the test by pressing the FAULT push-button on the control unit momentarily.

  12. Observe that the red indications A, B and Z1 on the relay will light up.

  13. Change the value % Potential tap up or down until the relay operates in second step.(Z2 on the relay will appear on the screen in the relay)

  14. Record this pickup point on test sheet.

Repeat the Above Test for Phases B-C and C-A

  1. Repeat the above test from step 1 to 14 but this time, calculate % potential corresponding to the relay setting of the second zone for phase to phase fault. Set this value on the test set and then change it up or down until the relay operates in third step. (Z3 will appear on the screen in the relay)

  2. Switch off, switch Sl on the supply unit.

Testing the Phase to Ground Measuring Elements

  1. Calculate from line impedance Z, the checking value for phase to ground impedance corresponding to zone one setting:
    Zph-to ground = (1+K) Z

  2. Calculate % Potential - which will set on switches S5 and S6 on control unit test set by the same method as shown in step 2 to 6 in pages 3 and 4 but for phase to ground.

  3. Select phase to ground fault A-N by using relay selector switch on supply unit test set.

  4. Connect the outgoing voltage from the supply unit to the terminal marked E/F

  5. Switch on the switch S1.

  6. Start the test by pressing the "FAULT" push-button on the control unit momentarily

  7. Observe that the red indications A, N and Z1 on the screen in the relay.

  8. Change the value % Potential tap up or down until the relay operates in second step. (Z2 will appear on the screen in the relay)

  9. Record this pickup points on test sheet

Repeat the Above Test for Phases B-N and C-N

  1. Repeat the above test, from step 1 to 9 but this time, calculate % Potential corresponding to the relay setting of the second zone for phase to ground fault. Set this value on the test set and then change it up or down until the relay operates in third step.

  2. Switch off switch S1.

Measuring the Operating Times

  1. Set % potential switches S5 and S6 to 50% of the zone 1 reach.

  2. Select phase to phase fault A-B by using relay selector switch on supply unit test set

  3. Connect the outgoing voltage from the supply unit to the terminal Ph/F.

  4. Connect a timer having the start contact operated by the start timer contact on the ZFB control unit.

  5. Connect test leads between contact on the relay to the stop terminal of the timer.

  6. Reset the interval timer.

  7. Switch on S1.

  8. Start the test by pressing the "FAULT" the timer will stop once the relay has picked up

  9. Record the zone 1 tripping time on test sheet

Repeat Tests for Zone 2 and Zone 3 Operating Times

  1. Repeat the above test from step 1 to 9 to measure the second zone tripping time, but this time set % potential S5 and S6 to a value approximately midway between zone 1 and zone 2.

  2. Repeat the above test from step 1 to 9 to measure the zone three tripping time, but this time set % potential S5 and S6 to a value approximately midway between zone 2 and zone 3.

Repeat Tests for Phase to Earth Operating Times

  1. Repeat the above test from step 1 to 10 to measure the times of the three zones but for phase to earth.

  2. Switch 6.

Power Swing Blocking Test

  1. Set the relay on Option 08

  2. Set the taps on the impedance unit of the ZFB tester to a value greater than zone 6 impedance.

    Note Z5 and Z6 are special zones built into the relay to detect power swings.

  3. Switch on the switch S1

  4. Press push-button FAULT.

  5. Decrease the % potential switches S5 and S6 until the relay picks up and 1 appears under the column on the monitor.

    Monitor             0 0 0 0 0 0 1 0

    OPT 08             0 0 0 0 0 0 1 0

Measuring the Time Z6 (TZ6)

  1. Set the relay on option 08

  2. Insert the plug connected to the test box into the parallel socket on the front of the relay.

  3. Connect the timer to the test box.

  4. Set the taps on the impedance unit of the ZFB tester to a value greater than zone 6 impedance.

  5. Decrease the % potential switches S5 and S6 until the relay operates in zone 6.

  6. The timer will start when comparator A - B picks up and will stop when Z6 time has expired

  7. Record this value on the test sheet.

Simulated Power Swing Blocking

  1. Return to the default display "Power Swing Test Enabled"

  2. It is now necessary to simulate power swing for the A-B element of the relay. Apply an impedance just out side zone 2 or 3 as appropriate and reduce it without switching off (say 1 %) until power swing indication is given.

  3. Check that the PSB alarm contact also closes.

  4. Re-select monitor option 08 which makes available information on all zones giving trip out puts after any set time delay

    Monitor         2 3 4 5 6 7 8 9

    Opt 08          0 0 0 0 0 0 0 0

  5. With this it is easy to check which zones are being blocked by the PSB feature in the menu selection. The power swing will be simulated by applying an A-B fault just out side Zone 3 or Zone 2 as appropriate, moving the fault to just inside the Zone and then moving it again to inside Zone 1.

  6. Apply the fault as above and again move in to just inside Zone 2 or Zone 3 as appropriate and then move the fault again to inside zone 1 in less time than the lowest time delay setting. On the ZFB, this is done by first applying the fault and moving it with the fine percentage taps. Then short out the fault impedance with a switch. Make sure that the fault impedance is at least 12 ohms for this test

  7. Only the forward Zones not blocked by the PSB feature will change from 0 to 1 on the LCD display.

Voltage Transformer Supervision

Operation occurs when the zero sequence voltage above a set level is detected without any zero sequence current being detected above a set level. If set "TO ALLOW TRIP" instantaneous indication will be given if operation conditions are satisfied and a comparator is operated.

Otherwise, time delayed indication will be given. If set for blocking, blocking can tack place instantaneously if a comparator operates and the indication will be "RELAY BLOCKED V~ FAIL". The relays have the facility for self resetting of the VTS alarm contact and relay blocking. This feature can be enabled or disabled via the menu.

Operation on Zero Sequence Volts

The operation time 5.5 second+0.1 seconds. The zero sequence voltage setting is 15% of 635 v, thus the required voltage reduction on one phase is 45% of 63.5v

Apply balance 3 phase neutral volts and reduce the voltage on one phase until v ~FAIL indication is given. The reduction in one phase - neutral voltage required for operation should be between 22.9 and 34.3 volts

Timing Check

Timing is check by using a double pole switch to remove one healthy voltage, and start a timer, which is stopped by the VTS alarm contact. The time obtained should be as stated above.

Instantaneous Indication

To perform this check it is necessary to provide the relay with zero sequence voltage but no zero sequence current and zone 1 comparator operation this can be achieved with a ZFB as follows:

  1. Select an A-N fault and 100% potential setting on the current output terminal remove the neutral connection and connect the B current connection the neutral terminal as in figure 3.

  2. Press the fault button and via a double pole switch remove A and B voltage from the relay.

  3. Instantaneous indication of V ~ FAIL should be given ( the test could be done in a similar way with other dynamic test sets )

Blocking Check

If VTS is now set to BLOCK TRIP, check that the indication RELAY BLOCKED is obtained when the above conditions are satisfied and no trip is given. the RELAY BLOCKED indication can not be reset or other information viewed (other than date and time of V ~ FAIL) until normal volts are restored

If self-resetting has been selected in the menu the RELAY BLOCKED indication will automatically reset when volts are restored leaving the V~FAIL indication only. If blocking is to be used only select it after all tests are finished.

Return Relay and Associated Equipment to Service

  1. Return relay to service by withdrawing the test plug handle from the relay.
  2. If any equipment associated with the relay has been isolated without the test plug, it should be returned to service by the following restoration steps:
    •  Breaker trip coils and transfer trip scheme.
    •  Breaker failure scheme associated with this relay.
    •  Disturbance alarms and fault recorder associated with this relay.
    •  Auto-reclosure associated with this relay.

Request Power Line be Returned to Service

  1. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those for normal load conditions.

  2. Check that readings agree with station meters: (MW/ MVAR/Power factor/Amperes/Voltages)

  3. Place a copy of the test result in the field maintenance file.
