Introduction

This document lists the testing procedure P1, which must be performed on an annually bases as indicated by Y1 in the above title block

The inverse time over current of IKC relays have been designed also for the detection of  earth faults & leakage. The IKC911 relays are single phase and the IKC913 relay are three phase.

 Safety Precautions

  • A work permit must be issued & the staff  of test must be qualified and carry the suitable safety class (category).

  • Safety fence (rope) with caution marks surrounding the work area is required.

    •

  • The auxiliary transformer must be out of service.

  • Wear safety shoes.

Tools and Equipment

  • Test Plug handle,

  • Clamp ammeter,

  • Test leads,

  • Secondary injection test.

Work to be Carried Out

Preparation

  1. Make sure CT secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

 

Testing the Relay 

Relays IKC911   (Figure 1)

  1. Connect test leads between  the current terminals ( A2 – A3  ).

  2. Connect the test leads between the trip contact  ( A4 – A5  )

  3. Increase the current slowly until the relay picks up . ( I Start )

  4. Record the current values and the time values at each current value at which the  relay trips  in the test sheet  .

  5. Increase the current to the values ( 1 I start – 2 I start & 3 I start  ) .

  6. Record the current values and the time values at each current value at which the  relay trips  in the test sheet  .

Relays IKC913      (Figure 2)

 Over Current Test

  1. Repeat the step no. 1 for phase R .

  2. Connect the test leads between the current terminals ( 6 –7 ) for phase S.

  3. Repeat the steps from 2 to 6 for phase S .

  4. Connect test leads between the current terminals ( 10 –11 ) for phase T .

  5. Repeat the steps from 2 to 6 for phase T .

 Short Circuit Test

  1. For phase R ( A2 – A3 ) increase the current quickly to high value until the relay trips instantaneously

  2. Record the current value in the test sheet .

  3. For phase S ( A6 – A7 ) repeat the steps 12 & 13 .

  4. For phase T ( A10 – A11 ) repeat steps 12 & 13 .

 

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set i.e.
    -     Disconnect the test leads from the terminals ( A2- A3 ) , ( A6 - A7 ) , ( A10 - A11 )
    - Disconnect the test leads between the trip contact ( A4 - A5 ). 

  2. Make sure CT and PT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
           - Breaker trip coils.
           - Breaker failure scheme associated this relay.
           - Disturbance alarm and fault recorder associated with this relay.
