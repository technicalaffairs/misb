Introduction

Applications for the DLP system is designed to be used on transmission lines, the DLP provides four Zones of distance protection and six different protection schemes.

  -Step distance,       - Zone 1 extension,        - Permissive Over Reach Transfer Trip (POTT),     
 - Permissive Under,              Reach Transfer Trip( PUTT )          - Blocking              -Hybrid

Safety Precautions 
  • Work order must be issued,
  • Checks that the Bus and line disconnect switches are open and tagged.
  • Make sure CT secondary is shorted.
  • VT miniature circuit breaker (M-C-B’s) is isolated (opened)
  • Isolate DC supply to the relay when doing only adjustments or maintenance on the DLP relay.
  • Isolate breaker failure scheme associated with relay.
  • Isolate disturbance alarms or fault recorder associated with DLP relay.
  • Isolate breaker trip coils and transfer trip scheme.
  • Isolate Auto-reclosing associated with the relay DLP
  • Test plugs isolated for all the above points.

Tools and equipment

  • Test set type XS92a.
  • Ammeter
  • Test plug and test leads
  • Timer

Work to be Carried Out

  1. Review technical reference maternal manufactures manual version for digital line protection (DLP)

  2. Obtain copy of relay setting from field maintenance file.

  3. Obtain outage for line protected by this relay (DLP)

  4. Check that the bus and line disconnected switches are opened and tagged.

  5. Black protection signals to breaker trip coils and transformer trip schemes

  6. Go to the switchgear and inspect the CT’s and V.T miniature circuit breaker (M-C-B’S) and all cabling supplying the DLP relay.

  7. Check the cables are in good condition and that all wires connections are tight and clean.

Testing procedure

Zone reaches

Zone I                            phase to earth

  1. Restore D.C Supply to DLP relay scheme

  2. Prepare the set test XS92a as following:


        MOD                          ZZ                        (Distance)

          ZL                   UP to setting Z1 at PZL = 85˚ 

          K0                        0.4

          FC                    Fault code Forward (111) “phase to earth”

          SQ                   1F1 

           A                      2

          TF                    Time fault Zone (250 ms) Zone 1

         DZL                 Step parameters and Range limits (0.1)

         Start (-)             Injection of testing signals


Note

  1. We enter the above parameters according to the flow chart of test  set  XS92a        

  2. When the relay DLP gives trip signal press ◙ to stop injection.

  3. Read the values In hand terminal of  test set XS92a and compare this value with the value of setting in test sheet of the relay.

  4. Press the reset button on the relay front key board (= ↓ + CLR ) , check that the indicator order trip on screen the DLP and alarm contacts are open.

  5. Repeat the above test with the phase S by using FC “ fault code” (112) & T by using FC (113) for Z1 at angle 85˚. Make sure the relay DLP gives trip signal ling with all phases R, S, T

Phase to  Phase 

  1. Entered the all above parameters, by using FC "fault code" 

             FC    “fault code “      Phase R-S             121        forward direction ( trip )

                                               Phase S-T             122                     ,,      

                                               Phase T-R             123                     ,,

            In the reverse direction (021) & (022) & (023) block.

 

 

Zone II 

  1. Repeat the above test for Zone I but change the value of Z2 at PZL = 85˚ ,TF up to time of Z2

  2. Make sure that the relay DLP in forward direction (trip) and reverse direction (block)

Zone III 

  1. Repeat the above for Zone II, but Z3 at PZL = 85˚ TF up to time of Z3

  2. Make sure that the relay DLP in forward direction (trip) and reverse direction (block)

Zone IV 

  1. Repeat that the above test for Zone III ,but Z4 at PZL = 85˚ and TFup to the time of Z4.
  2. Make sure that the relay (DLP) In for ward direction (trip) and reverse direction (block) with all phase R,S,T.

Notes:

 How to find “free contact for relay (DLP) ?

  1. Select any contact (free contact) and measured D.C (Not found)
  2. Measured (Buzzer) by using Avometer in this contact (Not found)
  3. Make trip by using test set XS92a for relay DLP, then this free contact measured buzzer by Avometer
  4. Then connect this free contact to test set XS92a in to point (red)'+",(yellow) in the front.

End – To – End Test For Relay DLP

  1. Repeat the test of zone II and Make sure that the trip accrue at time of zone II (t2) "500ms".
  2. Make bridge on receive signal.
  3. Make sure that the relay DLP Trip zone II at time of Zone I (100) m.sec

Switch - on -To – Fault (SOTF ) Test

  1. Repeat the test of zone 4 but (SQ = OF1) In test set XS92a
  2. Make sure that the zone IV trip at time of zone 1

Return protection scheme and associated equipment to service

  1. Return the relay DLP to service by with drawing the test plug handle from the relay

  2. Make sure CT secondary are opened.

  3. Return the relay to service by restoring supplies from voltage transformer (V.T) (M.C.B) is closed.

  4. Place the associated breaker failure (B.f.) protection scheme back in service.

  5. Restore the previously blocked disturbance alarms and fault recorder associated with this protection.

  6. Return the relay power line carrier (PLC) links to their nominal position.

  7. Turn on the associated recluse scheme.

  8. Turn on the power supply D.C to the relay DLP

  9. Take crossed reading of all A.C current and voltage and power in the relay DLP and compare them to those for nominal load condition.

  10. Check that these readings agree with station meters MW, MVAR, PF, Ampres, voltages.

  11. Place a copy of the test results in the field maintenance file and test sheets
