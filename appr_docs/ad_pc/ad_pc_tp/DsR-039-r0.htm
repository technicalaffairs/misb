Introduction

    This procedure details the steps required to completely check the proper operation and calibration of the Micromho distance relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

    The first draft initiated and tested at 6 November1997by eng. Alaa Eldeen Mohamed, accepted by general director eng. Afaf Elbadoy, and approved by eng .Mahmod Hasan .

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet .

  • Ensure that Line isolation is done.

  • Make sure CT secondary circuits are shorted and VT secondary circuits are opened before injection.

Tools and Equipment
  • ZFB test set.This test set consists of supply unit, control unit, fault impedance unit and CT unit.
  • Timer, Multimeters, Test leads
  • Test Plugs 1V and 2V
  • Secondary injection test set
  • The ZFB test set is fully described in publication R5228.
  • Screwdriver.  

 

Work to be carried out

Preliminary Steps to Prepare for Relay Testing

  1. This procedure is used to perform off load relay test.

  2. Review technical reference material: Manufacturer's manual.

  3. Obtain copy of relay setting from field maintenance file.

  4. Obtain outage for line protected by this relay.

  5. Check that the bus and line disconnect switches are opened and tagged.

  6. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB'S), and all cabling supplying the relay.

  7. Use station drawing to isolate:

  • Breaker trip coils and transfer trip scheme.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

  • Auto-reclose associated with this relay.

Testing the Relay, Relay Scheme, and Protection Outputs

  1. Carefully follow these test procedure steps

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Perform checks of the acceleration of the relay, breaker failure and reclosure schemes, separate technical procedures should be used to test each of these systems.

  4. Test trip all breakers associated with the relay.

Prior to Starting Testing Procedure (s) Ensure the Following Steps

  1. Look at the EEA panel nameplate and check that this EEA system designation nameplate agrees with the line that has been taken out of service for maintenance.

  2. Check there is no AC current from CT's by using a clip on ammeter.

  3. Check there is no AC voltage from VT's by using a voltmeter.

Preparation

  1. Connect the ZFB test set as shown in fig. 1

  2. Connect test plug 1V as shown in fig. 2 ( a ) and test plug 2V as shown in fig 2(b).

  3. Ensure that the CT secondary are correctly short circuited on the case side of test plug 1V

  4. These connections must be made before the test plug is inserted.

  5. Connect the current terminals IA, IB, IC and IN on the test plug to the corresponding current terminals on the supply unit test set

  6. Connect the voltage terminals VA, VB, VC and VN on the test plug to the corresponding voltage terminals on the supply unit test set.

  7. Remove relay isolating plugs 1V on Card RFC04102001E and 2V on Card RVH02401000B

  8. Insert the test plug 1V in the relay on rack number 31- Card RFC04102001E.

  9. Insert the test plug 2V in the relay on rack number 29- Card RVH02401000B

  10. Set the main on / off switch, S1, on the supply unit to off position.

  11. The test set type ZFB can now be energized by plugging the (Main Input) Card into 380 Vac supply source.

Testing the Phase to Phase Measuring Elements

  1. Calculate from line impedance Z, the value for phase to phase impedance corresponding to zone one setting. ZL-L= 2* Z

  2. A table of the range of fault impedance available from the test set along with the available fault angles appears inside the lid of the fault impedance unit.

  3. Select a value of fault impedance close to ZL-L from the table and it should have:

  • A phase angle as near as possible to the relay characteristic angle.

  • Fault impedance just above the calculated value.
     Note:  The corresponding X (choke tap) and R (res. tap) from the table.

  1. On the fault impedance test set unit, connect the resistance and reactance taps which correspond to the fault impedance.

  2. Select the source impedance required by connecting the flexible link from the terminal marked COMMON to the appropriate ohm tap of the source impedance. If it is required to test the relay under maximum voltage conditions the source impedance should be short circuited by linking the COMMON to the 0 terminal.

  3. Divide the fault impedance selected by the calculated value. % potential = Fault impedance selected / calculated value

  4. Set the % potential switches S5 and S6 on the control unit to 100%.

  5. Set the current reverse switch S3 on the control unit to the normal position.

  6. Select phase to phase fault A-B by using the relay selector switch on the supply unit test set.

  7. Connect the outgoing voltage lead from the supply unit to the terminal marked Ph/F.

  8. Switch on switch Sl on the supply unit.

  9. Start the test by pressing the FAULT push-button on the control unit momentarily.

  10. Observe that the red indications A, B and Z1 on the relay will light up.

  11. Change the value % Potential tap up or down until the relay operates in second step. (The red indication z2 on the relay will light up with the last indications)

  12. Record this pickup point on test sheet.

Repeat the Above Test for Phases B-C and C-A

  1. Repeat the above test from step 1 to 14 but this time, calculate % potential corresponding to the relay setting of the second zone for phase to phase fault. Set this value on the test set and then change it up or down until the relay operates in third step. (The red indication Z3 on the relay will light up with the last indication)

  2. Switch off switch Sl on the supply unit.

Testing the Phase to Ground Measuring Elements

  1. Calculate from line impedance Z, the checking value for phase to ground impedance corresponding to zone one setting
    Zph-to ground = (1+K)Z

  2. Calculate % Potential, which will set on switches S5 and S6 on control unit test set by the same method as shown in step 2 to 5 but for phase to ground .

  3. Select phase to ground fault A-N by using relay selector switch on supply unit test set.

  4. Connect the outgoing voltage from the supply unit to the terminal marked E/F

  5. Switch on the switch S1.

  6. Start the test by pressing the "FAULT" push-button on the control unit momentarily

  7. Observe that the red indications A, N and Z1 on the relay will light up.

  8. Change the value % Potential tap up or down until the relay operates in second step. (The red indication z2 on the relay will light up with the last indications)

  9. Record this pickup points on test sheet

Repeat the Above Test for Phases B-N and C-N

  1. Repeat the above test, from step 1 to 9 but this time, calculate % Potential corresponding to the relay setting of the second zone for phase to ground fault. Set this value on the test set and then change it up or down until the relay operates in third step.

  2. Switch off switch S1.

Directional Test

  1. Set the source impedance to 6 ohms and the fault impedance to zero.

  2. Set the % potential switches S5 and S6 to zero.

  3. Select a fault condition using the fault selector switch.

  4. Switch on the switch S1.

  5. Start the test by pressing the "FAULT" push-button on the control unit and hold down checking that the faulty phase operates in zone 1,2 and 3. Release the "FAULT" Push-button

  6. Repeat steps 5 and 6 for all phases of the protection.

  7. Set the current reversing switch S3 on the control unit to the REVERSE position.

  8. Repeat steps 5, 6 and 7. Keep the FAULT push-button depressed for a little longer than Zone 3 time. Operation should not occur in Zones I and 2, but can occur in Zone 3 only if the relay has reverse reach

  9. Switch off the switch S1.

  10. Return the current reversing switch S3 on the control unit to the NORMAL position.

Measuring the Operating Times

  1. Set % potential switches S5 and S6 to 50% of the zone 1 reach.

  2. Select phase to phase fault A-B by using relay selector switch on supply unit test set

  3. Connect the outgoing voltage from the supply unit to the terminal Ph/F.

  4. Connect a timer having the start contact operated by the start timer contact on the ZFB control unit

  5. Connect test leads between contact number, on the relay to the stop terminal of the timer.

  6. Reset the interval timer.

  7. Switch on S1.

  8. Start the test by pressing the "FAULT"

  9. The timer will stop once the relay has picked up

  10. Record the zone 1 tripping time on test sheet.

Repeat Tests for Zone 2 and Zone 3 Operating Times

  1. Repeat the above test from step 1 to step 9 to measure the zone two tripping time, but this time set % potential S5 and S6 to a value approximately midway between zone 1 and zone 2.

  2. Repeat the above test from step 1 to step 9 to measure the zone three tripping time, but this time set % potential S5 and S6 to a value approximately midway between zone 2 and zone 3.

Repeat Tests for Phase to Earth Operating Times

  1. Repeat the above test from step 1 to step 10 to measure the times of the three zones but for phase to earth.

  2. Switch off 51.

Synchronous Polarizing Check

  1. Connect test plug 2V.

  2. Set the source impedance to 24 ohms.

  3. Select the fault impedance to zero.

  4. Connect a time interval meter, selected to the dwell mode, having the start circuit connected to terminals VFI and VF2 on the protection.

  5. Switch on switch S1.

  6. Select a B-N fault with fault selector switch.

Note: Do not attempt this test using an A-N fault.

  1. Operate the FAULT push-button. Note the time indicated on the interval meter, which should read something in excess of 170ms, indicating that the synchronous polarizing is functioning. It will also be noticed that, during this test, the relay operation will appear to be transient. The reason for this is that the relay will drop off when the synchronous polarizing has disappeared. The maximum time may be much greater than 170ms, but only the minimum time is important.

  2. Switch off switch S1.

Switch on to Fault (SOTF) (With Level Detectors

  1. Connect the ZFB as shown in Figure 1.

  2. Connect test plug 1V as shown in Figure 2(a), and test plug 2V as shown in Figure 2(b).

  3. On module 114-of the protection sqt: Link 0 =I

  4. Set the source impedance to zero.

  5. Set the fault impedance to a value beyond the Zone 3 Reach for an A-N fault.

  6. Set the voltage switches S5 and S6 to 75%.

  7. Select an A-N fault condition on the ZFB.

  8. Connect a voltmeter 0 - 150V ac to the voltmeter terminals on the ZFB control unit.

  9. Ensure that ZFB is switched off.

  10. Hold down the fault push-button and then switch on the ZFB.

  11. If no SOTF operation occurs, switch off S1, reduce the voltage in 1 % steps and repeat 9 until operation occurs.

  12. Record the voltage.

  13. Switch off the ZFB.

  14. Voltage for operation should be 44.5V.

  15. Repeat steps 9 to 14 10 for other Ph-N fault conditions.

Switch on to Fault (Using Zone Comparators)

  1. Connect the ZFB as shown in Figure 2.

  2. Connect test plug IV as in Figure 2 (a).

  3. Connect test plug 2V as shown in Figure 2 (b).

  4. On module 14 of the protection set: Link 0 = 0

  5. Set the source impedance to 6 ohms.

  6. For a blocking scheme or any scheme in which Zone 3 is reversed, set the fault impedance to a value approximately midway between Zone1 and Zone 2 reach. For any other scheme set the fault impedance to a value between Zone2and Zone3.

  7. Select an A-N fault condition on the ZFB.

  8. Connect a voltmeter 0 - 150V to the voltmeter terminals on the ZFB.

  9. Ensure that the ZFB is switched off.

  10. Set the voltage switches on the ZFB to 100%.

  11. Hold down the fault push-button and then switch on the ZFB.

  12. SOTF operation should occur and indication will also be given of the faulted phase together with the zone of operation.

  13. Switch off the ZFB.

  14. Repeat steps from 9 to 11 for other Ph-N fault conditions.

  15. With conditions as for 4 above with Link I = I repeat 9 and check that the protection operates instantaneously indicating A-N, Zone 3, Trip on Close.

  16. Reset the indication on the protection and repeat 9. The protection should trip immediately and indicate as above.

  17. Switch off the switch S1.

Voltage Transformer Supervision

  1. Set the protection on Option 46.

  2. Now all the red indicators will be flashing.

  3. Remove the current test leads between the test plug 1V and the current terminals on the supply unit test set.

  4. Short circuit the current terminals on the supply unit test set.

  5. Connect test plug 2V as shown in figure 2(c).

  6. Select an A-N fault condition on the ZFB.

  7. Switch on switch S1.

  8. Set the % potential switches S5 and S6 to 30%.

  9. Increase % potential switches S5 and S6 in 1% steps but wait 6 seconds between each step.

    Note: The red indicator N will light up at each step until a certain % potential is reached

  1. Record the value where the red indicator does not light up.

  2. Switch off switch S1.

Low Set Current Block and High Set Current Block

  1. Set the protection on Option 46.

  2. Now all the red indicators will be flashing.

  3. Connect test plug 1V as shown in figure 3 with the current terminals connected to a secondary injection test set.

  4. Set the % potential switches S5 and S6 to a value greater than that recorded in step 9.10.

  5. Ensure an A-N fault is selected on the ZFB.

  6. Switch on switch S1 on the ZFB.

  7. Start the test by pressing the "FAULT" push-button on the ZFB.

  8. The red indicator "Fail" on the relay will be flashing.

  9. Slowly increase the current from the secondary injection test set until the "Fail" indicator stops flashing for 6 seconds and then starts flashing again.

  10. Record this value on the test sheet.

  11. Increase the current until the "Fail" indicator stops flashing permanently.

  12. Record this value on the test sheet.

  13. Decrease the current and switch off both test sets.

Voltage Level Detected

  1. Set the protection on Option 46.

  2. Now all the red indicators will be flashing.

  3. Hold in the reset push-button on the relay continuously during this test. The indicators will now stop flashing.

  4. Select an A-N fault condition on the ZFB.

  5. Switch on switch S1 on the ZFB.

  6. Start the test by pressing the "FAULT" push-button on the ZFB.

  7. Set the % potential switches S5 and S6 to a value which causes the red indicators Z1, Z2 and Z3 to light

  8. Decrease % potential switches S5 and S6 in 1% steps until the red indicator Z1 switch off.

  9. Measure the voltage on the ZFB control unit terminals labeled "Voltmeter".

  10. Record this value on the test sheet.

  11. Repeat the above test from steps 1 to 9 for phase B-N but this time red indicator Z2 will switch off.

  12. Repeat the above test from steps 1 to 9 for phase C-N but this time red indicator Z3 will switch off.

  13. Switch off switch S1 on the ZFB.

High Set Current Detector

  1. Set the protection on Option 46.

  2. Continue to hold in the reset push-button on the relay continuously during this test. The indicators will now stop flashing.

  3. Connect test plug 1V as shown in figure 4 to the current terminals on the secondary injection test set.

  4. Slowly increase the current from the secondary injection test set until the red indicator A lights up.

  5. Record this value on the test sheet.

  6. Repeat the above test from steps 1 to 6 for phases B-N and C-N.

Neutral Current Detector

  1. Keep the relay set as in step 1 and 3.

  2. Slowly increase current until the red indication of phase A lights up.

  3. Continue to increase the injected current until red indication N lights up.

  4. Record this value on the test sheet.

  5. Switch off the secondary injection test set.

Low Set Current Detector

  1. Set the protection on Option 47.

  2. Red indicators A, B and C will light immediately.

  3. Connect the relay as step 3.

  4. Slowly increase the current from the secondary injection test set until the red indicator Z1 lights up and red indicator A will switch off.

  5. Record this value on the test sheet.

  6. Repeat the above test from steps 1 to 5 for phases B-N but this time red indicator Z2 will light up and red indicator B will switch off.

  7. Repeat the above test from steps 1 to 5 for phase C-N but this time red indicator Z2 will light up and red indicator C will switch off.

  8. Switch off the secondary injection test set.

Restore the Relay to Service

  1. Unplug the test set ZFB from the power source.

  2. Remove the test leads from the relay and the test equipment.

  3. Remove the test plugs 1V and 2V from the relay.

  4. Return the relay to the permanent settings as indicated in the maintenance file.

  5. This Completes the Test Procedure

Testing the Relay, Relay Scheme, and Protection Outputs

  1. Carefully follow these test procedure steps

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Perform checks of the acceleration of the relay, breaker failure and reclosure schemes, separate technical procedures should be used to test each of these systems.

  4. Test trip all breakers associated with the relay.

Return Relay and Associated Equipment to Service

  1. Return relay to service by withdrawing the test plug handle from the relay.

  2. If any equipment associated with the relay has been isolated without the test plug, it should be returned to service by the following restoration steps:

  • Breaker trip coils and transfer trip scheme.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

  • Auto-reclosure associated with this relay.

Request Power Line be Returned to Service

  1. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those for normal load conditions.

  2. Check that readings agree with station meters:
    (MW/ MVAR/Power Factor/Amperes/voltages)

  3. Place a copy of the test result in the field maintenance file.
