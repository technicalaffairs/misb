Introduction

This document lists the testing procedure P1, which must be performed every six months as indicated by M6 in the above title block

This procedure details the steps required to completely check the proper operation and calibration of the relay. The relay must be tested at intervals of six months . This relay is  a negative – phase – sequence time over current relay used primarily to protect generators from unbalanced phase currents, it is providing essentially back – up protection.

Safety Precautions

  • A work permit must be issued & the staff  of test must be qualified and carry the suitable safety precautions.

  • The generator must be out of service.

Tools and Equipment

  • Test Plug handle.

  •  clamp ammeter.

  •  and test Leads.

  • Secondary injection test.                   

Work to be Carried Out

Preparation

  1. Make Sure C.T secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test

  2. Look at field maintenance file, compare with the settings value on relay, and  record  it on the test sheet.

 

Testing the Relay     Figure 1

Phase to Phase Test

  1. Connect the test leads between  current terminals ( 3 – 5 ).

  2. Connect the test leads between the trip contact ( 1 – 2 ).

  3. Increase the current slowly until the relay pick up ( start ).

  4. Record this  current value as the start value in the test sheet.

  5. Increase the current to the values ( 2 In & 5 In & 10 In ) from start.

  6. Record the current values & the time values at which the relay trips at every current value at the test sheet .

  7. Repeat the following steps at the current leads ( 3 – 7 ).

Phase to Earth Test

  1. Connect the test leads between the current terminals ( 3-4) & ( 5-6 ) & ( 7-8 ).

  2. Repeat the steps from 2 to 6 for the terminals ( 3 – 4 ) & ( 5 - 6 ) & ( 7 – 8 ) .

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set i.e.
     -     Disconnect the test leads from the terminals ( 3 - 4 ), ( 5 - 6 ), ( 7 - 8 )
     - Disconnect the test leads between the trip contact ( 1- 2 )
     - Disconnect the leads from the D.C. terminals  ( 10 - 19 ).

  2. Make sure C.T and P.T secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
           - Breaker trip coils.
           - Breaker failure scheme associated this relay.
           - Disturbance alarm and fault recorder associated with this relay.
