Introduction

The 7SL97 relay is an electromechanical relay multi purpose distance protection with measuring elements for each phase is a time graded protective system, depending on impedance measuring, which clears all short circuit faults, It is so designed for general use on high voltage and extra high voltage systems

The measuring principle employed is described in the publication (the technical basis of impedance measuring).
This relay is used to protect 220 K V lines, the change over facilities allow the characteristics to be altered and zone to be set in the reverse direction. It is manufactured by siemens.

Safety Precautions

  • Work permit required.
  • Make sure line is isolated
  • Make sure CT secondary circuits are scored  before isolating the current circuits for test current injection
Tools and Equipment
  • Siemens test set type 6VP800 and 7VP4900
  • Clip-on ammeter
  • Test leads

Work to be Carried Out
Steps To Prepare for Relay Testing
  1. Review technical reference material: manufacturer’s manual.
  2. Option copy of relay setting from field maintenance filed and compare it with setting on the relay.
  3. Option outage for line protected by this relay.
  4. Check that the bus and line protected by this relay.
  5. Inspect the CT secondary circuit the VT miniature circuit breakers (MCB’S) and all cabling supplying the relay.
  6. Use the station drawing to isolate the following:

  • Breaker trip coil

  • Breaker failure scheme associated with relay

Preparation

  1. This procedure is used to perform off load relay testing.
  2. Look at the EEA panned nameplate and check that this EEA system designation agrees with the line that has been taken out service maintenance.
  3. Check there is no AC current from CT’s using a clip on ammeter.
  4. Check there is no AC voltage from VT’s by using a voltammeter.

  • Disturbance alarmed and fault recorder associated with this relay.

Testing the relay and protection output

  4. Connect the current circuits between the relay and the tester 7VP4800

  6. Connect the voltage circuit between the relay and the tester 7VP4900

Zone 1 Test
Phase To Earth CAL
  1. Put The Test Switch at the test position
  2. Insert the plug into the test jack.
  3. Press the push button ILI and in on current unit 7VP4800 to test phase R-N fault.
  4. Adjust the voltage knob to maximum level.
  5. Set current selecting switch (S1o) to the rated current.
  6. Inject current by pressing the first push button and rotating the knob il until the ammetr reads two times rated current (2*IN).
  7. Adjust the test angle zero degrees.
  8. Decrease the voltage by rotating the voltage knob counter clockwise until the relay operates by first zone time.
  9. Record this value on the test sheet no1.
  10. Compare this value (practical value) with the calculated value (the critical value)
  11. If the result within the ±10% it is acceptable otherwise consult technical services.
  12. Repeat the above test from step 4 to11 but change the angle to (30, 45, 60, 80, 90) angles.
  13. Repeat the above test from step 4 to11 to step for phases S-N and T-N.

Phase To Phase Piles Test
  1. Press the buttons IL1 and IL2 on the current unit 7VP4800 to test RS-s fault.
  2. Adjust the voltage-rotating knob to maximum level.
  3. Put switch (S15) on L1 unit 7VP4800.
  4. Put switch (S6) on (L1-L2) on unit 7 VP4900.
  5. Inject current by pressing push-buttons IL1 and Il2 on unit 7VP4800 and rotate knobs IL1&IL2 until double rated current is reached I-e 2In.
  6. Adjust the test angle to zero degrees.
  7. Decrease the voltage by rotating the voltage knob counter clockwise until the relay operates by the first zone time.
  8. Record this value of voltage on test 1.
  9. Compare this value (practical e value) with (the critical value) which was calculated before through the law.
  10. If the result is within ± 10% it is acceptable other wise, consult technical services.
  11. Repeat the above test from step 5 to 10 for phase to phase but change the test angle (30º,45º,80º,90º)
  12. Repeat the above test from step 5 to 11 for phases S-T and T-R.
  13. Check the operating direction by rotating the phase shifter handle until obtaining forward direction limits.

Trip Test

  1. Put the test switch at service position.
  2. Isolate the voltage circuit and current circuit because the test position, connect the voltage circuit and current circuit.
  3. Request the operator to switch on the main CB e.b of the line kina.
  4. Inject with voltage value and current that’s fumbled any zone, which will operate the relay c and trip CB.
  5. Ensure that the CB tripped.
  6. Remove the test socket from the test jack and remove all flags at the relay and put the test switch at the service position.
  7. Reconnect the voltage circuit and current circuit.

Return Relay and associated equipment to service
  1. Verify the test result match the relay seeing if not consult technical services.
  2. Test trip the CB associated with the relay.
  3. Return relay to service by removing the test leads connections from the relay.
  4. All equipment associated with relay has been isolated should be return to service by the following steps:
        -Breaker trip coils
        -Breaker failure scheme associated with the relay
        -Disturbance alarms and fault recorder associated with relay
  5. Check that CT and VT secondary circuits are in normal condition.

Request Power Line to be Returned to service

  1. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those normal load conditions

  2. Check that reading agree with station meter (MW/MVR/PE/AMP/VOLT)

  3. Place a copy of the test result in the filed maintenance file.
