Introduction

LZ 96 is a non-switched high speed static distance relay capable of protecting three forward distance zones.

ABB utilizes a modular configuration where individual plug-in modules are combined to from complete distance relay. For a complete description of all LZ96 modules and their functions

The LZ96 relay includes a starting modules and 3 measurement modules for three zone measurements. The distance measuring unit M1 for the first zone is set at 85 percent of the line, while unit M2 for the second zone and M3 for the third zone are set at 120 percent and 200 percent of the line respectively.

The M2 unit is used as a: carrier initiating directional unit in DCU scheme. The LZ96 relay also functions as a backup distance relay employing all three forward zone. The first zone is not time delayed. Zone 2 and Zone 3 are time delayed (Device no. 21P/21GP)

Safety Precautions

  • Line isolation and work permit required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

Tools and Equipments

  • Test unit XS920 [secondary injection].

  • Voltammeter.

  • Clip on ammeter.

  • Test leads.

  • Phase rotation meter.

Work To be Carried Out

Preliminary steps to prepare for relay testing

  1. Review technical reference material: Manufacturer's manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Inspect the CT secondary circuits, the VT miniature circuit breakers (MCB's) and all cabling supplying the relay.

  6. Use the station drawing to isolate the following:
    -      Breaker trip coils
    - Breaker failure scheme associated with this relay.
    -      Disturbance alarms and fault recorder associated with this relay.

Testing the relay and protection output

  1. Carefully follow these test procedure steps .

  2. Verify the test results match the relays if not consuls technical services.

  3. Test trip the breaker associated with there.

Prior to starting testing procedures ensure the following steps

  1. This procedure is used to perform off load relay testing.

  2. Check that the panel name plate agrees with the EEC designation of the line that has been taken out of service for maintenance.

  3. Check there is no AC current from CT by using clip-on  ammeter/

  4. Check there is no AC voltage from VT by using voltmeter.

  5. Check all external connections to the relay are in good condition and check that the terminal block wires are tight.

Preparing the tester unit (XS92a) as a figure (2-1) & (2-2) as the following

  1. It should be earthing the tester unit (9XS92a).

  2. D.C of the relay is off [NF93 is off] DC-DC converter.

  3. Remove the UL 91 card and diagnostic card XV 92 

  4. Put the test connector YX91 in the test unit in XX91 for the relay.

  5. Put the jumper between the two units [3 Jumpers].

  6. Put the jumper between + I & + I in the lower unit.

  7. Put the unit (YB92) in the hand terminal in the control unit (XC92) in the upper unit.

  8. Put the jumper between the red point and No.(11)& yellow point and No (12).

  9. Put the key in the connector unit (YX91) on (ON position).

  10. Connect the DC-DC converter (NF93) on the relay.

  11. Connect the supply of the test unit (XS92a)[220VA].

  12. Connect the switch on the lower unit and on the upper unit for test unit XS92a.

  13. Use the (YB92) hand terminal according to the test step.
