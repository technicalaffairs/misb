Introduction

Application and scope for PD551:
PD551 distance protection devices are used for selective short - circuit protection in high - voltage systems. The system can be operated with impedance grounding , with ground fault compensation or with isolated neutral. The PD551 ,a single - system distance protection device has the following protective functions 

Over current and under voltage starting
Under impedance starting with load masking
Distance measurement with polygonal characteristic

Four distance modules, including one that can be used as spectial module
Six timer modules, including two that act as backup time stages
Direction voltage memory

Power swing blocking
Circuit breaker failure protection
Neutral displacement voltage time protection

Switch on to fault protection
Backup over current time protection
Auto - reclosing control

Protective signaling (teleportation)
Parameter set selection
Over current signal

The following functions are also available, depending on the model
                    * Transient ground fault direction determination
                    * Synchronism check

Besides the functions listed above , as well as measuring circuit monitoring and comprehensive sell - monitoring, the following function are always available in the PD 551 for optimum fault evaluation and system management:
                    * Operating data measurement
                    * Event counting
                    * Time - tagged fault logging
                    * Fault data acquisition ( including fault localization )
                    * Fault recording

Information is exchanged either through the integrated local control panel , the integrated PC interface,or the optional  ILSA interface as desired.

The PD551, a numerical protective device , can also be integrated into an ILS system (integrated protection and control system for substations) through the ILSA interface

Safety Precaution

  • Work order must be issued.

  • Check that the bus and line disconnect switches are opened and tagged.

  • Make sure CT secondary are shorted  

  • V.T Miniature circuit breaker (MCB`S) is isolated (opened)

  • Isolate DC supply to relay. When doing any Adjustments or Maintenance on the PD 551 relay

  • Isolate :
                * Breaker failure Scheme associated with relay 
                * Disturbance alarms or fault recorder associated with PD 551 relay
                * Breaker trip coils and transfer trip scheme
                * Auto - reclosing associated with the relay PD 551
    ( Test plug isolate the all above points ) 

Tools and Equipment

  • Test set type (XS92 a)

  • Ammeter

  •  Test pluge and test leads

  • Timer

Work to be carried out

  1. Review technical reference material: manufacturer's manual version 304-401/2/3/5/6-622 for PD551 distance protection device

  2. Obtain copy of relay setting form field maintenance file 

  3. Obtain outage for line protected by this relay PD551

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Block protection signals to : breaker trip coils and transfer trip scheme.

  6. Go to the switchgear and inspect the CT`s and VT miniature circuit breaker ( MCB`S), and all cabling supplying the PD551 relay

  7. Check that the cables are in good condition and that all wire connections are Tight and clean

 

 

Testing Procedures

Zone Reaches

Zone I                                            

 Phase to earth

  1. Restore DC supply to PD551 relay scheme

  2. Prepare the set test XS92a as following


                                                 MOD                 ZZ            (Distance)
                                                 ZL                      up to setting X1             at PZL = 78º 
                                                                         or up to setting R1                at PZL = 0º     
                                                 K0                     0.7
                                                 FC                     fault code forward 111 (phase R to E) 
                                                 SQ                    1F1
                                                 A                        2
                                                 TF                     Time fault zone (250) m.sec
                                                 DZL                 step parameters and range limits (0.1)
                                                 Start (-)              injection of testing signals  

Note   We entered the all above parameters according to the flowchart of set test XS 92a

  1. When the relay PD551 gives tripping signal press ◙ to stop injection

  2. Read the values in hand terminals of set test XS92 a and compare this value with the values of setting in test sheet of relay  

  3. Press the reset button an the relay front panel and check that the LED indicator resets and the alarm contacts open

  4. Repeat the above test with the phase (S) by using FC ( fault code ) (112) &  (T) by using FC  (113). for X1 at angle 78º & R1 at angle PZL 0º Make sure the relay PD551  give trip signaling with all phases R,S,T.

  5. Repeat the above test, but in the reverse direction by using (FC) fault code (011) & (012) & (013), and Make sure that the relay PD551 is blocked in reverse direction

 Phase / Phase  

  1. Entered the all above parameter, by using FC (fault code ) 
    Fault code
        phase  R- S         121    Forward direction (trip)
        phase  S- T         122    Forward direction (trip)
        phase  T- R         123    Forward direction (trip) 
  2. In the reverse direction (021) & (022) & (023) (Block)

Zone II

  1. Repeat the above test for zone I, but change the value of X2 at PZL = 78º,R2 at PZL = 0º, TF up to time of Z2  

  2. Make sure that the relay PD551 in forward direction (trip) and reverse direction (block)

Zone III

  1. Repeat the above test for zone II, but X3 at PZL =  78º,and R3 at PZL = 0º , TF up to time of Z3   

  2. Make sure that  the relay PD551 in forward direction (trip) and reverse direction (Block)

Zone IV

  1. Repeat the above test for zone III, but X4 at  PZL = 78º, R4 at PZL = 0º , TF up to the time of Z4    

  2. Make sure that the relay PD551 in forward direction (trip) and reverse direction (Block) with all phases R,S,T.

 Measuring the operating times

  1. The operating time of a distance relay should be Measured using the tripping contact (free contact). Connected the free contact to "+" and "   __\__ "  "YELLOW" in the front panel of set test XS92 a 

  2. Enter the all parameters of Z1 by ZL = 0.5 Z1 setting

  3. When the trip occur the injection automatically stopped by free contact. Then read the value of time zone I in the hand terminal 

  4. Repeat the above step to measuring the time of zone II & III & IV

                                            Time of Z1            at        ZL = 0.5 Z1

                                            Time of Z2            at        ZL = 0.5 ( Z1 + Z2)

                                             Time of Z3           at        ZL =  0.5 ( Z2 + Z3)
                                                                                             
                                            Time of Z4            at        ZL = 0.5 ( Z3 + Z4)

Note :   How to find " free contact " for relay PD551 ? 

  1. Select any contact (Free contact) and measured DC (Not found)

  2. Measured (buzzer) by using avometer in this contact (Not found)

  3. Made trip by using set tester XS 92 a for relay PD 551, them this free contact measured buzzer by avimeter 

  4. Them connect this free contact to set test XS 92 a in to point : Red "+" and yellow "    " in the front.

End to End Test For Relay PD551

  1. Relay the test of zone II and make sure that the trip occur at time of zone II (T2). (500 m sec) 

  2. Make bridge on receive signal

  3. Make sure that the relay PD 551 trip zone II at time of zone I (100 m sec)

 Switch on to fault ( SOIF ) test

  1. Repeat the test of zone4 but ( SQ = 0 F1) in set test XS 92 a

  2. Make sure that zone IV trip at time of zone I

Return Protection Scheme and Associated Equipment to Service

  1. Return the relay PD 551 to services by with drawing the test plug handle from the relay

  2. Make sure C.T secondary are opened

  3. Return the relay to service by restoring supply from : voltage transformer (V.T) (MCB) is closed  

  4.  Place the associated breaker failure (B.F) protection scheme back in service

  5. Restore the previously blocked disturbance alarms and fault recorder associated with this protection

  6. Return the relay power line carrier (PLC) links to their normal position

  7. Turn ON the associated auto - reclosing scheme

  8. Turn ON the power supply DC to relay PD 551

  9. Take crossed readings of all AC currents and voltages supplied to the relay PD 551 and compare them to those for normal load conditions.

  10. Checks that these readings agree with station meters: MW , MVAR , power factor , Amperes , voltages

  11. Place a copy of the test results in the field maintenance file, and test sheet
