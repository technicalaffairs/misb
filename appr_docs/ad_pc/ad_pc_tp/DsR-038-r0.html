Introduction

    This procedure details the steps required to completely check the proper operation and calibration of the 7SA511 distance relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

    The first preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section Preparation for testing describes the steps to set up the equipment for performing the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

    The first draft initialed & tested by eng. Abdel Azeez Mohamed at 29 Dec 1997 at Network training center.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet.

  • Ensure that Line isolation is done.

  • Make sure CT secondary circuits are shorted and VT secondary circuits are opened before injection.

Tools and Equipment

  • Siemens test set type 7VP48 and 7VP49.

  • Voltmeter,

  • Clip-on ammeter, Test leads.

Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical references material Manufacturer’s manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are open and tagged.

  5. Inspect the CT secondary circuits, the VT Miniature circuit breakers (MCB’S), and all cables supplying the relay.

  6. Use the station drawings to isolate the following items

  • Breaker trip coils.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection Outputs

  1. Carefully follow all test procedure steps.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Test trip the breaker associated with the relay.

Prior to Starting Testing Procedure (s) Ensure the Following Steps

  1. This procedure is used to perform off load relay testing.

  2. Look at the EEC panel nameplate and check that this EEC system designation agrees with the line that has been taken out of service for maintenance.

  3. Check there is no AC current from CT’s by using a clip on ammeter.

  4. Check there is no AC voltage from VT’s by using a voltmeter.

Preparations

  1. Connect the two units of the tester together.

  2. Connect the tester to a 3 phase 380v power source.

  3. Check the phase sequence by pressing push-buttons (I) ON for current tester unit, if the green lamp lights up, the phase sequence is correct, if the red one lights, the phase sequence is NOT correct and must be changed.

  4. Connect the current circuit between the relay and the tester 7VP4800.

  5. Connect the voltage circuit between the relay and the tester 7VP4900

  6. Connect a free contact from the relay to the current unit 7VP4800. (Relay terminals (95, 96) to tester terminals (21, 22), as shown in figure (1)

Testing Procedure (s)

Zone 1 Test

  1. To test first step, block the time steps for all other zones (except first zone) and other function (by computer or keyboard on the relay).

  2. Calculate the operating voltage corresponding to each angle from 0 to 90 degrees for all the four zones tests as follows:

    3. For Phase To Earth
     V= 1 [ X + X (XE/XL) ] R/X      for angle zero
      V= 1 ( X + X ( XE/ XL) )           for angle 90

    Phase To Phase
     V= 1 (X + X) R/X      for angle zero
      V= 1 ( X + X )           for angle 90

Phase To Earth Test

  1. Press the push-button IL1 and In on current unit 7VP4800 to test phase R-N.

  2. Adjust the voltage knob to maximum level.

  3. Set the current selecting switch (S10) to the rated current of relay (1A or 5A)

  4. Inject current by pressing the first push-button and rotating the knob IL1 until the ammeter reads two times rated current (2*In).

  5. Adjust the test angle to zero degrees.

  6. Decrease the voltage by rotating the voltage knob counter-clockwise until the relay operates.

  7. Record this value on the Test Sheet 1.

  8. Compare this value (practical value) with (theoretical value), which was calculated, if the result is within ± 10%, it is acceptable, otherwise, consult technical services.

  9. Repeat the above test from step 1 to step 8 but change the test angle to 90 degrees.

  10. Repeat the above test from step 1 to step 9 for phases S-N and T-N.

Phase To Phase

  1. Press the push-buttons IL1 and IL2 on the current unit 7VP4800 to test phaseR-S.

  2. Adjust the voltage rotating knob to maximum level.

  3. Put switch (S15) on L1 on unit 7VP4800.

  4. Put switch (S6) on (L1-L2) on unit 7VP49.

  5. Inject current by pressing push-buttons 1LI and 1L2 on unit 7VP4800 and rotate knobs1L1 and 1L2 until double rated current 2In is reached.

  6. Adjust the test angle to zero degrees.

  7. Decrease the voltage by rotating the voltage knob counterclockwise until the relay operates.

  8. Record this value of voltage on Test Sheet 1.

  9. Compare this value (practical value) with (theoretical value), which was calculated.

  10. If the result is within ± 10%, it is acceptable, otherwise, consult technical services.

  11. Repeat the above test from step 1 to step 10 but change the test angle to 90 degrees.

  12. Repeat the above test from step 1 to step 11 for phases S-T and T-R

Test Sheet 4 (Zone 4 Test Results)

 

Time Test for All Zones

  1. To measure the zone 1 time, set the tester to an impedance value equal to half the zone impedance value and a phase angle of 90 degrees.

  2. To measure the zone 2 time, set the tester to an impedance value equal to ( Z1+ (Z2 –Z1)/2 ) and a phase angle of 90 degrees.

  3. To measure the zone 3 time, set the tester to an impedance value equal to (Z2+ (Z3 –Z2)/2) and a phase angle of 90 degrees.

Prepare the Relay for Service

 

Rest all function on the relay 7SA511 by pressing reset on the keyboard of the relay.

 

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminals block and the test set.

  2. Make sure CT and VT secondary circuits are in normal condition.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps

  • Breaker trip coils.

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder associated with this relay.

Request Power Line be returned to Service

  1. Take crossed reading of all AC currents and voltages supplied to the relay and compare them to those for normal load condition.

  2. Check that readings agree with station meters (MW/MVAR/Power factor/Ampere/Voltages).

  3. Place a copy of the test results in the field maintenance file.
