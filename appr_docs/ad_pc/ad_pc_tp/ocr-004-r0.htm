Introduction

RACID is normally used as main or back - up protection in distribution or industrial net works for transformer, capacitor banks, electric boilers and small size generators, and motors as well as for feeders. The relay is also used as a back - up protection for transmission net works , large size generator and trans formers.

RACID`S built - in flexibility large setting ranges for current and delay, makes it possible to order the relay early in the planning of the plant or to store it as a universal time - over current relay.

ACCUR at measurement, low transient overreach, high reset ratio and short reset time contribute to the use of short time steps and smaller margins for current settings in the selective plan.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence (rope) with caution marks surrounding the work area is required,

  • Safety and healthy equipment must be used as safety shoes, safety Helmet ..etc

  • Transformer isolation must be done.

  • Make sure CT secondary circuits are shorted before the relay current circuits for test current injection.

Tools and Equipment

  • Secondary injection test set

  • Test leads, Volt meter

 Works to be Carried Out

  1. Two - phase or three phase time over current and earth felt relay with starting delayed and instantaneous functions

  2. Two measuring ranges 0.1-0.5 or 0.5-2.5 times rated current, for both over current and earth fault relay

  3. Adjustable time characteristic:
        * Inverse time, normal inverse, very inverse, extremely inverse
        * Definite time 0.1-1.2 or 1-12 sec

  4. Phase indication

  5. The instantaneous function can be definite time delayed

  6. Display showing set values service current and fault currents at tripping

  7. Continuous supervision of internal circuits

  8. Indication of faulty hand ling

  9. Auxiliary voltage 48-220 V + 20 % dc or ac

  10. Comb flex plug - in or screw connection

Over Current Relay

The start function is used to block other relay in the network or as an alarm, the delayed function for selectivity against other over current relays in the network can be selected with definite or inverse time delay the instantaneous function is normally set for close - up faults incases where selectivity against fuses is required, the definite time delay is used, the low transient over each also allows for extended reach with smaller settings margins

Setting

Setting is done by means of switches and potentiometers set values are shown on the display by pushing repeatedly on the " Mode " button , At the same time, the light emailing diodes (LED) to the left of the potentio meters, for each protective function will light up with a steady glow

Normal service

The green LED mocked " in service " in on , if the internal supervision discovers an internal fault or the supply voltage is interrupted , the LED will go out and an alarm relay pick - up. If the " reset " button ( service value ) is pushed repeatedly , the display will show the actual current in phases R, S, T and N, while the cores pending R, S, T or N LED will light up.

Preparation the Relay for test

  1. Make sure CT secondary circuits are shorted before isolating the relay current circuits for test

  2. Switch on the MeB for DC supply

Current setting test

  1. Connect test leads between the current terminals on the relay (3A,4A) and the current terminals on the test set this for phase R.

  2. Slowly increase the current until the relay operate the relay must be operate with a current between + 10 % of the setting value

  3. Record this value on the test sheet

  4. Increase the injected current to value 25 % greater than the operating value

  5. Slowly decrease the current until the relay rest

  6. Record this value on the test sheet

  7. Divided the reset current by the operating current to obtain the reset factor which must be less than 85 %

  8. Change the test leady between the current terminals on the relay (5 A , 6A) and (7A , 8A) for phases S and T

  9. Repeat the steps from 1 to 7 for phases S and T

Time Setting test

  1. Inject current into relay until the relay picks up. Keep this value and switch off the test set

  2. Connect the relay time delayed out put contacts (1A,7A) to the timer contact of the test set

  3. Switch on the test set the timer start and will stop once the relay has picked up

  4. Record this value on the test sheet

Return the Relay to Service

  1. Make a final check that all switches on the relay are set according to the setting sheet

  2. Make sure the CT secondary circuits is in normal condition

  3. Return the relay to service
