Introduction

Its a highly sensitive moving coil relay type RQS4T1 is used as measuring system which is applied with voltages over a rectifier bridge circuit which are proportional to the sum current and difference current - The current proportional voltages are gained on the burdens of the input transformer. The units are constructed according to the modular unit principle and accommodated in standard housings, which consist of alight metal base plate. The metal frame and the removable moduled cover with transport glass window.

The response current at holding current zero can be infinitely adjusted by means of good accessible adjusting knobs - in case of fault a neutral relay with drop indicator is actuated by the moving coil relay. This meutral relay can be reset from the outside by means of a pushbutton

This procedure, details the steps required to completely check the proper operation and calibration of the transformer differential relay. It is recommended this procedure be carried out on a yearly bases.

This document is the formatted way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

Work to be Carried Out

Checking the Minimum Pick Up

  1. Connect the test circuit as shown in figure (1)
  2. Slowly rotate the control knob in a clockwise direction and raise the test current IAC until the relay operate ,and the relay trip target should operate.
  3. Read and record the value of the pick up current (minimum pick up current).
  4. Return the control knob to (0) position and switch the power off.
  5. Repeat steps (2) to (4) by injecting the test current in terminals 2 and 8. This checks the minimum pick- up via the other half of the (R) phase restraint circuit. These two values should be approximately the same.
  6. Using the schematic diagram for the relay, determine relay terminal connections for testing the S and T phases.
  7. Repeat steps 2 to 5 for phases S and T observing that the target drops for each of the six tests.
  8. Record this values in the test results sheet.

Check Of the Percentage Differential Characteristics Slope
    1. Connect the test circuit as shown in figure (2).
    2. Slowly rotate the control knob to increase the restraining current (IR) until an indication of (1) Ampere on the ammeter (A2).
    3. Slowly rotate the control knob to increase the operating current (l op) on the ammeter (A) until the relay operates.
    4. Record the operating and restraining currents ( l op & I R )
    5. Repeat for the restraint current IR = 2, 3, 4 amps.
    6. Change the connection point from terminal 1to 2 and verify that similar values are obtained
    7. Repeat the above tests for the (2) restraint coil associated with the S & T phases, and record the results.
    8. Using all of the recorded values for each of the phases. Draw the characteristic slop for this relay.
          
    1. Record this values in the test results sheet.

Check Of the 2nd Harmonic Restraint
  1. Connect the test circuit as shown in figure 3.
  2. Slowly rotate the control knob in the clockwise direction and raise the test DC current until an indication of one ampere is observed on the DC ammeter.
  3. Slowly rotate the control knob in the clockwise direction and raise IAC until the relay operates.
  4. Read and record IDC and IAC
  5. Return all control I knob to 0
  6. Repeat steps 2 and 3 with IDC = 2,3,4 amperes.
  7. Repeat the above test using the (S) & (T) coil winding (other Phases).
  8. Use the all of the recorded results to calculate the percent second Harmonic restraint using the formula:-
  9. Record all this values in the test sheet.

            Percent Second Harmonic = (0.424 x IDC ) / ( IDC + 0.9 x IAC ) x 100 %)
