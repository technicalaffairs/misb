Introduction

This document outlines the testing procedure I1, which must be carried on an annually bases as indicated by Y1 in the above title block,

The three phase instantaneous and definite time over current relays have been designed for the phase and earth fault protection of electrical plant and networks.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety precautions.

  • The auxiliary transformer must be out of service.

Tools and Equipment

  • Test Plug handle.

  • Clamp ammeter.

  • Test leads.

  • Secondary injection test.

Work to be Carried Out

Preparation

  1. Make Sure C.T secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test .

  2. Look at field maintenance fill and compare the position of the setting switches on relay with those shown on the setting sheet.

Testing the Relay

  1. Connect test leads between  the current terminals ( A2 – A3  ) for phase R. figure 1

  2. Connect the test leads between the trip contact  ( A4 – A5  )

  3. For O.C Test :

            Increase the current slowly untill the relay picks up.

     4.  Continue increasing the current until the relay trips

     5.   Record the current value and the time value at which the relay trips in the test sheet.

      6.   For S.C Test:

            Decrease the S.C setting to the value (4) .

       7.  Continue increasing the current until the relay trips instantaneously.

       8.   Record the current value in the test sheet.

        9.   Repeat the steps from (1 to 8) to phase S but the current terminals between ( A6 & A7 ), and between

              terminals ( A10 & A11 ) for phase T.

Return Relay and Associated Equipment to Service

  1. Disconnect the test leads from the terminals (A10 - A11)

  2. Disconnect the test leads between the trip contact ( A4 - A5 )

  3. Make sure CT  secondary circuits are in normal condition.

  4. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
           - Breaker trip coils.
           - Breaker failure scheme associated this relay.
           - Disturbance alarm associated with this relay.
