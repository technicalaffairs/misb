Introduction

It's a unit protection for two or three terminal feeders, phase segregated diff current with high sensitivity, single or three pole tripping suitable for use with dedicated optical fiber link or multiplexed systems.
The first draft initiated by Eng Mohamed Helmy El Qady from WDL, The document tested at Dec. 2001 at El EZZ subststion by eng. Mohamed Helmy El Qady and approved at 22 May. 2002

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Safety and healthy equipment must be used as safety shoes, safety Helmet ..etc

Tools and Equipment

  • Secondary injection type Sverker or any set injection

  • Multimeter (0 -----> 250 rac & 0  -----> 10 Aac)

  • Communication local loop-back- link, (direct link to connect t x by R x locally )

  • MMLBO 1 test plug for use with MMLG test black

  • Optical power meter to measure the optical signal level IF required (sensitivity - 10  ----> - 50 dBm) 

Works to be Carried Out

Steps to Prepare for Relay Testing

  • Review technical reference material manufacturer's manual

  • Obtain outage for line protected by the relay

  • Inspect the secondary circuits of C.T and all cabling supplying the relay

 

  • Before inserting the test plug into the relay check from the station drawing that the test plug will isolate :-
                    - Breaker trip coils
                    - Breaker failure scheme associated with this relay
                    - Fault recorder, event recorder and disturbance alarm associated with this relay
                    - If the test plug not isolate one of the previous steps you have to isolate it

  • Link the local Tx output with the local Rx input with a suitable communication loop-back link

Testing the Relay and the Relay Outputs

  1. Check the relay setting accuracy. (diff prot u)

  2. Check the diff prot u operating time

  3. Current i/p accuracy and polarity checks

  4. Lamp test

  5. Communication supervision alarm

  6. Bias ch s 

  7. Test trip C.B (both sides) associated with the relay

Relay Setting Accuracy

  1. Inject a current into the A-phase at using sverker test set. (How to use sverker test set phase see document No. pc-ss-003-r0 which approved in 23/12/1996

  2. Slowly increase the current till the relay operates rd the current magnitude and check that it is equal to
    [0.5xIS1/(1-0.5K1)] P.U

  3. Check that the trip lamp turns steadily and the alarm lamp flashes.

  4. Slowly decrease the current until the relay contacts reset

  5. Check that the reset / operate current ratio is approx (0.75/1)

  6. Push accept / read and check display " diff prot trip phase A "     

  7. Push accept / read and check display " diff inter trip phase A "     

  8. Push accept / read & reset to clear alarms 

  9. Press   to display " protu alarm records "

  10. Press  and check and alarm record # 1 is " diff prot trip Phase A"

  11. Press  to display "prot fault records"

  12. Press  and check fault record # 1 is "diff phase A"

  13. To display (( current differential ))

  14. Repeat steps 1-14 for phase B.&C.  

Diff Prot Operating time

  1. Inject a current equal 1A (see fig)

  2. Close sw1 and check that the average operating time is less than 30 ms.

 

Current i/p accuracy and Polarity checks

  1. Inject 0.5 a current (0.5 A for a 1A relay or 2.5 A for a 5 A relay) into the relay as show in fig (2) the relay will trip and give steady trip. And a flashing ALARM indication, ALL trip and protection operated output contacts will close

  2. Accept the alarm by pressing ACCEPT / READ twice to display " *** ALARM *** "

  3. Them check the following current measurements:
                * Press       to display " MEASUREMENTS "
                * Press      and check that I1 is 0.00-0.02 pu
                * Press   and check that I2 is 0.00-0.02 pu                            
                * Press   and check that I0 is 0.48-0.52 pu
                * Press    and check that A-phase LOCAL is 0.48-0.52 pu
                * Press    and check that A-phase REMOTE is 0.48-0.52 pu
                * Press   and check that A-phase DIFF is 0.96-1.04 pu
                * Press   and check that A-phase bias  is 0.48-0.52 pu
                * Press and check B-phase as step N 3
                * Press  and check C-phase as step N 3
                * Press to return to the default display

  4. The previous test cane be carried out by injecting 0.5 A into one phase only (for ex into terminal A21 / A22 ) then the readings should be as follows:
                    I1                              0.15 - D - 18 pu
                    I2                              0.15 - D - 18 pu
                    I0                              0.15 - D - 18 pu

                                * Injected phase                                             * Other phases

LOCAL                     0.48-0.52 pu                                                   0.0-0.02 pu

REMOTE                  0.48-0.52 pu                                                   0.0-0.02 pu

Lamp Test

  1. Push     to display " PUSH     FOR LAMP TEST "

  2. Push        constantly and check all 4 led turn ON after one second the green RELAY HEALTHY lamp goes off

  3. Release the  key and check that the RELAY healthy lamp turns ON and the other 3 lamps go off after a short while

Communication Supervision Alarm

  1. Disconnect the communication loop-back link and check that the alarm lamp flashes after the expiry of the communication alarm time9.9 see (typical setting)

  2. Check also that the alarm contact (G7 -G8) closes

  3. Reconnect the communication loop-back link and check that the alarm contact resets

  4. Push ACCEPT / READ and check display " prot SCHEME FAIL "

  5. Push ACCEPT / READ and check display "communication CHAN FAILED - CHAN 1 FAIL "

  6. Clear these alarm to display " current differential "

  7. Press   to display " prot ALARM RECORDS "

  8. Press and check alarm record # 1 display "COMM. CHAN FAILED. CHAN 1 RX FAIL"

  9. Press  to return to " CURRENT DIFFERENTIAL "

Bias c/cs

A current is injected into A-phase which is used as the bias current and an other current in injected into B-phase. which is used as diff current (fig 3)

Lower Slope

  1. Inject a current of 1 pu in the A-phase. the relay will trip and the contacts associated with A-phase will operate

  2. Push ACCEPT / READ twice to accept the alarms which indicate diff prot and inter trip phase A have operated

  3. Push ACCEPT / READ twice more and check that the display show " *** ALARMS *** "message and the ALARM lamp is steadily ON

  4. Slowly increase the current in the B-phase until the phase B contacts operate and the ALARM lamp flashes again

  5. Accept the alarms and check that the display shows phase A and B have operated

  6. Record the phase B current magnitude and check that it is 0.25 pu + 10 %
                Assuming        IS1 = 0.2 pu                        K1 =30 %
                                      IS2 = 0.2 pu                        K2 =150 % 
               Generally the operate current is 0.5 x (IS1 + K1) pu

  7. Switch off the ac supply and reset the alarms

Upper Slope

  1. Repeat test 6-1 steps 1-- 3 and inject current- 3 pu

  2. Slowly increase the current in B-phase until the relay operates for both A and B phases

  3. Record the current magnitude and check that it is 1-15 + 20 % At the same values step 6-1-6 generally the operate current is : 0.5 x [(3 x K2) + IS1 - (K2 - K1) IS2]

  4. Switch off the ac supply and reset the alarms
