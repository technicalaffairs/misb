Introduction

    The static transformer differential protection SQT is intended for the protection of three-phase power transformers, independent of the size and rating.

    The protection relay works on the principle of measured valve comparison. The measured valves drawn from the current transformers are passed to the current input circuit, where the measured values are processed and switched with galvanic isolation to the measuring and blocking modules.

    The measuring nodule detects the appearance of a differential current and operates on a set valve being exceeded (measured valve comparison Ian/IN). If the ratio of the magnitude of the harmonic with twice the rated frequency to the magnitude of the fundamental oscillation exceeds the set value, the blocking module operates (inrush current stabilization FS). If a measuring decision exists and no blocking criterion, the output logic decides for tripping.

Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable ID safety category,

  • Maintenance staff should be trained well, qualified or directly supervised by some one who is.

  • Safety fence with caution marks surrounding the work area is required.

  • Personal safety and healthy equipment must be used as safety shoes, safety helmet ... etc

    •

  • Power transformer isolation is required.

  • Make sure that CT secondary circuits are shorted before isolating the current circuit for test current injection.

  • Switch off DC Power supply before inserting or withdrawing any card the relay.

Work to be carried out

Preliminary Steps to Prepare for Relay Testing

  1. Obtain outage for transformer protected by this relay.

  2. Check that all the two bus disconnect switches are opened and tagged.

  3. Inspect the CT secondary circuits and all cables supplying the relay including matching CT

  4. Use the station drawing to isolate the following

    • Breaker trip coils.

    • Breaker failure scheme associated with the relay.

    • Disturbance alarms and fault recorder associated with the relay.

Testing the Relay and Protection OUtputs

  1. Verify the results match the relay setting, if not consult technical services.

  2. Trip the circuit breakers associated with the relay.

  3. Carefully follow these test procedure steps

Pritor to Beginning Actual Testing Procedure Ensure the Following Steps

  1. This procedure is used to perform off load relay testing.

  2. Look at the EEA panel nameplate and check that this EEA system designation agrees with the power transformer that has been taken out of service for maintenance.

  3. Check there is no AC current from CT's by using a clip on ammeter.

  4. Check there is not AC voltage from VT's by using voltmeter.

Checking the Minimum Pick Up

  1. Connect the test circuit as shown in fig (3).

  2. Slowly increase the current until the relay operates, as indicated by light emitting diode (LED) on the front panel.

  3. Check the relay trip contacts are closed (terminals 13, 14)

  4. When the relay picks-up and those contacts open as the current is reduced.

  5. Record the minimum Pick up valve current.

  6. Repeat steps (2) and (3) above by injecting the test current in terminals (4) and (10). This checks the minimum pick up via the other half of the "R" phase restraint circuit these two valves should be approximately the same.

  7. Repeat steps (2), (3) and (4) above for phases "S" and "T"

Check the Percentage Differential Characteristics

  1. Connect the test circuit as chown in fig. (1) .

  2. With the relay set to I an /IN = (0.15 ……. 0.45)

  3. Adjust the restraint current (ID) to 1A. Increase the operating current Id until the relay operates.

  4. Record the operating and restraint current (Id, ID)

  5. Repeat for the restraint ID = 2, 3, 4, ……A

  6. Change the connection point from terminal (1) to (4) and verify that similar values are obtained.

  7. Repeat the above tests for the two restraint coils associated with the "S" and "T" phases. Record the results.

  8. Using all of the recorded valves for each phases,

  9. Draw the characteristic slope for this relay and match it with fig. ( 2 ).

Checking the Second Harmonic Restraint

  1. Connect the test circuit as shown in fig. (3 ).

  2. With the relay set to Fs = (15, 25, 35%)

  3. The relay may be tested with a simulated wave form representing magnetizing inrush by connecting a diode in series with relay to produce a half wave save rectified.

  4. Wave form as shown in figure no. , Close switch s1 and s2 and the current to the operate valve.

  5. Check that the relay operates.

  6. Open switch S2, close S 1, and check that the relay does not operate.

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminals block and the test set make sure CT Secondary circuits are in normal condition.

  2. If any equipment associated with relay has been isolated, it should be returned to service by the following restoration steps:

    • Breaker trip coils.

    • Breaker failure scheme associated with this relay.

    • Disturbance alarm and fault recorder associated with this relay.

    • Place a copy of the test result in the field maintenance file.

Request Power Transformer to be Returned to Service

  1. Take crossed readings of all AC currents and voltages from both HV and LV sides and compare them to those for normal load condition

  2. Check that these reading agree with station meters (MW, MVAR, Amperes and Power Factor)
