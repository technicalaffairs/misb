Introduction

This document outlines the testing procedure P1, which must be performed on an annually bases as indicated by Y1 in the above title block

The type IFC51AD is a single phase extended range, very inverse time over current relay with voltage restraint. This relay protects the generator against prolonged contribution to a fault on the system.

The first draft  initiated by eng. Fayza El-Sharkawy, tested by eng. Shirin Hamdy & eng. Azza Salama at 29 April 2003, revised by EDEPCo. quality control eng. Adel El-Nagar, accepted by General Director eng. Mahmoud Saker, and approved by eng. Moustfa Allam Head Sector of Talkha Power Station.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • The generator must be out of service.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuit for current injection

Tools and Equipment

  • Secondary injection test set OR Double - F2200 test set

  • Ammeter, and Voltmeter

  • Test leads .

Work to be Carried Out

Preparation

  1. Make Sure C.T secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

Testing the relay

  1. Connect test leads between the current terminals ( 5 and 6 ) on the relay. figure 1

  2. Connect test leads between the voltage terminals ( 7 and 8 ) on the relay .

  3. Slowly increase the current until the contacts close .

  4. Record this value on the test sheet .

  5. Increase the injected current with a value ( two,  four times )  of the operating value .

  6. Record this value on the test sheet .

  7. Increase the injected voltage with a value 100 % of the operating voltage .

  8. Repeat the above test steps ( from  1 to  6 ) .

  9. Slowly decrease the voltage from  ( 120V, 100 V,  80 V, 60V, 40V & 20V ) and record the pick up value of the current at every voltage value .

  10. Record this value on the test sheet .

  11. Repeat the above steps for other phases

Time Test

  1. Connect the time delayed contacts numbers ( 1  and  2 ) on the relay to timer terminals on the test set .

  2. Start the test by injecting current and voltage into the relay .

  3. The timer will stop once the relay has picked up .

  4. Record this value of time on the test sheet .

  5. Repeat the above test steps for other phases .

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminal block and the test set i.e.
    -     Disconnect the test leads from the current terminals ( 5 - 6 )
    - Disconnect the test leads between the terminals ( 1 - 2 )
    - Disconnect the test leads from the voltage terminals ( 7 - 8 )

  2. Check that the CT and PT secondary connections return to its normal position as before the test.

  3. If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps:
           - Breaker trip coils.
           - Breaker failure scheme associated this relay.
           - Disturbance alarm and fault recorder associated with this relay.
