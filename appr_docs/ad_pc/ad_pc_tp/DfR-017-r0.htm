Introduction

This document out lines the testing procedure P1, which must performed on an annually bases as indicated by Y1 in the above title block

The TMAD relay is an electronic relay for three-phase diff. Protection with instantaneous operation. It is intended for the protection of rotating machines against internal faults. These faults must entail the interruption of generator excitation and the opening of load circuit

Operation

The TMAD relay is a percentage diff. relay. Percentage "A" is defined for one phase as ratio of the difference between the source side current l1 and the line-side current l2 to half the sum of the two

A = 100 * (l1 -l2) / (l1 + l2) / 2

The currents l1 and l2 in the secondary of the current transformers source side and line side

Safety Precautions

  • A work permit must be issued and qualified maintenance staff should carry the suitable safety category.

  • Safety fence (rope) with caution mark surrounding the work area.

  • Line isolation

Tools and Equipment

  • Secondary injection test set

  • Ammeter, Voltmeter

  • Test lead

 

 

Work to be Carried Out

  1. Make sure C.T secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test

  2. Obtain copy of relay setting from field maintenance file

  3. Connect test leads between the current terminals (50 and 60) on the relay Fig(1)

  4. Connect test leads between the time delayed contacts (11 and 12) on the relay.

  5. Connect test leads between the voltage terminals (-1and +2) on the relay

  6. Increase the current slowly until the contacts close

  7. Record this value on the test sheet

  8. Record the time response of relay where the time will stop once the relay has picked up

  9. Compare the recorded values with setting values

  10. Repeat the above test steps for other phases by putting the current terminals for phase S (60 and 61) and phase T (50 and 70A)

Return Relay and associated Equipment to Service

  1. Disconnect the test leads from the current terminals (50 and 60)

  2. Disconnect the test leads between trip contact terminals (11 and 12)

  3. Disconnect the test leads between trip voltage terminals (-1 and +2).

  4. Check that the C.T and P.T secondary connections return to its normal position as before the test

  5. If any equipment associated with the relay has been isolated it should be returned to the service
