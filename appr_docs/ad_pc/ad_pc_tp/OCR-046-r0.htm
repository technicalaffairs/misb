   Introduction

This document outlines the testing procedures P1, which must be performed on an annually bases as indicated by Y1 in the above title block.

This procedure, details the steps required to completely check the proper operation and calibration of the relay ( TAL3111 ). 

The first draft initiated & tested at Sharkia North substations by eng. Salah Elsaied Abd-Elaal from Sharkia North protection management, reviewed by protection manager eng. Magdy Salem, accepted and approved by general manager eng. Mohamed El-Shahhat, revised by quality control manager eng. Mohamed Omran, and certified by eng. Safety Precautions

  • A work permit must be issued, the qualified maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding work area is required.

  • Wear your safety equipment as helmet, shoes, breathing mask, safety glass...etc.

  • Line isolation is required

  • Isolate the relay DC supply before drawing out any relay card .

  • Isolate the relay associated breaker trip Coils, breaker failure system, disturbance alarm and fault recorder systems

Tools and Equipment

  •  Secondary Injection test set XS92a

  •  Tow test plugs ( voltage and current ).

  •  Test leads, Ammeter, Voltmeter, Avometer &AC supply

Work to be Carried Out

Relay Preparation

  1. Review technical reference material: Manufacturer's manual.

  2. Obtain copy of relay settings from field maintenance file and compare it with the relay settings .

  3. Obtain outage for line protected by this relay .( work permit must be issued )

  4. Check that the bus and line disconnect switches are opened and tagged

  5. Make sure CT's Secondary circuits are shorted before isolating the relay current circuits for current injection.

  6. Make sure VTs  miniature circuit breakers ( MCB's ) is disconnected .

  7. Isolate the relay DC supply .

  8. Check that all cables supplying the relay and all wire connections are in good conditions.

Tester unit XS92a Preparation

  1. Position the two units of XS92a as in figure (1) and figure (2).

  2. The tester unit ( XS92a) should be earthed.

  3. Connect the three jumpers between the two units.

  4. Connect the hand terminal unit (YB92), figure (3) to the control unit (XC92) in the upper unit.

  5. Connect current and voltage terminals unit to ( output and switching logic unit AX91/92 ).

  6. Connect current and voltage terminals unit to relay current and voltage test socket terminals as in fig (4).

  7. Connect the test unit (XS92a) power supply ( 220 Va.c ).

  8. Turn ( ON )  the switches on the lower and upper units of tester (XS92a).

  9. Use the hand terminal ( YB 92 )  according to test steps.

Parameters Preparation

  1. Restore DC supply to the relay scheme while the tester (XS92a) is ( 0N ) i.e ready.

  2. For Ph -Ph fault as example, On  Hand Terminal Unit ( YB92 ) Set:

  3. MODE: FF then ENTER

  4. FREQUENCY = 50 Hz

  5. DFR = 0 (  Frequency Change )

  6. Static or Dynamic Injection     SQ = 2  i.e  Static injection.

  7. Automatic search  A = 2   i.e  with Interruption for Decreasing injection signal

  8. FC : Fault codes  (121 , 122 , 123 )

  9. TL : Duration of off -load voltage injection = 100 m.s

  10. To : Duration of zero injection = 100 m.s

  11. Mode  : UI  then  Enter ( in case of injection on directional over current relay )

  12. U : Fault voltage = 10 V

  13. I  : Fault current range = ( 0.0 :  20 ) A

  14. PHI : Angle between fault voltage and current  = Value inside relay  operating zone .

  15. TF  : Duration of fault injection  >  Relay time setting .

  16. DU : (0.1 : 10 ) V     Voltage change rate

  17. DI  : (0.01 : 1.0 ) A   Current change rate

  18. DPH : ( 1° : 30° ) Angle change rate

Relay Testing

  1. The test is accomplished in three stages:

  2. Finding relay operating current value while the voltage and current angle are constant

  3. Finding relay operating angle zone while the voltage and current are constant

  4. Finding relay operating time

Finding Relay Operating Current 
(The voltage and current angle are constant )

  1. Set:

    MODE = UI      i.e  Directional over current relay
     U         = 10  V  (constant voltage )    ;  U = ( 0.0 : 70 ) V
     PHI     = 80°   The relay current phase angle setting  (constant angle )  ; PHI = ( 0°  :  359° )
                   Or a value inside the relay operating angle zone ( 60° : 240° )  according  figure  (5)
     I        = 1.5 A   > The relay operating Current setting  ;  Operating Current = ( 0.0 : 20 ) A
     TF     > The relay operating time setting (up to 100 ms)
     Duration of fault injection =( 0 : 9999 ) ms
     DI     = 0.1 A   ;   Current change rate = ( 0.01 : 1.0 ) A
     SQ    =  2       i.e    Static injection signal
     A       =  2      i.e   Automatic search with interruption for Decreasing
                                   injection signal.

  2.  Press start button  then  minus (-) button to start injection of testing signal.

  3. Wait until auto. stop trigger point of relay starting current is shown on display.

  4. Record in test sheet : the value of relay starting current ( I ) associated to phases ( R - S ) fault  and the lighted leds on the relay.

  5. Press ( 0 ) button then Enter.

  6. Press the relay Reset.

  7. Repeat the above steps to other ph-ph faults for ( S - T) , (T - R)

 Finding Relay Operating Angle Zone

( The voltage and current are constant )

  1. Set :
    U = 10 V   (constant voltage )    ;  U = ( 0.0 : 70 ) V
     I  = 1 A     ( The relay operating Current setting ) 
    Operating Current = ( 0.0 : 20 ) A
    PHI = 40°assumed value outside the relay operating angle zone (60°: 240° )
    DPH = 5°  (Angle change rate)

  2. Press start button then minus (-) button to start injection of testing signal.

  3. Wait until auto-stop trigger point of relay starting angle is shown on display.

  4. Record in test sheet: the value of relay starting angle and the relay lighted leds.

  5. Press ( 0 ) button then Enter.

  6. Press the relay Reset.

  7. Repeat the above steps for another assumed angle values to find the relay operating angle zone. As in figure (5)

Finding Relay Operating Time

The operating time of a directional over current relay should be measured using the trip contact ( free contact ) . connected the free contact to ( + ) and ( -\ - )  on the front panel of  tester ( XS92a ) .

  1. Set :

U = 10 V   (constant voltage )    ;  U = ( 0.0 : 70 ) V

I  = 1 A     ( The relay operating Current setting );  Operating Current = ( 0.0 : 20 ) A
PHI = 80°   assumed value inside the relay operating angle zone ( 60° : 240° )

TF = 1.7 sec. > The relay operating time setting; Duration of fault injection =( 0 : 9999 ) ms

  1. Press start button  then  minus (-) button to start injection of testing signal.

  2. Wait until auto. stop trigger point of relay starting current is shown on display.

  3. Record in test sheet : the value of relay operating time and the lighted leds on the relay.

  4. Press ( 0 ) button then Enter.

  5. Press the relay Reset .

NOTE : How to find" free contact" on relay ( TAL3111 )

  1. Select any free contact and measure it's DC voltage ( must have not DC voltage )

  2. Check that it is normal open contact using avometer buzzer .

  3. Check that it is close with trip , using tester ( XS92a ) and avometer buzzer .

Return Protection Scheme and Associated Equipment to service

  1. Return relay ( TAL3111 ) to service by withdrawing test plug handle from the relay test socket .

  2. Make sure CT secondary are opened.

  3. Return relay to service by restoring supply from voltage transformer (V.Ts miniature circuit breakers ( MCB’s ) is closed.

  4. Place the associated barker failure ( B.F) protection scheme back in service.

  5. Restore the previously blocked disturbance alarms and fault recorders associated with this protection.

  6. Turn ON the power supply DC to relay ( TAL3111 )

  7. Take crossed readings of all AC currents and voltages supplied to the relay ( TAL3111 ) and compare them to those for normal load conditions.

  8. Check that these readings agree with station meters MW, MVAR, Power Factor, Amperes, and Voltages.

  9. Place a copy of the test results in the field maintenance file.
