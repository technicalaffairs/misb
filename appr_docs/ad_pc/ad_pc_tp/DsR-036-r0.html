Introduction

    The static distance protection SD135A is a switched relay system, which means it, contains only one measuring unit for both distance and direction. The starting unit can be selected for either for over current starting or for under impedance starting. It is intended for the selective protection of overhead lines or cables in high voltage systems. It can also be employed as back up protection for generators and transformers in the very high voltage system.

    This procedure details the steps required to completely check the proper operation and calibration of the SD135A relay. It is recommended this procedure be carried out on a yearly bases. This work be carried out section of this document is divided into 3 sections.

    The first section is the preliminary steps to prepare for relay testing describes the activities required prior to testing. Some of these steps could be performed days before the actual test. The second section is the preparation for testing describes the steps to set up the equipment to perform the actual tests. The third section is the actual test.

    The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet ... etc.

  • Ensure that Line isolation is done.

  • Make sure CT secondary circuits are shorted and VT secondary circuits are opened before injection.

Tools and Equipment
  • AEG relay test set RED14
  • Voltmeter,
  • Clip - on ammeter
  • Test leads
  • DC auxiliary relay with the same voltage rating as the control voltage.

Work to be Carried Out

Preliminary Steps to prepare for relay testing
  1. Review technical references material Manufacturer’s manual.
  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.
  3. Obtain outage for line protected by this relay.
  4. Checks that the bus and line disconnect switches are open and tagged.
  5. Inspect the CT secondary circuits, the VT .Miniature circuit breakers (MCB’S ), and all
  6. Cabling supplying the relay.
  7. Use the station drawings to isolate the following items:
  • Breaker trip coils.
  • Breaker failure scheme associated with this relay.
  • Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection Outputs
  1. Carefully follow these test procedure steps from page 2 to page 7.
  2. Verify the test results match the relay setting, if not consult technical services.
  3. Test trip the breaker associated with the relay.

Preparation
  1. This procedure is used to perform off load relay testing.
  2. Look at the E.E.A panel nameplate and check that this EEA system designation agrees with the line that has been taken out of service for maintenance.
  3. Check there is no AC current from CT’s by using a clip on ammeter.
  4. Check there is no AC voltage from VT’s by using a voltmeter.

Preparation of Test Set
  1. Set the two circuit breaker e1, e2 in, OFF, position.
  2. Rotate the current regulating knob r2 to the minimum current level (zero) by rotating it to the fully counterclockwise ( ccw ) position.
  3. Set b5, b7, b8, and b9 switches in zero position.
  4. Set b6 switch in the minimum current level position.
  5. Set b10 switch according to the relay rated current ( In = 1 Amp or 5 Amp ).
  6. Change b4 switch to select the “kind of C.T earthing” used on the protection.
  7. Change - over b3 switch to “OFF “ position.
  8. Short the terminals (R, Y, B) on the test set.
  9. Connect the current terminals on the test set R, Y, B and N to the current terminals on the relay.
  10. Connect the voltage terminals on the test set U, V, W, and mp to the voltage terminals on the relay.
  11. The AEG RED 14 test set can now be energized by plugging the main input cord in to a normal 220 v AC source

Testing the Over-current Starting Unit

Current Test

  1. Look at the over-current setting (IF/ IN):
  • If it value is a low setting, put b10 switch to 1A.
  • If it value is a high setting, put b10 switch to 5A
  1. Set the fault selector switch b3 on position RR’
  2. Set the MCB e1 to “on” ‘ position.

    3. Note: Flag indicator h1 will be white colour

  3. Push the push-button b1
  4. Inject current by rotating the knob r2 in clockwise direction until the relay picks up and the LED “ A R” lights up on the relay.
  5. Record this value on the test sheet.
  6. Decrease the current by rotating the knob r2 in (ccw) until the relay resets.
  7. Record this value in the test sheet.
  8. Push the push-button b2.
  9. Repeat the above test (Current test) from steps 1 to 9 for other phases (bb’ and yy’ on test set)

    10. Time Test
  10. Set b3 on RR’ position.
  11. Push the push-button b1.
  12. Inject current greater than the setting until the relay picks up.
  13. Push the push-button b2.
  14. Connect test leads between the output contact on the relay to the terminal of coil of the DC auxiliary relay as shown in figures 1 and 2.
  15. Connect test lead between a free normally closed contact on the DC auxiliary relay to “Remove Control” terminals on the test set as shown in figures 1 and 2.
  16. Push the push-button b1.
  17. The timer will stop once the relay has picked up.
  18. Record this value in the test sheet.
  19. Push the push-button b2.
  20. Repeat the above test (Time test) from steps 1 to 9 for the other phases (bb and yy on test set)

Testing The Phase To Earth Impedance (Characteristics)
  1. Turn the switches b7, b8 and b9 to agree with the zone one setting.
  2. Turn the switches b10 on 5 Amp. position.
  3. Turn the switch b6 to maximum current level
  4. Set the MCB, e2 to “on” position.
  5. Turn the switch b3 on RR’ position.
  6. Start the test by pushing the push-button b1.
  7. Observe that the red indications AR , AM, T1 and As light up .
  8. Change the value of zone one impedance which is set on the test set by switches b7, b8 and b9, starting by turning switch b8 up or down until the relay operates in second step.

    9. (The red indication T2 on the relay will light up with the last indications).
  9. The second step time will be recorded on the screen of the timer
  10. Record this valve on test sheet.
  11. Repeat the above test, from step 1 to 9, but this time, set the second zone impedance setting on the test set. Then, change it up or down until the relay operates in third step.
  12. The red indication T3 on the relay will light up
  13. Repeat the above test, from step 1 to 9, once again but this time, set the third zone impedance setting on the test set. After that chit up or down until the relay operates in the fourth step.
  14. The red indication T4 on the relay will light up.
  15. Push the push-button b2.
  16. Repeat the above test from step 1 to 10 for phases S and T.

Testing the Phase to Phase Impedance Characteristic
  1. Turn the switch b3 to RY position.
  2. Turn the switch b7, b8, and b9 to agree with zone one setting.
  3. Start the test by pushing down the push button b1.
  4. Observe that the red indications AR, Ay, T1 and Aus will light up.
  5. Change the value of zone one impedance which is set on the test set by turning switches b7, b8 and b9 up or down until the relay operates in second step.
  6. The red indication T2 on the relay will light up with the last indications.
  7. Record this value on test sheet.
  8. Repeat the above test from step 1 to 5 step but this time, set the second zone
  9. Impedance setting on the test set. Next, change it up or down until the relay operates in third step.
  10. The red indication T3 on the relay will light up.
  11. Repeat the above test from step 1 to step 5 once again but this time, set the third zone impedance setting on the test set. After that change it up or down until the relay operates in fourth step.
  12. The red indication T4 on the relay will light up.
  13. Push the push-button b2.
  14. Repeat the above test from step 1 to 7 for phases YB and BR.

Reverse Direction Test
  1. Turn switch b4 to a position opposite to the current transformer earthing.
  2. Start the test by pushing the push-button b1
  3. Observe that the red indication Q lights up. This means the fault in the backward direction.

Measuring the Operating Times
  1. Set the impedance on the test set to a value which equals half the normal pickup value in the first time step for phase to earth fault.
  2. Turn the switch b3 to RR’ position.
  3. Start the test by pushing the push-button b1.
  4. The timer will stop once the relay has picked up.
  5. Repeat the above test from step 1 to 4 to measure zone 2 tripping time, but this time set the impedance on the test set to a value equal to the normal pickup value in the second time zone.
  6. Repeat the above test from step 1 to 4 to measure zone 3 tripping time, but this time set the impedance on the test set to a value equal to the normal pickup value in the third time zone.
  7. Repeat the above test from step 1 to 5 but for phase to phase fault.

 

Performance and Tripping Test

     

  • Press push buttons PR, PS, PT or PM the starting signals are simulated and the corresponding LED’s will light.
  • Press push-button PA simultaneously with push-button PR, PS, PT or PM and the trip signal will be formed.

Return Relay and Associated Equipment to Service

     

  1. Test trip the breaker associated with the relay.

  2.   Return relay to service by removing the test leads from the relay terminals block and the test set.

  3.   Make sure CT and VT secondary circuits are in normal condition.

  4.   If any equipment associated with the relay has been isolated, it should be returned to service by the following restoration steps

  • Breaker trip coils.

  • Breaker failure scheme associated this relay.

  • Disturbance alarm and fault recorder with this relay.

Request Power Line be returned To Service

     

  1. Take crossed reading of all AC currents and voltages supplied to the relay and compare them to those for normal load condition.

  2. Check that readings agree with station meters (MW/MVAR/Power Factor/Ampere/Voltages)

  3. Place a copy of the test results in the field maintenance file.
