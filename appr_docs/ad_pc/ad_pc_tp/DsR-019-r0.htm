Introduction

Type (DZ–2) two distance protection relay is used either for basic or back–up protection of high tension transmission lines on lines with high short circuit currents to earth. The relay provides for selective isolation of line to line short circuits in systems of any lay out . (DZ–2 ) protective relay can be Transformed into a three–step distance relay by supplementing it with an additional device type (KPC 1) comprising three starting directional impedance units. The proactive relay is designed to operate together with out–of–step Blocking device type (kp6-126) which also services as the starting element.

Safety Precautions

  • Line isolation and work permit required.

  • Make sure CT secondary circuits are shorted and VT secondary circuits are open.

Tools and Equipment 

  • Injection secondary tester [YII3-1 & YII3 - 2]

  • Voltmeter, Clip on ammeter

  • Test Leads,

  • Megger.

Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical reference material : Manufacturer’s manual .

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay .

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB`S), and all cabling supplying the relay.

  6. Use the station drawing to isolate the following: 

     - Breaker trip coils.

     - Breaker failure scheme associated with this relay.

     - Disturbance alarms and fault recorder associated with this relay.

Testing The Relay And Protection Output 

  1. Carefully follow these test procedure steps.

  2. Verify the test results match the relay setting, If not consult technical services.

  3. Test trip the breaker associated with the relay.

Prior to Starting Testing Procedure (s) Ensure the Following Steps
  1. This procedure is used to perform off load relay testing.
  2. Check that the panel Nameplate agrees with the EEC designation of the line that has been taken out of service for maintenance.
  3. Check there is no AC current from CT by using clip-on ammeter.
  4. Check there is no AC voltage from VT by using voltmeter.
  5. Check all external connections to the relay are in good condition and check that the terminal
    block wires are tight.
