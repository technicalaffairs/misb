Introduction

The protection relay type SMK351 is static 2-stages definite over current relay. This procedure details the steps required to completely check the proper operation of the relay.

The first section is the activities required prior to testing the second section is to set up the equipment for performing the actual tests

The third section is the actual test. This document is formatted the way to minimize the possibility of error causing personal injury or damage equipment

Safety Precautions

  • A work permit must be issued, the qualified maintenance staff should carry the suitable safety category.

  • Safety fence with caution marks surrounding work area is required.

  • Wear your safety equipment as helmet, shoes, breathing mask, safety glass...etc.

  • Line isolation is required

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for current injection

Tools and Equipment

  • Secondary injection test set.

  • Ammeter.

  • Test leads.

    •

Work to be Carried Out

Testing the Relay and Protection output

  1. Carefully follow all test procedure steps

  2. Verify the test result match the relay setting, if not consult technical servies.

  3. Test trip the breaker associated with the relay

 

 

 

Preparation

  1. Make sure CT secondary circuits are shorted at the terminal blocks before isolating the relay current for test

  2. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting sheet

Testing Procedure(s)

Current Test

  1. Determine the CT terminal blocks connected to the relay from feeder drawings

  2. Open the internal links of the links of the above terminals so the injection will not affect the other relay connected to the same one

  3. Connect test leads between the current terminals on relay side related to R and N according to drawings

  4. Slowly increase the current until the red indication on over - current element of phase R on the relay appears. The relay must operate with a current between ± 10% of the setting value

  5. Record this value on the test sheet

  6. Increase the injected current with a value 25% greater than the operating value

  7. Slowly decrease the current until the relay resets

  8. Record this value on the test sheet

Time Test

  1. Inject current higher than the setting value so pick up firmly

  2. Keep the injected current and switch off the test set

  3. Connect the terminals related tie delayed contacts on the relay according to drawings to the timer terminals on the test set

  4. Switch on the test set

  5. Start the test by injection current into the relay. The timer will stop once the relay has picked up

  6. Record this value of the timer on the test sheet

  7. Repeat the above test fro step 1 to step 6 for other phases, but inject current at terminals related to S.N then to T.N

Trip Test

  1. Request the operator to switch on the main circuit breaker

  2. Inject current to at any phase to a value above the set value

  3. Check that the circuit breaker trip

Return the Relay to service

  1. Verify the test results match the relay setting. If not consult the technical services

  2. Re-close the internal links of current terminals mentioned before
