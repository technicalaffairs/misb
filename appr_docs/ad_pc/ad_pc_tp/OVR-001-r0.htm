Introduction

The over voltage that can occurs on the EEHC system owing to the following causes:
- Lighting impulses.
- Switching surges,
- Temporary power frequency over voltages as a result sudden change in load generator over speed and system resonance. there for the system is provided with over voltage protection. in the over voltage protection scheme, a reverse reactive power relay (32R) is utilized to supervise the operation of the two over voltage relays (59-1 and 59-2), simultaneous pickup of the 32R relay and 59 relays is required to imitate over voltage timing relays.

Safety Precautions

  • Line isolation and work permit required.

  • Make sure CT secondary circuits are shorted and VT secondary voltage is open.

Tools and Equipment

  • Test unit (XS92a without yx91).

  • Voltmeter.

  • Clip on ammeter.

  • Test leads.

 Work to be Carried Out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical reference material: Manufacturer's manual.

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay.

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Inspect the CT secondary circuits, the VT Miniature Circuit Breakers (MCB's) and all cabling supplying the relay.

  6. Use the station drawing to isolate the following:

  • Breaker trip coils.

  • Breaker failure scheme associated with this relay.

  • Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection output

  1. Carefully follow these test procedure steps .

  2. Verify the test results match the relay setting if not consult technical services.

  3. Test trip the breaker associated with the relay.

Prior to starting Testing Procedures ensure the following steps

  1. This procedure is used to perform off load relay testing.

  2. Check that the Panel Nameplate agrees with the EEA designation of the line that has been taken out of service for maintenance.

  3. Check there is no AC current from CT by using clip-on  ammeter.

  4. Check there is no AC voltage from VT by using voltmeter.

  5. Check all external connections to the relay are in good condition and check that the terminal block wires are tight.
