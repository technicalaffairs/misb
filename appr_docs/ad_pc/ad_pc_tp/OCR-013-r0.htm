Introduction

The feeder protection relay SPAS 348 is designed to be used in applications requiring directional phase over current, directional short circuits and directional earth-fault protection. Typically, the relay is used for the over current and earth-fault protection of in feeders and busbars in distribution substations provided with multiple in feeders supplied from the same high-voltage busbar system. 

This procedure details the steps required to completely check the proper operation and calibration of the relay. It is recommended this procedure be carried out on an annually bases. The document is formatted the way to minimize the possibility of error causing personal injury or damage to equipment.

 Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area,

  • Line isolation and work permit required

  • Make sure that CT secondary circuits are shorted before isolating the current circuits for test distance protection

Tools and Equipment

  • Test set type HARALD

  • Harald test set

  • Voltmeter, Ammeter

  • Test leads 

Work to be Carried Out

Steps to Prepare for Relay Testing

  1. Review technical reference material and manufacturer's manual

  2. Obtain copy of relay setting from field maintenance file and compare it with the setting on the relay

  3. Obtain outage for line protected by this relay

  4. Check that the circuit breaker is opened and tagged

  5. Inspect the CT secondary circuits, the V T miniature circuit breakers ( mcb.s) and all cabling supplying the relaying

  6. Use the station drawings to isolate the following:
         * Breaker trip coils
         * Breaker failure scheme associated with the relay and disturbance alarms

Testing the Relay

  1. Carefully follow this test procedure

  2. Verify the test results match the relay setting

  3. Trip the circuit breaker associated with the relay

Preparation

  1. This procedure should be used for performing off load line

  2. Look the EEC panel name plate and check this EEC system designation name plate agrees with the line that has been taken out of service for maintenance 

  3. Make sure CT secondary circuits are shorted at the incoming terminal blocks before isolate the relay current circuits for test

Reading the Setting from the Relay by using the PC

  1. Connecting the personal computer to the relay

  2. Run the SPAS 348 C program

  3. Read the setting

Over current Protection Test

  1. Be sure that the D.C power of the relay is correct by applying a D.C voltmeter across points X0/61 and X0/62 on the rear of the relay

  2. Connect the current test leads between the current test leads between the current teminals on the rear of the current relay X0/1-3 of the phase R

  3. Connect the voltage test leads between the driectional volt terminals on the rear of the relay X0/13-14

  4. Connect A.C power to the test set

  5. Inject current to the relay

  6. Increase slowly the current until reach to the operating value and then the red "LEDs" light the operate current is shown on the display at the using ammeter

  7. Record this value (I operate) on the test sheet

  8. Connect leads between the time delayed trip output contact terminal X1/15 on the rear of the relay to the time terminals on the tester

  9. Switch on the timer of the test set

  10. The value of the trip time lisplay on the test set

  11. Record this value (t" operate) on the test sheet

  12. Increase the current injection to 25% larger than the operating value (I operate)

  13. Slowly decrease the injection current until the relay reset tack the value of this current (I reset)

  14. Record this value (I reset) on the test sheet

  15. Connect the current test leads between the current teminals on the rear of the current relay X0 / 4-6 of the phase S

  16. Repeat steps from 3 to 13 to test phase S

  17. Connect the current test leads leads between the current teminals on the rear of the current relay X0/7-9 of the phase T

  18. Repeat steps from 3 to 13 to test phase T

Return Relay and Associated Equipment to Service

  1. Return relay to service by removing the test leads from the relay terminals and the test set

  2. Make sure (T and V T secondary circuits are in normal condition

  3. Remove the disturbance alarm LED`s on the relay which is caused by the test

  4. Request feeder to be returned in service

  5. Take phasor readings of all A C currents and voltages and compare them to those for normal load condition

  6. Check that these reading agree with station meters (MW, MVAR, Amperes and Power Factor)

  7. Place a copy of the test sheet in the field maintenance file
