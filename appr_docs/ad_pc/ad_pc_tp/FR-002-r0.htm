Introduction

    This document details the check steps, which are performed on the fault recorder type  TPE 2000 manufacturer Enertic, on a monthly basis M1.

Safety Precautions
  • Work order should be issued,
  • The fault recorder should be in service,
  • Appropriate personnel protective equipment must be used.

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks bust be surrounding the work area,

  • Wear Safety helmet, Safety shoes

Tools and Equipment
  • Normal multimeter or digital Multimeter,
  • Complete set of screwdrivers.
  • Blower.

Work to be Carried Out

Visual Inspection of the Fault Recorder

  1. Clean the outside casing of the fault recorder from the dust using the blower.

  2. Ensure that the operation lamps are in the correct position reference to the Daily MPIS doc's no. FR-001-r0.

  3. Check that all connections of the complete fault recorder system "not loose".

  4. Check the DC/DC converter and its output to UA250 & UR2000, the correct values as indicated in figure 1.

For UR2000 must be find the values as shown on the front of the unit
For UA250 must be find the measuring values as follows:

- Between terminal ( 6-1 ) ( +13V )
- Between terminal ( 6-7 ) ( +5V )
- Between terminal ( 6- 8 ) ( -13V )
- Between terminal ( 6-2 ) , ( 6-3 ) , ( 6-4 ) , ( 6-5 ) , ( 6-9 ) ( zero Volt )
- If the values of DC/DC Converter Output not as the same result, 
   must be change the DC/DC Converter.

    5.    Clean the mechanism of the printer by cleaner oil.
