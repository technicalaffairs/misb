Introduction

This document outlines the testing procedures P1, which must be performed on an annually bases as indicated  by Y1 in the above title block.

The over current earth fault H 125 is an static relay manufactured by MERLIN GERIN. This procedure details the steps required to completely check the proper operation and calibration of the relay.

Safety Precautions

  • A work permit must be issued, qualified maintenance staff carry the suitable safety ID category,

  • Safety fence with caution marks surrounding the work area is required,

  • Personal protective equipment must be used as safety shoes, and safety Helmet ... etc.

  • Ensure that the Line isolation is done.

  • Make sure CT secondary circuits are shorted before isolating the relay's current circuit and injection the test current.

Tools and Equipment

  • Secondary injection test set type SR 76a

  • Ammeter, Voltmeter, test leads

Work to be Carried Out

Preparation

  1. Carefully follow all test procedure steps.

  2. Verify the test results match the relay setting, if not consult technical services.

  3. Test trip the breaker associated with the relay.

  4. Make sure CT secondary circuit are shorted at the incoming terminal blocks before isolating the relay current circuit for test.

  5. Look at field maintenance file and compare the position of the setting switches on relay with those shown on the setting sheet.

  6. Use the station drawings to isolate the following:
        - Breaker trip coils
        - CT secondary circuit associated with the relay and disturbance alarms

Current Test

  1. Remove the input current terminals number H3,H4 from the relay

  2. Connect test leads between the current terminals number H3 and H4 on the relay to the current terminals on the test set to test the relay

  3. Slowly increase the current until the red indication ((on)) earth fault unit appears

  4. Record this value on the test sheet (pick up value)

  5. Increase the injected current higher than the operating value

  6. Slowly decrease the current until the relay resets

  7. Record this value on the test sheet (Reset Value)

Time Test

  1. Inject current higher than the setting value so the relay picks up firmly

  2. Keep the injected current and switch off the test set

  3. Connect delayed contacts number S4 and S5 on the relay to timer terminals on the test set

  4. Switch on the test set

  5. Start the test by injecting current into the relay. The timer will stop once the relay has picked up

  6. Record this value of time on the test sheet

Return the Relay to Service

  1. Verify the test results match the relay setting. if not consult the technical services

  2. Connect the input current terminals number C1,C2,C3,C4,C5,C6 and delayed contact terminal S4,S5 to the relay

  3. Test trip the circuit breaker associated with the relay

Trip Test

  1. Request the operator to switch on the main circuit breaker

  2. Operate the output-contact by hand to trip the circuit breaker

  3. Check that the circuit breaker trip
