Introduction

This procedure details the steps required to complete and check the proper operation and calibration of the over current relay type 7SJ600 using secondary injection test set

The document is formatted the way to minimize the possibility of error causing personal injury or damage to cquipment

The first draft initiated by eng. Mohamed Yahia, tested by eng. Marwa Abdel-Naby Ismaiel, accepted by eng. Hiat Farouq Ahmed, approved by eng. Mohamed Abu Taleb: Head Sector of Cairo Zone Protection, and certified by eng. Abdel Maqsoud Ahmed Mahmoud: general director of North Stations & Cairo Zone MPIS coordinator.

Safety Precautions

  • A work permit must be issued.

  • The qualified maintenance staff should carry the suitable ID safety category.

  • Safety fence with caution marks surrounding the work area is required.

  • Equipment isolation (disconnect switches are opened and tagged).

  • Measure the current circuit of the relay by clip ammeter to be sure that is no current.

  • Make sure ct secondary are shorted before isolating  relay current circuits for test current injection.

Tools and Equipment

  • Mulit meter, clip meter.

  • Test leads, toolkit.

Work to be Carried Out

Preparation

  1. Review technical reference material (manufacturer's manual).

  2. Obtain copy of relay setting from field maintenance file.

  3. Obtain outage of the equipment (line or transformer) protected by relay.

  4. Review station drawing.

  5. Check that the bus and equipment disconnect switches are operned and tagged.

  6. Prepare the test sheet for the tested relay.

  7. Short the current circuit of the relay at the C.T side on the relay terminals.

  8. Isolate the current circuit of the relay for current injection, be sure the injection current will flow on the relay under test only.

  9. Isolate the trip, B.F. fault recorder and any associtated equipment.

  10. Obtain the input terminals of the relay (current terminals) and also the output terminals (trip terminals) from the station from the station drawing.

Testing the over current element

  1. Inject current phase terminal (1,3),(3,5) or (5,1)

  2. Slowly increase the current until the relay pick up and record it as a pick up value.

  3. Slowly decrease the current until the relay drop off and record it as a drop off value.

  4. To measure the tripping time connect the out trip contact to stop terminals of the tester then inject current equal twice the pick up value and record it as an operating time.

  5. Repeat the above steps (form 1 to 4) on the others phases.

If other stage is used (I>>), repeat step (1 to 5) for this stage, but after display the low set stage (I>)

Testing the earth fault element

  1. Inject any phase to neutral for example (R-N)(1,2),(3,4) or (5,6).

  2. Slowly increase the current until the relay pick up and record it is a pick up value (for earth fault).

  3. Measure the time by injection current equal twice the pick up value and record it as an operating time.

Note

  • Compare the recorded value with the setting values of the relay and give your comment.

  • Check all he used of any displayed stage should be enabled.

  • Check the other used output contest.

If other stage is used (IE>>), repeat step (1 to 3) for this stage, but after display the low set stage (IE>)

Testing Thermal Overload Protection

 

  1. Using thermal overload with memory or thermal overload without memory.

  2. Test as the over current element.

  3. Repeat step (1 to 5) but with thermal time longer than over current t = 1 min to 999 min.

Trip Circuit Supervision

There are two ways:

  1. Supervision using two binary input.

  2. Supervision using one binary input.

Supervision using two Binary input

Connect as shown in fig (1), so this checked up every 200 ms.

 

Supervision using one Binary input

  1. Connect as shown in fig (2), so this checked up every 30 s

So continuously this indicates that either the T.C is interrupted or the trip circuit is short circuited or control voltage for tripping is absent or breaker has not properly operated

 

Testing trip circuit Supervision

Return relay and associated equipment to Service

  1. Disconnect the connection between the relay and the test equipment

  2. Connect the current circuit of the relay then remove the short circuit at the C.T side

  3. Connect the trip B.F fault recorder and associated equipment

Return Protection Relay and associated Equipment to Service

  1. Take crossed reading of all AC current

  2. Compare this to the normal loading

  3. Place a copy of the test resulting on the old maintenance file
