Introduction

The type (KZ-1) relay group employs two relays in a two phase circuit and is intended for instantaneous current cut off. The elementary diagram of the type KZ-1 relay set is given in figure 1. The relay set includes two type (PT-40) current relays 1PT& 2PT type PII 251 auxiliary relays PII and one type PY-21 indicating relay (PY).

The first draft initialed by Eng. Mohamed Mostafa, and tested at 01/02/02, Samalout 500, ME

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area.

  • Line isolation,

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

Tools and Equipment

  • Test set type (YII3-1).

  • Avometer, Clip-on ammeter, Test leads.

Work to be carried out

Preliminary Steps to Prepare for Relay Testing

  1. Review technical reference material: Manufacturer's manual

  2. Obtain copy of relay setting form field maintenance file and compare it with the setting on the relay.

  3. Obtain outage for line protected by relay.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Inspect the CT secondary circuits, the VT miniature circuit breakers (MCB`S) and all cabling supplying the relay.

  6. Use the station drawing to isolate the following :   
     - Breaker trip coils.
     - Breaker failure scheme associated with this relay.
     - Disturbance alarms and fault recorder associated with this relay.

Testing the Relay and Protection Out Put

  1. Carefully follow these test procedure steps from page 3 to page 5
  2. Verify the test results match the relay setting if not consult technical services.
  3. Test trip the breaker associated with the relay.

Prior to Starting Testing Procedure (S) Ensure the Following Steps

  1. This procedure is used to perform off load relay testing.
  2. Cheek that the panel nameplate agrees with the EEA designation of the line that has been taken out of service for maintenance.
  3. Check there is not AC current from CT by using clip-on ammeter.
  4. Check there is no AC voltage from VT by using voltmeter.
  5. Cheek all external connections to the relay are in good condition and check that terminal block wires are tight.

Checking the Tripping of the Line by Protection

  1. Connect the trip and signal terminal.

  2. Leave the (B.F.) terminal is off.

  3. Connect the CB of the line and the Isolators is off.

  4. Connect the strap (H) for auto reclosure (3F).

  11. Finch the permission.

General:

The type " K31, K32, K33, K34, and K35 " relay groups are protection devices incorporating various relay necessary for the protection of a certain circuit. All the component elements of each group are housed in a common casing. The relay groups are designed for direct current operation.

The type " K31, K32, K33, K34 and K35" protection devices are intended for operation in stationary installations al ambient temperature between - 5 and + 35" C in the general industrial and export designs and between - 10 and + 40 " C in the tropical designs.

Relay having letter " I " in the type designation ( K3 1-T, K3 2-T ) are intended for operation in tropical climate. The tropical design differs from general industrial one in materials protection coatings of parts and finishing.
