Introduction

This document produced after the 1st revision by a new doc number DsR-001-r1 according to the approved MPIS coding system, will be used instead of document # PC-SS-001-r0 

This document lists the testing procedures P1 for L8b distance relay, which must be performed on an annually bases Y1 as indicated in the title block.

Safety Precautions

  • Line Isolation and work permit required.
  • Make sure CT secondary circuits are shorted.
  • Voltage Supply to the relay may be alive even after the line is isolated.
  • Isolate DC supply to relay. When doing any adjustments or maintenance on the L8b relay itself
  • Ensure all CT path elements are secure after any test disturbance.
Tools and Equipment
  • ABB Relay Test Set Type QZW 415 Voltmeter,
  • Ammeter 5 amp, Phase rotation meter,
  • Small slot screwdriver, Test leads, timer, and
  • 5 amp. clip-on ammeter (optional)

Work to be Carried Out

Steps to Prepare for Relay Testing

  1. Review technical reference material: Manufacturer's manual and the P5 Course-L8b available from Technical Services.

  2. Obtain copy of relay's setting specification from field maintenance file

  3. Obtain outage for line protected by this relay.

  4. Check that the bus and line disconnect switches are opened and tagged.

  5. Block protection signals to: Breaker trip coils and transfer trip scheme(s).

  6. Go to the switchgear and inspect the CT's, the VT miniature circuit breakers (MCB's), and all cabling supplying the relay:

  7. Check that the cables are in good condition and that all wire connections are tight and clean.

Depending on the Particular Field Site, the Following Steps May be Required

  1. Isolate the breaker failure scheme associated with this protection.

  2. Block any disturbance alarms or fault recorders associated with this protection.

  3. If not required for end to end tests, open the relay power line carrier (PLC) links

  4. Turn off the associated auto-reclosure scheme.

Testing Of Relay, Relay Scheme, and Protection Outputs

  1. Carefully follow this testing procedure: Source P5 Course - Network Training Center, Cairo South Reference: NTC LAB DATA - L8b Relay

  2. Verify the test results match the relay setting specifications previously obtained from field maintenance rile. If not consult Technical Services/Central protection
    a) Restore DC supply to relay scheme.
    b) Perform function test on relay scheme to ensure all associated auxiliary relays in L8b relay case are working properly.

  3. If equipped and ready for service, check the acceleration of the L8b relay and breaker failure and reclosure of the associated breakers. Separate Technical Procedures should be used to test each of these systems

  4. Test trip all breakers associated with this protection scheme.

This Procedure Should be used for Performing Off Load /Not In Service Relay Tests

  1. Look at the EEC panel nameplate and check that this EEC system designation nameplate agrees with the line that has been taken out of service for maintenance.

  2. Perform one of the following ammeter checks to verify there is no current flowing through this relay and that the testing will occur on the correct relay:

    (i) Using a clip-on ammeter, check there is no current flowing in the L8b relay circuits. Wires going to the rear terminals number 1, 2, 3, and 4 should have no current flowing in them.

                                                                                           OR

         (ii) Carefully open the L8b relay cover and connect a regular digital or analog ammeter across one of the following Test Block Terminals (Across terminal 2 or across terminal 3, or across terminal 4.)

    Using a small slot screwdriver, turn that same test block pin until it is in the horizontal "Test Position".
    There should be no  current flow in the ammeter. Return the pin to its normal vertical (1) position and
    check the other terminals.

    Note: Always check the continuity of the ammeter before using it in a (possible) live current circuit

  3. Remove the relay DC isolating plug (located in the lower left comer of the relay)

  4. Connect the two CT shorting links in the test block terminals as shown in Figure 2(b).

  5. Use a small slot screwdriver to turn all test terminals to the horizontal (-) position.

  6. Check all external connections to the relay and check terminal block wires are tight.

  7. Inspect relay internally for any abnormalities: Loose connections, discolored or burned resistors, leaking capacitors, corrosion, unclean contacts (Clean with commercial Contact Cleaner. Burnish only if pitted.)

  8. Using the relay settings supplied by EEC. and the manufacturer's information, check that all the appropriate settings have been correctly applied to the L8b relay.

  9. Attach the multi-contact plug to the rear of the QZW415 Test Set and then connect the other end into the L8b relay test block terminals

    Note: Carefully Align the Plug Numbers with the Matching Test Terminal

  10. Arrange the switches and links on the QZW-415 test set as per the table below:

   Switch              Position                      Explanation
        c                  1 Amp              Rated Relay Current of One Amp
        u                       II                 Selection of External Potential Source 
        b                   L3 - 80            Setting for One cycle L8b Relay
        y                       I                  Allows Testing with Main Switch "n".
        d                       x                  Lamp Indication of Relay Trip Operation
        w                      I                  Allows Testing of Electro-Mechanical Relays
        x                     +I                  Setting for Forward Direction Testing
        0                    1, L                Position for Distance Relay Tests
        p                    1, L                Position for distance Relay Tests
Links q and r     Closed Short       Out External Resistance and Ammeter

Use one of the following two steps to check that a 220Vdc supply is available for testing and calibrating the L8b relay:

(i) Using a DC voltmeter, check for 220Vdc on the rear terminals of the relay.
    Terminal #13 is positive (+) and terminal #14 is negative

                                                                        OR

 (ii) The following power supply method can be used when the relay is initially commissioned or,
         when the relay is "bench-tested" such as at NTC:

  1. Connect fused test leads to the rear terminals [Terminal # 1 3 (+) and Terminal #14 (-)] of the L8b relay. After this, carefully connect the other ends of these test leads to a 220Vdc-test source. Be careful to observe the correct polarity (+) , (-) for these test leads and connections.

  2. Push in and turn the QZW415 voltage control switch n to the "OFF" position (0).

  3. Supply AC power to the QZW415 Tester, according to the one of the following methods:

    (i) If the location of PT's on the bus -bar, Request from shift engineer in substation to connect the bus disconnect switch.

  4.  Check the three internal L8b supply fuses are good and also secure in their holders

  5. Switch (U) must be turned to position (I) to select this internal voltage source

                                                              OR

    (ii) If the location of PT's on the line:

  6. Connect three fused test leads to the group "a" set of test terminals (R, S, T) in the upper left corner of the QZW415 tester. Observe the correct polarity and connect the other end of these leads to a three phase 110 VAC supply.

  7. Leave switch "U" in position II to select this external source

    Caution: Do not connect any disconnect switch for this circuit during the test, to avoid any feedback voltage to PT.
     

  8. Use a phase Rotation Meter to check the correct Phase sequence of the (3) voltages.

Testing the Combined (Functional) Operation of the L8b Components

  1. Place the %Z switches "h" and "k" in the zero (0) position.

  2. Place the %Zx switch "i" in the (xl) position.

  3. Insert the DC isolating plug into the relay to energize the DC circuits.

  4. Push down and turn the voltage control switch n to the ON position (I).

  5. Operate the L8b relay by pressing pushbutton nl.

  6. Test the operation of the various relay components by changing the phase selection switch "f' and the type of fault switch "g" according to the 4bh test table

  7. Check that the relays pick up and drop out according to the 4bh table, and the notes given at the bottom of table.

  8. Before repeating any Tests, Check that all the Contactors have dropped out, If button "n" is left closed, then voltage is constantly supplied to the relay. Pushbutton "nl" controls the current flow to the relay.

  9. Push down and turn the voltage control switch n to the OFF position (0).

  10. Remove the DC isolating plug from the relay to de-energize the DC circuits.

Checking the Pick up Value of the CM3 UNIT

  1. If this procedure is used for the L8a relay, short the 3-4 contact of the Pta 5 relay.

  2. Keep the contact of the CM4 phase-to-phase relay open with a small piece of clean plastic when this unit is not being checked. This will avoid CM3 calibration errors.

  3. For the phase being tested, remove the blue coupling plug on the HG32 Transformer and connect this plug to the QZW415 test terminal on the extreme left of group "e".

  4. Referring to the calibration curves and the values calculated on the 3h test sheet (Pages no. 16, 17), determine the Z% value corresponding to the relay setting "A" of the first time-step.

  5. Turn the %Z switches on the test set to agree with the calculated Zone 1 setting.

  6. Select single-phase-to-ground faults by turning switch "g" to position 2.

  7. Select the phase fault by turning switch "f' to the positions R, S, T, respectively.

  8. Insert the DC isolating plug into the relay

  9. Push down and turn the voltage control switch n to the ON position (I).

  10. Press button nl and observe when the CM3 Unit, the PTAW lockout contactor and the tripping contactor PD3 all pickup. (The red trip indication lamp on the test set will light when the tripping contactor operates).

  11. Check for the next higher % Z setting at which the CM3, PFAW, and PD3 units do not operate, when the nl button is pressed.

  12. Find the next lower setting at which the these units always pick up.

  13. Repeat the above test about 5 times to determine the average pickup point.

  14. Record the pickup point for this phase of the CM3 relay on the 3h test sheet.

  15. Repeat the above tests for all three phases (R, S, T).

  16. Repeat the above test, Steps 4 to 15 above, once again, but this time, increase the % Z setting on the tester until the relay is operating in the second time-step. The CM3 and PD3 Units should operate and the lockout contactor PTAW should not operate.

    For those relay schemes with an added third time step (zone) repeat Steps 4 to 15 above but increase the % Z setting on the tester to the third step operating level.

  17. When checking the pickup value for the second time-step, the CUO under voltage contact of the "test" phase must be kept closed with a clean paper.

    The mean value of all the above settings should agree with the curve (appendix-6r) or edeviate by a maximum of +/-5%.

  18. When the CM3 pickup tests are completed, push down and turn the voltage control switch n to the OFF position (0).

  19. Remove the DC isolating plug from the relay to de-energize the DC circuits.

Checking the PICKUP Value of the CM4 Phase to Phase UNIT

  1. Remove (withdraw) the three blue plugs on the HG32 Units and leave the ends free.

  2. Select phase-to-phase faults by turning switch "g" to position 1.

  3. Turn switch " f ' to R, S, T to select different combinations of phase-to-phase faults

  4. Use the corresponding calibration curves located in (appendixes) which enable the % Z value to be determined for the relevant "B" settings of the HG32 Units.

  5. Perform tests similar to those for the CM3 units as given in the steps above.

  6. Record all test results on the 3h-relay test sheet.

Checking the Zone 1 and Zone 2 Tripping Times

Originally, the operating times of the L8b relay components were checked using the FAVAG chronoscope Models M430 or M500. The appendix provides two figures showing the connection of these units to the QZW415 tester.

  1. These older electromechanical motor/clock units have been superceded by more accurate digital timers such as the PROGRAMMER TM 14, or timers on the SVERKER 650 or the SIEMENS 7VP4800.

  2. All of these modem electronic timers only require start and stop signals to accurately measure elapsed times.

The following Procedure Gives the Steps for using these Modem Timers

  1. Toggle switch "d" on the QZW415 to the sec position to select time measurements.

  2. Use (2) test leads to connect the digital timer START inputs to the two QZW415 terminals #3 and #4 located in the bottom right comer just above switch n.

  3. Use (2) test leads to connect the digital timer STOP inputs to terminal #12 and terminal # 13 (Yl) located in the top right comer of the QZW415 tester.

  4. Connect an AC supply to power the electronic digital timer circuit.

  5. Press the reset button for the digital timer display.

Time Tests for the CM3 Phase to Earth / Three Phase Unit

  1. Repeat Steps from 1 to 15 as given earlier in this procedure to measure the Zone 1, operation time for each of the CM3 units. Note: Set the line impedance % Z switches to a value which equals half the normal pickup value in the first time-step of the CM3 unit. This will allow a good operation of the Zone I of the relay.

  2. To perform the time test, press and hold switch nl in the closed position.

  3. Press down and turn voltage control switch n to the ON position (I). The timer will start and then stop once the PD3 trip relay has picked up.

  4. Measure and record the CM3 Zone I pickup time.

  5. Repeat the test for the other two phases.

  6. Increase the %Z setting on the QZW415 tester until the relay is operating in the second time step (as previously discussed in STEP 40) and measure the time. For those relay schemes with an added third time (zone) increase the %Z setting on the tester to the third step operating level and repeat timing tests to be as set out in the setting specifications for the third step.

    While testing the second (and third) time step, the contact of the CUO relay of the phase being tested should be kept closed.

  7. When the CM3 pickup tests are completed, push down and turn the voltage control switch n to the OFF position (0).

  8. Remove the D.C. isolating plug from the relay to de-energize the DC circuits.

Note: The Zone I tripping time should be about 25 milliseconds, but during the test it will be approximately 5 to 10 ms longer than the actual tripping time, because the relays are already energized before the fault occurs.

Time Tests For the CM4 Phase to Phase Unit

  1. Perform the timer initialization Using Modem Timers as given previously or using any available timer

  2. Remove (withdraw) the three blue plugs on the HG32 units and leave the ends free.

  3. Select Phase-to-Phase faults by turning switch "g" to position 1.

  4. Perform timing tests for the CM4 unit by repeating procedures similar to those given for the CM3 Unit above.

  5. When the CM4 timing tests are completed, push down and turn the voltage control switch n to the OFF position (0).

  6. Remove the DC isolating plug from the relay to de-energize the DC circuits.

Testing the Product Relays CUO

  1. A continuously variable voltage source is required to properly test the CUO under voltage relays. When using switches h and k of the ABB QZW415 tester, voltages of up to 120 volts can be produced, but the variable voltages occur with interruptions due to using switches.

  2. This can result in inaccurate calibration of these units. Since a variable rheostat is not available on the QZW415 tester, there are no procedures given here. Future test equipment procedures detailing the testing of under voltage relays can be used to perform these checks.

  3. Ensure That PT Circuit ON - Ensure That CT Circuit Closed - Ensure That DC is ON

Return Protection Scheme and Associated Equipment to Service

  1. Return relay to service by restoring supply from: Voltage transformers, Current transformers (Ensure all CT current paths are complete and secure) DC (Station battery) if not left in service after 14(b)

  2. Depending on the field site, the following restoration steps may be required:

  3. Place the associated breaker failure protection scheme back in service.

  4. Restore the previously blocked disturbance alarms and fault recorders associated with this protection.

  5. Return the relay power line carrier (PLC) links to their normal position.

  6. Turn ON the associated auto-reclosure scheme.

  7. Remove the isolation tags and close the line and bus disconnect switches.

  8. Request power line be returned to service

  9. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those for normal load conditions.

  10. Check that these readings agree with station meters: MW, MVAR, Power factor, Amperes, Voltages

  11. Place a copy of the test results in the field maintenance file.
