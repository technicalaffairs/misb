Information and Directives

Ion Chromatograph System ( Complete + Software kit )

High performance for all types of IC separation anions, cations, transition metals, cyanide, phenols in water.

  • Built-in inert, nonmetallic PEEK (polyether-ether ketone) components through the system.

  • Provides auto-suppression for eliminating the background and provides a high signal to noise ratio through the use of self-regenerating micro membrane suppressor.

  • Easily upgrade to IC-reagent free system to use only water as eluent.

  • Columns and suppressors specified in EPA method 300.

  • Full control and digital data collection available with Windows based chromatography workstations.

 

 

 

 

Pump Delivery System

Dual processor-controlled gradient pump to blend and pump mixtures of up to 4 mobile phases.

  • Dual piston series pump microprocessor controlled constant stroke, variable speed.

  • Flow rate range: 0.05-10.00 ml/min in 0.01 ml/min

  • Flow accuracy: ± 2 µL/min.

  • Flow precision: < 0.2 %.

  • Maximum pressure: 5000 psi.

  • Eluent selection: four different mobile phase components each can be proportioned from 0-100 % in 0.1%.

  • Delay volume: < 800 µL.

  • Programmable with up to 100 methods. High and low pressure limits automatically stop the pump in the event of leaks, flow restrictions or an empty solvent reservoir.

  • Power requirements: 220V, 50/60 Hz.

Electrochemical Detector

Provides the three major forms of electrochemical detection: conductivity, DC amperometry and integrated amperometry for anions, cations, cyanide, phenols sulfide, sulfite and iodide.

  • Incorporates a 1-mm diameter working electrode for a high signal to noise ratio.

  • Corrosion resistant titanium flow path and counter electrode.

  • Combination pH-Ag/AgCl reference electrode to minimize baseline shift with pH gradient.

  • Microprocessor controlled digital signal processing.

  • Power requirements: 220V, 50/60 Hz.

Conductivity Flow Cell

  • Electrodes: passivated 316 stainless steel microelectrodes.

  • Cell Body: chemically inert polymeric material

  • Cell volume: 1.0 µL.

  • Resolution: .01 nS.

  • Range: eleven separate ranges from 0.01 to 3000 µS.

  • Maximum operating pressure: 300 psi.

  • Output voltage: User-selectable of 10, 100 or 1000 mV.

Amperometry Flow Cell

  • Electrodes: interchangeable working electrodes: gold, platinum, silver or glassy carbon.

  • Counter Electrode: Titanium.

  • Reference Electrode: pH-Ag/AgCl

  • Cell Volume: < 0.5 µL

  • Maximum operating pressure: 100 psi.

  • Noise: 1 pA (DC amperometry), 20 pC (integrated amperometry).

  • Potential Ranges: ± 2.00 V in 0.01 V increments.

  • Analog signal out put: 0.01, 0.1 or 1 V.

  • Auto-ranging digital signal monitoring by software.

UV–VIS Detector

  • Dual beam, variable wavelength absorbance detector (dual beam monochromators with a concave holographic grating and reflective beam splitter).

  • Two light sources, deuterium and tungsten lamps.

  • Validation of wavelength accuracy is accomplished efficiently using a built-in holmium oxide filter test.

  • Wavelength range: 190-800 nm in 1 nm increments.

  • Wavelength accuracy: ± 1 nm.

  • Bandwidth: < 5 nm.

  • Linearity: > 2 AU.

  • Noise: ± 10.0 µAU / h peak to peak.

  • Drift: < 100 µAU / h.

  • Cell path length: 10 mm.

  • Cell volume: 10 µL.

  • Cell operating pressure: 300 psi.

  • Full dynamic digital absorbance signal monitoring with software.

  • Control modes: local or remote through software.

  • Power requirements: 220V, 50/60 Hz.

Software Workstation

  • System Description: Chromatography data system for data acquisition, processing and reporting for Ion Chromatograph.

  • Maximum instrument control: up to six independent instruments can be controlled from a single workstation (four per workstation recommended).

  • Instrument control: Real-time bi-directional control via DX-LAN.

  • System Wellness feature provides on-line diagnostics and calibration operations of ion chromatograph system modules to ensure optimum performance. Software reminders prompt users when to perform calibrations for improved quality.

  • Wizards help with method creation, scheduling and validation.

  • Virtual "control panels" can be customized to each user's preferences

  • Storage of results in a database allows queries based on virtually any stored parameter.

  • Extensive audit trail logs details of instrument use all data manipulations are recorded and kept in a history file

GLP/GALP features include access control, audit trails, and automated validation, system wellness and electronic signatures and sign-offs.
