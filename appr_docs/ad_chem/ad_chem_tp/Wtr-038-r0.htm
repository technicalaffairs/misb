Introduction

This document outlines the equipment inspection I1, which must be performed on an annually bases or at unit shutdown.

The equipment function is to Sampling System pH Analyzer.

Safety Precautions

 

At the Normal Operation

  • Put control station at manual position.

  • Constant unit load.

Tools and Equipment

  • Safety helmet, shoes, tools box, avometer, piece of cloth, pail and transparent hose. 

Work to be Carried out

  1. Open the work permit to start the maintenance procedure and receipt the equipment place from the operation after work permit finished.

  2. Check the equipment for the dust.

  3. Calibration entails comparison and adjustment of the measured pH with that of a known buffer solution. Either one or two buffer solution is used.

 One buffer solution:  1 Point Calibration  calibrating the zero point (asymmetry potential).
 Two buffer solution:  2 Point Calibration  calibrating the zero point and the slope.

 

 

 

 

 

 

 

 

 

 

1 Point Calibration

  1. Access the programming mode by simultaneously pressing SELECT and ENTER.

  2. Proceed with programming until CAL appears, followed by the display of the currently measured sample pH.

  3. Immerse the electrodes into the buffer solution. Wait  for the pH reading to stabilize. Using the rollkeys, adjust the display until the indicated pH corresponds to the pH of  the buffer solution.

  4. Confirm with ENTER.  

  5. Now the symbol Er1 temporarily appears, followed by the corrected pH value.

2 Point Calibration 

  1. Immerse the electrodes into the second buffer solution. Wait  for the pH reading to stabilize. Using the rollkeys, adjust the display until the indicated pH corresponds to the pH of  the buffer solution.

  2. Confirm with ENTER.

  3. Now the symbol Er1 temporarily appears, followed by the corrected pH value.

  4. Returning the equipment place to the operators after maintenance schedule completed.

  5. Closing the work permit and returning the safety working at the normal position.

Operating the Instrument

 

Functional Control

Four keys enable the selection of the various parameters relevant to the measuring mode,

 

- Recall parameters in the measuring mode

- Select functions in the programming mode

- Confirm selected parameters

 

 

- Recall temperature data in the measuring mode

- Confirm the calibration coefficients in the calibration mode

- Return into the measuring mode and confirm the selected parameters

 

 

 

- Rollkeys to program status parameters and key data

 

- Load the default values. Press both keys, then turn the power on

 

 

 

 

 

 

 

 

 

 

Display

All data (status parameters and measurement data) may be recalled in the order of the main-menu sequence by pressing SELECT.

The instrument will return into the measuring mode by any of the following actions:

  • Automatically, at the end of the programming sequence

  • Press ENTER at any time while programming

  • Automatically, if no key is pressed for 10 minutes (this also applies for the programming or calibration modes)

Press ENTER while in the measuring mode displays the sample temperature. Releasing ENTER causes the instrument to return into the measuring mode a few second later.

Programming

During programming or calibration, the relay outputs are inactivated and the output signal represents the last measured value prior to the interruption.
