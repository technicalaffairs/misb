Introduction

This apparatus is used to measure the strength of light in different places using different sources of light such as tungsten, fluorescant, daylight or mercury. The process of measurement depends on zero adjustment at first.

Safety Precautions

  • The instrument has "Auto Power Shutt-off" in order to prolonged battery life. The meter will switch-off automatically if none of the buttons are pressed within 10 minutes. To de-activate this feature, select memory record function during measurement by pressing "RECORD' button.

  • When the left corner of LCD display show "LBT", it is necessary to replace the battery. However, in-spec measurement may still be made for several hours after low battery indicator appears before the instrument become inaccurate.

Measuring Procedures

  1. Push the "Power Off/On Button"(3-2, Figure:1) to switch the instrument on.

  2. Zero Adjust Procedures

  • Cover the Light Sensor (3-12, Figure:1) using the Sensor

  • Cover provided (3-13, Figure:1).

  • Slide the "Range switch" (3-ll, Figure:1) to the 2000 LUX position.

  • Push the "Zero Button" (3-9, Figure:1), then display will show zero values.

  • Upon completion, remove the sensor cover.

  1. Select the desired measuring unit by pressing the "LUX/FC Button" (3-4,Figure:1). The display will indicate the selected unit of "LUX" or "Ft-cd"

  2. Determine the lighting type ( Daylight, Tungsten, Fluorescent or Mercury lamp) by pressing the light Source Select Button (3-8, Figure:1)

  • The LCD will indicate the selected lighting type using the following symbols:

            L = Tungsten,       F = Fluorescent      S = Day Light,    C =Mercury

  1. Select the max. range using the "Range Switch" (3-ll,Figure: l).

  • If the display shows "--- ", it indicates an overload condition, select the next higher range.

  • If the display shows "--- ", it indicates an out-of-range, select the next lower range.

  1. Position the Light Sensor(3-12, Figure:1) directly under the light source.

  2.  

  • On the 20000 LUX range, the last digit will be shown on the lower line of LCD display.

  • On the 50000 LUX range, the last two digits will show on the lower line of LCD display.

  • For example:  On the 20000 range, if the display shows 1562  LUX

                                                                                             0

    that means the real display is 15620 LUX.                                            

  1. Data Hold:

  • During measurement, pushing the "Data Hold Button" ( 3-3. Figure: l) will hold the display values & the LCD will show the "D.H" symbol.

  • To cancel the Data Hold function. Press the Data Hold Button, once more.

  1. Relative % light measurement:

  • During measurement, press the % Button" (3-10, Figure:1). The current measured value will he indicated as "100% ".

  • All the subsequent measurements will be indicate as a percentage, relative to the value when the button was pressed.





The formula used is as shown below :

 

The new light values         ×  100

The light values when the "%" button was pressed

 

 

 

  • To de-activated this feature. Press the "% Button" (3-10, Figure:1) again

  1.  Data Record (Max.,Min., Average reading)

  • The DATA RECORD function displays the maximum, minimum and average readings. To start the DATA RECORD function, press the "Record button" (3-6,Figure: l)once- "REC" marker with appear on the LCD display.

  • With the 'REC" symbol indicated on the display

  1. Push the "CALL Button" (3-7, Figure: 1) once, then the "Max" symbol with the maximum values recorded will appear on the LCD display.

  2. Push the "CALL Button" once again, the "Min" symbol win the minimum values recorded will appear on the LCD display.

  3. Push the "CALL Button" once more. the "AVG" symbol with the average values will appear on the LCD display.

  4. To de-activate the Data Record function. Press the "Record Button (3-6-Figure:1) once again. All associated anunciators will disappear from the display.
