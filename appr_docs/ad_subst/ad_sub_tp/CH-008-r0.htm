Introduction

This document lists the equipment inspection, which must be performed on an annually bases as indicated by Y1 in the title block for Battery Charger:
 

Model                   :ARF130SE12

AC  input              :380 V , 10 A, 50 HZ, 1 PH

DC output             :129-140V, 12A

No of cells: 56

The first draft initiated by Eng Metwaly Mahmoud El_Sayed El_Hendawy and revised by Operation and maintenance's General Director: eng Mohamed Reda Abd El-Azez

 

Safety Precautions

  • Work order must be issued.

  • Switch off input & output supply

 Tools and Equipment

  • Suitable Tool Set

  • Digital Voltmeter

  • Vacuum Cleaner or Blower

  • Brush

Work to be Carried Out

  1. Switch off input and output supply.

  2. Clean from dust deposits, dust of heat radiating surface.

  3. Check area around charger should be clean.

  4. Inspect over heating compartment of control board.

  5. Inspect visually for Scr's and Diodes.

  6. Connections at the terminals should be clean, tithing.

  7. Switch on input and output supply.

  8. Check float and equalize voltage.

  9. Check the current limit operation.

  10. Check input and output voltage.
