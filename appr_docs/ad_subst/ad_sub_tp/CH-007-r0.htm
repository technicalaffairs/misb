Introduction

This document lists the equipment inspection, which must be performed on a monthly bases as indicated by M1 in the title block for Battery Charger:

Model                : ARF130SE12

S.N                    :ES781633

AC  input            :380 V , 10 A, 50 HZ, 1 PH

DC output           :129-140V, 12A

NO.OF CELLS : 56


The first draft initiated by Eng Metwaly Mahmoud El_Sayed El_Hendawy and revised by Operation and maintenance's General Director: eng Mohamed Reda Abd El-Azez

Safety Precautions

  • Work order must be issued.
  • Switch off input & output supply

Tools and Equipment

  • Suitable Tool Set
  • Digital Voltmeter
  • Vacuum Cleaner or Blower
  • Brush

Work to be Carried Out

  1. Switch-off input and output supply.

  2. Clean it from air dust and dirty.

  3. Check over heating compartment of control board.

  4. Inspect visually the Scr’s and Diodes.

  5. Switch on input and output supply.

  6. Check input and output voltage.
