Safety Precautions

  • A work permit must be issued.

  • Caution marks must be hanged on a work place.

Tools and Equipment

  • Suitable Tool Set
  • Digital Voltmeter, Brush, Blower.

Work to be Carried Out

  1. Check for correct operation of  DC room air condition.

  2. Check ventilation fan of rectifier.

  3. Read input and output voltage and record.

  4. Switch off input and output supply.

  5. Check over heating compartment of rectifier cards.

  6. Check all abnormal cases inside the rectifier.

  7. Inspect visually fuses, contactors, capacitors, rectifier card and diodes.

  8. Clean the rectifier from dusty air.

  9. Clean the floor of the DC room

  10. Switch on input and output supply.

  11. Check input and output voltage and record.
