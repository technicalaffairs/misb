Introduction

This document outlines the inspection checklist I2, which must be carried out on an annually basesas indicated by Y1 in the above title block. The first draft has been initiated by eng. El-Sayed Abdel Moneam El-Geddawy and tested at 11 Aug 2002 for charger no.1 at Sheraton El Matar 66kv Substation, Cairo zone.

 

The Charger technical data:

Input voltage 3*380 v AC,    Output voltage 220 v DC,     Output current 50A DC

Input voltage tolerance + 15%,    Input  frequency tolerance + 5%

Safety Precautions

  • Work order must be issuedand the qualified maintenance staff should carry the suitable safety category,

  • Switch off the power supply,

  • No smoking is permitted,

  • Wearing goggles, safety shoes, helmet, and rubber gloves is necessary

Tools and Equipments

  • Digital voltmeter

  • Suitable tool set

  • Vacuum Cleaner or blower

  • Brush

Work to be carried out 

  1. Transfer the battery to the standby charger

  2. Disconnect the power supply.

  3. Check ventilation system, and grease exhausting fans bearings,

  4. Check fuses,

  5. Check contacts and terminals and tighten it,

  6. Check control board for overheating components,

  7. Check SCR, ECU and diodes visually,

  8. Clean transfer switches (mode switch, voltage switch, etc )

  9. Clean relays (low voltage relay, fault indicating relay, ..etc .. using spray liquid free of oil, video cleaner)

  10. Then switch on power supply and measure the operation FLOAT voltage using digital DC voltmeter (must be 245V + 1%), if the values of measured voltage are below or above the standard values as in brackets, you can adjust it accurately by FLOAT control potentiometer P3 in the upper electronic control unit as indicated by the following steps:
       1- Put the mode switch S1 on FLOAT position.
       2- Increase the floating voltage by turning the FLOAT potentiometer P3 clockwise
       3- Decrease the floating voltage by turning the FLOAT potentiometer P3 anticlockwise.
       4- Repeat steps no. 2, 3 until get the desired value.

  11. Measure the HI Rate charging using digital DC voltmeter (must be 253.75V + 1%), if the values of measured voltage are below or above the standard values as in brackets, you can adjust it accurately by HI-Rate potentiometer P1 in the upper electronic control unit SCR, ECU, as indicated by the following steps:
    1- Put the mode switch S1 on HI-Rate position.
    2- Increase the HI-Rate voltage by turning the HI-Rate potentiometer P1 clockwise
    3- Decrease the floating voltage by turning the HI-Rate potentiometer P1 anticlockwise.
    4- Repeat steps no. 2, 3 until get the desired value.
