Introduction

This document lists the equipment inspection, which must be performed on yearly bases as indicated by Y1 in the title block for Battery Charger:

 

Model                : ARE130B5C35F

S.N                    :CPS960399

AC  input            :220 V , 30 A, 50 HZ, 1 PH

DC output           :132V, 35A

No.of cells : 60


The first draft initiated by Eng Metwaly Mahmoud El_Sayed El_Hendawy and revised by Operation and maintenance's General Director: eng Mohamed Reda Abd El-Azez

Safety Precautions

  • Work order must be issued.
  • Switch off input & output supply

Tools and Equipment

  • Suitable Tool Set
  • Digital Voltmeter
  • Vacuum Cleaner or Blower
  • Brush

Work to be Carried Out

  1. Switch off input and output supply.

  2. Clean from dust deposits, dust of heat radating surface.

  3. Check aea around charger should be clean.

  4. Inspect over heating compartment of control board.

  5. Inspect visually for Scr's and Diodes.

  6. Connections at the terminals should be clean,tithing.

  7. Check ventilation fan.

  8. Clean battery charger house

  9. Switch on input and output supply.

  10. Check float and equalize voltage.

  11. Check the current limt operation.

  12. Check input and output voltage.
