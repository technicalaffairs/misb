 Introduction

This document lists the equipment inspection, which must be performed on yearly bases as indicated by Y1 in the title block for Battary Charger:

Model: CLC -130-400 - 5, S.N.:13166-1,13166-2,13166-313166-4, AC input:380±10% V, 158A, 50HZ, 
3 PH, DC output :130V 400A, NO.OF CELLS: 58

 

The first draft initiated by Eng Metwaly Mahmoud El_Sayed El_Hendawy and revised by Operation and maintenance's General Director: eng Mohamed Reda Abd El-Azez

 

Safety Precautions

  • Work order must be issued.
  • Switch off input & output supply

Tools and Equipment

  • Suitable Tool Set
  • Digital Voltmeter
  • Vacuum Cleaner or Blower
  • Brush

Work to be Carried Out

  1. Switch off input and output supply.
  2. Clean from air dust and dirty.
  3. Check ventilation system and replace fans bearings.
  4. Inspect over heating compartment of control board.
  5. Check fuses, Contacts and Terminal Tithing.
  6. Inspect visually for Scr's and Diodes.
  7. Check charger ventilation area.
  8. Check and clean transfer switch.
  9. Switch on input and output supply.
  10. Check input and output voltage.
