Safety Precautions

  • A work permit must be issued and the maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area is required.

  • Safety and healthy tools must be used  as safety Helmet, safety shoes ... etc

  • Switch off input and output supply

Tools and Equipment

  • Suitable Tool Set

  • Digital Voltmeter, brush

  • Vacuum Cleaner or Blower

Work to be Carried Out

  1. Switch-off input and output supply.

  2. Clean it from air dust and dirty.

     

  3. Check the ventilation system.

  4. Check over heating compartment of control board.

  5. Inspect visually the Scr’s and Diodes.

  6. Clean battery charge house.

  7. Switch on input and output supply.

  8. Check input and output voltage.
