Introduction

This procedure details the outlines the Inspections activities steps required for the SF6 outdoor Circuit Breaker 132kv FX11, GEC Alsthom I1, which are performed every Year as indicated by the Y1, as indicated in the above titled block.

Safety Precautions

  • A  work permit must be issued

  • CB must be isolated and proper earthing at the both sides of the CB

  • Interrupt the control circuit (DC volt).

  • Discharges closing spring 70.02 and tripping spring 40.04 by operating manual lever trip and manual lever close.

  • Wearing safety helmet is required.

Tools and Equipment

  • Densimeter test device W41.

  • Leak detector W51, Thermometer W51.

  • 1 set of metric open end box wrenches.

  • Cleaning cloths, Lubricating grease.

Work to be carried out

  1. Inspect break insulator (20.02 fig.1) and supporting isolator (30.01 fig.1) for pollution and any cracks and clean it with dry wipers.

  2. Compare the reading of pressure gauge (80.03 fig,1) with filling pressure of CB where  (filling pressure =Pn + ∆P) 
    Pn =Rated Pressure (6bar in this case) ∆P = Correction value which can be determined using the chart fig  (2) 
    according to the approximate gas temperature which     can be defined with using a thermometer (w52 fig.2) closely fitted to the CB in shade.

  3. Check the function of heater (70.15 fig.1) and check the door gasket of the control cabinet.

  4. Record counter reading of operation counter.

  5. Carryout gas inspection.
